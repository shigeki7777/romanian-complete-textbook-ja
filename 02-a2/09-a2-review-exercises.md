# 第9章:A2 総復習練習【CEFR A2】

## この章の目標

この章を終えると、次のことができるようになる。

- 第1〜8章で学んだ動詞活用・時制・代名詞・場面表現を、章の順番ではなく**技能別に横断**して確認できる
- 自分の弱点分野(現在形か、複合過去か、o să か、弱形代名詞か、場面表現か)を具体的に特定できる
- A2 チェックポイントテスト(第11章)に向けて、抜けている知識を洗い出せる

この章は**問題集**である。新しい文法や語彙は一切登場しない。すべての問題は A1 全範囲と A2 第1〜8章の範囲内で作られている。ここで解けなかった問題は、あなたが「まだ定着していない箇所」を教えてくれる貴重な手がかりである。

## この章の使い方

1. **時間を計らず、辞書を使わずに解く**。総復習の目的は「今の自分が何を覚えていて、何を忘れているか」を正直に知ることである。分からない問題は飛ばさず、まず「たぶんこうだろう」と書いてみてほしい。間違いこそが復習の地図になる。
2. **8 つの技能セクションを順に解く**。①一般動詞の現在形・再帰動詞 → ②時刻・頻度 → ③複合過去 → ④未来表現 → ⑤弱形代名詞と a plăcea → ⑥場面表現 → ⑦ミニ会話完成 → ⑧和文羅訳、の順である。問題は通し番号(1〜59)になっている。
3. **解き終わったら [第10章:A2 解答集](./10-a2-answer-key.md) で答え合わせをする**。この章には解答を載せていない。第10章は単なる正答一覧ではなく、全問に「なぜそうなるか」の解説を付けているので、合っていた問題も解説を読むと理解が深まる。
4. **間違えた分野の章に戻る**。たとえば③でよく間違えたら [第3章](./03-basic-past-tense.md) へ、⑤でつまずいたら [第5章](./05-objects-and-pronouns.md) へ。分野と章の対応は第10章末尾の「復習マップ」にまとめてある。

> 表記の約束: ș と ț は**コンマ付き**(ș=U+0219, ț=U+021B)で書く。答えを書くときも ă/â/î/ș/ț を省かないこと(第8章)。

## セクション① 一般動詞の現在形・再帰動詞(問題 1〜8)

参照: [第1章:一日の生活と動詞の現在形](./01-daily-routines.md) / [文法参照:現在形](../grammar/present-tense.md)

### 問題1
a lucra(働く・-ez 型)を 6 人称すべて活用せよ(eu / tu / el,ea / noi / voi / ei,ele)。

### 問題2
a citi(読む・-esc 型)を 6 人称すべて活用せよ。

### 問題3
正しい活用形を選べ。

1. Noi (lucrăm / lucrează) la birou. 2. Voi (citiți / citesc) mult? 3. Ei (dorm / doarme) opt ore. 4. Tu ce (faci / face)?

### 問題4
再帰代名詞(mă / te / se / ne / vă)を補え。

1. Eu ___ trezesc la șase. 2. Tu ___ speli pe mâini. 3. Ea ___ îmbracă frumos. 4. Noi ___ culcăm târziu. 5. Voi ___ treziți devreme?

### 問題5
a dormi(眠る)と a se culca(床につく)の使い分けに注意して、ルーマニア語にせよ。

1. 私は11時に寝る(=床につく)。 2. 私は8時間眠る。

### 問題6
次の 2 単の形で、語幹末の子音はどう発音されるか(/k/ か /t͡ʃ/、/ɡ/ か /d͡ʒ/)。

1. faci(fac の tu 形) 2. mergi(merg の tu 形)

### 問題7
次の文の誤りを直せ。

1. Noi citescim o carte. 2. Trezesc la șapte.(「私は7時に起きる」のつもり) 3. Mă culc și mă dorm.

### 問題8
自分の一日を、次の 3 つの動詞をすべて使って 3 文で書け:a se trezi / a merge / a se culca。(時刻を添えること)

### 問題23

### 問題24

1. a lucra 2. a merge 3. a veni 4. a face 5. a pleca

1. a pleca(私は) 2. a vizita(彼は) 3. a lucra(彼らは)

### 問題28

### 問題29

## セクション⑤ 弱形代名詞と a plăcea(問題 30〜39)

参照: [第5章:目的語と代名詞 — îmi place の世界](./05-objects-and-pronouns.md)

### 問題30
対格弱形代名詞(〜を)を 8 つ、意味とともに書け(私を / 君を / 彼を / 彼女を / 私たちを / 君たちを / 彼らを / 彼女らを)。

### 問題31
与格弱形代名詞(〜に)を 6 つ、意味とともに書け(私に / 君に / 彼・彼女に / 私たちに / 君たちに / 彼ら・彼女らに)。

### 問題32
下線の弱形代名詞は「〜を(対格)」か「〜に(与格)」か。

1. **Îl** văd. 2. **Îmi** place. 3. **Te** ajut. 4. **Îți** spun. 5. **O** cunosc.

### 問題33
place / plac の正しいほうを選べ。

1. Îmi (place / plac) cafeaua. 2. Îmi (place / plac) fructele. 3. Îmi (place / plac) să citesc. 4. Îi (place / plac) filmele. 5. Îmi (place / plac) muzica.

### 問題34
適切な弱形代名詞を入れよ。

1. ___ place muzica?(君は〜が好き?) 2. ___ văd.(彼を見る) 3. ___ spun.(彼に言う) 4. ___ ajut.(君を手伝う)

### 問題35
口語の縮約形に直せ(nu + 与格弱形)。

1. Nu îmi place.(→ 好きじゃない) 2. Nu îmi trebuie.(→ 要らない)

### 問題36
複合過去にせよ(縮約と o の後置に注意)。

1. Îl văd.(→ 昨日) 2. O văd.(→ 昨日) 3. Îmi place.(→ 過去) 4. Te văd.(→ 昨日)

### 問題37
与格を変えて応用せよ。

1. Îmi pare rău.(→「彼は残念だ」に) 2. Îmi trebuie o pauză.(→「君に休憩が要る」に) 3. Îmi place ciocolata.(→「彼らはチョコが好き」に)

### 問題38
次の文の誤りを直せ(人の直接目的語)。

> Văd Maria.(「マリアに会う/マリアが見える」のつもり)

### 問題39
日本語をルーマニア語にせよ。

1. 私は音楽が好きだ。 2. その映画は気に入った。(過去) 3. 君はコーヒーが好き?

## セクション⑦ ミニ会話完成(問題 50〜53)

参照: [第2章](./02-shopping-and-restaurants.md) / [第3章](./03-basic-past-tense.md) / [第4章](./04-basic-future-expressions.md) / [第7章](./07-basic-medical-situations.md)

### 問題50(市場で)
> A: Bună ziua! ___ dori un kilogram de mere.(丁寧に「いただきたい」)
> B: Poftiți!
> A: Cât ___?(いくらですか)
> B: Zece lei.

### 問題51(昨日の報告)
> A: Ce ___ făcut ieri?(君は)
> B: ___ fost la piață și ___ cumpărat legume.(私は・2回とも)

### 問題52(診察室で)
> 医師: Bună ziua. Ce vă ___?(どうされましたか)
> 患者: ___ doare capul de ieri.(頭が痛い)
> 医師: De ___?(いつから?)
> 患者: De ieri.

### 問題53(週末の予定)
> A: Ce ___ ai în weekend?(どんな予定?)
> B: Sâmbătă o să ___ la munte cu familia.(a merge, 私は)
> A: Frumos! O să ___ vreme bună.(a fi)
