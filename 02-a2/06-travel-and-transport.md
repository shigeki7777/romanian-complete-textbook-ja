# 第6章:旅行と交通【CEFR A2】

## この章の目標

この章を終えると、次のことができるようになる。

- 道を尋ね(Unde este…? / Cum ajung la…?)、方向の答え(la dreapta / la stânga / drept înainte)を聞き取れる
- 交通手段を cu を使って言える(merg cu trenul)
- 切符を買い(un bilet dus-întors)、発車時刻を尋ねられる(La ce oră pleacă trenul?)
- タクシー・配車アプリで行き先を伝えられる
- ホテルでチェックインし、部屋・朝食について確認できる(o cameră pentru două nopți)

## 本文解説

### 「チャンク」を続ける

この章では**丁寧な命令の言い方**(Mergeți… / Luați-o…)が出る。ルーマニア語の命令法(imperativ)の体系は B1 で扱うので、ここでは道案内で頻出する形だけを**チャンク**として丸ごと覚える。チャンクには必ず(チャンク)と明記する。第1・2章で習った現在形(merg, plec, ajung…)や Aș dori… は分析して使える。

### 道を尋ねる

- **Unde este…?** /ˈun.de ˈjes.te/ —「〜はどこですか?」。este は口語で e に短縮(Unde e gara? = 駅はどこ?)。
- **Cum ajung la…?** /kum aˈd͡ʒunɡ la/(チャンク)—「〜へはどう行けばいいですか?」。ajung は a ajunge(到着する・たどり着く)の1人称。la=〜へ。Cum ajung la gară?(駅へはどう行けば?)。
- **Scuzați, unde este stația de metrou?** /sku.ˈzat͡sʲ …/ —「すみません、地下鉄の駅はどこですか?」。Scuzați(すみません)は声かけの定番。

### 方向の答えを聞き取る

尋ねたら、次のような方向語が返ってくる。聞き取れることが目標である。

| 方向 | ルーマニア語 | IPA | 意味 |
|---|---|---|---|
| 右 | la dreapta | /la ˈdre̯ap.ta/ | 右に |
| 左 | la stânga | /la ˈstɨn.ɡa/ | 左に |
| まっすぐ | drept înainte | /drept ɨ.naˈin.te/ | まっすぐ前へ |
| 近い | aproape | /aˈpro̯a.pe/ | 近い・近くに |
| 遠い | departe | /deˈpar.te/ | 遠い・遠くに |

- **Este aproape.** —「近いです」。**Este departe.** —「遠いです」。
- **la prima stradă**(最初の通りで)、**la a doua intersecție**(2つ目の交差点で)のように la + 順序で位置を示すこともある。

### 丁寧な命令チャンク — 道を教える側の言い方

自分が道を教えるときや、教えられた内容を確認するときに使う。**命令法はチャンクとして覚える**(体系は B1)。相手が敬称(または複数)のときの形である。

- **Mergeți drept înainte.** /merˈd͡ʒet͡sʲ drept ɨ.naˈin.te/(チャンク)—「まっすぐ進んでください」。
- **Luați-o la dreapta.** /luˈa.t͡sjo la ˈdre̯ap.ta/(チャンク)—「右に曲がってください」。直訳は「それを右に取れ」。曲がる指示の定番。
- **Luați-o la stânga.**(チャンク)—「左に曲がってください」。
- **Continuați până la…**(チャンク)—「〜まで進み続けてください」。

これらは丸ごと音で覚える。「まっすぐ=Mergeți drept înainte」「曲がる=Luați-o la…」の2型を押さえれば道案内の大半に対応できる。

### 交通手段 — cu + 交通手段

「〜で行く」は動詞 a merge(または a călători=旅する)に **cu + 交通手段(定冠詞付き)** を添える。

| 手段 | 単数(裸) | 「〜で」の形 | IPA |
|---|---|---|---|
| 電車 | tren | cu trenul | /ku ˈtre.nul/ |
| バス | autobuz | cu autobuzul | /ku a.u.toˈbu.zul/ |
| 路面電車 | tramvai | cu tramvaiul | /ku tramˈva.jul/ |
| 地下鉄 | metrou | cu metroul | /ku meˈtro.ul/ |
| タクシー | taxi | cu taxiul | /ku taˈksi.ul/ |
| 飛行機 | avion | cu avionul | /ku a.viˈo.nul/ |

- **Merg cu trenul.** /merɡ ku ˈtre.nul/ —「電車で行く」。cu の後は**定冠詞付き**(trenul)になる点に注意。裸の tren ではない。
- 例外的に「徒歩」は **pe jos** /pe ʒos/(歩いて):Merg pe jos.(歩いて行く)。cu は使わない。

日本語話者への注意:cu は「〜と一緒に」の意味も持つ(cu colegii=同僚と)。手段の cu と同伴の cu は同じ語だが、後ろに交通手段が来れば「〜で」と判断できる。

### 駅・空港 — 切符と発車

- **bilet / bilete** /biˈlet/ —「切符」。
- **un bilet dus** /un biˈlet dus/ —「片道切符」。dus=行き。
- **un bilet dus-întors** /… dus ɨnˈtors/ —「往復切符」。dus-întors=行き帰り。
- **peron / peroane** /peˈron/ —「(駅の)ホーム」。
- **La ce oră pleacă trenul?** /la t͡ʃe ˈo.ra ˈple̯a.kə ˈtre.nul/(チャンク)—「電車は何時に出ますか?」。pleacă は a pleca(出発する)の3人称(o→ea 交替: plec, pleci, pleacă)。第1章の ora の復習でもある。
- **La ce oră ajunge trenul?** —「電車は何時に着きますか?」。ajunge=着く。
- **De pe ce peron pleacă?** —「何番ホームから出ますか?」。

「〜行きの切符」は **un bilet până la…**(〜までの切符):un bilet până la Cluj(クルージュまでの切符)。値段は第2章の Cât costă biletul?(切符はいくら?)で尋ねる。

### タクシー・配車アプリ

- **Sunteți liber?** /sunˈtet͡sʲ ˈli.ber/ —「(タクシーが)空いていますか?」。
- **La aeroport, vă rog.** /la a.e.roˈport və roɡ/ —「空港までお願いします」。行き先は la + 場所。
- **Cât costă până la centru?** —「中心街までいくらですか?」。
- **Puteți să opriți aici?**(チャンク)—「ここで止まってもらえますか?」。opriți=止める。
- 配車アプリでは行き先を入力するので会話は最小限だが、Aici e bine.(ここで大丈夫です)、Mulțumesc!(ありがとう)は覚えておく。

### ホテル — チェックイン

- **cazare** /kaˈza.re/ —「宿泊」。Aș dori o cazare pentru două nopți.(2泊の宿泊をお願いしたい)。
- **o cameră pentru două nopți** /o ˈka.me.rə ˈpen.tru ˈdo.wə nopt͡sʲ/ —「2泊の部屋」。noapte(夜)の複数は nopți。
- **o cameră single / dublă** —「シングル/ダブルの部屋」。
- **mic dejun inclus** /mik deˈʒun inˈklus/ —「朝食込み」。inclus=含まれる。Micul dejun este inclus?(朝食は込みですか?)。
- **La ce oră este micul dejun?** —「朝食は何時ですか?」(第1章の ora)。
- **Aveți wi-fi?** —「Wi-Fi はありますか?」(第2章の Aveți…?)。

## 重要ポイント

- 道を尋ねる:Unde este…? / Cum ajung la…?(チャンク)。答えは la dreapta / la stânga / drept înainte / aproape / departe
- 道を教える命令はチャンク:Mergeți drept înainte. / Luați-o la dreapta.
- 交通手段は **cu + 定冠詞付き**(cu trenul)。徒歩だけ pe jos
- 切符:bilet dus(片道)/ dus-întors(往復)/ până la…(〜まで)。La ce oră pleacă trenul?(チャンク)
- ホテル:o cameră pentru două nopți / mic dejun inclus / cazare
- 行き先は la + 場所(La aeroport, vă rog.)

## フォーマル/カジュアルの違い

道尋ねや切符購入は見知らぬ相手なので、基本は敬称(2複)を使うのが安全である。

- **フォーマル(見知らぬ人・窓口)**: Scuzați, puteți să-mi spuneți unde este gara?(すみません、駅がどこか教えていただけますか)/ 相手は Mergeți… / Luați-o… と2複で答える
- **カジュアル(友人に)**: Auzi, cum ajung la tine?(ねえ、君の家までどう行く?)/ 相手は Mergi drept… / Ia-o la dreapta.(tu の命令)と2単で答える

道尋ねの現場では、**まず Scuzați(すみません)で丁寧に声をかける**のが最も大事で、その後の細部の丁寧度差は二次的である。

## 文化的注意点

- **ブカレストの地下鉄**:ブカレストには地下鉄(metrou)が走っており、市内移動に便利な手段として使われている。路線番号(M1, M2…)で案内されることが多い。運賃や乗車方式は変わりうるので、最新の案内板・公式情報で確認するのが確実である(具体的な料金はここでは書かない)。
- **CFR(国鉄)**:都市間の鉄道は CFR(Căile Ferate Române=ルーマニア国鉄)が代表的な運行主体である。長距離は列車種別(IR=都市間急行 など)で所要時間が変わる。時刻・料金は季節や改定で変動するため、購入時に窓口(casă de bilete)や公式で確認する。
- **タクシーと配車アプリ**:都市部では配車アプリも一般的に使われている。街で拾うタクシーは、乗る前にメーター(aparatul de taxat)が動いているか確認するのが安心とされる。料金の目安は路線・時間帯で変わるため、断定的な数値は避け、Cât costă până la…? で事前に尋ねる習慣をつけるとよい。

## 練習問題

### 問題1:cu を使って交通手段を言え(定冠詞に注意)

1. 電車で(tren)→ 2. バスで(autobuz)→ 3. 地下鉄で(metrou)→ 4. 徒歩で →

### 問題2:方向語を日本語にせよ

1. la dreapta 2. la stânga 3. drept înainte 4. aproape 5. departe

### 問題3:場面に合う表現を選べ(Cum ajung la…? / La ce oră pleacă trenul? / Sunteți liber? / mic dejun inclus)

1. 駅への行き方を尋ねる。 2. タクシーが空いているか確認する。 3. 電車の発車時刻を尋ねる。 4. ホテルで朝食が込みか確認する。

### 問題4:次のダイアログを完成させよ(切符購入)

A: Bună ziua! ___ dori un bilet ___ la Cluj.(往復切符をお願いしたい)
B: Desigur. La ce ___ pleacă trenul?(何時に出るか尋ね返す想定→客が尋ねる)
A: La ce oră ___ trenul?(発車時刻を尋ねる)

### 問題5:次の日本語をルーマニア語にせよ(チャンク使用可)

1. すみません、空港へはどう行けばいいですか?
2. 2泊の部屋をお願いしたいです。朝食は込みですか?

## 解答

### 問題1
1. **cu trenul** 2. **cu autobuzul** 3. **cu metroul** 4. **pe jos**(徒歩だけ pe jos)。
— cu の後は定冠詞付き。徒歩は例外。

### 問題2
1. 右に 2. 左に 3. まっすぐ前へ 4. 近い/近くに 5. 遠い/遠くに。

### 問題3
1. **Cum ajung la gară?** 2. **Sunteți liber?** 3. **La ce oră pleacă trenul?** 4. **Micul dejun este inclus?**(mic dejun inclus)。

### 問題4
A: **Aș** dori un bilet **dus-întors** la Cluj.(Aș dori はチャンク、往復は dus-întors)
A: La ce **oră** **pleacă** trenul?(pleacă は3人称。ora は第1章)。
— B の返しは文脈上の想定なので、客側の2文が要点。

### 問題5
1. **Scuzați, cum ajung la aeroport?**(Scuzați で声かけ、Cum ajung la…=チャンク) 2. **Aș dori o cameră pentru două nopți. Micul dejun este inclus?**(nopți は複数、inclus=込み)。

## 復習チェックリスト

- [ ] Unde este…? / Cum ajung la…?(チャンク)で道を尋ねられる
- [ ] la dreapta / la stânga / drept înainte / aproape / departe を聞き取れる
- [ ] Mergeți drept înainte. / Luați-o la dreapta.(チャンク)で道を教えられる
- [ ] cu + 定冠詞付き交通手段(cu trenul)、徒歩は pe jos が言える
- [ ] bilet dus / dus-întors / până la… と La ce oră pleacă trenul? が使える
- [ ] タクシーで行き先(La aeroport, vă rog.)を伝えられる
- [ ] o cameră pentru două nopți / mic dejun inclus でチェックインできる

## 次章への橋渡し

移動と宿泊——旅の骨格を扱えるようになった。旅先では体調を崩すこともある。[第7章:医療の基本場面](./07-basic-medical-situations.md) では、薬局や診療所で「どこが痛いか」を伝え、助けを求める最低限の表現を学ぶ。本章で身につけた Unde este…?(〜はどこ?)や Aș dori…(〜をいただきたい)は、そのまま「薬局はどこ?」「この薬をください」に応用できる。
