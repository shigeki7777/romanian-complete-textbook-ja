# 第8章:メールとメッセージ【CEFR A2】

## この章の目標

この章を終えると、次のことができるようになる。

- 書き言葉の丁寧度を **3段階**(カジュアル SMS / 標準メール / フォーマルメール)で使い分けられる
- 相手に応じた書き出し(Dragă... / Bună ziua... / Stimate domnule...)と結び(Pa / Toate cele bune / Cu stimă)を選べる
- 用件を切り出す定型(Vă scriu pentru că... / Aștept răspunsul dumneavoastră.)をチャンクとして使える
- 友人への誘い・語学学校への問い合わせ・ホテルへの予約確認を、それぞれ適切なレジスターで書ける
- チャットの省略形(k, sal, ms など)を **読めて**、自分では正書法で書くという線引きができる
- ダイアクリティクス(ș/ț/ă 等)を省くメールの現実を知りつつ、学習者・フォーマル文書では正書法を守れる

## 前提知識

- [第2章:基本の挨拶](../01-a1/02-basic-greetings.md)(A1) — Salut / Bună ziua / La revedere の丁寧度の軸。メールの書き出し・結びはこの延長にある
- [第4章:未来表現の基礎](./04-basic-future-expressions.md) — `o să + 接続法`。誘いのメールで「〜する予定」「〜しよう」を書くのに使う
- [第7章:医療の基本場面](./07-basic-medical-situations.md) — 予約 programare。予約確認メールで再利用する
- 話し言葉の tu / dumneavoastră の使い分け(第2章)。書き言葉では、この軸がさらに **書き出し・結び・定型句** の3か所に広がる

## 本文解説

前章までは主に「話す」場面を扱ってきた。本章のテーマは **書く** ことである。ルーマニア語の書き言葉は、話し言葉以上に **丁寧度(レジスター)** が形にはっきり表れる。同じ「用件を伝える」でも、相手が親友か、担当者か、目上の役職者かで、書き出し・本文・結びのすべてが変わる。この章は文法の新項目を増やすより、**すでに知っている表現を「書き言葉のレジスター」で並べ替える** ことに集中する。したがって本章は、標準テンプレートの「フォーマル/カジュアルの違い」を本文全体で扱う構成になっている。

### 書き言葉の丁寧度 3 段階

ルーマニア語の書き言葉は、大きく次の3段階で捉えると整理しやすい。

1. **カジュアル(SMS / WhatsApp / チャット)** — 親しい相手。短く、くだけて、絵文字も可。
   例:**Salut! Ce faci?**(やあ!元気?)
2. **標準メール(semi-formal)** — 面識はあるが親しくない相手、店・学校・担当者など。丁寧だが堅すぎない。
   例:**Bună ziua, ...**(こんにちは、〜)
3. **フォーマルメール** — 目上・役職者・公式文書。敬称と定型を厳守する。
   例:**Stimate domnule ...**(拝啓 〜様)/ **Stimată doamnă ...**(拝啓 〜様〔女性〕)

この3段階は独立した3つの箱ではなく、**同じ軸(相手との距離)** の上に並ぶグラデーションである。迷ったら **一段階丁寧なほう** を選べば失礼にはならない——話し言葉で「迷ったら dumneavoastră」だったのと同じ原則が、書き言葉にも貫かれている。

### 書き出し(呼びかけ)

メールの最初の1行は、丁寧度を最も強く決める。

- **Dragă Maria,** /ˈdra.ɡə maˈri.a/ — 親愛なるマリアへ(親しい相手)。dragă =「親愛な」。ファーストネームで呼ぶ間柄
- **Bună ziua, domnule Popescu,** /ˈbu.nə ˈzi.wa ˈdom.nu.le poˈpes.ku/ — こんにちは、ポペスクさん(標準)。話し言葉の挨拶をそのまま書き出しに使える
- **Stimate domnule Director,** /stiˈma.te ˈdom.nu.le diˈrek.tor/ — 拝啓 ディレクター様(フォーマル・相手が男性)
- **Stimată doamnă Ionescu,** /stiˈma.tə ˈdo̯am.nə jo.ˈnes.ku/ — 拝啓 イオネスク様(フォーマル・相手が女性)

**Stimate / Stimată** は「尊敬する」を意味する形容詞で、**相手の性で語尾が変わる**(男性 -e / 女性 -ă)。この語尾一致は日本語話者が最も間違えやすい点なので、後の「よくある誤用」で改めて扱う。呼びかけの後には **コンマ** を打ち、次の行から本文に入るのがルーマニア語の書式である。

呼びかけに使う敬称・肩書き(domnule / doamnă)は、第2章で学んだ呼びかけの domnul / doamna の **呼格形**(呼びかけの形)で、語尾が -ul → -ule、-a → -ă に変わる点にも触れておく(domnul → domnule、doamna → doamnă)。今は「呼びかけるときはこの形」とチャンクで覚えてよい。

### 結び(締めの言葉)

本文を書き終えたら、丁寧度に合った締めの一言を置き、その下に自分の名前を書く。

| レジスター | 結び | 意味・ニュアンス |
|---|---|---|
| 超カジュアル | **Pa!** / **Te pup** | バイバイ / チュッ(親しい間柄のみ。te pup は直訳「君にキスする」) |
| カジュアル〜親しみ | **Cu drag,** | 愛情を込めて(親しい相手への温かい締め) |
| 標準 | **Toate cele bune,** | ご多幸を(直訳「すべての良きことを」。無難で使いやすい) |
| 標準〜ややフォーマル | **Cu stimă,** | 敬具(直訳「尊敬をもって」) |
| フォーマル | **Cu respect,** | 敬具(直訳「敬意をもって」。最も硬い) |

- **Te pup** /te pup/ は非常に親密で、家族・恋人・ごく親しい友人に限る。ビジネスで使うと不適切なので注意。
- 標準的な締めに迷ったら **Toate cele bune,** が最も汎用性が高い。
- フォーマル文書は **Cu stimă,** または **Cu respect,** で締め、コンマの後に改行して署名する。

### 用件を切り出す定型表現(チャンク)

本文の骨組みになる定型句は、丸ごと覚えておくと文章が一気に書けるようになる。

- **Vă scriu pentru că...** /və skri.u ˈpen.tru kə/ — 〜のためにご連絡しています(直訳:あなたに書きます、なぜなら〜)【チャンク】。用件を切り出す最初の1文に使う
- **Vă scriu în legătură cu...** /və skri.u ɨn le.ɡəˈtu.rə ku/ — 〜の件でご連絡しています(より硬い言い方)
- **Aș vrea să vă întreb dacă...** /aʃ vre̯a sə və ɨnˈtreb ˈda.kə/ — 〜かどうかお尋ねしたいのですが(丁寧な問い合わせ)
- **Mulțumesc pentru răspuns.** /mul.t͡suˈmesk ˈpen.tru rəsˈpuns/ — お返事ありがとうございます
- **Aștept răspunsul dumneavoastră.** /aʃˈtept rəsˈpun.sul dum.ne̯aˈvo̯as.trə/ — お返事お待ちしています(直訳:あなたの返事を待ちます)【チャンク】。フォーマルメールの締めの定番
- **Cu mulțumiri,** /ku mul.t͡suˈmirʲ/ — 感謝を込めて(結びとしても使える)

tu 系のくだけた版もある:**Îți scriu ca să...**(君に書くのは〜のため)/ **Aștept răspunsul tău.**(君の返事を待ってる)。dumneavoastră(あなたの)を tău(君の)に替えるだけでレジスターが下がる。

### モデルメール 3 本

同じ「連絡する」でも、レジスターで文面がどう変わるかを3本の実例で見る。

#### ① 友人への週末の誘い(カジュアル + 未来 o să)

```
Salut, Andrei!

Ce faci în weekend? O să merg la un concert sâmbătă seara.
Vrei să vii cu mine? O să fie super!
Scrie-mi când poți.

Pa,
Vlad
```

対訳:
> やあ、アンドレイ!
> 週末どうしてる?土曜の夜コンサートに行くんだ。
> 一緒に来ない?最高だと思う!
> 都合ついたら連絡して。
> じゃあね、ヴラド

ポイント:tu 系(faci, vrei, vii)、未来は **o să merg / o să fie**(第4章)、締めは Pa。命令の **Scrie-mi**(連絡して)は「書く+私に」のチャンクとして覚えてよい。

#### ② 語学学校への問い合わせ(標準)

```
Bună ziua,

Vă scriu pentru că sunt interesat de cursurile de limba română.
Aș vrea să vă întreb dacă mai aveți locuri pentru grupa de începători
și cât costă un curs.

Mulțumesc pentru răspuns.

Cu stimă,
Kenji Sato
```

対訳:
> こんにちは。
> ルーマニア語の講座に興味があってご連絡しています。
> 初級グループにまだ空きがあるか、講座の料金がいくらかをお尋ねしたいのですが。
> お返事ありがとうございます。
> 敬具、佐藤ケンジ

ポイント:標準の書き出し Bună ziua、用件チャンク **Vă scriu pentru că...** と **Aș vrea să vă întreb dacă...**、丁寧な締め Cu stimă。話し手が男性なので interesat(女性なら interesată)。

#### ③ ホテルへの予約確認(フォーマル)

```
Stimate domnule,

Vă scriu în legătură cu rezervarea mea pentru data de 15 august.
Aș dori să confirm rezervarea pentru o cameră dublă, două nopți.
Vă rog să îmi confirmați dacă rezervarea este în regulă.

Aștept răspunsul dumneavoastră.

Cu respect,
Kenji Sato
```

対訳:
> 拝啓
> 8月15日の私の予約の件でご連絡いたします。
> ダブルルーム1室、2泊の予約を確認したく存じます。
> 予約に問題がないかご確認いただけますようお願いいたします。
> お返事お待ちしております。
> 敬具、佐藤ケンジ

ポイント:フォーマルの書き出し Stimate domnule、硬い用件切り出し Vă scriu în legătură cu...、締めの定番 **Aștept răspunsul dumneavoastră.** + Cu respect。rezervare(予約)は前章の programare(診察の予約)と使い分ける(宿泊・座席は rezervare)。

### SMS / チャットの省略形(読めればよい)

若者や友人同士のチャットでは、正書法を崩した省略形が多用される。**自分では正書法で書くのが原則** だが、相手のメッセージを **読めるように** 代表的なものを知っておく。

| 省略形 | 元の語 | 意味 |
|---|---|---|
| **k** | ca / că | 〜のように / 〜ということ |
| **sal** | salut | やあ |
| **ms** | mersi / mulțumesc | ありがとう |
| **npc** | nicio problemă | 問題ないよ / どういたしまして |
| **pt** | pentru | 〜のために |

これらは **くだけたチャット専用** である。若者言葉・俗語のより詳しい世界(スラング全般)は `vocabulary/slang-and-colloquial.md` に送る。本章では「読めれば十分、自分は正書法」という線引きだけ押さえる。

### ダイアクリティクス省略の現実

ルーマニア語には ș ț ă â î というダイアクリティクス(発音区別符号)があるが、**日常のメール・チャットではこれらを省いて書く人が非常に多い**(例:mulțumesc → multumesc、ș/ț を s/t で代用)。キーボード設定や手間の問題で定着した習慣である。しかし:

- **学習者は正書法で書く** — 崩れた綴りに慣れると、正しい形が身につかない
- **フォーマル文書は必ず正書法** — 公式メール・申込書・履歴書などでダイアクリティクスを省くと、雑・非公式な印象を与える
- **読むときは省略綴りも想定する** — 相手が multumesc と書いてきても mulțumesc だと分かる

つまり **「読むときは寛容に、書くときは(特にフォーマルでは)正確に」** が学習者の構えである。本教材の例文はすべて DOOM3 準拠の正書法(ș/ț はコンマ付き)で示している。

## 重要ポイント

- 書き言葉は丁寧度 **3段階**:カジュアル(Salut!)/ 標準(Bună ziua,)/ フォーマル(Stimate domnule,)
- 書き出しの **Stimate(男)/ Stimată(女)** は相手の性で語尾一致。呼びかけの後はコンマ
- 結び:超カジュアル Pa / Te pup、親しみ Cu drag、標準 Toate cele bune、フォーマル Cu stimă / Cu respect
- 用件チャンク:**Vă scriu pentru că...** / **Aștept răspunsul dumneavoastră.**(フォーマルの締め)
- 丁寧度を1段下げるには dumneavoastră → tu(Aștept răspunsul tău. など)
- 省略形(k / sal / ms / npc / pt)は **読めればよい**、自分は正書法で書く
- ダイアクリティクス省略は現実に多いが、学習者・フォーマル文書は正書法を守る
- 日本式の過剰な前置き(「お世話になっております」相当)は不要。用件に直接入るのが標準

## よくある誤用

| ❌ 誤り | ✅ 正しく | 説明 |
|---|---|---|
| Stimată domnule Popescu,(男性に -ă) | **Stimate domnule Popescu,** | Stimate/Stimată は相手の性で語尾一致。男性 -e / 女性 -ă |
| Stimate doamnă Ionescu,(女性に -e) | **Stimată doamnă Ionescu,** | 上と逆パターン。女性には Stimată(-ă) |
| フォーマルメールを Salut! で始める | **Bună ziua, / Stimate domnule,** | Salut はチャット専用。目上・公式には使わない |
| フォーマルメールを Pa! で締める | **Cu stimă, / Cu respect,** | Pa は超カジュアル。公式文書の締めにはならない |
| 冒頭に長い前置き(「お世話になっております」を直訳)を書く | **Vă scriu pentru că...** で用件に直接入る | ルーマニア語では過剰な前置きは不要。挨拶+用件が標準 |
| フォーマルメールを multumesc(ダイアクリティクスなし)で書く | **mulțumesc**(ș/ț を正しく) | 学習者・公式文書は正書法必須。省略綴りは非公式な印象 |
| 目上への締めで Aștept răspunsul tău.(tău=君の) | **Aștept răspunsul dumneavoastră.** | tău は tu 系。フォーマルは dumneavoastră(あなたの) |

## フォーマル/カジュアルの違い

本章のテーマそのものだが、同じ「返事がほしい」を3段階で並べると差が一目で分かる。

- **カジュアル(友人へ SMS)**: Scrie-mi când poți! Pa!(都合ついたら連絡して!じゃ!)
- **標準(担当者へメール)**: Mulțumesc pentru răspuns. Toate cele bune,(お返事ありがとうございます。ご多幸を)
- **フォーマル(役職者へメール)**: Aștept răspunsul dumneavoastră. Cu respect,(お返事お待ちしております。敬具)

丁寧度は主に **①書き出し(Salut / Bună ziua / Stimate)②用件切り出し(Îți scriu / Vă scriu)③締め(Pa / Toate cele bune / Cu respect)** の3か所に集中して表れる。この3点を相手に合わせて揃えれば、文面全体のレジスターは自然に一貫する。

## 文化的注意点

- **前置きは不要、用件に直接入る** — 日本のビジネスメールでは「お世話になっております」「先日はありがとうございました」といった定型の前置きから入るのが礼儀だが、ルーマニア語(および欧米のメール文化一般)では、こうした儀礼的な前置きは **不要** である。書き出しの挨拶(Bună ziua, / Stimate domnule,)の後、**Vă scriu pentru că...** ですぐ用件に入るのが標準的で、かえって前置きが長いと回りくどい印象を与える。日本語話者が最も調整を要する点なので、意識して簡潔にする。
- **省略綴りは相手を見て** — 友人同士のチャットではダイアクリティクスや省略形が飛び交うが、これは親密さの表れでもある。相手が正書法で丁寧に書いてくる場面(仕事・目上)では、こちらも正書法で応じるのが礼儀である。**相手のレジスターに合わせる** のが無難。
- **名前と署名** — フォーマルメールでは結びの下に自分のフルネームを書く。必要なら肩書き・所属を添える。カジュアルなら下の名前だけで十分である。

## 練習問題

### 問題1: 丁寧度3段階に振り分けよ

次の書き出しを「カジュアル / 標準 / フォーマル」に分類せよ。
1. Stimate domnule Director, 2. Salut, Andrei! 3. Bună ziua,

### 問題2: 結びを選べ

1. 親友への SMS の締め。 2. 語学学校への標準メールの締め。 3. 会社役員への公式メールの締め。
選択肢: a. Cu respect, b. Pa! c. Toate cele bune,

### 問題3: Stimate / Stimată を選べ

1. ___ domnule Popescu,(相手は男性)
2. ___ doamnă Ionescu,(相手は女性)
3. ___ domnule Director,(相手は男性の部長)

### 問題4: 省略形を正書法に直せ

1. sal 2. ms 3. npc 4. pt 5. k

### 問題5: 次のカジュアルなメールを、語学学校への標準メールに書き換えよ

```
Salut!
Vreau să știu dacă mai aveți locuri. Cât costă?
Pa!
```
(書き出し・用件の丁寧化・締めをすべて標準レジスターに整える)

## 解答

### 問題1
1. **フォーマル**(Stimate domnule + 肩書き) 2. **カジュアル**(Salut + ファーストネーム) 3. **標準**(Bună ziua,)。
— 書き出し1行で丁寧度がほぼ決まる。Stimate 系=フォーマル、Salut=カジュアル、Bună ziua=標準。

### 問題2
1–b(Pa! は超カジュアル) / 2–c(Toate cele bune は標準の万能締め) / 3–a(Cu respect は最も硬いフォーマル)。
— 締めは書き出しとレジスターを揃える。カジュアルに Cu respect、公式に Pa は不一致。

### 問題3
1. **Stimate**(男性 → -e) 2. **Stimată**(女性 → -ă) 3. **Stimate**(男性 → -e)。
— Stimate/Stimată は相手の性に語尾一致。日本語話者が最も間違える点。

### 問題4
1. **salut** 2. **mersi**(または mulțumesc) 3. **nicio problemă** 4. **pentru** 5. **ca / că**。
— 省略形は読めればよい。自分で書くときはこの正書法の形にする。

### 問題5
```
Bună ziua,

Vă scriu pentru că aș vrea să știu dacă mai aveți locuri
și cât costă un curs.

Mulțumesc pentru răspuns.

Cu stimă,
[名前]
```
— 書き出し Salut → Bună ziua、用件に Vă scriu pentru că... を足して丁寧化、tu 系を dvs. 系の丁寧表現に、締め Pa! → Cu stimă,。3か所(書き出し・用件・締め)を揃えるのがコツ。
