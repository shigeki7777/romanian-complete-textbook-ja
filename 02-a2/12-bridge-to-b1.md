# 第12章:B1 への橋渡し【CEFR A2】

## この章の目標

この章を終えると、次のことができるようになる。

- A2 で自分が「何をできるようになったか」を具体的に言葉にできる
- A2(場面をこなす)と B1(考えを述べる)の違いを説明できる
- B1 で最初に出会う 3 つの新しい仕組み(接続法・半過去・条件法)を事前に知り、心の準備ができる
- A2 の知識を維持する復習習慣と、多聴多読の始め方を持って B1 に進める

この章は問題集ではなく**読み物**である。肩の力を抜いて読んでほしい。A2 の締めくくりとして、これまでの道のりを振り返り、次のレベルへの地図を手に入れる章である。

## 前提知識

- [第11章:A2 チェックポイントテスト](./11-a2-checkpoint-test.md) — 80 点以上で A2 修了・B1 進級推奨。まだの人はテストと復習を先に
- A2 全体([第1章](./01-daily-routines.md)〜[第8章](./08-basic-email-and-messages.md))

## 本文解説

### まず、あなたができるようになったこと(棚卸し)

A1 を終えたとき、あなたは「存在と属性」(〜である・〜を持つ)を言えるようになっていた。A2 を終えたいま、あなたは**時間軸の上で動く一人の生活者**として話せるようになっている。具体的に、A2 を終えたあなたは次のことができる。

**動詞と時間**
- [ ] 一般動詞の現在形を 4 群・-ez/-esc 型込みで 6 人称活用できる(lucrez, citesc, merg, fac, dorm)
- [ ] 再帰動詞(mă trezesc / mă culc)で一日の動作を言える
- [ ] 複合過去(am/ai/a/am/ați/au + 過去分詞)で「〜した」「〜したことがある」を言える
- [ ] 否定の縮約 n-am、再帰の縮約 m-am trezit / s-a trezit を作れる
- [ ] 未来を 3 通り(現在形 / o să / voi 型)で言い分けられる
- [ ] o să の 3 人称交替(lucreze / meargă / vină / fie / aibă)を作れる

**代名詞**
- [ ] 対格弱形(mă, te, îl, o, ne, vă, îi, le)と与格弱形(îmi, îți, îi, ne, vă, le)を使える
- [ ] Îmi place / Îmi plac を対象の数で使い分けられる(主語逆転の仕組みが分かる)
- [ ] 複合過去との縮約(L-am văzut / Mi-a plăcut)と o の後置(Am văzut-o)を作れる

**場面**
- [ ] 買い物・レストランで値段を尋ね、Aș dori… で丁寧に注文し、会計まで済ませられる
- [ ] 時刻を尋ね・答え、自分の一日を朝から夜まで語れる
- [ ] 切符を買い、道を尋ね、cu trenul / pe jos で移動手段を言える
- [ ] Mă doare capul / Am febră で症状を伝え、薬局・受診・緊急(112)の基本をこなせる
- [ ] カジュアル SMS からフォーマルメールまで、レジスターを 3 段階で書き分けられる

**読解・作文**
- [ ] 過去と未来を含む 100 語程度の手紙・日記を辞書なしで読める
- [ ] Vă scriu pentru că… / Aștept răspunsul dumneavoastră. で用件のあるメールを書ける

これだけのことが、できるようになった。ルーマニアの市場で Aș dori un kilogram de mere. と注文し、昨日のことを Ieri am fost la piață. と報告し、Mâine o să merg la Cluj. と予定を語り、体調が悪ければ Nu mă simt bine. と伝えられる——**生活の基本場面は、もうひととおり回せる**。まず、そのことを誇ってほしい。

### A2 と B1 の違い — 「場面をこなす」から「考えを述べる」へ

A2 と B1 の違いを一言でいうと、こうなる。

> **A2 = 「場面をこなす」。B1 = 「考えを述べる」。**

A2 であなたが身につけたのは、買い物・移動・受診・メールといった**場面の中で決まった目的を達成する**力である。何を言うべきかは場面がほぼ決めてくれた(値段を聞く、切符を買う、症状を言う)。

B1 では、場面ではなく**あなた自身**が話の中心になる。「なぜそう思うのか」「どちらが良いと思うか」「もし〜だったらどうするか」——意見・理由・仮定・提案を組み立てる段階である。

| | A2(場面をこなす) | B1(考えを述べる) |
|---|---|---|
| 話の中心 | 場面の目的(買う・行く・治す) | 自分の意見・理由・希望 |
| 時制・法 | 現在+複合過去+未来 | +半過去(背景・物語り)+接続法+条件法 |
| 文の形 | 単文の積み重ね+チャンク | 従属節(〜ということ・〜するために)で文を重ねる |
| 典型フレーズ | Cât costă? / Mă doare capul. | Cred că…(〜だと思う)/ Vreau să…(〜したい) |

急に別の言語になるわけではない。それどころか、**B1 の主役たちには、あなたはすでに全員会っている**。次でその「正体」を明かす。

### B1 の 3 つの主役 — 実はもう出会っている

**① 接続法(conjunctiv)— o să の「să」の正体**

B1 最大のテーマは接続法である。……と聞くと身構えるかもしれないが、あなたは A2 でずっと使ってきた。**o să merg の să**、**Vreau să cumpăr… の să**、**Vreau să fac o programare. の să**——あれが接続法である。

第4章で「o să に続く動詞は、ほぼ現在形だが 3 人称だけ語尾が変わる」と学んだ。あの「3 人称だけ変わる形」こそ接続法現在の姿だ。B1 では、この să 構文が未来や「買いたい」だけでなく、願望・目的・必要・提案(〜してほしい、〜するために、〜しなければならない)へと一気に広がる。**動詞が 2 つ連続する文の多くが să を挟む**——この感覚をつかむのが B1 前半の山場である。o să と Vreau să… を体に入れてあるあなたは、すでに登り口に立っている。

**② 半過去(imperfect)— 物語りの過去**

A2 の複合過去は「〜した」という**完結した出来事**を語る形だった。B1 では、もう 1 つの過去——半過去(imperfect)——が加わる。役割分担はこうだ。

- **複合過去** = 一回きりの完了した出来事(昨日、市場に行った)
- **半過去** = 過去の習慣・続いていた状態・背景描写(子供の頃、クルージュに住んでいた)

日本語の「た」はこの 2 つを区別しないので、日本語話者はどちらか一方に頼りがちになる。だが半過去が使えると、「あの頃は毎朝コーヒーを飲んでいた」「外は雨が降っていた。私は本を読んでいた」のような**物語り**ができるようになる。過去がぐっと立体的になる、B1 で一番楽しい文法である。

**③ 条件法(conditional)— Aș dori の「Aș」の正体**

もう 1 人の主役にも、あなたはとっくに会っている。第2章からずっと使ってきた **Aș dori…(〜をいただきたいのですが)の Aș** ——あれが条件法である。条件法は「もし〜なら〜するだろう」という仮定に加えて、**丁寧なお願い・控えめな希望**(Aș vrea… = 〜したいのですが)として日常会話に頻出する。B1 では、チャンクとして丸暗記していた Aș dori を、自分で自由に組み立てられる形(aș / ai / ar / am / ați / ar + 不定詞)へと展開する。

つまり B1 の新文法は、**A2 で「チャンク」として先に使ってきた表現の中身を開けていく**作業でもある。「あ、これが o să の să か」「Aș dori の aș はこう動くのか」——この「正体の回収」は本書が A1 からずっと採ってきた学び方(先に使う → 後で仕組みを理解する)の集大成である。

### B1 の章構成

B1 の全体像は [B1 概要](../03-b1/00-b1-overview.md) にまとまっている。B1 を始める前に必ず一度読んでほしい。章構成はおおまかに次の流れである。

- **半過去と物語り**(文法)— 複合過去との使い分けで過去を立体化する
- **意見表明**(会話)— Cred că…(〜だと思う)で自分の考えを言う
- **接続法導入・条件法導入**(文法)— o să / Aș dori の正体を回収し、自由に使えるようにする
- **仕事と行政・医療コミュニケーション**(会話)— A2 の場面力を「手続き・説明」レベルへ引き上げる
- **ビジネスメール**(作文)— 第8章のレジスター 3 段階を業務水準へ
- **社会的話題の読解**(読解)— 100 語の手紙から、社会的な文章へ
- **復習・チェックポイントテスト・B2 への橋渡し**で締めくくる

A2 と同じく「文法 → 会話 → 作文 → 読解」を織り交ぜて進む構成であり、進め方のコツも変わらない。**声に出す・書いてみる・間違えたら該当章に戻る**。

### A2 の知識を維持する習慣

新しいことを学ぶと、古いことは自然に薄れていく。対策は「思い出す機会を意図的に作る」ことに尽きる。おすすめの復習スケジュールを挙げる。

| タイミング | やること |
|---|---|
| **1 週間後** | [第9章:総復習練習](./09-a2-review-exercises.md) をもう一度解く(答えを覚えていても、手が動くか確認) |
| **1 か月後** | [第11章:チェックポイントテスト](./11-a2-checkpoint-test.md) を再挑戦し、前回との得点差を見る |
| **毎日 5 分** | 複合過去の助動詞 am/ai/a/am/ați/au と、o să の 3 人称(lucreze / meargă / vină / fie / aibă)を口で唱える。Îmi place / Îmi plac を身の回りの物で言ってみる |
| **B1 の各章の前** | 対応する A2 の章を 5 分だけ読み返す(例: B1 の接続法の前に [第4章](./04-basic-future-expressions.md) の o să を復習) |

特に**複合過去の助動詞**と**弱形代名詞**は、B1 でも毎日使う「筋肉」である。半過去も接続法も、複合過去と代名詞の土台の上に載る。使わないと衰えるので、短くてよいから毎日触れてほしい。

### 多聴多読の始め方 — 教科書の外へ半歩

B1 からは、教科書だけでなく**生のルーマニア語に触れる量**が伸びを左右する。とはいえ、いきなりニュースや映画に挑むと挫折する。A2 修了時点の現実的な始め方を挙げる。

1. **まず本書のモデルテキストを「多読の種」にする** — [第3章](./03-basic-past-tense.md) の Ziua mea de ieri、[第8章](./08-basic-email-and-messages.md) のモデルメール 3 本、[第11章](./11-a2-checkpoint-test.md) の Cristina の手紙。すでに読んだ文章を**声に出して繰り返し読む**のが、いちばん確実な「聴く・読む」の筋トレになる。
2. **子ども向け・学習者向けのやさしい素材から** — 難しい素材で 3 割分かるより、やさしい素材で 9 割分かるほうが力がつく。「知らない語が 1 ページに数個」が適正レベルの目安である。
3. **ルーマニア語の音楽・短い動画を「ながら聴き」する** — 全部聞き取ろうとしなくてよい。知っている語(am fost, îmi place, o să…)が耳に飛び込んでくる体験を積むことが目的である。
4. **毎日 10 分、やめないこと** — 週末に 1 時間より、毎日 10 分。多聴多読は総量がすべてなので、続けられる形に小さくする。

## 重要ポイント

- A2 = 「場面をこなす」、B1 = 「考えを述べる」(意見・理由・仮定・提案)
- B1 の 3 つの主役には既に出会っている:①接続法 = **o să / Vreau să… の să の正体** ②半過去 = 物語り・背景の過去 ③条件法 = **Aș dori の Aș の正体**
- B1 の新文法は、A2 のチャンクの「中身を開ける」作業。ゼロからではない
- 維持の習慣: 1 週間後に第9章、1 か月後に第11章、毎日 5 分(助動詞・o să 3 人称・îmi place)
- 多聴多読は「やさしい素材を毎日 10 分」から。モデルテキストの音読が種になる

## 学習相談 Q&A

これまで多くの学習者が A2 修了時に抱いてきた悩みに、具体的な助言で答える。

### Q1. 複合過去と現在形の使い分けが、まだ怪しいです。

判断の軸は 1 つでよい——**「終わった出来事か、いまの習慣・状態か」**。終わった出来事(昨日・さっき・先週の 1 回)は複合過去、いまの習慣・状態・確定した近い予定は現在形である。コツは**時の表現とセットで文を作る癖**をつけること。ieri / aseară / săptămâna trecută が付くなら複合過去(Ieri am lucrat.)、de obicei / în fiecare zi が付くなら現在形(De obicei lucrez de acasă.)、mâine なら現在形か o să(Mâine plec. / O să plec.)。時の表現が時制を選んでくれる。迷ったら文頭に時の表現を置いてから動詞を選ぶ、を 2 週間続けると反射になる。

### Q2. 弱形代名詞が反射的に出てきません。文を作るとき毎回止まってしまいます。

全員がそうなる。原因は、日本語に「動詞の前に置く短い代名詞」という部品が存在しないからで、理解の問題ではなく**運動の問題**である。処方箋は「考えて組み立てる」のをやめて、**高頻度のかたまりを口癖にする**こと。Îmi place. / Nu-mi place. / Îl văd. / Te ajut. / Mi-a plăcut. / L-am văzut. ——この 6 つを、意味を考えずに言えるまで毎日唱える。次に名詞だけ入れ替える(Îmi place cafeaua. → Îmi place muzica. → Îmi plac fructele.)。文法表([第5章](./05-objects-and-pronouns.md))は「困ったときに引く地図」であり、話すときは地図ではなく体で動く。B1 でも代名詞は毎日出てくるので、この投資は必ず回収できる。

### Q3. o să の 3 人称交替(vine → vină)をよく間違えます。B1 の接続法が不安です。

まず安心してほしい——**間違える場所が「3 人称だけ」だと分かっている**時点で、山の 8 割は越えている。1・2 人称は現在形と同じなのだから、覚え直すのは頻出動詞の 3 人称形だけでよい。**o să lucreze / o să plece / o să meargă / o să facă / o să vină / o să fie / o să aibă** ——この 7 つを 1 セットにして、朝の 5 分で唱える。そして B1 の接続法の章は、まさにこの形を体系として整理し直してくれる章である。いま完璧でなくてよい。「3 人称は –ă と –e が入れ替わる、fie と aibă は特別」という感覚だけ持って B1 に入れば、接続法の章で一気に腑に落ちる。

### Q4. 聞き取りが全然追いつきません。多聴多読はどこから始めればいいですか?

A2 修了時点で生の会話が聞き取れないのは**正常**である。ネイティブの速度では n-am / nu-mi / ce-ai のような縮約が連続し、知っている語でも音の形が変わるからだ。対策は 2 段階。**第 1 段階(いま)**: 本文で正体を知っている音源=自分の音読を使う。モデルテキスト(Ziua mea、Cristina の手紙)を毎日音読し、縮約を 1 音節で言えるようにする。自分の口で言える音は、耳でも拾えるようになる。**第 2 段階(B1 と並行)**: やさしい素材の「ながら聴き」を毎日 10 分。全部聞き取ろうとせず、知っているチャンク(o să fie bine, îmi pare rău…)が聞こえた回数を数えるゲームにする。「聞こえた!」の成功体験が継続の燃料になる。

### Q5. B1 は難しくなりすぎませんか? 接続法・半過去・条件法と聞くだけで不安です。

不安になるのは、真剣に取り組んできた証拠である。安心してほしい理由が 3 つある。**第一に**、B1 の 3 つの主役すべてに、あなたはもう会っている(o să の să、Aș dori の aș、そして「過去を語る」経験そのもの)。ゼロから 3 つ覚えるのではなく、知っている形の中身を開けていくだけだ。**第二に**、本書は B1 でも「チャンクとして先に使う → 後で仕組みを理解する」という A1・A2 と同じやさしい順序で進む。**第三に**、詰まったらいつでも A2 に戻れる——各章に相互リンクがあり、第10章の復習マップも使える。A1 の終わりにも同じ不安があったはずだ。そして、あなたはいまここにいる。一歩ずつで大丈夫である。
