# 第7章:医療の基本場面【CEFR A2】

> **免責**: 本章は語学学習を目的とした教材であり、医学的・法的助言ではない。
> 実務で使用する場合は、必ず資格を持つ専門家およびネイティブ話者の確認を経ること。
> 症状の自己判断・自己治療の指針として本章を用いてはならない。体調不良や緊急時には、必ず医療機関・救急サービスに連絡すること。

## この章の目標

この章を終えると、次のことができるようになる。

- 「〜が痛い」を中核構文 **a durea**(Mă doare capul.)で言え、痛む部位の数に動詞を一致させられる
- 身体部位の基本語彙(12語程度)を使い、どこが痛いかを指し示せる
- 熱・吐き気・咳・風邪・めまいなど、基本的な症状をひとことで伝えられる
- 薬局(farmacie)で「頭痛に効くものはありますか」と尋ね、処方箋(rețetă)の有無を確認できる
- 医院で予約(programare)を取り、医師の定番質問(Ce vă supără? / De când?)を聞き取って答えられる
- 緊急時に **112** へ通報し、「救急車を呼んでください」「助けが必要です」と言える

## 前提知識

- [第4章:未来表現の基礎](./04-basic-future-expressions.md) — `o să + 接続法`。予約や受診の予定を言うのに使う
- [第3章:過去形の基礎](./03-basic-past-tense.md) — 複合過去(perfectul compus)。「昨日から(痛い)」De ieri. のように、いつからかを言うのに応用する

- 挨拶(第2章 A1)と店に入るときの Bună ziua は、そのまま薬局・医院でも使う

この章の症状語彙・基本受診表現は、専門編の医療セクションの入り口でもある。より詳しい解剖語彙(臓器・器官の細かい区分)は `medical/anatomy.md` に、程度・頻度を含む詳しい症状描写は `medical/symptoms.md` に送ってある。本章はあくまで **A2 段階の「基本場面」** に絞る。

## 本文解説

### 症状を言う中核構文 — a durea(痛む)

日本語の「頭が痛い」は「頭」が主語で「痛い」という状態を表す。ルーマニア語の発想はこれと大きく異なる。中核となる動詞 **a durea**(/a duˈre̯a/、痛ませる)は、「(痛みが)私を痛める」という他動詞的な構造を取る。つまり主語は **痛む身体部位** のほうで、痛みを感じている「私」は目的語(mă = 私を)として現れる。

```
Mă     doare      capul.
私を    痛める     頭が(定形)
└─弱形代名詞  └─a durea 三単  └─主語(痛む部位)
```

直訳すると「私を痛める、頭が」となる。日本語話者にとって最大のポイントは、**主語が身体部位のほうだ**という点である。したがって:

- 痛む部位が **1つ**(単数)なら、動詞は三人称単数 **doare** /ˈdo̯a.re/
- 痛む部位が **複数**(両脚など)なら、動詞は三人称複数 **dor** /dor/

この「部位の数に動詞が一致する」規則は、日本語の「痛い」が変化しないのと対照的で、非常に間違えやすい。

- **Mă doare capul.** /mə ˈdo̯a.re ˈka.pul/ — 頭が痛い(cap は単数 → doare)
- **Mă dor picioarele.** /mə dor piˈt͡ʃo̯a.re.le/ — 脚が痛い(両脚 = 複数 → dor)
- **Mă doare piciorul.** /mə ˈdo̯a.re pi.ˈt͡ʃo.rul/ — (片方の)脚が痛い(単数 → doare)
- **Mă doare în gât.** /mə ˈdo̯a.re ɨn ɡɨt/ — のどが痛い(în を挟む言い方も自然)

「私を」の部分は弱形代名詞(第5章)で、相手を変えれば主体も変わる:

- **Te doare capul?** — 頭痛いの?(te = 君を)
- **Îl doare stomacul.** — 彼はお腹(胃)が痛い(îl = 彼を)

A2 では、まず自分のことを言う **Mă doare / Mă dor** の2形を、部位の数で選び分けられれば十分である。

### 身体部位の基本語彙

痛む場所を言うには、身体部位の名詞が要る。ここでは受診で頻出する **12語程度** に絞る(細かい臓器・器官の語彙は `medical/anatomy.md` 送り)。名詞なので性と複数形を意識してほしい。a durea 構文では **定形**(語尾に冠詞が付いた形)で使う点に注意する。

| 部位 | 単数(定形) | 複数(定形) | 意味 |
|---|---|---|---|
| 頭 | capul | — | 頭 |
| のど・首 | gâtul | — | のど/首 |
| 胃・みぞおち | stomacul | — | 胃 ⚠ |
| お腹 | burta | — | 腹 ⚠ |
| 背中・腰 | spatele | — | 背中/腰 |
| 脚・足 | piciorul | picioarele | 脚(股下全体) |
| 手・腕 | mâna | mâinile | 手/腕 |
| 目 | ochiul | ochii | 目 |
| 耳 | urechea | urechile | 耳 |
| 歯 | dintele | dinții | 歯 |

「胸(piept)」「鼻(nas)」なども足せるが、A2 の最初はこの範囲で困らない。

### 症状を伝える定型表現

痛み以外の症状は、動詞 **a avea**(持つ)や **a fi**(である)を使った短い型で言える。

- **Am febră.** /am ˈfe.brə/ — 熱がある(直訳:私は熱を持っている)
- **Am greață.** /am ˈɡre̯a.t͡sə/ — 吐き気がする(greață = 吐き気)
- **Tușesc.** /tuˈʃesk/ — 咳が出る(a tuși「咳をする」の一人称)
- **Sunt răcit.** /sunt rəˈt͡ʃit/(男)/ **Sunt răcită.** /sunt rəˈt͡ʃi.tə/(女) — 風邪をひいている ⚠
- **Amețesc.** /a.meˈt͡sesk/ — めまいがする(a ameți の一人称)
- **Nu mă simt bine.** /nu mə simt ˈbi.ne/ — 気分が良くない(万能の一言。第5章の mă を使う)

**răcit / răcită** は形容詞なので、話し手の性で語尾が変わる(男 -it / 女 -ită)。これは第2章で見た bun→bună と同じ、性による語尾変化である。

### 薬局(farmacie)での会話

ルーマニアでは軽い不調はまず薬局に行き、薬剤師(farmacist)に相談するのが一般的である。処方箋なしで買える市販薬を **medicamente fără rețetă**(処方箋なしの薬)という。定番の尋ね方を **チャンク**(丸ごと覚える表現)として持っておくとよい。

- **Aveți ceva pentru durere de cap?** /aˈvet͡sʲ ˈt͡ʃe.va ˈpen.tru duˈre.re de kap/ — 頭痛に効くものはありますか?(直訳:頭の痛みのために何かありますか)【チャンク】
- durere de cap(頭痛)、durere de gât(のどの痛み)、durere de stomac(胃痛)のように **durere de + 部位** で言い換えられる
- **Am nevoie de o rețetă?** /am neˈvo.je de o reˈt͡se.tə/ — 処方箋は必要ですか?
- **Se dă fără rețetă?** /se da ˈfə.rə reˈt͡se.tə/ — 処方箋なしで買えますか?(直訳:処方箋なしで出されますか)
- **rețetă** /reˈt͡se.tə/ = 処方箋。同じ語が料理の「レシピ」も意味する(文脈で判断)

### 医者の予約と受診の流れ

医院・診療所での基本の流れは「予約 → 受付 → 問診」である。

- **programare** /pro.ɡraˈma.re/ = 予約(名女)。**Vreau să fac o programare.**(予約を取りたい)/ 未来表現なら **O să fac o programare.**(第4章)
- 受診で医師が最初に言う定番質問を、**聞き取れる**ことが A2 の目標である(自分で言えなくてよい):
  - **Ce vă supără?** /t͡ʃe və ˈsu.pə.rə/ — どうされましたか?(直訳:何があなたを困らせていますか)【聞き取りチャンク】
  - **De când?** /de kɨnd/ — いつからですか?
- これに答える型:
  - **De ieri.** /de ˈjerʲ/ — 昨日からです(ieri = 昨日。複合過去「Am început să tușesc ieri.=昨日から咳が出始めた」の短縮的な応答)
  - **De trei zile.** /de trej ˈzi.le/ — 3日前からです
  - **De azi-dimineață.** /de ˈazʲ di.miˈne̯a.t͡sə/ — 今朝からです

De + 時点で「〜から(ずっと)」を表す。「昨日から今も続いている」というニュアンスは、複合過去(第3章)で「いつ始まったか」を言えることと地続きである。

### 緊急時 — 112

ルーマニアの緊急通報番号は **112** である。112 は EU 共通の単一緊急番号で、EU 域内の固定・携帯電話から無料でかけられ、警察・救急・消防につながる。ルーマニアは 112 を **唯一の国内緊急番号** として採用している(出典: European Commission, europa.eu / ec.europa.eu, 2026年時点)。旅行者・在住者ともに、覚える番号はこの1つでよい。

緊急時の最重要フレーズは、正確さより **確実に言える** ことが優先される。次を **チャンク** として暗記しておく。

- **Chemați o ambulanță!** /keˈmat͡sʲ o am.buˈlan.t͡sə/ — 救急車を呼んでください!(chemați = a chema「呼ぶ」の敬称命令)【チャンク】
- **Am nevoie de ajutor!** /am neˈvo.je de a.ʒuˈtor/ — 助けが必要です!(ajutor = 助け)
- **Sunați la 112!** /suˈnat͡sʲ la o su.nə do.ˈis.pre.ze.t͡ʃe/ — 112 に電話して!(112 は口語で「unu-unu-doi」と1桁ずつ読むのが確実)
- **Este o urgență.** /ˈjes.te o urˈd͡ʒen.t͡sə/ — 緊急です

番号 112 は日本語話者には「イチイチニ」と読みたくなるが、ルーマニア語では **unu, unu, doi**(ウヌ ウヌ ドイ)と1桁ずつ言うのが最も誤解がない。

## 重要ポイント

- 痛みは **a durea**:主語は「痛む部位」、私は目的語 mă。**Mă doare capul.**(頭が痛い)
- 部位の数に動詞が一致:単数 → **doare** / 複数 → **dor**(Mă dor picioarele.)
- 部位は **定形**(冠詞付き)で使う:cap → capul、picioare → picioarele
- 症状の型:**Am febră / Am greață / Tușesc / Sunt răcit(ă) / Amețesc / Nu mă simt bine**
- 風邪 **răcit(ă)** は性で語尾が変わる(男 -it / 女 -ită)
- 薬局:**Aveți ceva pentru durere de cap?** + **rețetă**(処方箋)の有無を確認
- 医師の定番質問 **Ce vă supără?** / **De când?** は聞き取れれば十分。答えは **De ieri.**
- 緊急番号は **112**(EU 共通・ルーマニアの唯一の番号)。**Chemați o ambulanță! / Am nevoie de ajutor!**

## ⚠ 誤訳・混同に注意する語

医療語は取り違えると別の症状として伝わる。A2 で特に注意すべき3点を挙げる。

- **⚠ stomac(胃)と burtă(腹)** — stomac /stoˈmak/ は解剖学的な「胃」、burtă /ˈbur.tə/ は口語で「お腹・腹部」全般を指す。「胃が痛い」と正確に言いたいなら **Mă doare stomacul**、「お腹(下腹含む)が痛い」なら **Mă doare burta**。日本語の「お腹が痛い」を機械的に stomac に訳すと、下腹部の痛みまで「胃」と伝わりうる。医療現場では位置がずれると診断に影響するため、慎重に使い分ける。
- **⚠ răcit / răcită(風邪)と gripă(インフルエンザ)** — răcit(ă) は「(軽い)風邪をひいた」状態を指す形容詞、**gripă** /ˈɡri.pə/ は「インフルエンザ」という別の病気(名詞)である。**Sunt răcit.**(風邪だ)と **Am gripă.**(インフルエンザだ)は重症度も対応も異なる。「風邪」を安易に gripă と訳すと、実際より重い病気を申告することになる。逆も然り。
- **⚠ a durea(痛む)と a suferi(苦しむ・患う)** — a durea は「(局所が)痛む」。**a suferi de...** は「(慢性的に)〜を患っている」で用途が違う(例:sufăr de diabet = 糖尿病を患っている)。「痛い」を a suferi で言うと不自然か、慢性疾患の申告に聞こえることがある。A2 の症状申告は a durea で統一するのが安全。

## ネイティブらしい言い換え

| 教科書的 | ネイティブが実際によく言う形 | 注 |
|---|---|---|
| Nu mă simt bine. | **Nu mă simt prea bine.** / **Mă simt rău.** | prea(あまり)を足すと柔らかい。Mă simt rău は「気分が悪い」とはっきり |
| Am durere de cap. | **Mă doare capul.** | 名詞 durere より a durea 構文のほうが会話では自然で多い |
| Sunt bolnav.(病気だ) | **Nu mă simt bine.** / **Sunt răcit.** | 具体的な症状で言うほうが実用的。bolnav は「病気だ」と大きく構える語 |
| Chemați o ambulanță. | **Sunați la 112!** | 緊急時は番号を直接言うのが速い。ambulanță より通じやすい場面も |

## フォーマル/カジュアルの違い

医療の場面は基本 **丁寧(dumneavoastră 系)** で臨むのが無難だが、家族や友人には tu 系になる。

- **医師・薬剤師・受付に(丁寧)**: Bună ziua. Aveți ceva pentru durere de cap? / Mă doare stomacul de ieri.
- **家族・友人に症状を話す(くだけた)**: Mă doare capul, iau o pastilă.(頭痛い、薬飲む)/ Ce ai? — Sunt răcit.(どうしたの?—風邪ひいた)
- 医師の質問は相手が丁寧形で来る: **Ce vă supără?**(vă = あなたを)。友人同士なら **Ce ai?**(何かあるの?/どうした?)

迷ったら丁寧形。医療機関では初対面かつ目上に準じる相手なので、第2章の「迷ったら dumneavoastră 系」の原則がそのまま効く。

## 文化的注意点

- **薬局は身近な相談窓口** — ルーマニアの都市部には薬局(farmacie)が多く、緑十字の看板が目印である。軽い不調(頭痛・のど・胃の不快感など)は、まず薬局で薬剤師に相談するのが一般的な習慣で、医師の診察を待たずに市販薬(medicamente fără rețetă)を求められることが多い。24時間営業(farmacie non-stop)の薬局もある。ただし本章の免責の通り、市販薬の選択も最終的には薬剤師・医師の判断に委ねること。
- **欧州健康保険カード(EHIC)** — EU/EEA 圏内では、加盟国が発行する **欧州健康保険カード(EHIC = European Health Insurance Card)** を提示すると、旅行・滞在中に公的医療を現地住民と同条件で受けられる仕組みがある。ルーマニアもこの制度の対象国である。**保険適用の範囲・費用負担・手続きの詳細は制度として複雑で変化もあるため、本章では深入りしない**。制度の具体的な内容は `medical/overview.md` に送る。渡航前に自国の保険当局で確認することを勧める。
- **料金・保険制度は本章では扱わない** — 診察料・薬価・公的保険(CNAS 等)の仕組みは高リスクかつ変化する情報のため、A2 の本章では **112 以外の数値・制度を書かない** 方針である。制度面は医療セクションを参照すること。

## 練習問題

### 問題1: a durea の動詞形(doare / dor)を選べ

1. Mă ___ capul.(頭が痛い)
2. Mă ___ picioarele.(脚が痛い/両脚)
3. Mă ___ un dinte.(歯が1本痛い)
4. Mă ___ ochii.(目が痛い/両目)

### 問題2: 症状を表すルーマニア語を書け

1. 熱がある。 2. 吐き気がする。 3. 咳が出る。 4. めまいがする。 5. (女性が)風邪をひいた。

### 問題3: 場面に合う一言を選べ(薬局 / 医院受付 / 緊急)

a. Chemați o ambulanță!  b. Aveți ceva pentru durere de gât?  c. Vreau să fac o programare.

1. のどの痛みに効く薬を薬局で探す。 2. 医院で予約を取る。 3. 路上で人が倒れている。

### 問題4: ⚠ 誤訳注意 — 正しい語を選べ

1. 解剖学的な「胃」が痛い: Mă doare(**stomacul** / **burta**)。
2. 「インフルエンザにかかった」: (**Sunt răcit** / **Am gripă**)。
3. 「気分が良くない」: Nu mă(**simt** / **doare**)bine.

### 問題5: 対話を完成させよ(医師と患者)

医師: Bună ziua. Ce vă supără?
患者: (「昨日から頭が痛い」と丁寧に答える)
医師: De când vă doare?
患者: (「昨日からです」と答える)

## 解答

### 問題1
1. **doare**(cap は単数) 2. **dor**(picioare は複数) 3. **doare**(dinte 1本=単数) 4. **dor**(ochi は複数)。
— 主語は「痛む部位」。部位が複数なら動詞も複数 dor に一致する、が全問共通の理由。

### 問題2
1. **Am febră.** 2. **Am greață.** 3. **Tușesc.** 4. **Amețesc.** 5. **Sunt răcită.**(女性形 -ită)。
— 症状名は a avea(febră/greață)、動詞は一人称(tușesc/amețesc)、状態は a fi + 形容詞(răcită)。

### 問題3
1–b(薬局で「のどに効くものは?」) / 2–c(受付で予約) / 3–a(緊急=救急車)。
— 場面の緊急度で言葉が変わる。路上の急変は迷わず 112 と Chemați o ambulanță!。

### 問題4
1. **stomacul**(解剖学的な胃) 2. **Am gripă**(インフルエンザは gripă。räcit は軽い風邪) 3. **simt**(a se simți=感じる。a durea ではない)。
— ⚠ の3点そのもの。stomac/burtă、răcit/gripă の取り違えは症状の誤申告につながる。

### 問題5
患者(1回目): **Mă doare capul de ieri.**(昨日から頭が痛いです)
患者(2回目): **De ieri.**(昨日からです)。
— De + 時点で「〜から」。De ieri は複合過去「昨日から続いている」の応答形。医師の Ce vă supără? / De când? を聞き取れることが目標。

## 復習チェックリスト

- [ ] Mă doare + 部位(定形)で「〜が痛い」と言え、語順が日本語と逆だと分かる
- [ ] 部位が単数なら doare、複数なら dor と動詞を一致させられる
- [ ] Am febră / greață、Tușesc、Amețesc、Sunt răcit(ă) を使い分けられる
- [ ] 薬局で Aveți ceva pentru durere de cap? と尋ね、rețetă の有無を確認できる
- [ ] 医師の Ce vă supără? / De când? を聞き取り、De ieri. と答えられる
- [ ] 緊急番号 112 と Chemați o ambulanță! / Am nevoie de ajutor! を暗記している
- [ ] ⚠ stomac/burtă、răcit/gripă を混同しないと理解している
- [ ] 本章は語学教材であり、実際の受診・緊急対応は専門家・救急に委ねると心得ている
