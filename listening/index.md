# 聴解

このセクションは、リスニング力を段階的に養うための戦略・練習教材をまとめた参照集である。本編(`01-a1/` 〜 `06-c2/`)には各レベルの会話パートに音声的要素が含まれるが、聴解は読解以上に「聞き取れる」と「理解できる」の間にギャップが生じやすい技能であり、専用の反復練習が必要になる。このセクションは、発音の識別(ミニマルペア)からディクテーション、そして実際の会話速度での聴解まで、段階を分けて集中的に鍛える場として機能する。

`listening-strategy.md` は他のすべてのファイルの土台となる方法論であり、トップダウン処理(文脈・話題からの推測)とボトムアップ処理(音の識別)の両方をどう組み合わせるかを扱う。`pronunciation-minimal-pairs.md` はルーマニア語特有の音素対立(たとえば ș/s、ț/t、â・î/a、母音の長さに影響されない位置での ă/a など)を聞き分ける訓練であり、[00-orientation/02-pronunciation-overview.md](../00-orientation/02-pronunciation-overview.md) や [grammar/phonetics-and-orthography.md](../grammar/index.md) で学んだ音韻知識を実際の聴取力に変換する橋渡しの役割を持つ。`dictation-practice.md` は聞いた音声を正書法(DOOM3 準拠、ダイアクリティクス正確表記)に書き起こす訓練で、聴解と正書法の両方を同時に強化する。

`listening-dialogues-a1-a2.md` と `listening-dialogues-b1-b2.md` は生活場面別の対話音声教材、`advanced-listening-c1-c2.md` はニュース・インタビュー・自然な会話速度の音声を扱う上級教材である。音声そのものの収録・生成方法は別途 `tools/` 配下の整備方針に従うが、このセクションではスクリプト・設問・解説を中心にテキストベースで教材を設計する。

## 学習の目安

- **A1〜A2 の学習者**: `listening-strategy.md` で聴解の基本姿勢を学んだ上で、`pronunciation-minimal-pairs.md` と `dictation-practice.md` で音の識別と表記の対応を固め、`listening-dialogues-a1-a2.md` で生活場面の対話に慣れる。
- **B1〜B2 の学習者**: `listening-dialogues-b1-b2.md` に進み、自然な会話速度・より複雑な話題への対応力をつける。
- **C1〜C2 の学習者**: `advanced-listening-c1-c2.md` を中心に、ニュースや討論のような編集されていない自然な音声に取り組む。[speaking/discussion.md](../speaking/index.md)・[speaking/debate.md](../speaking/index.md) と組み合わせることで、聴解とアウトプットを連動させて鍛えることができる。
