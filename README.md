# 日本語話者向けルーマニア語完全教科書

**Manual complet de limba română pentru vorbitorii de japoneză**

完全初心者(A1)からネイティブ相当(C2)までを一本の学習導線でつなぐ、日本語話者のためのルーマニア語教科書です。文法・語彙・会話・読解・作文・聴解・文化理解を連動させた「読み進められる教科書」として設計されており、GitHub 上で継続的に拡張・改善されます。

> 🚧 **本書は段階的に構築中です。** 現在の完成状況は [PROGRESS.md](./PROGRESS.md) と [ROADMAP.md](./ROADMAP.md) を参照してください。

---

## この教材の目的

- 日本語話者がゼロからルーマニア語を体系的に学び、最終的にネイティブに近い運用能力(C2 相当)を目指せる教材を提供する
- 単語帳・フレーズ集ではなく、**学習順序が設計された教科書**として機能する
- 日常会話に加えて、医療・ビジネス・行政・文化・歴史まで、実際にルーマニア語圏で生活・仕事をするために必要な言語運用を扱う

## 対象読者

- ルーマニア語を初めて学ぶ日本語話者
- ルーマニア・モルドバ共和国への移住者、駐在員、留学生
- 医療・ビジネス・行政の現場でルーマニア語を使う必要がある人
- 中上級で伸び悩み、体系的に学び直したい人

## 学習の進め方

1. **はじめに読む**: [00-orientation/00-how-to-use-this-textbook.md](./00-orientation/00-how-to-use-this-textbook.md)
2. **学習計画**: [study-plan.md](./study-plan.md)(週あたりの学習時間別モデルプラン)
3. **本編**: `01-a1/` から番号順に読み進める。各レベルは
   `導入 → 発音 → 語彙 → 文法 → 例文 → 会話 → 読解 → 作文 → 聴解 → 練習 → 解答 → 復習 → チェックポイントテスト → 次レベルへの橋渡し` の順に構成
4. **参照系**: 学習中に疑問が出たら [grammar/](./grammar/index.md)(参照文法)、[verb-conjugation/](./verb-conjugation/index.md)(活用表)、[vocabulary/](./vocabulary/index.md)(語彙集)を引く
5. **専門編**: B1 以降、必要に応じて [medical/](./medical/index.md)・[business/](./business/index.md) を並行して学ぶ

## ファイル分割方針

- 1ファイル1テーマ。長すぎる場合は `part-1` / `part-2` に分割
- レベル別本編は `NN-` 番号付きで学習順序を固定
- 練習問題と解答は A1〜A2 は同一章内、B1 以降は分離可
- 各セクションは `index.md` を起点にリンクでつながる
- 執筆規約の詳細は [STYLE_GUIDE.md](./STYLE_GUIDE.md)

## 専門セクションについて

- **[medical/](./medical/index.md)** — 症状・診察・薬・救急・同意書・医療通訳。誤訳リスクの高い表現には ⚠ を付し、曖昧表現の回避を扱います
- **[business/](./business/index.md)** — メール・会議・交渉・契約・採用・ルーマニア企業文化。断り方や信頼構築の言い回しまで
- **[culture/](./culture/index.md) / [history/](./history/index.md)** — 価値観・宗教・タブー・地域差・モルドバ共和国との社会言語的差異、ダキアから EU 加盟までの歴史と言語史

## 出典ポリシー

歴史・法制度・医療制度・統計・現代事情などの事実情報には信頼できる出典(政府機関・EU 公式・学術機関・Academia Română の辞書/文法書など)を明記します。詳細は [SOURCE_POLICY.md](./SOURCE_POLICY.md)、出典一覧は [bibliography.md](./bibliography.md)。
