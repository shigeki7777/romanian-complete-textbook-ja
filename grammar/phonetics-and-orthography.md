# 音韻と正書法(fonetică și ortografie)【対象 CEFR: A1〜A2】

## この参照文法の位置づけ

ここはレベル別本編の [01-a1/01-pronunciation-and-alphabet.md](../01-a1/01-pronunciation-and-alphabet.md) で学んだ発音の知識を、体系的な「音の地図」として引き直すための辞書的な章である。本編の第1章は、学習の出発点として「まず声に出して読めるようになる」ことを目的としていた。この章はその先を扱う。すなわち、①ルーマニア語の音素(意味を区別する最小の音の単位)を母音・半母音・子音に整理して総覧し、②綴りと音の対応規則(正書法)を DOOM3(Academia Română, 2021)準拠でまとめ、③名詞の複数形や動詞の活用で繰り返し現れる**音交替(母音交替・子音交替)を一望する表**を用意する。

本編との役割分担をはっきりさせておく。**本編は「学ぶ」ため**の章であり、音を1つずつ導入して練習させる。**この参照文法は「引く」ため**の章であり、たとえば名詞の章で `casă → case` の母音交替に出会ったとき、「これは何交替だったか」をここで確認する、という使い方をする。両者は矛盾しない。本編で述べた事実(母音は7つ、â と î は同じ音、語頭・語末は î・語中は â、語末 -i は口蓋化 /ʲ/ など)を、ここではより網羅的な表の形で再提示する。

音交替を独立した節で扱うのは、この教材の設計思想による。ルーマニア語の名詞複数形・女性単数 G/D 形([cases.md](./cases.md))・動詞活用は、語幹の母音や子音が規則的に入れ替わることで作られる。`carte → cărți`(本)一語の中に、母音交替 a→ă と子音交替 t→ț が同時に起きている。この「交替の仕組み」を音韻の視点で一度整理しておくと、以後の形態論(名詞・動詞の形づくり)がぐっと見通しよくなる。

## 1. 母音体系 — 7母音

ルーマニア語の母音は7つ。日本語の「アエイオウ」5母音に、**ă /ə/ と â・î /ɨ/ の2つを加える**だけである。â と î は綴りが違うだけで**同じ音 /ɨ/** を表す(綴り分けは §5.1)。

| 母音 | IPA | 舌の位置(前後) | 舌の高さ | 唇 | 日本語話者向けの出し方 |
|---|---|---|---|---|---|
| i | /i/ | 前 | 高 | 非円唇 | 「イ」でよい |
| e | /e/ | 前 | 中 | 非円唇 | 「エ」でよい |
| a | /a/ | 中〜後 | 低 | 非円唇 | 「ア」でよい |
| ă | /ə/ | 中央 | 中 | 非円唇 | 口の力を抜いた曖昧な「ア」。英語 about の a |
| â / î | /ɨ/ | 後寄り中央 | 高 | 非円唇 | 「ウ」の口のまま「イ」と言う。舌を奥に引く |
| o | /o/ | 後 | 中 | 円唇 | 「オ」より唇を丸める |
| u | /u/ | 後 | 高 | 円唇 | 「ウ」より唇を突き出す |

日本語話者が新たに獲得すべき母音は **ă と â/î の2つだけ**である。この2音は意味を区別する(音素である)ため、代用がきかない。本編第1章で示したとおり、`rău` /rəw/(悪い)と `râu` /rɨw/(川)、`fată` /ˈfa.tə/(娘)と `fata` /ˈfa.ta/(その娘=定形)は、母音1つの違いで別語になる。

### 1.1 母音の対立(ミニマルペア)

| 対立 | 語1 | 語2 | 何が違うか |
|---|---|---|---|
| /ə/ vs /ɨ/ | rău /rəw/(悪い) | râu /rɨw/(川) | ă か â か |
| /a/ vs /ə/ | fata /ˈfa.ta/(その娘) | fată /ˈfa.tə/(娘) | 語末が a か ă か(定冠詞の有無) |
| /e/ vs /ə/ | mere /ˈme.re/(リンゴ複数) | mără…(該当語なし、練習用) | 交替の方向を意識する |

`fata`(定冠詞付き=その娘)と `fată`(無冠詞=娘)の対立は、単なる発音練習ではなく**文法の区別**([articles.md](./articles.md) の後置定冠詞)そのものである。ă を「ア」で読むと定形と無冠詞形を取り違えるため、母音の正確さは文法の正確さに直結する。

## 2. 半母音と二重母音・三重母音

ルーマニア語には、母音に添えて1拍で発音する**半母音(semivocală)**があり、母音との組み合わせで二重母音(diftong)・三重母音(triftong)を作る。半母音は音節の核を作らない点で、完全な母音と区別される。

### 2.1 半母音

| 半母音 | IPA | 対応する母音 | 例 |
|---|---|---|---|
| i の半母音 | /j/ | i | iarnă /ˈjar.nə/(冬)、mai /maj/(5月) |
| u の半母音 | /w/ | u | sau /saw/(または)、ou /ow/(卵) |
| e の半母音 | /e̯/ | e | seară /ˈse̯a.rə/(夕方) |
| o の半母音 | /o̯/ | o | școală /ˈʃko̯a.lə/(学校) |

IPA で半母音は、下に小さな弧 /e̯/ /o̯/ を付けるか、/j/ /w/ で表す。この教科書は本編と同じく `ea` を /e̯a/、`oa` を /o̯a/ と表記する。

### 2.2 頻度の高い二重母音

| 綴り | IPA | 1拍で | 例 |
|---|---|---|---|
| ea | /e̯a/ | 「エァ」 | seară /ˈse̯a.rə/(夕方) |
| oa | /o̯a/ | 「オァ」 | floare /ˈflo̯a.re/(花) |
| ia | /ja/ | 「ヤ」 | iarbă /ˈjar.bə/(草) |
| ie | /je/ | 「イェ」 | ieri /jerʲ/(昨日) |
| iu | /ju/ | 「ユ」 | iubire /juˈbi.re/(愛) |
| ai | /aj/ | 「アイ」 | mai /maj/(5月) |
| au | /aw/ | 「アウ」 | sau /saw/(または) |
| ei | /ej/ | 「エイ」 | lei /lej/(レイ・通貨) |
| oi | /oj/ | 「オイ」 | noi /noj/(私たち) |
| ou | /ow/ | 「オウ」 | ou /ow/(卵) |
| ui | /uj/ | 「ウイ」 | pui /puj/(ひな鳥) |

`ea` と `oa` は本編第1章でも最重要とされた二重母音で、**必ず1拍**で滑らかに発音する。これらは後述の母音交替(§4)で単母音 e・o と入れ替わるため、音韻の骨格として特に重要である。

### 2.3 三重母音

半母音+母音+半母音で3つの音が1拍にまとまる形もある。

| 綴り | IPA | 例 |
|---|---|---|
| eai | /e̯aj/ | (完了形などで)voiai …(揺れあり、代表例は少ない) |
| ioa | /jo̯a/ | aripioară /a.ri.pjo̯a.rə/(小さな翼) |
| eau | /e̯aw/ | beau /be̯aw/(飲む・1人称) |
| iai | /jaj/ | (2人称単純過去語尾など) |

三重母音は上級で頻度が上がる。A1〜A2 では `beau` /be̯aw/(私は飲む)のような高頻度語で出会う程度でよい。

## 3. 子音体系

子音は22音素ある。日本語にない、または綴りと音がずれるものを中心に整理する。

| 綴り | IPA | 日本語話者向けの注意 |
|---|---|---|
| p / b | /p/ /b/ | パ行・バ行の子音 |
| t / d | /t/ /d/ | タ行・ダ行の子音(母音を添えない) |
| c(a,o,u,子音の前) | /k/ | カ行。ただし e・i の前は /t͡ʃ/(§5.2) |
| g(a,o,u,子音の前) | /ɡ/ | ガ行。ただし e・i の前は /d͡ʒ/(§5.2) |
| k / q | /k/ | 外来語のみ |
| f / v | /f/ /v/ | 上歯+下唇。日本語のファ行より摩擦を強く |
| s | /s/ | サ行(濁らない) |
| z | /z/ | ザ行 |
| ș | /ʃ/ | 「シュ」の子音 |
| j | /ʒ/ | 「ジュ」の子音(舌先を付けない有声音) |
| ț | /t͡s/ | 「ツ」の子音(破擦音) |
| h | /h/(語中で /x/ 気味) | 「ハ」の子音。やや強め |
| m / n | /m/ /n/ | マ行・ナ行の子音 |
| l | /l/ | 舌先を歯茎に付ける明るい l |
| r | /r/ | はじき音〜ふるえ音(巻き舌)。英語の r ではない |
| ce / ci | /t͡ʃe/ /t͡ʃi/ | 「チェ・チ」 |
| ge / gi | /d͡ʒe/ /d͡ʒi/ | 「ヂェ・ヂ」 |
| che / chi | /ke/ /ki/ | 「ケ・キ」(h は硬さを守る印) |
| ghe / ghi | /ɡe/ /ɡi/ | 「ゲ・ギ」 |
| x | /ks/ または /ɡz/ | 「クス」/「グズ」(母音間で有声化する語がある) |

日本語話者が特に注意すべきは、①`l` と `r` の区別(日本語のラ行は `r` に近く、`l` は舌先を歯茎に付けて保つ)、②`ș`(シュ)と `s`(ス)、`j`(ジュ)と `z`(ズ)の区別、③破擦音 `ț` /t͡s/ と `c(e,i)` /t͡ʃ/、`g(e,i)` /d͡ʒ/ である。

## 4. 母音交替(alternanțe vocalice)— 語形変化の心臓部

ここからがこの参照文法の主眼である。ルーマニア語は名詞の複数形・女性単数 G/D 形・動詞活用を、**語幹の母音を規則的に入れ替える**ことで作る。交替のパターンは限られているので、一覧で押さえておくと形態論全体が楽になる。

### 4.1 母音交替の一覧表

| 交替 | 方向 | 起きる場面 | 例 |
|---|---|---|---|
| **a ↔ ă** | 単数 a → 複数/派生 ă | 名詞複数、動詞 | mare(海)→ mări、a spăla → spăl |
| **a ↔ e** | 単数 a → 複数 e | 女性名詞複数 | masă(机)→ mese、fată(娘)→ fete |
| **ă ↔ e** | 語幹 ă → e | 女性名詞複数、動詞 | vară(夏)→ veri、seară(夕)→ seri |
| **e ↔ ea** | 単母音 e ↔ 二重母音 ea | 動詞現在、名詞 | a vedea → văd/vede、negru → neagră |
| **o ↔ oa** | 単母音 o ↔ 二重母音 oa | 女性形、動詞、複数 | frumos(美しい・男)→ frumoasă(女)、a purta → poartă |
| **oa ↔ o** | 二重母音 oa → 単母音 o | 複数化・派生 | floare(花)→ flori、poartă(門)→ porți |
| **â/î ↔ i** | まれ | 一部の語 | cuvânt(語)→ cuvinte |

### 4.2 特に重要な2つ — o↔oa と e↔ea

この2つは**形容詞の男性形・女性形の作り分け**([adjectives.md](./adjectives.md))と**動詞の活用**に頻出する、最も見返りの大きい交替である。

- **o ↔ oa**: 男性 `frumos`(美しい)→ 女性 `frumoasă`。動詞 `a purta`(運ぶ・着る)の3人称 `poartă`(強勢が来ると o が oa に開く)。
- **e ↔ ea**: 男性 `negru`(黒い)→ 女性 `neagră`。動詞 `a vedea`(見る)の1人称 `văd` に対し3人称 `vede`。

覚え方の核はこうだ。**強勢が母音に乗って開くと、単母音 o・e が二重母音 oa・ea になる**。逆に強勢が語尾側へ移ると二重母音が単母音へ戻る(floare → flóri で強勢が -i 側へ、oa が o に縮む)。「強勢が来ると開く・離れると閉じる」という原理で、多くの交替が説明できる。

### 4.3 女性名詞複数と G/D 形での a↔e・ă↔e

[cases.md](./cases.md) §2.3 で扱う「女性単数 G/D 形は複数形と同形になる」という最重要規則は、この母音交替に支えられている。`casă`(家)の複数が `case`(a→e 交替)であり、その G/D 単数 `casei` もこの `case` を語幹にする。したがって**女性名詞は「複数形の母音交替」を覚えれば、G/D 形も自動的に作れる**。

| 単数 | 複数(交替) | G/D 単数(定) | 交替の型 |
|---|---|---|---|
| casă(家) | case | casei | a→e |
| masă(机) | mese | mesei | a→e |
| fată(娘) | fete | fetei | a→e |
| vară(夏) | veri | verii | ă→e |
| floare(花) | flori | florii | oa→o |
| țară(国) | țări | țării | a→ă |

## 5. 正書法(ortografie)

DOOM3(Academia Română, 2021)準拠の綴り規則を整理する。本編第1章で述べた原則を、より体系的に再掲する。

### 5.1 î と â の綴り分け(1993年正書法)

â と î は同じ音 /ɨ/ を表す。**どちらで綴るかは語中の位置で決まる。**

| 位置 | 綴り | 例 |
|---|---|---|
| 語頭 | **î** | început(始まり)、în(〜の中に)、înțeleg(分かる) |
| 語末 | **î** | a coborî(降りる)、a urî(憎む) |
| 語中 | **â** | român(ルーマニア人)、cânta(歌う)、mână(手) |
| 複合語・接頭辞付きで元の語頭の î が語中に来た場合 | **î を保つ** | neînceput(未着手の)、reîntâlnire(再会) |

最後の規則が盲点になりやすい。`neînceput` は `ne-`(否定接頭辞)+ `început` で、元の語頭 î がそのまま保たれる。単純に「語中だから â」とはしない。派生元の綴りを尊重する、と覚える。

> **例外的に â で始まる語**: 固有名詞や借用の一部に例外がある(Ântr… の類は現代標準では避けられる)。標準語では「語頭・語末=î」で困らない。

### 5.2 ce/ci・ge/gi 系(軟音化)

`c` と `g` は後続の文字で音が変わる。この規則はイタリア語と同じ仕組みで、**綴りの読みで唯一「暗記が必要」な部分**である。

| 綴り | IPA | 読み | 例 |
|---|---|---|---|
| ca, co, cu, c+子音 | /k/ | カ・コ・ク | casă /ˈka.sə/、cu /ku/ |
| ce, ci | /t͡ʃe/ /t͡ʃi/ | チェ・チ | cine /ˈt͡ʃi.ne/(誰)、ceai /t͡ʃaj/(茶) |
| che, chi | /ke/ /ki/ | ケ・キ | chelner /ˈkel.ner/(給仕)、chibrit(マッチ) |
| ga, go, gu, g+子音 | /ɡ/ | ガ・ゴ・グ | gară /ˈɡa.rə/(駅) |
| ge, gi | /d͡ʒe/ /d͡ʒi/ | ヂェ・ヂ | ger /d͡ʒer/(厳寒)、girafă(キリン) |
| ghe, ghi | /ɡe/ /ɡi/ | ゲ・ギ | ghid /ɡid/(ガイド)、ghețar(氷河) |

**h は「硬い音を保つ印」**であり、それ自体は音を持たない。「/t͡ʃe/ にしたくない・/ke/ と読ませたい」なら `che` と綴る。この仕組みは名詞複数化・動詞活用の子音交替(§6)と表裏一体である。`c` /k/ で終わる語幹に複数語尾 -i が付くと、`c` が軟音化して `ci` /t͡ʃ/ になる(amic → amici)。

### 5.3 語末の -i の3つの読み

本編第1章で扱った、語末 -i の読み分けを表で再掲する。これは単数・複数の聞き分け([nouns.md](./nouns.md))に直結する。

| 環境 | 読み | 例 |
|---|---|---|
| 子音+語末 -i(1つ) | /ʲ/(直前子音の口蓋化) | lupi /lupʲ/(狼複数)、ani /anʲ/(年複数) |
| -ii(2つ重なる) | /i/(完全母音)または /ij/ | copiii /koˈpi.ij/(その子供たち) |
| 子音+l/r+i | /i/(完全母音) | membri /ˈmem.bri/(メンバー複数) |
| 強勢が -i にある | /i/(完全母音) | a citi /a t͡ʃiˈti/(読む) |

### 5.4 ハイフンと縮約の正書法

速い口語で起きる母音の脱落・融合は、書き言葉では**ハイフン**で表記する。これは省略ではなく、正書法上の正式な綴りである。

| 縮約形 | 元の形 | 意味 | 読み |
|---|---|---|---|
| **s-a** | se + a | (彼が)〜した(再帰複合過去) | /sa/ |
| **n-am** | nu + am | 私は〜を持たない/していない | /nam/ |
| **într-o** | într(u) + o | ある(女)〜の中で | /ˈɨn.tro/ |
| **dintr-o** | dintr(u) + o | ある(女)〜の中から | /ˈdin.tro/ |
| **printr-un** | printr(u) + un | ある(男)〜を通って | /ˈprin.trun/ |
| **mi-a** | îmi + a | 私に〜した | /mja/ |
| **și-o** | și + o | そして(女)1つを | /ʃo/ |
| **l-am** | îl + am | 彼を〜した | /lam/ |

ハイフンで結ばれた2語は、発音上は1拍に融合するが、**書くときは必ずハイフンを入れる**。`s-a dus`(行った)を `sa dus` と書くのは誤り、`sa`(彼女の)という別語になってしまう。前置詞 `într-o` `dintr-o` `printr-un` の類は、母音で始まる不定冠詞(o / un)の前で `într(u)` の語末母音が落ちてハイフンで結ばれる形で、日常文に極めて多く現れる。

## 6. 子音交替(alternanțe consonantice)

母音交替と並んで、語末の子音が規則的に入れ替わる**子音交替**がある。名詞複数形(-i の前)と動詞活用に集中して現れる。本編第3章で `student → studenți`(t→ț)として導入済みの現象を、ここで表にまとめる。

### 6.1 子音交替の一覧表

| 交替 | 環境 | 例(単数 → 複数) |
|---|---|---|
| **t → ț** | 語幹末 t + -i | student → studenți、frate → frați、carte → cărți |
| **d → z** | 語幹末 d + -i | brad(モミ)→ brazi、verde(緑)→ verzi |
| **s → ș** | 語幹末 s + -i | urs(熊)→ urși、gros(太い)→ groși |
| **t → ț(語中)** | 派生 | perete(壁)→ pereți |
| **c → ci /t͡ʃ/** | 語幹末 c + -i | amic(友)→ amici、sac(袋)→ saci |
| **g → gi /d͡ʒ/** | 語幹末 g + -i | drag(愛しい)→ dragi |
| **sc → șt** | 語幹末 sc + -i(主に動詞活用) | 動詞 a crește → (tu) crești | 
| **st → șt** | 語幹末 st + -i | trist(悲しい)→ triști /triʃtʲ/ |

### 6.2 母音交替と子音交替が同時に起きる語

最も学習者を戸惑わせるのは、**1語の中で母音交替と子音交替が同時に起きる**場合である。本編第3章で `carte → cărți` を「二段変化」として注意したのがこれである。

| 単数 | 複数 | 母音交替 | 子音交替 |
|---|---|---|---|
| carte(本) | cărți | a→ă(語幹母音) | t→ț |
| poartă(門) | porți | oa→o | t→ț |
| fereastră(窓) | ferestre | ea→e | (str のまま) |
| negru(黒・男) | neagră(女) | e→ea | (逆方向:女性化で開く) |

`carte /ˈkar.te/ → cărți /kərt͡sʲ/` を分解すると、①語幹母音 a → ă(cart → cărt)、②語末 t → ț(cărt → cărț)、③複数語尾 -i の口蓋化 /ʲ/、の3つが重なっている。1語に見えるが、実は3つの規則の合成である——このように分解できると、不規則に見える形も暗記でなく理解で処理できる。

## 7. 強勢(accent)

ルーマニア語の強勢の位置は**語ごとに決まっており、綴りからは完全には予測できない**。最後から2番目の音節(paroxytone)に来ることが最も多いが、例外も多い。この教科書は例文の IPA に必ず強勢記号 `ˈ` を音節の直前に付けるので、単語は「音の形ごと」覚えること。

### 7.1 強勢が意味を分ける例

同じ綴りでも強勢の位置だけで意味が変わる語がある。

| 綴り | 語1(強勢位置) | 語2(強勢位置) | 意味の違い |
|---|---|---|---|
| veselă | **ve**selă /ˈve.se.lə/ | vese**lă** /ve.seˈlə/ | 前寄り=陽気な(形・女)/後寄り=食器類(名) |

本編第1章でも触れたとおり、強勢の位置は綴りから完全には予測できない。だからこの教科書は例文の IPA に必ず強勢記号 `ˈ` を付ける。単語は「音の形ごと」覚えることが肝心である。

強勢は §4 の母音交替とも連動する。強勢が母音に乗ると o・e が二重母音 oa・ea に開き(§4.2)、強勢が語尾側に移ると閉じる。したがって強勢と母音交替はセットで捉えるのが理にかなっている。

### 7.2 IPA 表記の約束(この教科書)

- 強勢は音節の**直前**に `ˈ`。例: `limbă` /ˈlim.bə/、`mulțumesc` /mul.t͡suˈmesk/。
- 音節境界は `.` で示す。
- 半母音は `/e̯/ /o̯/`(弧付き)または `/j/ /w/`。
- 口蓋化した語末子音は上付き `ʲ`。例: `lupʲ`。

## 8. よくある誤り(❌→✅)

これは発音・綴りの章のため、「日本語話者の綴り・音の誤り」を対で示す。

| ❌ 誤り | ✅ 正しく | 説明 |
|---|---|---|
| roman(român のつもり) | rom**â**n | 語中の /ɨ/ は â。î にしない |
| inceput(語頭に i) | **î**nceput | 語頭の /ɨ/ は î。â にしない |
| neanceput / neânceput | ne**î**nceput | 複合語は元の語頭 î を保つ |
| casa(「家」のつもり) | cas**ă** | 語末 -ă を -a にすると定冠詞付き(その家)になる |
| sa dus / n am | **s-a** dus / **n-**am | 縮約はハイフンで綴る。ハイフンなしは別語 |
| ştiinţă(セディーユ) | ș**t**iin**ț**ă | ș ț は必ずコンマ付き(§DOOM3) |
| studenti(t のまま) | studen**ț**i | 複数 -i の前で t→ț の子音交替 |
| carti(交替なし) | c**ă**r**ț**i | 母音交替 a→ă と子音交替 t→ț の二段変化 |
| școala を4拍「シュ・コ・ア・ラ」 | /ˈʃko̯a.lə/(2音節) | oa は1拍の二重母音 |

## 9. ネイティブらしい音・口語の縮約

発音章のため、言い換えの代わりに「話し言葉で起きる音の縮約」を挙げる。**自分では正式形を書き・言い**、耳のためだけに知っておく。

- **-sprezece → -șpe**: 数詞の口語縮約。unsprezece → unșpe /ˈun.ʃpe/、doisprezece → doișpe。書き言葉では必ず -sprezece(→ [numerals.md](./numerals.md))。
- **nu-i / nu e**: `nu este`(〜ではない)の縮約。
- **-a の脱落**: 速い会話で語末の弱い母音がさらに弱まるが、教科書レベルでは全母音をはっきり読んでよい。
- **ダイアクリティクスの省略**: SNS やチャットで `sa`(← să)、`inteleg`(← înțeleg)のように書かれるが、これは非標準の省略綴りである。学習者は必ず正しい文字で書く。読解は B2 以降で扱う。

## 10. 練習問題

### 問題1:î / â の綴りを選べ
次の /ɨ/ を含む語を正しく綴れ(î か â か)。
1. r_u(川) 2. _nceput(始まり)3. rom_n(ルーマニア人)4. ne_nceput(未着手の)5. a cobor_(降りる)

### 問題2:母音交替の型を答えよ
次の単数→複数で起きている母音交替(a→e / oa→o / e→ea など)を答えよ。
1. masă → mese 2. floare → flori 3. țară → țări 4. vară → veri

### 問題3:子音交替を補って複数形を作れ
1. student(学生)2. brad(モミの木)3. urs(熊)4. amic(友)5. trist(悲しい・男複)

### 問題4:カルテ問題(二段変化)
`carte`(本)の複数 `cărți` で起きている変化を、①母音交替 ②子音交替 ③語末 -i の読み、の3点に分けて説明せよ。

### 問題5:縮約をハイフン付きで正しく綴れ
1. se + a dus(彼は行った)2. nu + am(私は持っていない)3. într(u) + o zi(ある日に)4. îmi + a spus(私に言った)

---

## 11. 解答

### 問題1
1. r**â**u(語中=â)2. **î**nceput(語頭=î)3. rom**â**n(語中=â)4. ne**î**nceput(複合語で元の語頭 î を保つ)5. a cobor**î**(語末=î)

### 問題2
1. masă → mese = **a→e** 2. floare → flori = **oa→o** 3. țară → țări = **a→ă** 4. vară → veri = **ă→e**
— いずれも女性名詞。この複数形の語幹が G/D 単数形にもそのまま使われる([cases.md](./cases.md))。

### 問題3
1. studen**ți**(t→ț)2. bra**zi**(d→z)3. ur**și**(s→ș)4. ami**ci** /aˈmit͡ʃʲ/(c→ci、/k/→/t͡ʃ/)5. tri**ști** /triʃtʲ/(st→șt)

### 問題4
`carte /ˈkar.te/ → cărți /kərt͡sʲ/`。①**母音交替 a→ă**(語幹 car → căr)②**子音交替 t→ț**(cărt → cărț)③**語末 -i の口蓋化 /ʲ/**(-ți を /t͡sʲ/ と読む)。3つの規則の合成であり、暗記でなく分解で理解できる。

### 問題5
1. **s-a** dus 2. **n-**am 3. **într-o** zi 4. **mi-a** spus
— いずれもハイフンが必須。ハイフンを落とすと `sa`(彼女の)のような別語と衝突する。
