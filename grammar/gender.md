# 性(genul)【対象 CEFR: A1〜A2】

## この参照文法の位置づけ

ここはレベル別本編の [01-a1/03-nouns-and-gender-intro.md](../01-a1/03-nouns-and-gender-intro.md) で導入した「性(gen)」の知識を、判定確率と職業名詞の作り方まで含めて掘り下げるための辞書的な章である。本編では、①名詞に3つの性(男性・女性・中性)があること、②un/doi テストで見分けること、③中性が「単数は男性・複数は女性」として振る舞うこと、を最小限で扱った。この章はそれを深化させる。

性(gen)は日本語話者にとって、そもそも母語にない概念である。日本語では「机」も「本」も「先生」も文法上まったく同じに扱う。ところがルーマニア語では、名詞の性が決まらないと**冠詞も形容詞も数詞も正しく付けられない**。`un/o`(不定冠詞)、`bun/bună`(形容詞)、`doi/două`(数詞)が、すべて性で形を変える。だから名詞は「意味+性+複数形」の3点セットで覚える必要がある——これが本編第3章の核心的な主張だった。

この章では、その「性をどう見抜くか」を、確実な方法(un/doi テスト)から確率的な方法(語尾)まで段階的に整理し、さらに**中性の本質**(なぜルーマニア語の中性は独自形を持たないのか)と、**職業名詞の男女形の作り方**(-ă / -că / -easă / -oaică など)を扱う。複数形の作り方そのものは [nouns.md](./nouns.md)、格変化は [cases.md](./cases.md) に譲り、この章は「性の見分けと振る舞い」に集中する。

## 1. 3性の振る舞い — なぜ性が全体に波及するか

性は名詞そのものの分類だが、その影響は名詞単独にとどまらない。名詞に「一致(acord)」する語がすべて性に従って形を変える。

| 一致する語 | 男性(băiat 少年) | 女性(fată 少女) | 中性(scaun 椅子) |
|---|---|---|---|
| 不定冠詞 | **un** băiat | **o** fată | **un** scaun(単数=男性型) |
| 数詞「2」 | **doi** băieți | **două** fete | **două** scaune(複数=女性型) |
| 形容詞 | băiat **bun** | fată **bună** | scaun **bun**(単数=男性型) |
| 定冠詞 | băiat**ul** | fat**a** | scaun**ul**(単数=男性型) |

この表から中性の本質が読み取れる。**中性は単数で男性の一致、複数で女性の一致を受ける**。したがって「中性専用の冠詞・形容詞形」は存在しない——中性は常に男性か女性の衣を借りている。これが「ルーマニア語の中性は独自形を持たない」という言い方の意味である(§3)。

## 2. 語尾からの性の判定 — 確率で押さえる

un/doi テスト(§5)が確実だが、辞書を引くたびにテストするのは非効率である。語尾から**8割方あたる経験則**で当たりをつけ、迷う語だけテストで確定する、というのが実際的な運用である。

### 2.1 語尾別の判定確率

| 語尾 | 判定 | 確度 | 例 | 主な例外 |
|---|---|---|---|---|
| **-ă** | 女性 | 非常に高い | casă, fată, mamă, apă | **tată(父)は男性**(最重要例外) |
| **-a / -ea**(強勢あり) | 女性 | 高い | cafea, stea, basma | — |
| **-ie** | 女性 | 高い | familie, bucurie, primărie | — |
| **-re / -are**(動名詞) | 女性 | ほぼ確実 | creștere, mâncare, reducere | — |
| **-tate / -ime / -ie**(抽象名詞) | 女性 | 高い | libertate, înălțime, bogăție | — |
| **-e** | 女性が多い | 中 | carte, floare, pâine | **frate, perete, munte** は男性 |
| **子音終わり** | 男性か中性 | — | bărbat(男・男性)、oraș(町・中性) | un/doi テストで確定 |
| **-u / -iu** | 男性か中性 | — | codru(森・男)、muzeu(博物館・中) | テストで確定 |
| **-tor**(行為者) | 男性 | 高い | vânzător, muncitor | 女性形は -toare |

**実務的な結論**(本編第3章と同じ): -ă と強勢 -a/-ea・-ie で終われば女性、それ以外の母音・子音終わりは「男性か中性」で、そこは un/doi テストで割る。抽象名詞・動名詞(-tate、-re、-ime、-ie)が女性であることを覚えておくと、B1 以降の語彙で性の推測が大きく当たるようになる。

### 2.2 tată — 最重要例外の再確認

`tată`(父)は語末が -ă なのに**男性名詞**である。`un tată`(1人の父)、`doi tați`(2人の父)と un/doi を取る。これは幼児語起源の親族語(tată, popă「司祭」など)に見られる例外で、-ă=女性という強い規則を裏切る唯一クラスの重要語である。本編第3章でも「最頻出の例外」として強調した。

## 3. 中性(neutru)の本質 — 両性名詞

ドイツ語・ロシア語の中性は独自の形(独自の冠詞・語尾)を持つが、**ルーマニア語の中性は独自形を持たない**。単数では男性名詞として、複数では女性名詞として振る舞う。この意味で、ルーマニア語の中性は「両性名詞(ambigen)」とも呼ばれる。

| 中性名詞 | 単数(男性の振る舞い) | 複数(女性の振る舞い) |
|---|---|---|
| scaun(椅子) | **un** scaun / scaunul / scaun bun | **două** scaune / scaunele / scaune bune |
| oraș(町) | **un** oraș / orașul / oraș mare | **două** orașe / orașele / orașe mari |
| tren(電車) | **un** tren / trenul | **două** trenuri / trenurile |

したがって中性名詞を覚えるときは、**「単数で un、複数で două」**という組み合わせを口に出すのがそのまま判定になる。`un scaun, două scaune` と言えれば、それは中性である。「un で始まり două で終わる」——これが中性の指紋である。

中性名詞は無生物に多いが、絶対ではない(例:`animal`「動物」は中性)。逆に無生物でも女性(masă 机、casă 家)や男性(perete 壁、munte 山)がある。**中性=無生物と決めつけない**こと。判定はあくまで un/doi テストで行う。

## 4. 職業名詞・人を表す名詞の男女形

人を指す名詞は、多くが自然の性に従って男性形・女性形の対を持つ。女性形の作り方には決まったパターンがある。本編第3章で `student/studentă`、`profesor/profesoară` を挙げた、その形成規則をここで整理する。

### 4.1 女性形の作り方 — 接尾辞のパターン

| パターン | 男性形 | 女性形 | 意味 |
|---|---|---|---|
| **-ă を足す** | student | student**ă** | 学生 |
| **-ă を足す** | prieten | prieten**ă** | 友人 |
| **-oară**(-or → -oară) | profesor | profeso**ară** | 教師 |
| **-iță / -ță** | doctor | doctor**iță** | 医師 |
| **-că**(-c/-ic → -că) | chimist…—、tehnician → tehnicia**nă** | (例) | 技師系 |
| **-easă** | croitor(仕立て屋)| croit**oreasă** | 仕立て屋(女) |
| **-easă** | bucătar(料理人)| bucătăr**easă** | 料理人(女) |
| **-toare**(行為者 -tor → -toare) | vânzător | vânză**toare** | 店員 |
| **-oaică**(民族・動物) | român(ルーマニア人)| româ**noaică** | ルーマニア人(女) |
| **-oaică** | lup(狼)| lu**poaică** | 雌狼 |

代表的な生産パターンは次の4つに集約できる。

- **-ă**: 最も普通。男性形に -ă を足す(student → studentă、prieten → prietenă)。
- **-oară**: -or 終わりの職業名(profesor → profesoară、doctor は口語で doctoriță も)。
- **-easă / -easă**: 手仕事・伝統的職業(croitor → croitoreasă、bucătar → bucătăreasă)。
- **-oaică**: 民族名・動物の雌(român → românoaică、urs → ursoaică)。

### 4.2 性が変わらない・両性で使う名詞

一部の職業名は男女で形を変えず、文脈や冠詞・形容詞で性を示す。

| 名詞 | 扱い | 例 |
|---|---|---|
| medic(医師) | 男性形のまま男女に使う | Ea este medic.(彼女は医師だ) |
| inginer(技師) | 女性形 ingineră もあるが、男性形のまま使うことも多い | Ea este inginer(ă). |
| ministru(大臣) | 標準は男性形。女性形は議論あり | doamna ministru |

本編第3章の文化注記で触れたとおり、職業名の女性形の使い方は社会的な話題にもなる。A1〜A2 では、**確立した標準ペア(student/studentă、profesor/profesoară、vânzător/vânzătoare)をそのまま覚え**、`medic`・`inginer` のように女性形が揺れる語は男性形で通じることを知っておけばよい。

### 4.3 例文で見る性の一致

| ルーマニア語 | IPA/発音 | 自然な日本語訳 | 直訳 | 解説 | 使用場面・レジスター |
|---|---|---|---|---|---|
| El este un student bun. | /jel ˈjes.te un stuˈdent bun/ | 彼は良い学生だ | 彼 である 1 学生 良い | 男性:un + bun | 日常・中立 |
| Ea este o studentă bună. | /ja ˈjes.te o stuˈden.tə ˈbu.nə/ | 彼女は良い学生だ | 彼女 である 1 学生(女) 良い(女) | 女性:o + studentă + bună | 日常・中立 |
| Un scaun e liber, două scaune sunt ocupate. | /un ˈska.un e ˈli.ber ˈdo.wə ˈska.u.ne sunt o.kuˈpa.te/ | 椅子が1つ空いて、2つは埋まっている | 1 椅子 空いた 2 椅子 占められた | 中性:un(単)/ două(複) | 会話・中立 |
| Tata este inginer. | /ˈta.ta ˈjes.te in.d͡ʒiˈner/ | 父は技師だ | 父 である 技師 | tată は -ă だが男性。inginer は職業叙述で無冠詞 | 家族紹介・中立 |
| Doamna profesoară a venit. | /ˈdo̯am.na pro.feˈso̯a.rə a veˈnit/ | 先生(女性)が来た | 奥様 教師(女) 来た | profesor → profesoară(-oară) | 学校・丁寧 |
| Vânzătoarea e amabilă. | /vɨn.zəˈto̯a.re̯a e aˈma.bi.lə/ | その店員(女性)は親切だ | 店員(女)-その である 親切(女) | vânzător → vânzătoare(-toare) | 買い物・中立 |

## 5. un / doi テスト — 確実な判定法(再掲)

語尾の確率で迷ったら、**必ずこのテストで確定**する。「1つの〜」と「2つの〜」を言ってみて、使う語の組み合わせで性が分かる。

| 性 | 1つ | 2つ | 判定の言い方 |
|---|---|---|---|
| 男性 | **un** | **doi** | un student, doi studenți → 男性 |
| 女性 | **o** | **două** | o casă, două case → 女性 |
| 中性 | **un** | **două** | un scaun, două scaune → 中性 |

判定の骨子は「**単数で un か o か**」と「**複数で doi か două か**」の2軸である。

- un + doi → 男性
- o + două → 女性
- un + două(単数は男性型・複数は女性型)→ 中性

新しい名詞に出会ったら、辞書の意味だけでなく必ずこの形で口に出して覚える。これが本編第3章から一貫する学習動作である。

## 6. よくある誤用(❌→✅)

| ❌ 誤り | ✅ 正しく | 説明 |
|---|---|---|
| o tată | **un** tată | tată は -ă で終わるが男性(最頻出例外) |
| doi case | **două** case | casă は女性。doi は男性専用 |
| doi scaune | **două** scaune | scaun は中性、複数は女性型の două |
| un carte | **o** carte | carte は -e 終わりの女性 |
| Ea este o student. | Ea este (o) student**ă**. | 女性には女性形 studentă。職業叙述なら無冠詞 studentă も可 |
| profesoră(-oară でなく -ă) | profeso**ară** | -or 終わりの女性形は -oară |
| 中性=無生物と決めつけ | テストで確定 | animal(中)も masă(女・無生物)もある。un/doi で確認 |
| bună + 男性名詞 | bun + 男性名詞 | 形容詞も性一致(→ [adjectives.md](./adjectives.md)) |

## 7. ネイティブらしい言い換え・注意点

- **「男の人/女の人」の丁寧形**: bărbat/femeie は中立、`domn/doamnă`(紳士/淑女)は丁寧。店や役所では `Un domn vă caută.`(男の方がお訪ねです)のように domn/doamnă を使うと丁寧(本編第3章)。
- **曜日・月の性**: 曜日名(luni, marți…)は女性名詞、月名(ianuarie…)は男性扱い。時刻の「1時・2時」が女性形 `ora una / ora două` になるのは ora(時)が女性だから(本編 A2 第1章)。
- **職業の女性形は揺れる**: 現代ルーマニア語では `medic`・`inginer`・`ministru` のように、女性でも男性形を使うことがある。会話では `doamna doctor`・`doamna inginer` のように doamna を前置して女性を示す言い方も一般的。
- **中性の指紋を口癖に**: 中性かどうか迷ったら「un ○○, două ○○」と唱える。un で始まり două で終われば中性、と体で覚える。

## 8. 練習問題

### 問題1:un/doi テストで性を判定せよ
例: pom → un pom, doi pomi → 男性
1. casă 2. hotel(ホテル)3. profesor(教師)4. cafea(コーヒー)5. muzeu(博物館)6. animal(動物)

### 問題2:語尾から性を推測し、例外に注意せよ
1. bucurie(喜び)2. tată(父)3. libertate(自由)4. munte(山)5. mâncare(食べ物)

### 問題3:女性形を作れ(職業名詞)
1. student 2. profesor 3. vânzător(店員)4. prieten(友人)5. român(ルーマニア人)

### 問題4:un / o / două / doi を入れよ
1. ___ scaun(椅子1つ・中)2. ___ scaune(椅子2つ)3. ___ fată(少女1人・女)4. ___ băieți(少年2人・男)5. ___ tată(父1人・男)

### 問題5:誤りを直せ
1. o tată 2. doi case 3. Ea este o student. 4. profesoră bună

## 9. 解答

### 問題1
1. o casă, două case → **女性** 2. un hotel, două hoteluri → **中性** 3. un profesor, doi profesori → **男性** 4. o cafea, două cafele → **女性** 5. un muzeu, două muzee → **中性** 6. un animal, două animale → **中性**(無生物でなくても中性でありうる)

### 問題2
1. bucurie → -ie=**女性** 2. tată → -ă だが**男性**(例外)3. libertate → -tate=**女性**(抽象名詞)4. munte → -e だが**男性**(frate/perete 類)5. mâncare → -re=**女性**(動名詞)

### 問題3
1. student → student**ă** 2. profesor → profeso**ară** 3. vânzător → vânză**toare** 4. prieten → prieten**ă** 5. român → româ**noaică**

### 問題4
1. **un** scaun(中性単数=男性型)2. **două** scaune(中性複数=女性型)3. **o** fată(女)4. **doi** băieți(男)5. **un** tată(男・例外)

### 問題5
1. **un** tată(tată は男性)2. **două** case(casă は女性)3. Ea este (o) student**ă**.(女性形)4. profeso**ară** bună(-oară)
