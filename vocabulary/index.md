# 語彙

このセクションは、ルーマニア語の語彙を「頻度別コアリスト」と「テーマ別リスト」の2軸で整理した参照集である。本編(`01-a1/` 〜 `06-c2/`)は各章のテーマに沿って必要最小限の語彙を導入していくが、語彙は文法と違って量そのものが運用能力に直結するため、本編の進度だけでは語彙量が不足しがちである。このセクションは、本編で扱いきれない語彙を体系的に補い、目標 CEFR レベルに必要な語彙量に到達するための学習リソースとして機能する。

頻度別コアリスト(`core-500.md` 〜 `core-5000.md`)は、レベルごとに到達すべき語彙量の目安と直結している。[cefr-map.md](../cefr-map.md) が定める「A1=基礎500 / A2=1000 / B1=3000 / B2=5000」という語彙量の目安に対応させ、各リストは前段階のリストを包含する累積的な構成にする(core-1000 は core-500 を含んだ上での1000語、という設計)。これにより、学習者は自分のレベルに応じたリスト1つを覚えればよく、複数のリストを重複して学習する無駄を避けられる。

テーマ別リスト(`daily-life.md` 〜 `slang-and-colloquial.md`)は、頻度だけでは拾いきれない実用語彙を場面ごとに補う。たとえば「家族」「食事」のようにA1段階から必要になるテーマもあれば、「法律・行政」「メディア・政治」のようにB2以降でないと文脈が理解できないテーマもある。したがって各テーマ別リストには対象 CEFR レベルを明示し、そのレベルに達する前に無理に暗記しないよう案内する。俗語・口語表現(`slang-and-colloquial.md`)は教科書的な語彙だけでは対応できないネイティブとの会話の壁を埋めるためのものであり、[idioms/](../idioms/index.md) の口語表現と連動する形で C1〜C2 で扱う。

## 学習の目安

- **A1 の学習者**: `core-500.md` を軸に、`daily-life.md`・`family.md`・`food.md` を並行して覚える。
- **A2 の学習者**: `core-1000.md` に進み、`travel.md`・`housing.md`・`shopping.md` で行動範囲を広げる語彙を補う。
- **B1 の学習者**: `core-3000.md` を目標に据え、`emotions.md`・`work.md` で意見表明や職場の話題に必要な語彙を固める。
- **B2 の学習者**: `core-5000.md` に到達しつつ、`technology.md`・`law-and-administration.md`・`media-and-politics.md` でニュースや専門的な話題への対応力をつける。
- **C1〜C2 の学習者**: `academic.md`・`slang-and-colloquial.md` を通じて、フォーマルからくだけたレジスターまでの両極を使い分けられるようにする。
