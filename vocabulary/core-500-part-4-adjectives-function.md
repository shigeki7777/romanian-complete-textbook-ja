# 基礎500語 コアボキャブラリー 第4分冊:形容詞・機能語【A1〜A2】

この分冊は core-500(基礎500語コアリスト)の**第4分冊であり、全4分冊の最終巻**である。第1〜3分冊で名詞と動詞という「文の中身」を固めたのに対し、本巻はそれらを結び付け、彩り、関係づける **形容詞** と **機能語**(代名詞・前置詞・接続詞・副詞・疑問詞・数詞)を扱う。形容詞は「大きい」「新しい」のように物や人の性質を述べ、機能語は「私は」「〜の中で」「そして」「なぜ」のように文の骨組みを作る。この二種がそろって、はじめて自分の言いたいことを組み立てられる。

ルーマニア語の形容詞は、修飾する名詞の**性(男性・女性)に一致して形が変わる**のが最大の特徴である。そのため本巻の形容詞は見出しを「**男性形 / 女性形**」の順で併記する(例: bun / bună)。男性形だけ覚えても女性名詞には使えないので、必ず両形をペアで覚えてほしい。mare(大きい)・verde(緑)・rece(冷たい)のように**男女同形で変化しない語**もあり、その旨を訳欄・例文で明示した。

**この分冊の使い方(縮約7列フォーマットの断り)**: 本書は基本、多列の詳細表で語を解説するが、コアリスト巻では語数を優先し、以下の**7列の縮約フォーマット**を用いる。各列は | ルーマニア語 | 品詞・性 | IPA | 日本語訳 | 例文 | 例文訳 | テーマ | である。形容詞の見出しは「男性形 / 女性形」、品詞欄は「形」。機能語は品詞欄に 副/前/接/代/数/疑 を明記する。IPA は形容詞では原則**男性形**を示す(女性形が大きく異なり確実な場合のみ併記)。例文はすべて A1〜A2 文法(a fi / a avea / 現在形 / 複合過去 am … -t まで)の 3〜7 語の短文で、収録語を必ず含む。形容詞の例文は、できるだけ性一致が目に見えるように作ってある。
