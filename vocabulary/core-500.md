# 基礎500語 コアボキャブラリー(core-500)【A1〜A2】

このファイルは、ルーマニア語の**頻度上位およそ500語**を集めた基礎語彙リストのハブである。本書の CEFR マップが定める「A1=基礎500語」という語彙量の目安に対応し、サバイバル会話・日常生活・自己表現に必要な最小限の中核語彙をここで固める。上位リスト([語彙索引](./index.md)の `core-1000` 以降)はすべてこの500語を包含する累積構成なので、まずはこの500語を確実に自分のものにすることが、その後のあらゆる学習の土台になる。

## 選定方針

収録語は「教科書のための人工的な単語集」ではなく、**実際の会話・生活で高頻度に出てくる日常語**を優先して選んだ。基準は次のとおりである。

- **場面カバレッジ**: 自己紹介・家族・体調・食事・買い物・移動・時間・感情・仕事・学校という、A1〜A2 の学習者が最初にぶつかる場面をひととおり言葉にできること。
- **本編との整合**: 本書 A1 本編(`01-a1/` の各章)にすでに登場した語をできるだけ含め、本編と語彙リストが互いを補強するようにした。
- **国際語の採用**: `taxi`・`hotel`・`telefon`・`metrou` のように、頻度が高く日本語話者にも馴染みのある国際語は積極的に採用した。
- **正直さ優先**: 複数形・活用形・IPA が確実でない語は**収録せず**、同程度に高頻度で確実な別語に差し替えた。不確かな情報を載せないことを、語数の見栄えより優先している。

## 使い方

### 1. SRS(間隔反復)で回す

500語は一度に覚えられる量ではない。**SRS(間隔反復システム / spaced repetition)**の使用を強く推奨する。Anki などのフラッシュカードアプリに、表(ルーマニア語→日本語)と裏(日本語→ルーマニア語)の両方向で入れ、毎日少しずつ回すのが最も効率的である。1日あたり新規10〜15語のペースなら、5〜6週間で一巡できる。

### 2. 「un/o + 単数、複数形」でセット暗記する(名詞)

ルーマニア語の名詞は**性**(男性・女性・中性)を持ち、性は不定冠詞 `un`(男・中)/ `o`(女)に現れる。名詞は裸で覚えず、**「不定冠詞 + 単数形、複数形」の3点セット**で覚えること。

- `o casă / case`(女性・家)、`un scaun / scaune`(中性・椅子)、`un frate / frați`(男性・兄弟)

こうしておくと、性・冠詞・複数形が一度に身につき、文を組み立てるときに迷わない。中性(neutru)は**単数で男性・複数で女性**として振る舞う点に常に注意する(`un telefon` だが `două telefoane`)。

### 3. 動詞は「3点セット」で覚える

動詞は本リストでは**「a 不定詞 (1人称単数現在, 過去分詞)」**の3点セットで示す。例: `a merge (merg, mers)`。1人称単数現在は「私は〜する」を、過去分詞は複合過去(`am mers` = 行った)を作るのに必須なので、この3つを一体で覚えると現在も過去もすぐ言えるようになる。

### 4. 形容詞は男女両形で覚える

形容詞は名詞の性・数に一致して形が変わる。本リストは**「男性形 / 女性形」**を併記する(例: `bun / bună`)。`mare`・`verde` のように男女同形の形容詞もあるので、その区別も一緒に覚える。

## 4分冊への案内

500語は品詞ごとに4つの分冊に分けてある。上から順に進めても、興味のあるテーマの分冊から始めてもよい。

| 分冊 | ファイル | 品詞 | テーマ | 収録語数 |
|---|---|---|---|---|
| 第1分冊 | [名詞(前半)](./core-500-part-1-nouns-a.md) | 名詞 | 人・家族・体・家・食 | 120語 |
| 第2分冊 | [名詞(後半)](./core-500-part-2-nouns-b.md) | 名詞 | 町・交通・時間・自然・仕事・学校・買い物・衣類 | 129語 |
| 第3分冊 | [動詞](./core-500-part-3-verbs.md) | 動詞 | 生活動詞中心(3点セット) | 123語 |
| 第4分冊 | [形容詞・機能語](./core-500-part-4-adjectives-function.md) | 形容詞・機能語 | 形容詞(男女形)+ 副詞・前置詞・接続詞・疑問詞・代名詞・数詞 | 124語 |

各分冊は、500語規模を扱いやすくするため**コンパクトな7列フォーマット**(`ルーマニア語 / 品詞・性 / IPA / 日本語訳 / 例文 / 例文訳 / テーマ`)を採用している。これは本書公認の縮約形式であり、スタイルガイド §4 の9列フル語彙表を、大量語彙のリスト用に簡約したものである。詳しい解説・直訳・レジスター注記が必要な語は、本編の該当章を参照してほしい。

> 各分冊の**正確な収録語数**は、それぞれの分冊末尾に明記してある。上表は目安であり、確実な語だけを収める方針の結果として品詞ごとに±数語の増減がある。

## 学習の順序(おすすめ)

1. まず[名詞(前半)](./core-500-part-1-nouns-a.md)の「人・家族」と[動詞](./core-500-part-3-verbs.md)の「基本・存在・移動」を並行して覚える。これだけで「私は〜だ / 〜を持っている / 〜へ行く」という骨格が言える。
2. 次に[名詞(前半)](./core-500-part-1-nouns-a.md)の「食・家」と[形容詞・機能語](./core-500-part-4-adjectives-function.md)の形容詞・前置詞を加え、身の回りを描写できるようにする。
3. [名詞(後半)](./core-500-part-2-nouns-b.md)の「町・交通・時間」で行動範囲を広げ、外出・約束の会話に備える。
4. 最後に残りの動詞・機能語・感情語彙で、意見や気持ちを短く言えるところまで持っていく。

本編と併走する場合は、A1 本編の「生活の基本語彙」の章(第7章)を終えたあたりからこのリストを回し始めると、本編で出た語がリストで再確認でき、定着が速い。
