# 本書の使い方

ようこそ。本書は、日本語を母語とする学習者が、ルーマニア語をまったくのゼロ(A1)からネイティブ相当(C2)まで、一本の学習導線でたどり着けるように設計された教科書である。単語帳でもフレーズ集でもなく、「上から順に読み進められる教科書」として書かれている。この最初の章では、本書がどのような部品でできていて、それをどう組み合わせて使えば最短で力がつくのかを説明する。

急いでいる人へ。最短の使い方は次の3行である。(1) この orientation の4章を読む。(2) [study-plan.md](../study-plan.md) で自分のペースのプランを選ぶ。(3) `01-a1/00-a1-overview.md` から番号順に読み始める。それ以外の細かい話は、必要になったときにこの章へ戻ってくればよい。

## 本書の全体構成

本書は大きく3つの層でできている。**レベル別本編**、**参照セクション**、**専門編**である。この3層は役割が違うので、使い方も違う。

**レベル別本編(`01-a1/` 〜 `06-c2/`)** が背骨である。ここは番号順に読む。A1(入門)から始まり、A2・B1・B2・C1・C2 と、CEFR(ヨーロッパ言語共通参照枠)のレベルを一段ずつ上がっていく。各レベルのフォルダの中でも、章にはすべて `00`, `01`, `02`… と番号が振ってあり、この順番自体が学習順序である。順番を飛ばすと、まだ習っていない文法や語彙が説明なしに出てきてつまずく設計になっている。だから本編は「番号順に、飛ばさずに」進めるのが原則である。全体像は [README.md](../README.md) の到達目標表、レベルごとの詳しい対応は [cefr-map.md](../cefr-map.md) を見るとよい。

**参照セクション(grammar・vocabulary・verb-conjugation など)** は、本編を読んでいる最中に疑問が出たときに「辞書のように引く」ための場所である。頭から順に読むものではない。たとえば本編で接続法が出てきて「他の使い方も一覧で確認したい」と思ったら [grammar/](../grammar/index.md) を引く。動詞の活用形を確認したければ [verb-conjugation/](../verb-conjugation/index.md) を、テーマ別の単語をまとめて仕入れたければ [vocabulary/](../vocabulary/index.md) を引く。参照セクションは「本編で出会った疑問に答える後方支援」であり、本編を進める代わりにはならない。

**専門編(medical・business など)** は、医療やビジネスといった特定の現場でルーマニア語を使う人のための応用パートである。これは B1 以降、本編と**並走**させて使う。A1・A2 の段階で専門編に手を出すのは早すぎる。まず本編で土台を作り、B1 でだいたいの日常運用ができるようになってから、自分の必要に応じて専門編を少しずつ足していく。医療・法律・行政の内容は語学教材であって専門的助言ではない点に注意してほしい(各章の冒頭に免責がある)。

## 各章の標準構成と、その使い方

本編の各章は、ほぼ共通のかたちを持っている。この「決まったかたち」を知っておくと、どの章でも迷わず学べる。標準的な流れは次のとおりである。

1. **この章の目標** — 章を終えたとき何が「できるようになる」かを、行動で示す。読み始める前にここに目を通し、「自分は何を身につけようとしているのか」を意識してから本文に入ろう。
2. **前提知識** — その章を読むのに必要な既習事項を示す。ここに挙がっている内容にあやふやなところがあれば、先に該当章へ戻るほうが結局は早い。
3. **本文解説** — その章の中心。日本語の解説文で、文法や語彙や場面を説明する。
4. **例文** — 解説した内容が実際の文としてどう現れるかを見る。例文の読み方は次の節で詳しく述べる。
5. **練習問題** — 学んだことを自分の手で使ってみる。
6. **解答** — 練習問題の答えと、「なぜそうなるか」の短い説明。
7. **復習チェックリスト** — その章で身につけるべき項目を箇条書きにしたもの。
8. **次章への橋渡し** — 次に何を学ぶか、そしてこの章とどうつながるかの案内。

ここで最も大事な使い方のルールを一つ強調しておく。**解答を見る前に、必ず自力で練習問題を解くこと。** 答えを先に見てしまうと、「分かったつもり」になるだけで力はつかない。間違えてよい。むしろ、自分がどこで間違えたかが分かることこそが学習の中身である。解いてから解答を読み、間違えた箇所は本文解説に戻って理由を確認する — この往復が、本書で最も効く学び方である。

章末の**復習チェックリスト**は、次の章へ進む前のゲートとして使う。チェックリストの各項目を見て、「これは自分の言葉で説明できる」「これは例文が作れる」と確信できるなら次へ進む。ひとつでも自信がなければ、その項目を扱った本文へ戻る。学習は積み上げなので、下の段が崩れていると上の段は必ず崩れる。

## 例文の読み方

本書の例文には、ただルーマニア語と訳が並んでいるだけではなく、いくつかの情報が付いている。それぞれに意味があるので、読み飛ばさずに活用してほしい。表形式の例文では、おおむね次の列が並ぶ。

| 項目 | 何のためにあるか |
|---|---|
| ルーマニア語原文 | 学ぶ対象そのもの。正書法(綴りの規則)まで含めて正確に写す |
| IPA/発音 | どう発音するかを国際音声記号で正確に示す。カナの近似ではぶれる音を、記号で確定させる |
| 自然な日本語訳 | 日本語として自然な訳。「意味」を掴むための訳 |
| 直訳 | 語順や構造がそのまま見える逐語的な訳。ルーマニア語が「どう組み立てられているか」を掴むための訳 |
| レジスター | その表現の丁寧度・くだけ具合。誰にいつ使ってよいかの目印 |

**自然な訳と直訳の両方を載せているのには理由がある。** 自然な訳だけを読むと意味は分かるが、ルーマニア語の構造が日本語の中に溶けて見えなくなる。逆に直訳だけだと、ぎこちなくて意味が取りにくい。両方を並べて見比べることで、「この意味を、ルーマニア語はこういう順番・こういう部品で組み立てるのか」という感覚が育つ。たとえば「私は日本人だ」にあたる文 **Sunt japonez.** /sunt ʒa.poˈnez/(直訳:「(私は)である 日本人」)を見れば、ルーマニア語では「私」にあたる代名詞をふつう省くこと、動詞が先に来ることが、直訳から読み取れる。

**レジスター表示** は、その表現が「丁寧」なのか「中立」なのか「くだけた」ものなのかを示す。同じ「こんにちは」でも、初対面の相手に使う形と、友人に使う形は違う。レジスターを無視して丸暗記すると、丁寧すぎて他人行儀になったり、逆にくだけすぎて失礼になったりする。例文を覚えるときは、必ず「これは誰に対して使う形か」をレジスター表示で確認する習慣をつけよう。

**IPA(国際音声記号)** の読み方そのものは、この orientation の [発音の全体像](./02-pronunciation-overview.md) で凡例を示す。最初はすべての記号を覚える必要はない。ルーマニア語に特有の数個(`ə`, `ɨ`, `ʃ`, `t͡s` など)を押さえれば、あとは読みながら慣れていけばよい。

## チェックポイントテストの使い方

各レベルの本編の終わりには、そのレベル全体の到達度を測る「チェックポイントテスト」が用意されている(たとえば A1 なら本編の最後の章として置かれる)。これは腕試しであると同時に、**次のレベルへ進んでよいかの判定** に使う。

使い方の目安はこうである。**得点が 80% 未満だったら、次のレベルには進まず、間違えた領域を扱った章へ戻る。** テストは「合格したかどうか」を競うものではなく、「土台に穴がないか」を点検するものである。80% という線は、次のレベルの内容を無理なく載せられる土台があるかどうかの、実用的な目安として設けている。低い得点で先へ進んでも、結局は上のレベルでつまずいて戻ることになる。急がば回れで、穴を埋めてから進むほうが速い。

## 学習記録の付け方の提案

独学は、進んでいる実感が持ちにくいと続かない。だから、簡単でよいので学習記録をつけることを勧める。凝った仕組みは要らない。ノートでもスプレッドシートでもよいので、次のようなことを残しておくと、後で自分を助けてくれる。

- **日付と、その日読んだ章・かけた時間。** 積み上がった行数を眺めるだけで、続ける動機になる。
- **間違えた練習問題と、その理由。** 同じ間違いを繰り返していないかが見える。人は同じところで何度もつまずくものなので、これを可視化すると弱点対策が具体的になる。
- **覚えきれない単語・表現。** これは間隔反復(SRS)ソフト(Anki など)に入れるのがよい。詳しい使い方は [学習戦略](./01-learning-strategy.md) で述べる。
- **チェックポイントテストの得点と日付。** レベルが上がった記録は、何よりのモチベーションになる。

記録は「完璧な記録」を目指すものではない。三日坊主でも、また書き始めればよい。大事なのは、自分の歩みが目に見える形で残っていることである。

## この章のまとめ

- 本編(`01-a1/` 〜 `06-c2/`)は番号順に読む背骨。参照セクションは辞書のように引く後方支援。専門編は B1 以降に本編と並走させる。
- 各章は「目標 → 前提 → 本文 → 例文 → 練習 → 解答 → 復習 → 橋渡し」の順。**練習問題は解答を見る前に自力で解く。**
- 例文は、自然な訳と直訳を見比べ、レジスター表示で「誰に使う形か」を確認する。IPA は特有の数音から慣れる。
- チェックポイントテストが 80% 未満なら、進まずに該当章へ戻る。
- 簡単でよいので学習記録をつけ、続ける動機と弱点を可視化する。

執筆の細かな約束事(表記・出典など)に関心があれば [STYLE_GUIDE.md](../STYLE_GUIDE.md) を読むとよいが、学習者として本書を使うだけなら必須ではない。
