# 発音の全体像

この章は、ルーマニア語の発音の「鳥瞰図(ちょうかんず)」である。細かい発音のドリルや、一音ずつの丁寧な練習は A1 本編の第1章 `01-a1/01-pronunciation-and-alphabet.md` に譲る。ここでねらうのは、地図を上から眺めるように、「ルーマニア語にはどんな音があり、どこが規則的で、日本語話者は特にどこでつまずくのか」を最初に俯瞰しておくことである。全体像を先に持っておくと、A1 で個々の音を練習するとき、それが地図のどこに位置する音なのかが分かり、迷わない。

良い知らせを先に言っておく。ルーマニア語は綴りと発音の対応が規則的な言語であり([学習戦略](./01-learning-strategy.md)参照)、少数の規則さえ押さえれば、書いてあるとおりに読めば正しく発音できる。覚えるべき「例外的な音」は多くない。この章で、その少数の要点を通しておこう。

## アルファベットと文字数

ルーマニア語のアルファベットは 31 文字である。内訳は、基本ラテン文字 26 文字に、ダイアクリティクス(発音を区別する記号)の付いた 5 文字 — **ă**, **â**, **î**, **ș**, **ț** — を加えたものである。基本 26 文字のうち **q, w, y** は、主に外来語・固有名詞でしか使われず、ルーマニア語本来の語ではほとんど現れない。

ここで表記上の注意を一つ。**ș** と **ț** は「コンマ付き」の文字であり、Unicode としては `ș`(U+0219)・`ț`(U+021B)を使う。見た目のよく似た「セディーユ付き」の `ş` `ţ` とは別物で、本書ではセディーユ付きは用いない。手元で入力・検索するときは、正しいほうのコンマ付き文字を使ってほしい。

## 母音 — 7 つの音

ルーマニア語の母音は 7 音である。IPA(国際音声記号)で示すと /a e i o u ə ɨ/ となる。このうち 5 つ /a e i o u/ は、日本語の「ア・エ・イ・オ・ウ」に十分近く、日本語話者にとって新しく覚える必要はほとんどない。問題は残りの 2 つ、日本語にない **ə** と **ɨ** である。

**ă = /ə/(あいまい母音)。** 文字 **ă**(a の上に小さな半円が付いた形)は、あいまい母音 /ə/ を表す。口をあまり開かず、力を抜いて「ア」と「ウ」の中間のような、ぼんやりした音を出す。英語の about の最初の母音に近い。日本語の「ア」よりも口の開きが小さく、こもった感じになるのがコツである。強く「アー」と言わないように注意する。例:**casă** /ˈka.sə/(「家」)。語末の -ă はこの音であり、はっきり「ア」と言ってしまうと不自然になる。

**â・î = /ɨ/(日本語にない音)。** 文字 **â** と **î** は、どちらも同じ音 /ɨ/ を表す。日本語にまったくない音で、これがルーマニア語の母音で最大の難所である。作り方の目安は、「イ」を言う口の形のまま、舌を後ろ(奥)へ引いて、「ウ」と「イ」の中間のような音を出す、というものである。喉の奥のほうで響く、こもった音になる。例:**român** /roˈmɨn/(「ルーマニア人(の)」)、**în** /ɨn/(「〜の中に」)。

同じ音 /ɨ/ に **â** と **î** という 2 つの文字があるのは、綴りの規則によって使い分けるためである。原則として、**語頭と語末は î**、**語中は â** を使う(例:**în**〔語頭〕, **a coborî**〔語末〕, **român**〔語中〕)。接頭辞が付いた複合的な語では、元の語頭の î を保つことがある(例:**neînceput**)。音は同じなので、発音のうえでは「â も î も /ɨ/」とだけ覚えておけばよく、使い分けは綴りの問題である(詳細は A1 本編と [STYLE_GUIDE.md](../STYLE_GUIDE.md) で扱う)。

## 二重母音 — 音がなめらかにつながる

ルーマニア語では、2 つの母音が 1 つの音節の中でなめらかにつながる「二重母音」が頻繁に現れる。日本語では母音を 1 つずつはっきり切って発音するが、ルーマニア語の二重母音は、2 つの母音を切らずに滑らせて一息で出す。代表的なものに次がある。

**ea, oa, ia, ie, iu, ai, au, ei, eu, oi, ou, ui** など。

このうち特に重要なのが **oa** と **ea** である。この 2 つはルーマニア語で非常によく現れ、語の意味を分ける働きをすることもあるため、最初からきちんと区別して発音したい。

- **oa** /o̯a/:例 **școală** /ˈʃko̯a.lə/(「学校」)。o から a へ、切らずに滑らせる。
- **ea** /e̯a/:例 **seară** /ˈse̯a.rə/(「夕方」)。e から a へ、なめらかにつなぐ。

日本語話者は、これらを「オ・ア」「エ・ア」と 2 拍に割って発音しがちだが、それでは不自然になる。1 つの音として、ひと息で滑らせる感覚をつかむのが大切である。細かい二重母音の一覧と練習は A1 本編で扱う。

## 子音 — 押さえるべき規則

子音の多くは、ローマ字読みの感覚でおおむね通じる。ただし、日本語にない音や、綴りの規則を知らないと読めない音がいくつかある。要点だけ挙げる。

**特殊な文字の子音。**

- **ș** = /ʃ/:日本語の「シュ」に近い音。例 **și** /ʃi/(「〜と、そして」)。
- **ț** = /t͡s/:「ツ」に近い音。例 **țară** /ˈt͡sa.rə/(「国」)。
- **j** = /ʒ/:日本語にない、有声の「ジュ」(フランス語の j の音)。例 **jos** /ʒos/(「下に」)。

**c と g の軟音化。** これは英語やイタリア語にも似た規則があり、ルーマニア語でも重要である。c と g は、後ろにくる母音によって音が変わる。

- **ce, ci** = /t͡ʃe/, /t͡ʃi/:「チェ」「チ」の音。例 **cine** /ˈt͡ʃi.ne/(「誰」)。
- **ge, gi** = /d͡ʒe/, /d͡ʒi/:「ジェ」「ジ」の音。例 **ger** /d͡ʒer/(「厳しい寒さ」)。
- **che, chi** = /ke/, /ki/:h を挟むと硬い「ケ」「キ」に戻る。例 **chibrit** /kiˈbrit/(「マッチ」)。
- **ghe, ghi** = /ɡe/, /ɡi/:同様に硬い「ゲ」「ギ」。例 **ghid** /ɡid/(「ガイド」)。

つまり、e や i の前で c・g をそのまま書くと軟らかい音(チ・ジ系)になり、硬い音(カ・ガ行)を出したいときは h を挟んで **ch-**, **gh-** と書く、という仕組みである。この規則を知っていれば、初見の語でも正しく読める。

## 語末の -i — 口蓋化する「隠れた i」

日本語話者が見落としやすい、しかしとても大切な規則がある。**語末の -i は、多くの場合、はっきりした「イ」ではなく、直前の子音を口蓋化(こうがいか)させる /ʲ/ になる。** 口蓋化とは、子音を出すときに舌の中ほどを上あご(硬口蓋)に近づけて、その子音に「イ」の色を軽く添えることである。この語末の /ʲ/ は、複数形などで非常によく現れる。

例:**pom** /pom/(「木」・単数)に対して、複数形 **pomi** /pomʲ/(「木々」)。この場合、末尾は「ポミ」ではなく、m を口蓋化させた「ポミ゚」のような、母音の「イ」をほとんど発音しない音になる。つまり、**pomi** は 2 音節(po-mi)ではなく、実質 1 音節に近い。

日本語話者は、綴りに i があるとつい「イ」とはっきり読んでしまうが、多くの語末 -i はそうではない。ここを「イ」と読むと、たちまち外国語なまりが強くなる。単数と複数の区別が末尾の口蓋化だけで示されることも多いので、この /ʲ/ を聞き分け・言い分けられることは、思いのほか実用的である。詳しい練習は A1 本編で扱うが、この段階では「語末の -i は油断できない」とだけ心に留めておこう。

## アクセント — 位置は語ごとに覚える

ルーマニア語のアクセント(強く読む音節)の位置は、**固定ではない**。フランス語のように「常に最後の音節」と決まっているわけではなく、語ごとに位置が異なる。傾向としては、最後から 2 番目の音節にアクセントが来ることが多いが、例外も多数あり、規則だけからは決められない。

したがって、アクセントは**単語を覚えるときにセットで覚える**しかない。本書では、例文・語彙の IPA 表記の中で、アクセントのある音節の直前に記号 `ˈ` を付けて示す(例:**limbă** /ˈlim.bə/ =「言語」。最初の音節にアクセント)。新しい単語を覚えるときは、意味・綴りだけでなく、この IPA が示すアクセント位置も一緒に音読して身につけてほしい。アクセントの位置を間違えると、意味は同じでも「なまって」聞こえるし、まれに別の語と紛らわしくなることもある。

## 日本語話者の頻出課題(要注意リスト)

最後に、日本語話者がルーマニア語の発音で特につまずきやすい 4 点を、注意リストとしてまとめておく。A1 本編で発音を練習するとき、この 4 点を意識すると効率がよい。

1. **l と r の区別。** 日本語はこの 2 つを区別しないが、ルーマニア語では別の音であり、区別しないと意味が通じにくくなる。l は舌先を上の歯ぐきにしっかり付ける音。r はふるえ音/はじき音で、舌先を歯ぐきの近くで短く弾く(またはふるわせる)。この r は日本語のラ行に近い瞬間もあるが、意識してはじく・ふるわせる練習が要る。
2. **子音連結に母音を入れない。** str-, spr- のような子音の連なりに、無意識に母音を差し込まないこと(例:**strada** /ˈstra.da/ を「ストラダ = su-to-ra-da」にしない)。子音を続けざまに出す練習が必要である。
3. **ə と ɨ の区別。** 新しく覚える 2 母音 /ə/(ă)と /ɨ/(â/î)を混同しない。/ə/ は口の力を抜いたあいまい母音、/ɨ/ は舌を奥に引いたこもった音、と作り方が違う。この 2 つを取り違えると別の語になってしまうことがある。
4. **語末の /ʲ/ を「イ」と読まない。** 前述のとおり、語末 -i の多くは口蓋化であって、はっきりした「イ」ではない。ここを「イ」と読むと強いなまりになる。

これらはいずれも、知識として知っておけば意識でき、意識できれば練習で直せるものである。逆に、知らないままだと自分では気づけない。だからこそ、発音のドリルに入る前にこの鳥瞰図で全体を押さえておく意味がある。

## IPA 表記の凡例(本書での約束)

本書では、発音を国際音声記号(IPA)で示す。すべての記号を最初から覚える必要はないが、ルーマニア語で頻出する主要な対応を、以下に凡例としてまとめておく。読みながら少しずつ慣れていけばよい。

| 綴り | IPA | 音の目安 | 例 |
|---|---|---|---|
| ă | /ə/ | あいまい母音(力を抜いたア〜ウの中間) | casă /ˈka.sə/ |
| â, î | /ɨ/ | 舌を奥に引いたこもった音(日本語になし) | român /roˈmɨn/ |
| ș | /ʃ/ | シュ | și /ʃi/ |
| ț | /t͡s/ | ツ | țară /ˈt͡sa.rə/ |
| j | /ʒ/ | 有声のジュ | jos /ʒos/ |
| ce, ci | /t͡ʃe/, /t͡ʃi/ | チェ、チ | cine /ˈt͡ʃi.ne/ |
| ge, gi | /d͡ʒe/, /d͡ʒi/ | ジェ、ジ | ger /d͡ʒer/ |
| che, chi | /ke/, /ki/ | 硬いケ、キ | chibrit /kiˈbrit/ |
| ghe, ghi | /ɡe/, /ɡi/ | 硬いゲ、ギ | ghid /ɡid/ |
| 語末 -i | /ʲ/ | 直前の子音の口蓋化(「イ」を強く出さない) | pomi /pomʲ/ |

- 発音は音素表記として `/ /` で囲んで示す。
- アクセントのある音節の直前に `ˈ` を置く(例:limbă /ˈlim.bə/)。
- 音節の切れ目は `.` で示すことがある。
- 二重母音は、なめらかに続けて出す(oa /o̯a/, ea /e̯a/ など)。
- 必要に応じて日本語のカナで近似を添えることがあるが、カナはあくまで近似であり、正確な音は IPA と解説で確認してほしい。

## この章のまとめ

- アルファベットは 31 文字(基本 26 + ă â î ș ț)。ș・ț はコンマ付きを使う。
- 母音は 7 音。新規は ă=/ə/ と â・î=/ɨ/ の 2 つだけ。â/î は同じ音で綴りが違うだけ。
- 二重母音、特に oa・ea は切らずに滑らせる。
- 子音は c/g の軟音化規則(ce ci / ge gi / che chi / ghe ghi)を押さえれば読める。
- 語末 -i の多くは口蓋化 /ʲ/。アクセントは語ごとに覚える。
- 頻出課題は l/r、子音連結、ə/ɨ、語末 /ʲ/ の 4 点。
