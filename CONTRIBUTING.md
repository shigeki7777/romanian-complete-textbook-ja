# 貢献ガイド

本教材への貢献を歓迎します。特に価値が高いのは次の 3 つです。

1. **ネイティブ話者によるルーマニア語の自然さレビュー**(最重要)
2. 日本語話者としての「ここが分かりにくい」というフィードバック
3. 事実情報(歴史・制度)の出典付き訂正

## 貢献の流れ

1. Issue を立てる(誤りの指摘は Issue だけでも十分に価値があります)
2. 修正 PR を送る場合は、対象ファイルと理由を明記する
3. ルーマニア語の変更には「なぜその形が自然か」の説明(可能なら出典)を添える

## 執筆ルール

- [STYLE_GUIDE.md](./STYLE_GUIDE.md) に従う(章テンプレート・例文フォーマット・正書法)
- [SOURCE_POLICY.md](./SOURCE_POLICY.md) に従う(事実情報の出典)
- [QUALITY_CHECKLIST.md](./QUALITY_CHECKLIST.md) の必須項目を満たす
- 正書法は DOOM3 準拠。`ș`/`ț` はコンマ付き(U+0219/U+021B)を使用
- リンクは相対パス。`python3 tools/check-links.py` でリンク切れがないことを確認

## してほしくないこと

- 出典のない事実情報の追加
- 機械翻訳のみで検証していないルーマニア語例文の追加
- 1 PR での無関係な大量変更
