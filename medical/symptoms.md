# 症状の表現【CEFR B1】

> **免責**: 本章は語学学習を目的とした教材であり、医学的・法的助言ではない。
> 実務で使用する場合は、必ず資格を持つ専門家およびネイティブ話者の確認を経ること。
> 症状の自己判断・自己治療の指針として本章を用いてはならない。体調不良や緊急時には、必ず医療機関・救急サービスに連絡すること。

## この章の目標

この章を終えると、次のことができるようになる。

- 痛みの種類(**ascuțită** 鋭い / **surdă** 鈍い / **pulsatilă** ズキズキ)を形容詞で言い分けられる
- 発熱・悪寒、消化器・呼吸器・皮膚・全身の主要な症状を、B1 の語彙で描写できる
- 症状の**程度**(ușor / moderat / sever)と**頻度**(din când în când / permanent / de două zile)を伝えられる
- 医師の定番質問(**Ce vă supără? / De când aveți simptomele? / Aveți alergii?**)を聞き取り、応答できる
- 問診で**複合過去と時間表現**(De ieri. / Acum trei zile a început.)を使って経過を言える
- 誤訳リスクの高い症状語(amețeală と slăbiciune、a vomita と a-i fi greață、răceală と gripă 等)を区別できる

## 前提知識

- [第7章:医療の基本場面(A2)](../02-a2/07-basic-medical-situations.md) — a durea 構文、Am febră / greață、Sunt răcit(ă)、Ce vă supără? は既習。本章はこれを B1 の描写力へ引き上げる
- [身体部位・臓器の語彙(anatomy)](./anatomy.md) — 「どこが」を言う部位語彙。本章は「どう・どのくらい」を重ねる
- [医療の全体像(overview)](./overview.md) — 受診の流れ。本章は consultație(診察)の中身を担う
- 複合過去(perfectul compus)・形容詞の性数一致(本編 A2〜B1)。症状の経過と、症状名に付く形容詞の一致に使う

A2 では症状を「ひとこと」で言えた(頭が痛い、熱がある)。B1 では、医師が診断に必要とする情報——**どんな痛みか・どのくらい強いか・いつからか・どのくらいの頻度か**——を、自分から**描写して**伝えられることを目指す。症状の申告は誤訳が実害に直結する高リスク領域であり、本章では ⚠ を多めに置く。

## 本文解説

### 痛みを描写する ― durere に形容詞を重ねる

A2 の **Mă doare X**(X が痛い)は「痛い/痛くない」の二択だった。B1 では、名詞 **durere**(痛み)に形容詞を付けて**痛みの質**を描写する。これは診断で決定的に重要な情報である。

**o durere + 形容詞(女性形)** の形が基本。durere は女性名詞なので、形容詞も女性形で一致させる。

- **o durere ascuțită** /o duˈre.re a.ku.ˈt͡si.tə/ ― 鋭い痛み(刺すような)
- **o durere surdă** /o duˈre.re ˈsur.də/ ― 鈍い痛み(重く続く)
- **o durere pulsatilă** /o duˈre.re pul.saˈti.lə/ ― ズキズキする痛み(脈打つような)
- **o durere ascuțită ca un cuțit** ― ナイフのような鋭い痛み(ca un cuțit = ナイフのような、比喩を添える)
- **o arsură** /o arˈsu.rə/ ― 灼熱感・ヒリヒリ(焼けるような痛み。別語)
- **o apăsare** /o a.pəˈsa.re/ ― 圧迫感(胸の締めつけなど)

痛みの位置は、部位語彙(anatomy)と組み合わせる:

- **Am o durere surdă în spate.** ― 背中に鈍い痛みがある。
- **Simt o durere ascuțită în piept când respir.** ― 息をすると胸に鋭い痛みを感じる。
- **Am o durere pulsatilă de cap.** ― ズキズキする頭痛がある。

動詞 **a simți**(感じる)を使う **Simt o durere...**(痛みを感じる)は、a durea 構文と並んで自然な言い方である。

### 発熱・悪寒(febră și frisoane)

- **febră** /ˈfe.brə/(名女)= 熱。**Am febră.**(熱がある)は A2 既習。程度を足す: **febră mare**(高熱)/ **febră ușoară**(微熱)/ **subfebrilitate**(微熱状態、やや医学的)。
- **frisoane** /friˈso̯a.ne/(名中複)= 悪寒。**Am frisoane.**(悪寒がする)。単数 frison より複数で使うのが普通。
- **transpirație** /trans.piˈra.t͡si.e/(名女)= 発汗。**Transpir noaptea.**(夜に汗をかく)。
- **a-i fi frig / cald** ― 寒気/ほてり。**Îmi este frig.**(寒気がする、直訳「私に寒い」)/ **Îmi este cald.**(ほてる)。この「与格 + a fi + 感覚」の型は体感の申告に頻出する。

### 消化器症状(simptome digestive)

消化器は語が多く、A2 の greață をここで広げる。⚠ の区別(後述)に注意。

- **greață** /ˈɡre̯a.t͡sə/(名女)= 吐き気。**Am greață.** / **Îmi este greață.**(吐き気がする)。
- **vărsături** /vərsəˈturʲ/(名女複)= 嘔吐(吐いたもの・嘔吐という事象)。動詞 **a vărsa** / **a vomita**(吐く)。**Am vărsat.**(吐いた)。
- **diaree** /di.aˈre.e/(名女)= 下痢。**Am diaree.**。
- **constipație** /kon.stiˈpa.t͡si.e/(名女)= 便秘。**Am constipație.**。
- **balonare** /ba.loˈna.re/(名女)= 腹部膨満・張り。**Sunt balonat(ă).**(お腹が張っている)。
- **arsuri la stomac** /arˈsurʲ la stoˈmak/ = 胸やけ(直訳「胃の灼熱感」)。
- **lipsă de poftă de mâncare** = 食欲不振(直訳「食欲の欠如」)。

### 呼吸器症状(simptome respiratorii)

- **tuse** /ˈtu.se/(名女)= 咳。A2 の動詞 a tuși(咳をする)に対し、名詞 tuse に形容詞を重ねて種類を言う:
  - **tuse seacă** /ˈtu.se ˈse̯a.kə/ = 乾いた咳(痰の出ない乾性咳)。seacă = 乾いた。
  - **tuse productivă** /ˈtu.se pro.duk.ˈti.və/ = 痰のからむ咳(湿性咳)。productivă = 痰を伴う。
- **nas înfundat** /nas ɨn.funˈdat/ = 鼻づまり(直訳「詰まった鼻」)。**Am nasul înfundat.**。
- **secreții nazale / muci** = 鼻水(前者は中立〜医学的、muci は口語)。**Îmi curge nasul.**(鼻水が出る、直訳「私の鼻が流れる」)。
- **dificultăți de respirație** / **lipsă de aer** = 呼吸困難・息切れ(直訳「呼吸の困難」「空気の欠如」)。**Am dificultăți de respirație.**。
- **durere în gât** = のどの痛み(A2 既習)。**a răgușit** = 声がかれる(**Am răgușit.** 声がかれた)。

### 皮膚症状(simptome cutanate)

- **mâncărime** /mɨn.kəˈri.me/(名女)= かゆみ。**Am mâncărime.** / **Mă mănâncă.**(かゆい、直訳「私を食べる」— a mânca が「かゆい」の口語)。
- **erupție** /eˈrup.t͡si.e/(名女)= 発疹。**Am o erupție pe piele.**(肌に発疹がある)。
- **roșeață** /roˈʃe̯a.t͡sə/(名女)= 赤み。**pete** /ˈpe.te/(名女複)= 斑点・しみ。
- **umflătură** /um.fləˈtu.rə/(名女)= 腫れ・こぶ。**S-a umflat.**(腫れた)。
- **vânătaie** /vɨ.nəˈta.je/(名女)= あざ(青あざ)。

### 全身症状(simptome generale)

- **oboseală** /o.boˈse̯a.lə/(名女)= 疲労・だるさ。**Sunt obosit(ă).**(疲れている)/ **Am o stare de oboseală.**。
- **amețeală** /a.meˈt͡se̯a.lə/(名女)= めまい。**Am amețeli.** / **Amețesc.**(めまいがする)。⚠ 後述。
- **slăbiciune** /slə.biˈt͡ʃu.ne/(名女)= 脱力・力が入らない感じ。**Simt o slăbiciune.**。⚠ 後述。
- **insomnie** /in.somˈni.e/(名女)= 不眠。**Am insomnii.**(眠れない)。
- **frisoane / transpirații** = 悪寒/寝汗(発熱の項と重なる)。
- **stare generală proastă** = 全身の不調(直訳「悪い全身状態」)。A2 の **Nu mă simt bine.** の B1 版。

### 程度を伝える(intensitatea)

症状の強さは、形容詞・副詞で3段階を基本に伝える。

- **ușor / ușoară** /uˈʃor/ = 軽い(o durere ușoară = 軽い痛み)
- **moderat / moderată** = 中程度(o durere moderată)
- **sever / severă** /seˈver/ = 重い・強い(o durere severă)。**puternic(ă)**(強い)も同義でよく使う。
- 数値で言う言い方も広まっている: **De la 1 la 10, durerea e cam 7.**(1から10で、痛みは7くらい)。診察で聞かれることがある。
- 副詞で動詞を修飾: **Mă doare tare.**(すごく痛い、tare = 強く)/ **Mă doare puțin.**(少し痛い)。

### 頻度・経過を伝える(frecvența și evoluția)

「いつから・どのくらいの頻度か」は診断に不可欠で、B1 の最重要ポイントである。

- **din când în când** /din kɨnd ɨn kɨnd/ = ときどき
- **permanent** /per.maˈnent/ = ずっと・常に(**Mă doare permanent.** = ずっと痛い)
- **des** /des/ = 頻繁に / **rar** /rar/ = まれに
- **de două zile** = 2日前から(de + 期間 = 「〜前から続いている」)
- **de azi-dimineață** = 今朝から
- **când și când** = ときたま(din când în când のくだけた同義)
- 発作的か持続的か: **vine și pleacă**(来ては去る=断続的)/ **e continuu**(持続的)

「de + 期間/時点」で「〜から今も続く」を表すのは A2 の De ieri. の延長である。ここに複合過去を組み合わせると経過が言える(次項)。

### 医師の質問を聞き取り、答える(問診)

診察の核は医師の質問への応答である。定番質問を**聞き取れる**うえで、**応答例とセット**で持っておく。

- **Ce vă supără?** /t͡ʃe və ˈsu.pə.rə/ ― どうされましたか?(A2 既習)
  - 応答: **Am o durere surdă în spate de trei zile.**(3日前から背中に鈍い痛みがあります)
- **De când aveți simptomele?** /de kɨnd aˈvet͡sʲ simpˈto.me.le/ ― いつから症状がありますか?
  - 応答: **De ieri.** / **De acum trei zile.** / **Acum trei zile a început.**(3日前に始まりました)
- **Cât de des vă doare?** ― どのくらいの頻度で痛みますか?
  - 応答: **Din când în când.** / **Aproape permanent.**
- **Cum e durerea? Ascuțită sau surdă?** ― 痛みはどんな感じですか、鋭いですか鈍いですか?
  - 応答: **Mai degrabă surdă.**(どちらかというと鈍い)
- **Aveți alergii?** /aˈvet͡sʲ a.ˈler.d͡ʒij/ ― アレルギーはありますか?
  - 応答: **Da, sunt alergic(ă) la penicilină.**(はい、ペニシリンにアレルギーがあります)/ **Nu, nu am alergii.**(いいえ、ありません)
- **Luați vreun medicament?** ― 何か薬を飲んでいますか?
  - 応答: **Da, iau...** / **Nu iau nimic.**

⚠ **alergic は性で変わる**: 男性 **alergic** /aˈler.d͡ʒik/、女性 **alergică** /aˈler.d͡ʒi.kə/。「〜にアレルギー」は **alergic la + 物**。

### 問診で使う複合過去と時間表現

「いつ始まったか」を言うには複合過去(perfectul compus)が要る。**Acum + 期間 + 複合過去** で「〜前に(始まった)」を表す。

- **Acum trei zile a început.** /aˈkum trej ˈzi.le a ɨn.t͡ʃeˈput/ ― 3日前に始まりました(a început = a începe の複合過去三単)。
- **Am răcit acum o săptămână.** ― 1週間前に風邪をひきました。
- **Mi s-a făcut rău brusc.** ― 急に気分が悪くなりました(a se face rău = 気分が悪くなる)。
- **De ieri am febră.** ― 昨日から熱があります(de + 時点 = 継続)。

「acum trei zile(3日前=起点)」と「de trei zile(3日前から=継続)」の使い分けに注意。前者は**始まった一点**、後者は**続いている期間**を指す。医師には両方を組み合わせて言うと明快である: **Acum trei zile a început și de atunci mă doare permanent.**(3日前に始まって、それからずっと痛い)。

## 重要ポイント

- 痛みは **o durere + 形容詞(女性形)**: ascuțită(鋭い)/ surdă(鈍い)/ pulsatilă(ズキズキ)。arsură(灼熱)/ apăsare(圧迫)も
- 消化器: greață(吐き気)/ vărsături(嘔吐)/ diaree(下痢)/ constipație(便秘)⚠ 吐き気と嘔吐は別語
- 呼吸器: tuse seacă(乾いた咳)/ tuse productivă(痰の咳)/ nas înfundat(鼻づまり)
- 全身: oboseală(疲労)/ amețeală(めまい)/ slăbiciune(脱力)/ insomnie(不眠)⚠ めまいと脱力は別語
- 程度: ușor / moderat / sever(+ puternic, tare)。頻度: din când în când / permanent / de + 期間
- 医師の質問 **De când aveți simptomele? / Aveți alergii?** を聞き取り、応答例とセットで持つ
- 経過は **Acum + 期間 + 複合過去**(Acum trei zile a început)。acum(起点)と de(継続期間)を区別

## よくある誤用

| ❌ 誤り | ✅ 正しく | 説明 |
|---|---|---|
| o durere surd(男性形) | **o durere surdă** | durere は女性名詞。形容詞は女性形 surdă で一致 |
| Am greață.(嘔吐したと言いたい) | **Am vărsat.** / **Am vomitat.** | greață は「吐き気(まだ吐いていない)」。実際に吐いたなら a vărsa/a vomita ⚠ |

## ⚠ 誤訳・混同に注意する語

症状語は取り違えると別の状態として伝わる。B1 で特に注意すべき点を挙げる。

- **⚠ amețeală(めまい)と slăbiciune(脱力)** ― **amețeală** は「くらくらする・目が回る」めまい、**slăbiciune** は「力が入らない・体に力がない」脱力・衰弱である。日本語では両方「ふらつく」と言えてしまうが、医学的には別の所見を示唆する。「めまいがする」を slăbiciune と言うと、回転性の感覚が伝わらず、逆に「脱力」を amețeală と言うと目まいと誤解される。**Am amețeli.**(めまい)と **Simt o slăbiciune.**(脱力)を分けて使う。
- **⚠ a vomita / a vărsa(吐く)と a-i fi greață / a avea greață(吐き気がする)** ― 前者は「実際に吐く(嘔吐)」行為、後者は「吐き気を感じる(まだ吐いていない)」状態である。**Am greață.**(吐き気がする)と **Am vomitat.** / **Am vărsat.**(吐いた)は別の事実を伝える。吐いていないのに「vomit」と言うと嘔吐したことになり、逆に吐いたのに「greață」だけだと吐き気止まりに聞こえる。診断上、実際に嘔吐があったかは重要なので区別する。
- **⚠ răceală(風邪)と gripă(インフルエンザ)〔A2 再掲〕** ― **răceală** /rəˈt͡ʃe̯a.lə/(名詞「風邪」)・形容詞 **răcit(ă)**(風邪をひいた)は軽い上気道感染、**gripă** /ˈɡri.pə/ は「インフルエンザ」という別の病気である。**Sunt răcit(ă).**(風邪だ)と **Am gripă.**(インフルエンザだ)は重症度も対応も違う。安易に gripă と訳すと実際より重い病気を申告することになる(A2 第7章の ⚠ を継続)。
- **⚠ surd(鈍い痛み)と surd(耳が聞こえない)** ― 形容詞 **surd/surdă** は、痛みに付けば「鈍い(o durere surdă)」だが、人に付けば「耳が聞こえない(聴覚障害)」を意味する。**Sunt surd.** は「私は耳が聞こえない」で、痛みの描写ではない。痛みを言うときは必ず durere と結び付ける(o durere surdă)。単独で人に使わないよう注意する。
- **⚠ tuse seacă(乾いた咳)と tuse productivă(痰のからむ咳)** ― seacă は「乾いた(痰なし)」、productivă は「痰を伴う」。この区別は診断で重視される。「咳が出る」とだけ言うより、seacă / productivă を添えると情報量が大きく変わる。日本語の「乾いた咳/湿った咳」に対応するが、湿った咳を「umedă(湿った)」と直訳せず、医療では **productivă** を使うのが標準である。
- **⚠ a răci(風邪をひく・冷える)と a răcori(涼む・冷ます)** ― 綴りが近いが別語。**a răci** は「風邪をひく/冷える」(Am răcit = 風邪をひいた)、**a răcori** は「涼をとる・冷ます」で症状とは無関係。取り違えると意味が通じない。

## ネイティブらしい言い換え

| 教科書的 | ネイティブが実際によく言う形 | 注 |
|---|---|---|
| Am o durere de cap puternică. | **Mă doare capul rău / îngrozitor.** | rău(ひどく)・îngrozitor(すごく)で強調するのが口語的 |
| Am o senzație de greață. | **Îmi este greață.** / **Mi-e greață.** | mi-e greață(与格の縮約)が最も自然で頻出 |
| Simt o stare de oboseală. | **Sunt frânt(ă).** / **Sunt terminat(ă).** | frânt/terminat(へとへと)は口語で強い疲労を表す |
| Nu pot să dorm noaptea. | **Nu închid ochii toată noaptea.** | 「一晩中目を閉じられない」=不眠の生き生きした言い方 |

## フォーマル/カジュアルの違い

診察室では **丁寧(dumneavoastră 系)**、家族・友人には tu 系。

- **医師に(丁寧)**: Am o durere surdă în spate de trei zile și amețeli din când în când. Sunt alergică la penicilină.
- **友人に(くだけた)**: Mă simt varză azi — mă doare capul și mi-e greață.(今日ボロボロ、頭痛いし吐き気する。varză=直訳「キャベツ」、口語で「ひどい状態」)
- 医師の質問は丁寧形: **De când aveți simptomele?**(vă/aveți)。友人になら **De când te simți așa?**(いつからそんな感じ?)。

迷ったら丁寧形。ただし症状の**中身**(surdă / greață / de trei zile 等)はレジスターにかかわらず正確に言うことが最優先である。丁寧さより**正確さ**が命の場面だと心得る。

## 練習問題

### 問題1: 痛みの形容詞を女性形で入れよ

1. 鋭い痛み:o durere ___ 2. 鈍い痛み:o durere ___ 3. ズキズキする痛み:o durere ___ 4. 軽い痛み:o durere ___ 5. 強い痛み:o durere ___

### 問題2: ⚠ 区別 ― 正しい語を選べ

1. 吐き気がする(まだ吐いていない):(Am greață / Am vomitat)。
2. めまいがする(目が回る):(Am amețeli / Simt o slăbiciune)。
3. 乾いた咳:tuse(seacă / productivă)。
4. インフルエンザにかかった:(Sunt răcit / Am gripă)。

### 問題3: 経過を言う ― acum か de か

1. 3日前に始まった:___ trei zile a început.
2. 3日前からずっと痛い:Mă doare ___ trei zile.
3. 昨日から熱がある:Am febră ___ ieri.

### 問題4: 医師の質問に答えよ(応答を書く)

1. Ce vă supără?(「2日前から鈍い頭痛があります」)
2. De când aveți simptomele?(「今朝からです」)
3. Aveți alergii?(女性が「はい、ペニシリンにアレルギーがあります」)
4. Cum e durerea, ascuțită sau surdă?(「どちらかというと鈍いです」)

### 問題5: 症状を一文で描写せよ(部位+痛みの質+経過)

例にならって作れ。例: Am o durere surdă în spate de trei zile.
1. 「昨日から胸に鋭い痛みがある」 2. 「3日前からときどきめまいがする」

## 解答

### 問題1
1. **ascuțită** 2. **surdă** 3. **pulsatilă** 4. **ușoară** 5. **severă**(または puternică)。
― durere は女性名詞なので形容詞はすべて女性形(-ă)。男性形で書くのが典型的な誤り。

### 問題2
1. **Am greață**(吐き気=まだ吐いていない) 2. **Am amețeli**(めまい=目が回る。slăbiciune は脱力) 3. **seacă**(乾いた咳) 4. **Am gripă**(インフルエンザ。räcit は軽い風邪)。
― すべて ⚠ 混同注意そのもの。greață/vomita、amețeală/slăbiciune、seacă/productivă、răceală/gripă を分ける。

### 問題3
1. **Acum** trei zile a început.(起点=acum+期間+複合過去) 2. **de** trei zile(継続=de+期間) 3. **de** ieri(継続=de+時点)。
― acum=「〜前に(始まった一点)」、de=「〜から(続いている)」。組み合わせると Acum trei zile a început și de atunci mă doare.

### 問題4
1. **Am o durere surdă de cap de două zile.**(2日前から鈍い頭痛) 2. **De azi-dimineață.**(今朝から) 3. **Da, sunt alergică la penicilină.**(女性形 alergică) 4. **Mai degrabă surdă.**(どちらかというと鈍い)。
― 医師の質問を聞き取り、症状の質・経過・アレルギーを応答例の型で返せることが B1 の目標。

### 問題5
1. **Am o durere ascuțită în piept de ieri.**(胸に鋭い痛み+de ieri) 2. **Am amețeli din când în când de trei zile.**(ときどきめまい+de trei zile)。
― 部位(anatomy)+痛みの質(形容詞女性形)+経過(de+期間/頻度)を1文に束ねる。これが問診応答の基本骨格。

- [ ] 消化器(greață/vărsături/diaree/constipație)・呼吸器(tuse seacă/productivă/nas înfundat)・皮膚・全身症状を描写できる
