# 身体部位・臓器の語彙【CEFR A2〜B1】

> **免責**: 本章は語学学習を目的とした教材であり、医学的・法的助言ではない。
> 実務で使用する場合は、必ず資格を持つ専門家およびネイティブ話者の確認を経ること。
> 症状の自己判断・自己治療の指針として本章を用いてはならない。体調不良や緊急時には、必ず医療機関・救急サービスに連絡すること。

## この章の目標

この章を終えると、次のことができるようになる。

- 頭頸部・体幹・四肢・内臓・感覚器の身体部位語彙(約60語)を、性と複数形とともに使える
- ルーマニア語に多い**不規則複数**(ochi/ochi, ureche/urechi, mână/mâini, picior/picioare など)を意識して使える
- 日本語と**体の区切り方が違う語**(mână は「手+腕」/ picior は「足+脚」/ deget は手足の指両方/ gât は「首+のど」)を、誤解を避けて使い分けられる
- A2 で学んだ **Mă doare X**(X が痛い)を、より細かい部位へ広げて症状を言える
- 体の部位を含む慣用表現をいくつか知り、直訳では意味が取れないことに気づく

## 前提知識

- [第7章:医療の基本場面(A2)](../02-a2/07-basic-medical-situations.md) — **a durea** 構文(Mă doare capul.)と基本部位12語は既習。本章はそこから部位を約60語へ広げる
- [医療の全体像(overview)](./overview.md) — 受診の流れと制度。本章はその「診察でどこが痛いかを言う」語彙面を担う
- 名詞の性・定形・複数形(本編 A1〜A2)。ルーマニア語の中性(neutru)は**単数で男性・複数で女性**として振る舞う点を、本章の名詞表で繰り返し思い出してほしい

## 本文解説

### 部位語彙は「性・複数形・区切り」の3点で押さえる

体の部位を覚えるとき、単語だけを丸暗記するのは危険である。ルーマニア語の名詞には性(男・女・中)があり、複数形が不規則なものが多く、さらに日本語と体の**区切り方**そのものが違う語がある。本章ではこの3点を常にセットで扱う。とくに **a durea 構文では複数形と定形が要になる**(Mă dor picioarele = 両脚が痛い)ため、複数形は必ず一緒に覚えてほしい。

### Mă doare X の復習 ― 部位語彙を症状文へ

A2 の中核構文 [Mă doare +部位(定形)](../02-a2/07-basic-medical-situations.md) を、本章の広い部位語彙で使ってみる。復習ポイントは3つ。

1. **主語は「痛む部位」**、私は目的語 mă。語順は **Mă doare/dor + 部位(定形)**。
2. **部位が単数なら doare、複数なら dor**。
3. **部位は定形(冠詞付き)**で言う。

新しい部位を入れた例:

- **Mă doare umărul.** /mə ˈdo̯a.re ˈu.mə.rul/ ― 肩が痛い(umăr → 定形 umărul、単数)。
- **Mă dor genunchii.** /mə dor d͡ʒeˈnun.kij/ ― 膝が(両方)痛い(genunchi 複数定形 genunchii → dor)。
- **Mă doare gâtul când înghit.** /mə ˈdo̯a.re ˈɡɨ.tul kɨnd ɨnˈɡit/ ― 飲み込むとのどが痛い(gât → gâtul)。
- **Mă doare pieptul când respir.** /mə ˈdo̯a.re ˈpjep.tul kɨnd resˈpir/ ― 息をすると胸が痛い。

「〜のあたりが痛い」とぼかしたいときは **la + 部位** も使える: **Mă doare la rinichi.**(腎臓のあたりが痛い)。位置を断定できないときに便利で、内臓のように正確な位置が言いにくい場合に自然である。

## 重要ポイント

- 部位語彙は**性・複数形・区切り**の3点セットで覚える。とくに複数形は a durea の数一致に必須
- 不規則複数: **ochi/ochi、ureche/urechi、mână/mâini、picior/picioare、os/oase**、単複同形 **genunchi・rinichi・mușchi**
- ⚠ 日本語と区切りが違う: **mână**(手+腕)/ **picior**(足+脚)/ **deget**(手足の指両方)/ **gât**(首+のど)
- **stomac**(解剖学的な胃)と **burtă**(口語の腹全般)は A2 既習の区別を継続 ⚠
- Mă doare + 部位(定形)で症状文を作り、単数 doare / 複数 dor、位置がぼやけるなら la + 部位
- 内臓・感覚器は B1 寄りだが、症状申告と診断理解で必要になる

## ⚠ 日本語との区切り違い(誤解例つき)

日本語とルーマニア語では、体の**区切り方そのもの**が違う語がある。直訳すると別の部位を指してしまうため、⚠ を付けて誤解例を明記する。

- **⚠ mână ―「手」だけでなく「腕」まで指しうる** ― 日本語の「手」は手首から先(掌と指)だが、ルーマニア語の **mână** は文脈により手首から先も、肩から先の「腕」全体も指しうる。前腕・上腕を明確に言いたいときは **braț**(腕)を使う。
  - 誤解例: 「腕が痛い(上腕)」を伝えたいのに **Mă doare mâna.** と言うと、相手は「掌・手首あたり」を想像しうる。腕全体なら **Mă doare brațul.**、手首なら **încheietura mâinii** と絞る。
- **⚠ picior ―「足」だけでなく「脚」全体を指す** ― 日本語では「足」(足首から先)と「脚」(股下全体)を区別するが、**picior** は股関節から下の**脚全体**を含みうる。足首から先の「足(あし)」を特定したいなら **talpă**(足の裏)や **laba piciorului**(足の甲・足先)といった言い方で絞る。
  - 誤解例: 「足の裏が痛い」を **Mă doare piciorul.** と言うと、脚のどこが痛いのか特定されない。足の裏なら **Mă doare talpa.**、太ももなら **coapsă** と具体化する。
- **⚠ deget ―「手の指」も「足の指」も同じ語** ― **deget** は手の指も足の指も指す。足の指を特定したいときは **deget de la picior**(直訳「脚の指」)と言う。
  - 誤解例: 足の指をぶつけたのに **Mă doare un deget.** とだけ言うと、手の指と受け取られうる。足なら **Mă doare un deget de la picior.** と明示する。
- **⚠ gât ―「首」も「のど」も同じ語** ― **gât** は外側の「首」も、内側の「のど(咽喉)」も指す。A2 で見たとおり、のどの痛みは **Mă doare în gât.**(în を入れて内側=のどを示唆)、首の外側なら **Mă doare gâtul.** や **la ceafă**(うなじ)で区別できる。
  - 誤解例: 飲み込むと痛い「のど」を **Mă doare gâtul.** とだけ言うと、寝違えのような「首」の痛みと取られうる。のどなら **Mă doare când înghit**(飲み込むと痛い)と補うと誤解が減る。

| Mă doare ochii.(複数なのに doare) | **Mă dor ochii.** | ochi は単複同形。両目=複数なので動詞は **dor** に一致 |
| Mă dor genunchiul.(単数なのに dor) | **Mă doare genunchiul.** | 片膝=単数なら **doare**。両膝なら Mă dor genunchii |
| Mă doare mâna.(上腕を伝えたい) | **Mă doare brațul.** | mână は手〜腕まで曖昧。上腕・前腕は braț で明確化 ⚠ |
| Mă doare un deget.(足の指のつもり) | **Mă doare un deget de la picior.** | deget は手足共通。足なら de la picior で特定 ⚠ |
| Mă doare oasul.(os の複数・定形の誤り) | 単数 **osul** / 複数 **oasele** | os の複数は不規則 oase(定形 oasele)。oasul は誤形 |

## ネイティブらしい言い換え

| 教科書的 | ネイティブが実際によく言う形 | 注 |
|---|---|---|
| Am o durere în piept. | **Mă doare în piept.** / **Simt o apăsare în piept.** | a durea 構文や「圧迫感(apăsare)」で言うほうが自然なことが多い |
| Mă doare stomacul. | **Mă doare la stomac.** / **Mă doare burta.** | la + 部位でぼかす、口語では burta も多用(⚠ 位置は曖昧化する) |
| Am o durere la genunchi. | **Mă doare genunchiul.** | 名詞 durere より a durea 構文のほうが会話で優勢 |
| Am pielea uscată. | **Am pielea uscată / iritată.** | iritată(荒れた)など具体形容詞を足すと伝わりやすい |

## フォーマル/カジュアルの違い

診察室では基本 **丁寧(dumneavoastră 系)**。医師の指示は敬称命令で来る。

- **医師の指示(丁寧・命令形)**: Deschideți gura!(口を開けて) / Ridicați brațul!(腕を上げて) / Respirați adânc!(深呼吸して)。
- **家族・友人に部位を言う(くだけた)**: Mă doare spatele rău.(背中がすごく痛い) / Ai pățit ceva la mână?(手、どうかした?)。
- 医師の質問は丁寧形: **Unde vă doare?**(どこが痛みますか、vă = あなたを)。友人になら **Unde te doare?**(どこ痛いの?)。

迷ったら丁寧形。ただし、部位を指し示すジェスチャーを添えると、区切り違い(⚠)による誤解を大きく減らせる。**言葉で曖昧なら指差しで補う**のが、医療現場では最も安全である。

## 先取りコラム ― 体の部位を使った慣用表現

体の部位は慣用表現に多用され、直訳では意味が取れないものがある。ここでは深入りせず **2〜3個だけ**先取りとして紹介する(慣用表現全般は本編の idioms セクションで扱う)。

- **a da o mână de ajutor** /a da o ˈmɨ.nə de a.ʒuˈtor/ ― 手を貸す・手伝う(直訳「一つの手の助けを与える」)。日本語の「手を貸す」とほぼ同発想で、覚えやすい。
- **a fi cu ochii pe cineva** /a fi ku ˈo.kij pe ˈt͡ʃi.ne.va/ ― 誰かを見張る・注意して見る(直訳「誰かの上に目でいる」)。
- **a-i sări inima din piept** /aj sə.ˈri ˈi.ni.ma din pjept/ ― 非常に驚く(直訳「心臓が胸から飛び出す」)。日本語の「心臓が飛び出しそう」に近い。

これらは**医療の文脈ではなく日常会話**の表現である。診察室で「a da o mână」を字義通りに使うことはない点に注意。慣用句は部位語彙を覚える楽しい入口だが、医療の症状申告とは切り離して考えること。

## 練習問題

### 問題1: 不規則複数を書け(定形なしの複数形)

1. mână → ___ 2. picior → ___ 3. ochi → ___ 4. ureche → ___ 5. os → ___

### 問題2: doare か dor か(a durea の一致)

1. Mă ___ umărul.(肩が痛い) 2. Mă ___ ochii.(両目が痛い) 3. Mă ___ genunchii.(両膝が痛い) 4. Mă ___ dintele.(歯が痛い)

### 問題3: ⚠ 区切り違い ― より正確な語を選べ

1. 上腕が痛い:Mă doare(mâna / brațul)。
2. 足の裏が痛い:Mă doare(piciorul / talpa)。
3. 足の指をぶつけた:un deget(de la picior / de la mână)。
4. 飲み込むとのどが痛い:Mă doare(în gât / la ceafă)când înghit.

### 問題4: 系統を答えよ(頭頸部/体幹/四肢/内臓/感覚器)

1. plămân 2. genunchi 3. sprânceană 4. frunte 5. coastă

### 問題5: 部位を使って症状文を作れ(Mă doare/dor を使う)

1. 「肩が痛い」 2. 「両手(腕)が痛い」 3. 「胃が痛い」 4. 「両耳が痛い」

## 解答

### 問題1
1. **mâini** 2. **picioare** 3. **ochi**(単複同形) 4. **urechi** 5. **oase**。
― いずれも不規則。とくに mâini / picioare / oase は綴りが変わるので要暗記。ochi は形が変わらず、定形 ochii で複数と分かる。

### 問題2
1. **doare**(umăr は単数) 2. **dor**(ochi 両目=複数) 3. **dor**(genunchi 両膝=複数) 4. **doare**(dinte 1本=単数)。
― 主語は「痛む部位」。部位が複数なら動詞も複数 dor に一致する、が全問共通の理由。

### 問題3
1. **brațul**(mână は手〜腕まで曖昧。上腕は braț で明確化) 2. **talpa**(picior は脚全体。足の裏は talpă) 3. **de la picior**(足の指) 4. **în gât**(în でのど内側を示唆。la ceafă はうなじ)。
― すべて ⚠ 区切り違いそのもの。日本語の1語に対しルーマニア語は部位を絞る語を持つ。

### 問題4
1. **内臓**(肺) 2. **四肢**(膝) 3. **感覚器**(眉) 4. **頭頸部**(額) 5. **体幹**(肋骨)。
― 系統で覚えると、診察で「どのあたりか」を整理して言いやすくなる。

### 問題5
1. **Mă doare umărul.** 2. **Mă dor mâinile.**(両手=複数 → dor + 定形複数 mâinile) 3. **Mă doare stomacul.** 4. **Mă dor urechile.**(両耳=複数 → dor + urechile)。
― 単数は doare、複数は dor。部位は必ず定形(冠詞付き)。mâini/urechi の不規則複数がここで効く。

## 復習チェックリスト

- [ ] 頭頸部・体幹・四肢・内臓・感覚器の主要部位を、性と複数形つきで言える
- [ ] 不規則複数 ochi/ochi・ureche/urechi・mână/mâini・picior/picioare・os/oase を暗記している
- [ ] 単複同形 genunchi・rinichi・mușchi を知っている
- [ ] ⚠ mână(手+腕)・picior(足+脚)・deget(手足の指)・gât(首+のど)の区切り違いを理解し、絞る語(braț/talpă/de la picior/în gât 等)を使える
- [ ] Mă doare/dor + 部位(定形)で症状文を作り、単数 doare・複数 dor を一致させられる
- [ ] 位置がぼやけるとき la + 部位(Mă doare la rinichi)が使えると分かる
- [ ] 慣用表現(a da o mână de ajutor 等)は日常会話のもので、医療の症状申告とは別だと心得ている
- [ ] 本章は語学教材であり、実際の受診では言葉の曖昧さを指差し等で補うべきだと理解している

## 次章への橋渡し

「どこが」を正確に言うための部位語彙がそろった。次はいよいよ「**どう痛むか・どんな症状か**」を細かく描写する番である。次章 [症状の表現](./symptoms.md) では、痛みの種類(ascuțită 鋭い / surdă 鈍い / pulsatilă ズキズキ)、発熱・悪寒、消化器・呼吸器・皮膚・全身の症状、そして程度(ușor/moderat/sever)と頻度(din când în când / permanent)の伝え方を学ぶ。本章の部位語彙(Mă doare stomacul)に「どう痛むか」(o durere surdă=鈍い痛み)を重ねられるようになる。
