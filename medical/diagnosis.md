# 診断・検査結果を読む【CEFR B2〜C1】

> **免責**: 本章は語学学習を目的とした教材であり、医学的・法的助言ではない。
> 実務で使用する場合は、必ず資格を持つ専門家およびネイティブ話者の確認を経ること。
> 本章は病名・検査結果に現れる **語彙と定型表現の読解** を扱うにとどまり、**その診断・数値が何を意味するか、正常か異常か、どう対応すべきかという医学的判断には一切踏み込まない。** 内容の解釈は必ず医師・薬剤師などの医療専門家に委ねること。
> 本章に登場する病名・検査値・診断はすべて **語学学習用の架空・例示** であり、実際の医療データ・指示ではない。体調不良や緊急時には、必ず医療機関・救急サービス(112)に連絡すること。

## この章の目標

この章を終えると、次のことができるようになる。

- 頻出する病名の **語形の作られ方**(infecție, inflamație, fractură, hipertensiune の接尾辞パターン)を体系として理解し、初見の病名を **語として** 読める
- 検査結果の定型(**în limite normale / peste valorile de referință / sub valorile de referință**)を、口頭ではなく **書かれた結果票の言い回し** として読める
- 医師の説明構文(**se recomandă / este necesar să / se impune / urmează să**)を読み・聞き分けられる
- 説明を受けた後に、理解を確定する **確認質問の型**(**Deci, dacă am înțeles bine, ...? / Vreți să spuneți că...?**)を能動的に使える
- 誤読リスクの高い医療語(**afecțiune/infecție、fractură/factură、tensiune arterială の高低表現** 等)を認識し、慎重に扱える

## 前提知識

- [症状の表現(B1)](./symptoms.md) — 痛みの描写、程度・頻度、⚠ amețeală/slăbiciune・greață/vomita・răceală/gripă。本章の直前。症状を「伝える」側から、結果を「読む」側へ進む
- [第5章:医療の説明を理解する(B2)](../04-b2/05-medical-explanations-b2.md) — **医師の口頭説明を聞き取る** 側。analize de sânge / ecografie / radiografie / **în limite normale** / ușor crescut(ă) / efecte secundare / a interna・internare / rețetă。本章は **これらを既習** とし、聞き取りではなく **読解と語彙体系** へ軸足を移す(重複させない)
- [医療の全体像](./overview.md) — 受診の流れ(programare → consultație → rețetă / trimitere → farmacie)、家庭医制度、trimitere(紹介状)。本章の検査・診断はこの流れの中で発行される
- [身体部位・臓器の語彙](./anatomy.md) — 病名は部位・臓器名の上に接尾辞が乗って作られる。臓器語彙が病名読解の土台になる

B2 の[医療の説明を理解する](../04-b2/05-medical-explanations-b2.md)では、患者は医師の **口頭の説明を聞き取り、質問で確定する** 側に立った。本章が新しく扱うのは、その説明が **語彙の体系** として、また **書かれた結果票の定型** として立ち現れる面である。医師は「感染です」と一言で言うが、その裏には **-ție(炎症・感染)/ -oză(状態)/ hiper-・hipo-(過剰・不足)** といった、初見の病名でも見当がつく **語形成の規則性** がある。本章はこの規則を読解の道具として渡し、そのうえで、聞いた・読んだ内容を **確認質問** で確定するところまでを扱う。ただし——B2・C1 の医療章と同じく——**本章は語形と定型の読解にとどまり、診断・数値の医学的判断には立ち入らない。**

## 本文解説

### 病名は「作られ方」で読む ― 接尾辞のパターン

ルーマニア語の病名は、無限に暗記する対象ではない。多くが **部位・原因を表す語幹 + 決まった接尾辞** で作られ、接尾辞が「どんな種類の異常か」を示す。日本語話者にとって、これは漢字の部首で意味の見当をつけるのに似た読解の手がかりになる。⚠ ただし **接尾辞から意味を推測できるのは「語として」の見当** であって、**その病気が実際に何であるか・どの程度かの医学的判断ではない**(免責)。

代表的な接尾辞パターンを、確信の持てるものに絞って挙げる。

- **-ție /-t͡si.e/(炎症・感染・状態を表す名詞化)** ― **infecție**(感染)/ **inflamație**(炎症)。ラテン語系の -tio に対応する女性名詞化で、医療語の中核をなす。
- **-ită /-i.tə/(〜炎)** ― 臓器・部位 + -ită で「その部位の炎症」を作る、非常に生産的な接尾辞。**bronșită**(気管支の炎症)、**gastrită**(胃の炎症)、**dermatită**(皮膚の炎症)、**artrită**(関節の炎症)。⚠ **-ită が付いていれば「〜の炎症(を指す語)」と読めるが、その炎症の原因・重さの判断はしない。**
- **-oză /-o.zə/(〜症・状態)** ― 変性・蓄積・慢性状態などを表す。**artroză**(関節の変性状態)、**tromboză**(血栓の状態)。-ită(炎症)と -oză(状態)の対比は語形読解の要点。
- **hiper- /hi.per-/(過剰・高い)/ hipo- /hi.po-/(不足・低い)** ― **hipertensiune**(高い緊張=高血圧)/ **hipotensiune**(低い緊張=低血圧)、**hiperglicemie**(高血糖)/ **hipoglicemie**(低血糖)。⚠ **hiper-=高い・hipo-=低い** は正反対で、綴りが近く誤読リスクが高い(後述)。
- **接尾辞のない外傷・状態語** ― **fractură**(骨折)、**entorsă**(捻挫)、**luxație**(脱臼)、**arsură**(やけど。symptoms の「灼熱感」と同語)。これらは接尾辞パターンではなく、語として覚える基本外傷語。

この規則性を持っておくと、初見の病名 **X-ită** を見て「X という部位の炎症(を指す語だ)」と **語としての当たり** がつく。⚠ 繰り返すが、当たりがつくのは **語義** であって、その診断の医学的意味・重さではない。意味に確信が持てなければ、推測で確定せず医師・辞書に確認する。

### 頻出病名 ― 語として押さえる(判断はしない)

index.md が挙げる「確実なもの」を中心に、頻出病名を **語として** 押さえる。**いずれもその病気の医学的説明・重さの判断はしない**(免責)。

- **infecție** /inˈfek.t͡si.e/(名女)= 感染(症)。**infecție bacteriană / virală**(細菌性/ウイルス性の感染)は B2 既習。
- **inflamație** /in.flaˈma.t͡si.e/(名女)= 炎症。B2 既習。**-ție** 名詞化の代表。
- **fractură** /frakˈtu.rə/(名女)= 骨折。⚠ **factură**(請求書)と一字違い(後述)。
- **hipertensiune (arterială)** /hi.per.tenˈsju.ne/(名女)= 高血圧(直訳「高い動脈の緊張」)。B1 の **tensiune**(血圧)に hiper- が乗った形。
- **afecțiune** /a.fek.t͡siˈu.ne/(名女)= 疾患・(体の)不調全般。⚠ **infecție**(感染)と別(後述)。C1 で **afecțiune acută/cronică** として既出。
- **alergie** /a.ler.ˈd͡ʒi.e/(名女)= アレルギー。symptoms・B1 既習。
- **anemie** /a.neˈmi.e/(名女)= 貧血(a-「無」+ -emie「血中」)。**glicemie**(血糖)と同じ **-emie** 系。
- **diabet** /di.aˈbet/(名中)= 糖尿病。B1 で **Sufăr de diabet** として既出。

⚠ これらは **語彙として読む** 対象である。診断名が文書・説明に現れたとき、その **意味の重大性ゆえに、内容の理解・説明は必ず医師に委ねる**(とくに後述の benign/malign のような重い対語)。本章は語形・語義の読解を支えるにとどまる。

### 検査結果を「読む」 ― 結果票の定型表現

B2 では、医師が口頭で言う **în limite normale**(正常範囲内)/ **ușor crescut**(やや高い)を **聞き取った**。本章では、同じ内容が **書かれた検査結果票** に現れる定型を **読む**。口頭と書面で言い回しが少しずれる点が、本章の新しい観点である。

- **în limite normale** /ɨn ˈli.mi.te norˈma.le/ = 正常範囲内(B2 既習・口頭でも書面でも使う)。
- **valori de referință** /vaˈlorʲ de re.feˈrin.t͡sə/(名女複)= 参照範囲・基準値(C1 の検査票見出しで既出)。結果票では「基準値」を指す列の見出しになる。
- **peste valorile de referință** /ˈpes.te va.loˈri.le de re.feˈrin.t͡sə/ = 基準値を超えて(直訳「参照値の上に」)。B2 の口頭表現 **peste limita normală**(正常値超え)に対する、**書面の結果票でよく使う言い回し**。
- **sub valorile de referință** = 基準値を下回って(sub =〜の下)。上の対。
- **în afara valorilor de referință** = 基準範囲外(C1 で既出。「範囲の外」の総称)。

**peste / sub valorile de referință** は、結果票で数値の隣に注記として現れることが多い。**peste**(上)/ **sub**(下)の一字で「基準より高い/低い」を分ける、読解上の要となる前置詞対である。⚠ **ただし、その値が高い/低いことが健康上どういう意味を持つかは、本章は一切判断しない。** 本章が教えるのは「peste valorile de referință = 基準値を超えている(という表示だ)」という **言語的な読み** だけであり、その解釈は医師に委ねる(免責。B2・C1 の検査票の原則と一致)。

書面の結果票では、次の **省略・注記** も読めるようにする(いずれも語としての読み)。

- **rezultat** /re.zulˈtat/(名中)= 結果(値の列の見出し)。B2 既習の複数 rezultate の単数。
- **U.M.**(= **unitate de măsură**)= 測定単位(C1 で既出の略。列見出し)。
- **normal / anormal** = 正常/異常(の表示)。⚠ **anormal**(異常)の表示があっても、その医学的意味の判断は医師へ。

### 医師の説明構文 ― se recomandă / este necesar să / se impune

B2 では **se recomandă**(推奨される)/ **Vă recomand să**(お勧めします)を聞き取った。本章では、説明・文書で並ぶ **推奨〜必要〜義務** の構文を、**強さの段階** とともに読み分ける。この段階の聞き分け・読み分けが、B2〜C1 の要点である。

強さの弱い順に:

- **se recomandă (să)...** /se re.koˈman.də/ ― 〜が推奨される(B2 既習)。もっとも一般的・穏当。**Se recomandă repaus.**(安静が推奨される)/ **Se recomandă să evitați efortul.**(無理を避けることが推奨される)。
- **este indicat să...** /ˈjes.te in.diˈkat sə/ ― 〜するのが適切・望ましい(indicat = 適応の・指示された)。**Este indicat să faceți controale periodice.**(定期検診が望ましい)。
- **este necesar să...** /ˈjes.te ne.t͡ʃeˈsar sə/ ― 〜する必要がある(necesar = 必要な)。recomandă より強い。**Este necesar să repetați analizele.**(検査を繰り返す必要がある)。
- **se impune...** /se imˈpu.ne/ ― 〜が求められる・必要とされる(a se impune = 課される)。硬く、文書的で、強い。**Se impune un consult de specialitate.**(専門医の受診が求められる)。
- **urmează să...** /urˈme̯a.zə sə/ ― これから〜することになっている(予定・段取り)。**Urmează să faceți o ecografie.**(これからエコーを受けることになっている)。

これらはいずれも **主語をぼかす非人称・受動の言い回し**(se + 動詞、este + 形容詞 + să)で、[04-b2 第6章:文体とレジスター](../04-b2/06-style-and-register.md)の se 構文の医療応用である。**接続法(conjunctiv)să + 動詞** が後続する点(să evitați / să repetați / să faceți)は B1〜B2 既習。⚠ 読解では「recomandă(推奨)< indicat(望ましい)< necesar(必要)< se impune(求められる)」という **表現の強さの差** を捉える。ただし、**その指示に実際にどう従うべきかは医師・薬剤師に確認する**(本章は表現の強さの読み分けを示すにとどめる)。

### 説明を確定する ― 確認質問の型(能動スキル)

読み・聞きした診断や指示は、**理解を確定する確認質問** で締めるのが、医療における誠実さである。B2 で学んだ **Ce înseamnă asta? / Puteți să îmi explicați mai simplu?**(意味・平易化を求める)を土台に、本章では **「自分の理解が合っているか」を言い返して確かめる** 一段上の型を加える(重複させず、新しい型に絞る)。

- **Deci, dacă am înțeles bine, ...?** /det͡ʃʲ ˈda.kə am ɨnˈt͡se.les ˈbi.ne/ ― つまり、私の理解が正しければ、〜ということですか?(自分の要約を言い返して確認)
- **Vreți să spuneți că...?** /vret͡sʲ sə spuˈnet͡sʲ kə/ ― 〜とおっしゃりたいのですか?(相手の意図を言い換えて確認)
- **Adică...?** /aˈdi.kə/ ― つまり…?(短い言い換え確認)
- **Am înțeles corect că trebuie să...?** ― 〜しなければならない、と正しく理解しましたか?(指示の理解確認)
- **Vă rog să repetați ce trebuie să fac.** ― 何をすべきか、もう一度言ってください(指示の再確認を依頼)。

**Deci, dacă am înțeles bine, trebuie să repet analizele peste o lună?**(つまり、私の理解が正しければ、1か月後に検査をやり直すのですね?)のように、**自分の言葉で要約して投げ返す** ことで、聞き違い・読み違いをその場で正せる。⚠ これは B2 の[会議と交渉](../04-b2/04-business-meetings-and-negotiation.md)で学んだ **言い換えによる確認(reformulare)** の医療版である。医療では、この確認を挟む手間が **安全** に直結する。分からないまま・思い込んだまま帰らないことが、本章を貫く原則である(実際の医療判断は医療者に委ねる)。

## 重要ポイント

- 病名は接尾辞で読む:**-ție**(感染・炎症の名詞化)/ **-ită**(〜炎)/ **-oză**(〜症・状態)/ **hiper-**(高)・**hipo-**(低)。初見でも「語としての当たり」がつく(医学的判断はしない)
- 頻出病名:**infecție / inflamație / fractură / hipertensiune / afecțiune / anemie / diabet**(語として押さえる。重い対語は内容を医師へ)
- 結果票の読解:**în limite normale / peste valorile de referință / sub valorile de referință / în afara valorilor de referință**。**peste(上)/ sub(下)** の一字で高低を分ける
- 医師の説明構文は強さで段階:**se recomandă(推奨)< este indicat(望ましい)< este necesar să(必要)< se impune(求められる)**。urmează să=予定
- 確認質問で理解を確定:**Deci, dacă am înțeles bine, ...? / Vreți să spuneți că...? / Am înțeles corect că trebuie să...?**
- ⚠ 誤読:**afecțiune(疾患)≠ infecție(感染)、fractură(骨折)≠ factură(請求書)、hiper-/hipo-(高/低)** を取り違えない
- **本章は語形・定型の読解のみ。診断・数値の医学的意味は判断せず、必ず医師に委ねる**(免責)

## 文法メモ ― 診断・結果に頻出する形

| 形 | 意味 | 例 | どこで読む・聞くか |
|---|---|---|---|
| 語幹 + **-ită** | 〜の炎症 | bronșită, gastrită, artrită | 病名 |

## よくある誤用(読解・運用上の取り違え)

| ❌ 誤り | ✅ 正しく | 説明 |
|---|---|---|
| **afecțiune** を「感染」と読む | afecțiune=疾患・不調(全般)。感染は **infecție** | 語が近いが別。afecțiune は原因を問わない不調全般 ⚠ |
| **fractură** を「請求書」と読む | fractură=骨折。請求書は **factură** | 一字違い(fr-/f-)で意味が全く別。文脈で判別 ⚠ |
| **hipotensiune** を「高血圧」と読む | hipo-=低い→低血圧。高血圧は **hipertensiune**(hiper-) | hiper-(高)/hipo-(低)は正反対。綴りが近く誤読リスク大 ⚠ |

| 病名 **X-ită** を見て自己診断する | 「X の炎症を指す語」と読むにとどめ、診断は医師へ | 語形の当たりは語義まで。医学的判断ではない ⚠ |

## ⚠ 誤読・混同に注意する語

医療語は取り違えると別の情報として伝わり、実害に直結しうる。**確信の持てる、頻出かつ重要な誤読リスクを挙げる。** 既出の ⚠(symptoms の amețeală/slăbiciune 等、B2 の analiză・a interna・efecte secundare・rețetă、C1 の acut/cronic・benign/malign・diagnostic/prognostic)とは **重複させず**、本章で新たに要注意となる語に絞る。

- **⚠ afecțiune(疾患・不調)と infecție(感染)** ― **afecțiune** /a.fek.t͡siˈu.ne/ は「(体の)疾患・不調」を **原因を問わず** 指す総称、**infecție** /inˈfek.t͡si.e/ は「感染(病原体による)」という **原因を特定した** 語である。語形が近い(af-/inf-、ともに -ție 系)ため混同しやすいが、**すべての afecțiune が infecție ではない。** 医師が **o afecțiune cronică** と言えば「慢性の疾患(感染とは限らない)」、**o infecție** と言えば「感染」で、指す範囲が違う。日本語話者は両方「病気」とまとめがちだが、読解では区別する。⚠ ただし、実際の診断が感染か否かの判断は医師に委ねる(本章は語の区別を示すにとどめる)。

- **⚠ fractură(骨折)と factură(請求書)** ― **fractură** /frakˈtu.rə/(骨折)と **factură** /fakˈtu.rə/(請求書・インボイス)は、**r 一字の有無だけ** の違いで意味が全く別の語である。医療文書で **fractură**、経理・行政文書で **factură** と、文脈で分かれるが、**綴りを取り違えると「骨折」を「請求書」と読む** ような致命的な誤読になる。⚠ 医療の文脈では fr- の **fractură**(骨折)、と一字を確実に読む。とくに手書き・不鮮明な文書では要注意。

- **⚠ hiper-(高い・過剰)と hipo-(低い・不足)** ― 接頭辞 **hiper-** は「高い・過剰」、**hipo-** は「低い・不足」を表し、**正反対** である。**hipertensiune**(高血圧)/ **hipotensiune**(低血圧)、**hiperglicemie**(高血糖)/ **hipoglicemie**(低血糖)のように、同じ語幹に付いて逆の意味を作る。⚠ 綴りが hiper-/hipo- と近く(共通の hip-)、**取り違えると高低が逆転する** 極めて誤読リスクの高い対である。読解では **hiper=高、hipo=低** と確実に区別する。意味に確信が持てなければ確認する(医学的判断は医師へ)。

- **⚠ tensiune arterială の「高い/低い」表現** ― B1 で **tensiune**(血圧)、B1 の ⚠ で「tensiune の多義(血圧/緊張)」を扱った。本章で新たに注意すべきは、**血圧の高低の言い方** である。「血圧が高い」は **tensiune mare / crescută** または **hipertensiune**、「低い」は **tensiune mică / scăzută** または **hipotensiune** で表す。⚠ 日本語話者は **Am tensiune.**(B1 既出)を字義どおり「血圧がある」と読みがちだが、口語ではしばしば「血圧が高い(高血圧だ)」の含意で使われる。**mare/mică(高/低)や hiper-/hipo- の有無** で高低が決まるので、そこを読み落とさない。ただし血圧値の医学的評価は医師に委ねる。

- **⚠ se recomandă(推奨)と se impune(要求)の強さの差** ― どちらも非人称の se 構文だが、**se recomandă** は「推奨される(穏当)」、**se impune** は「求められる・必要とされる(強い)」で、**指示の強さが違う**。⚠ 日本語話者は両方「〜されます」と同じ強さに読みがちだが、**se impune** は「ほぼ必須」に近い硬い要求である。読解では recomandă(弱)/ necesar(中)/ se impune(強)の段階を捉える。ただし実際にどう対応すべきかは医師・薬剤師に確認する(本章は表現の強さの読み分けを示すにとどめる)。

## ネイティブらしい理解のコツ

| つまずき | 読解・運用のコツ |
|---|---|
| 初見の病名が読めない | 接尾辞(-ită=炎/-oză=症)・接頭辞(hiper-/hipo-)で **語としての当たり** をつける。ただし診断は医師へ |
| 結果票の高低表示が読めない | **peste**(上)/ **sub**(下)+ valorile de referință の一字で高低を掴む。数値の意味は判断しない |
| 指示の強さが分からない | recomandă(推奨)< necesar(必要)< se impune(求められる)の順に強くなると捉える |
| 説明を分かった気で帰ってしまう | **Deci, dacă am înțeles bine, ...?** で自分の要約を言い返し、その場で確定する |
| 似た語形(afecțiune/infecție, fractură/factură, hiper/hipo)で迷う | 語形の近さで推測せず、**別語・対語** として確実に区別。確信がなければ確認 |
| 意味に確信が持てない | **推測で確定せず、医師・薬剤師・辞書・有資格の医療通訳者に確認する**(本章の原則) |

最後のコツが最も重要である。診断・検査結果は、**読み違いが人の健康・安全に直結する** 領域である。本章は語形・定型の読解を **語学の側から支える** ものであって、医学的判断を代替するものではない。分からなければ確認する——この姿勢を本章全体で保ってほしい。

## フォーマル/カジュアルの違い

診断・結果の説明は、**書面はほぼ非人称・硬い書き言葉**、口頭の医師説明は **丁寧(dumneavoastră 系)** で来る。患者の確認質問も丁寧形で行う。

- **結果票・文書(非人称・硬い)**: Valori peste limitele de referință. / Se impune un consult de specialitate. / Este necesar să se repete analizele.
- **医師の口頭説明(丁寧形)**: Aveți o valoare puțin peste normal, dar nu e nimic grav. / Vă recomand să repetați analizele peste o lună.
- **患者の確認(丁寧)**: Deci, dacă am înțeles bine, trebuie să revin peste o lună? / Vreți să spuneți că nu este nimic grav?
- **家族に伝える(tu 系・平易)**: Doctorul a zis că e o valoare puțin mare, dar nimic serios.(先生が、値が少し高いけど深刻じゃないって)

書面は非人称で強く、口頭は丁寧で穏当に言い換えられることが多い。⚠ 家族に伝える橋渡しでは、**推測で訳さず、分からない語は医師に確認してから** 伝えること(B2 の橋渡しの鉄則を継続。命に関わる場面は有資格の医療通訳者・医療者へ。職業通訳は `medical/medical-interpretation.md`)。

## 文化的注意点

- 一般に、ルーマニアの検査結果票・診断記載は、**ラテン語系の専門語と非人称の書き言葉** で書かれる傾向があるとされる(文書・機関・電子化の程度により差はある)。近年の電子カルテ・電子処方箋の普及で書式は変わりうるため、本章は特定の書式を規定しない。
- 一般に、家庭医(medic de familie)を起点に、専門医・検査へは紹介状(trimitere)を得て進む流れが言及される([医療の全体像](./overview.md)の骨格を参照)。検査(analize)の指示・結果の説明も、まず家庭医を通ることが多い。制度の詳細・料金は変動情報のため本章では扱わない(免責)。
- 診断・検査結果に **CNAS / asigurat / card de sănătate** といった保険・制度の語が併記されることがある(C1 で既出)。これらが現れたら「保険・制度に関わる記載だ」と **言語的に認識** し、制度の実務は一次情報・専門家に委ねる。
- 診断・数値の解釈は、**必ず資格を持つ医療専門家に委ねる。** 本章は語形・定型の読解を扱うものであり、医学的判断を代替するものではない(免責の再確認)。

## 練習問題(読解・語彙のみ・医学的判断を求めない)

### 問題1: 接尾辞・接頭辞から語の当たりをつけよ(語義のみ)

2.「深刻なものは何もない、ということですか?」(Vreți să spuneți că を使う)

## 解答

### 問題1
1. **〜の炎症**(-ită=部位の炎症。bronșită=気管支の炎症を指す語)。 2. **〜症・状態**(-oză=変性・状態。artroză=関節の状態を指す語)。 3. **高い**(hiper-=高・過剰)。 4. **低血圧**(hipo-=低。hipotensiune=低い血圧)。
— 接尾辞・接頭辞で **語としての当たり** はつくが、⚠ その診断の医学的意味・重さは判断しない(免責)。

### 問題2
1. **上**(peste=〜の上=基準値超)。 2. **下**(sub=〜の下=基準値未満)。 3. **正常範囲内**(în limite normale)。
— peste/sub の一字で高低を分ける。⚠ ただし値が高い/低いことの医学的意味は本章では判断しない(医師へ)。

### 問題3
弱い順: **se recomandă(推奨)→ este necesar să(必要)→ se impune(求められる)**。
— recomandă(穏当)< necesar(必要)< se impune(硬く強い)。表現の強さの差を読む。実行法は医師・薬剤師に確認。

### 問題4
1. **afecțiune=疾患・不調 / infecție=感染**(af-/inf- が近いが別。afecțiune は原因を問わない総称 ⚠)。 2. **骨折**(医療文書の fractură。factură=請求書とは r 一字違い ⚠)。 3. **高血圧**(hiper-=高。hipotensiune=低血圧 ⚠)。
— すべて本章 ⚠ の要点。afecțiune/infecție、fractură/factură、hiper-/hipo- を取り違えない。診断内容の判断は医師へ。

### 問題5
1. **Deci, dacă am înțeles bine, trebuie să revin peste o lună?**(自分の要約を言い返して確認)。 2. **Vreți să spuneți că nu este nimic grav?**(相手の意図を言い換えて確認)。
— 分かった気で帰らず、確認質問で理解を確定する。⚠ 医療では確認を挟む手間が安全に直結する(実際の判断は医療者に委ねる)。

## 次章への橋渡し

診断・検査結果を、語形と結果票の定型として **読み**、確認質問で理解を **確定** できるようになった。ここまでで、症状を「伝える」(symptoms)→ 説明を「聞く」(B2)→ 結果を「読む」(本章)という受診の情報の流れが一通りそろった。次は、この一連のやり取りを **病院という具体的な場の対話** の中で運用する番である。次章 [`hospital-dialogues.md`](./hospital-dialogues.md)(受付・診察室・入院病棟での対話)では、予約変更・検査説明・入院手続きといった実際の対話の流れの中で、本章の診断・結果の語彙と確認質問を使う練習に入る。読解で得た語彙を、双方向のやり取りへと開いていく。

## 出典

- 語義・正書法(infecție, inflamație, afecțiune, fractură, hipertensiune/hipotensiune, anemie, -ită / -oză / hiper- / hipo-, valori de referință, se recomandă / se impune / este necesar să, repaus, consult 等)は Academia Română DOOM3(2021)および DEX 系辞書(dexonline.ro)に準拠して確認した(参照 2026-07)。これらは規範的な語彙・正書法であり、**医学的助言ではない**。
- 検査・診断の読解観点(în limite normale / peste・sub valorile de referință)は [第5章:医療の説明を理解する(B2)](../04-b2/05-medical-explanations-b2.md) の口頭表現、および [医療文書(C1)](../05-c1/05-medical-documents-c1.md) の検査票読解を土台に、本章では **書面の読解と語彙体系** に絞った(重複させない)。
- 受診・検査・紹介状(trimitere)など制度面の骨格は [医療の全体像](./overview.md) に集約し、本章では言語運用に絞った(制度の数値・要件は本章では扱わない。免責参照)。
- 病名・検査値・診断に関する記述はすべて語学学習用の架空の例示であり、医療上の指示ではない。内容の解釈は必ず医療専門家に委ねる。
- 出典一覧の集約は [bibliography.md](../bibliography.md)(言語規範・辞書節)を参照。
