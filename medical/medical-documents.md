# 医療文書の実務読解【CEFR C1】

> **免責(強化)**: 本章は **語学学習を目的とした教材であり、医学的・法的助言では一切ない。**
> 本章が扱うのは、医療文書に現れる **言語の型(構造・定型表現・略号・語彙)の読解** だけである。
> **検査値・診断名・治療内容・病欠日数・接種内容などの医学的・行政的な判断には一切踏み込まない。**
> 医療文書の内容の解釈は、**必ず資格を持つ医師・薬剤師などの医療専門家、および必要に応じて有資格の医療通訳者に委ねること。**
> 本章の理解だけで自己判断・自己治療をしてはならない。体調不良や緊急時には、必ず医療機関・救急サービス(112)に連絡すること。
> 本章に登場する患者名・医師名・機関名・診断名・検査値・日付・処方はすべて **語学学習用の架空の例示** であり、実在の個人・医療データとは一切関係がない。

## この章の目標

この章を終えると、次のことができるようになる。

- **紹介状(bilet de trimitere)** の実務的な読解——「誰が・誰へ・何のために送るか」の構造を **言語として** 読み取れる
- **診断書(certificat medical)** と **病欠証明(concediu medical)** の言語(発行目的・記載項目・定型表現)を読み分けられる
- **予防接種記録(schema de vaccinare)** に現れる語彙を **語として** 読める(接種内容・スケジュールの医学的判断はしない)
- **検査依頼書(bilet de trimitere pentru analize / recomandare de investigații)** の構造を読み、「どこを最初に読むか」の当たりがつけられる
- 各文書の「**最初に読むべき箇所**」を知り、高密度の文書を効率よく把握できる
- **本章が言語の型の読解にとどまり、医学的・行政的判断には踏み込まないこと** を自覚して運用できる

## 前提知識

- [第5章:医療文書(C1)](../05-c1/05-medical-documents-c1.md)(C1) — **本章の土台。** 医療文書の主要種別(scrisoare medicală / bilet de externare / rezultatele analizelor / rețetă)、退院サマリーの見出し構造(diagnostic la internare/externare / antecedente personale・heredocolaterale / evoluție / recomandări)、検査結果票の項目名(hemoglobină, glicemie...)、処方箋の略号(Rp. / D.S.)、⚠ 誤読リスク(acut/cronic, benign/malign, diagnostic/prognostic, a se administra, buletin de analize, recomandări)。**これらは既習として重複させず、本章は別の文書種別(紹介状・診断書・病欠証明・接種記録・検査依頼書)へ拡張する**
- [医療の全体像(overview)](./overview.md)(A2〜B1) — 受診の流れ(programare → recepție → consultație → rețetă / trimitere → farmacie)、家庭医(medic de familie)制度、⚠ **trimitere(紹介状)と rețetă(処方箋)の別物性**、card de sănătate / asigurat / CNAS。本章の文書はこの流れの中で発行される
- [医療通訳の言語運用(medical-interpretation)](./medical-interpretation.md)(C1) — 「推測せず確認する/足さない引かない変えない」。文書を家族に口頭で橋渡しする場面では、この通訳原則がそのまま生きる
- [身体部位・臓器の語彙(anatomy)](./anatomy.md)・[症状の表現(symptoms)](./symptoms.md)(A2〜B1) — 文書に現れる部位・症状語の土台

前章[医療文書(C1)](../05-c1/05-medical-documents-c1.md)は、退院サマリー・検査結果票・処方箋という **診療の「結果」を伝える文書** を読解した。本章はその **深化・拡張** として、**受診の「流れ」を動かす文書**——専門医へ送る紹介状、診断を証明する診断書、病欠を証明する証明書、接種の記録、検査を依頼する依頼書——を扱う。前章と同じく——**本章の目的は「文書に何が書いてあるかを言語として読み解く」ことに厳格に限られ、「その内容が医学的・行政的に妥当か」という判断には立ち入らない。** 診断名の意味も、病欠日数の妥当性も、接種の要否も、本章はしない。それは医療専門家・行政の領域である。

## 本文解説

### 本章が扱う文書の位置づけ ― 前章との棲み分け

前章[医療文書(C1)](../05-c1/05-medical-documents-c1.md)で読んだ文書と、本章で読む文書の棲み分けを、まず地図として押さえる。**各語の意味は読解上の語義であり、その文書の医学的・行政的効力を保証するものではない。**

| 文書 | 主に扱う章 | 読解上の位置づけ |
|---|---|---|
| bilet de externare(退院サマリー) | [前章](../05-c1/05-medical-documents-c1.md) | 入院診療の「結果」の要約 |
| rezultatele analizelor / buletin de analize(検査結果票) | [前章](../05-c1/05-medical-documents-c1.md) | 検査の「結果」 |
| rețetă(処方箋) | [前章](../05-c1/05-medical-documents-c1.md) | 薬局へ向かう指示 |
| **bilet de trimitere(紹介状)** | **本章** | 別の医師・検査へ「送る」文書(流れを動かす) |
| **certificat medical(診断書)** | **本章** | 診断を「証明する」文書 |
| **concediu medical(病欠証明)** | **本章** | 病気による休業を「証明する」文書 |
| **schema de vaccinare(予防接種記録)** | **本章(語彙のみ)** | 接種の「記録」 |
| **bilet de trimitere pentru analize / recomandare(検査依頼書)** | **本章** | 検査を「依頼する」文書 |

前章が「診療の結果を要約・証明する文書」なら、本章は **「次のアクションへ人を送る/事実を証明する文書」** が中心である。**trimitere(送ること)** と **certificat(証明)** という二つの機能語が、本章の文書群の骨格を作る。

### 紹介状(bilet de trimitere)を実務的に読む

**bilet de trimitere** /biˈlet de triˈmi.te.re/(紹介状)は、[overview](./overview.md) で見たとおり、家庭医(medic de familie)が患者を **専門医・検査・入院へ「送る」** ための書類である。前章で「bilet de + 名詞で文書種別が作られる」ことは既習だが、本章では紹介状を **実務的に読む**——つまり「誰が・誰へ・何のために送るか」を構造として読み取る。**内容の医学的判断はしない。**

紹介状に頻出する構造要素を、**読み(語として)** で押さえる。

| 要素(語) | IPA/発音 | 読解上の意味 |
|---|---|---|
| **medic trimițător** | /ˈme.dik tri.mi.t͡səˈtor/ | 紹介元の医師(直訳「送る側の医師」。多くは家庭医) |
| **către / către dr.** | /ˈkə.tre/ | 〜宛(直訳「〜へ向けて」。紹介先を示す) |
| **specialitatea** | /spe.t͡ʃi.a.liˈta.te̯a/ | 診療科・専門(cardiologie, neurologie... の別) |
| **motivul trimiterii** | /moˈti.vul tri.miˈte.rij/ | 紹介理由(直訳「送ることの理由」。属格 trimiterii) |
| **diagnostic prezumtiv** | /di.aɡˈnos.tik pre.zumˈtiv/ | 推定診断(直訳「推定的な診断」。⚠ 後述) |
| **investigații solicitate** | /in.ves.tiˈɡa.t͡sij so.li.t͡ʃiˈta.te/ | 依頼する検査(直訳「要請された精査」) |
| **date de contact / CNP** | /ˈda.te de konˈtakt/ | 連絡先・個人番号(CNP=個人識別番号) |

読解の骨格は、紹介状を **「送る側 → 送られる先 → なぜ」** の三点で把握することである。**medic trimițător**(誰が)、**către + 診療科/医師**(誰へ)、**motivul trimiterii**(なぜ)——この三点を押さえれば、紹介状の骨組みが読める。⚠ **diagnostic prezumtiv**(推定診断)は特に注意を要する語である。**prezumtiv** は「推定の・仮の」を意味し、これは **確定診断ではなく「紹介元の医師が疑っている段階の診断」** を示す。読解上「prezumtiv = まだ確定していない、送り先で確かめてもらう段階」と受け取る。⚠ ただし、その推定診断の医学的な当否は **本章では一切判断しない**(免責)。

**この文書の「どこを最初に読むか」**:紹介状はまず **către(誰宛か=どの診療科・医師へ送られるか)** と **motivul trimiterii(なぜ送るか)** を読む。この二点で「この紹介状は何のための誰への紹介か」が掴める。診断名・検査の詳細は、その枠組みを掴んだ後に読む。⚠ 内容の医学的意味は判断せず、必ず紹介先の医師に委ねる。

### 診断書(certificat medical)の言語

**certificat medical** /t͡ʃer.ti.fiˈkat me.diˈkal/(診断書・医療証明書)は、**医師が診断・所見を「証明する」文書** である。**certificat**(証明書)という語が機能の核で、雇用主・学校・行政などへ提出する目的で発行されることがある。本章が扱うのは **言語の型** だけで、**その証明の効力・適用は行政・法律の領域であり判断しない。**

診断書に現れる定型語彙。

| 語 | IPA/発音 | 読解上の意味 |
|---|---|---|
| **certificat medical** | /t͡ʃer.ti.fiˈkat me.diˈkal/ | 診断書・医療証明書(certificat=証明書) |
| **se certifică prin prezenta că...** | /se t͡ʃer.tiˈfi.kə prin preˈzen.ta/ | 「本書により〜を証明する」(証明の定型) |
| **subsemnatul dr. ...** | /sub.semˈna.tul/ | 「下記署名の〜医師」(発行者の自称。硬い書式語) |
| **la cererea pacientului** | /la t͡ʃeˈre.re̯a pa.t͡ʃi.enˈtu.lui/ | 「患者の求めにより」(発行の契機) |
| **pentru a servi la...** | /ˈpen.tru a serˈvi la/ | 「〜に供するため」(提出先・用途) |
| **eliberat la data de...** | /e.li.beˈrat la ˈda.ta de/ | 「〜(日付)に発行」(a elibera=発行する) |
| **ștampila și semnătura** | /ʃtamˈpi.la ʃi sem.nəˈtu.ra/ | 印章と署名 |

読解の要点は、診断書が **「誰が(subsemnatul dr.)・何を(se certifică că...)・何のために(pentru a servi la...)証明するか」** の三点で構成される定型文書だと掴むことである。⚠ **se certifică prin prezenta că...**(本書により〜を証明する)は、行政・法律文書と共通する **証明の定型**([契約書と行政文書(C1)](../05-c1/04-contracts-and-administration.md)の書式語彙と地続き)であり、se 受動(B2 既習)の硬い書式版である。**pentru a servi la...**(〜に供するため)は用途・提出先を示す定型で、診断書が「何かの手続きのために出される」ことを言語化している。

**この文書の「どこを最初に読むか」**:診断書はまず **se certifică ... că の後ろ(何を証明しているか)** と **pentru a servi la ...(何のための証明か)** を読む。証明の中身と用途が掴めれば、文書の目的が分かる。⚠ 診断内容の医学的意味・証明の行政的効力は判断せず、医師・行政窓口に委ねる。

### 病欠証明(concediu medical)の言語

**concediu medical** /konˈt͡ʃe.di.u me.diˈkal/(病欠・疾病休業証明)は、**病気による休業を証明する文書** である。⚠ **concediu** /konˈt͡ʃe.di.u/ は多義語で、日常語では「休暇・バカンス」を意味するが、**concediu medical** は「病欠(病気休暇)」という別の制度的な意味になる(後述の ⚠)。本章は **言語の型** のみを扱い、**休業日数・給付・手続きの妥当性は行政・制度の領域であり判断しない。**

病欠証明に現れる語彙。

| 語 | IPA/発音 | 読解上の意味 |
|---|---|---|
| **concediu medical (CM)** | /konˈt͡ʃe.di.u me.diˈkal/ | 病欠・疾病休業(略 CM)⚠ concediu=休暇との多義 |
| **incapacitate temporară de muncă** | /in.ka.pa.t͡ʃiˈta.te tem.poˈra.rə de ˈmun.kə/ | 一時的労働不能(直訳。病欠の根拠となる状態) |
| **cod de indemnizație** | /kod de in.dem.ni.zaˈt͡si.e/ | 給付コード(直訳「補償のコード」。数値・分類) |
| **perioada de valabilitate** | /pe.riˈo̯a.da de va.la.bi.liˈta.te/ | 有効期間(直訳「有効性の期間」=休業の期間) |
| **de la ... până la ...** | /de la ... ˈpɨ.nə la/ | 〜から〜まで(期間の定型) |
| **seria și numărul** | /ˈse.ri.a ʃi ˈnu.mə.rul/ | 証明書の記号と番号 |

読解の要点は、病欠証明が **「いつからいつまで(perioada / de la ... până la ...)、どういう状態(incapacitate temporară de muncă)による休業か」** を証明する文書だと掴むことである。⚠ **incapacitate temporară de muncă**(一時的労働不能)は病欠の根拠を示す硬い定型句で、「一時的に(temporară)働けない状態(incapacitate de muncă)」を表す。**cod de indemnizație**(給付コード)は分類のための数値・記号であり、⚠ **その分類・給付の内容は制度の領域で、本章は判断しない。**

**この文書の「どこを最初に読むか」**:病欠証明はまず **de la ... până la ...(有効期間=休業の期間)** を読む。期間が休業証明の核だからである。次に incapacitate temporară de muncă(休業の根拠となる状態区分)を読む。⚠ 期間・分類・給付の妥当性は行政・雇用主・制度に委ね、本章は言語の型の読解にとどめる。

### 予防接種記録(schema de vaccinare)― 語彙のみ

**schema de vaccinare** /ˈske.ma de vak.t͡ʃiˈna.re/(予防接種スケジュール・記録)は、**受けた/受ける予定の接種の記録** である。本章はこの文書に現れる **語彙を「語として」読める** ことだけを扱う。⚠ **どの接種をいつ受けるべきか・受けたか、その内容が適切かといった医学的・スケジュール上の判断は一切しない**(免責。接種の要否・時期は必ず医師・公的接種案内に委ねる)。

読解の手がかりとなる語彙(語として)。

| 語 | IPA/発音 | 読解上の対応語 |
|---|---|---|
| **vaccin** | /vakˈt͡ʃin/ | ワクチン(名中。複数 vaccinuri) |
| **vaccinare** | /vak.t͡ʃiˈna.re/ | 予防接種(a vaccina の名詞化) |
| **schema de vaccinare** | /ˈske.ma de vak.t͡ʃiˈna.re/ | 接種スケジュール・記録 |
| **doză** | /ˈdo.zə/ | 接種回(直訳「用量・回」。prima doză=1回目) |
| **rapel** | /raˈpel/ | 追加接種・ブースター |
| **data administrării** | /ˈda.ta ad.mi.nisˈtrə.rij/ | 接種日(直訳「投与の日」。a administra の名詞化 administrare) |
| **lot / serie** | /lot/ | ロット・製造番号 |
| **carnet de vaccinări** | /karˈnet de vak.t͡ʃiˈnərʲ/ | 接種手帳(記録冊子) |

読解の要点は、接種記録が **「どの vaccin を・いつ(data administrării)・何回目(doza / rapel)」** を並べた表形式の記録だと掴むことである。**doză**(接種回)と **rapel**(追加接種)の別、**data administrării**(接種日)の位置を **語として** 読めれば、記録の構造は把握できる。⚠ **しかし、どの接種をいつ受けるべきか、記録が完全か、次に何が必要かといった判断は、本章は一切しない。** 接種の要否・時期・内容は、**必ず医師・公的な接種案内に委ねる。** 本章が扱うのは「vaccin / doză / rapel / data administrării といった語が接種記録に並ぶ」という言語的事実のみである。

### 検査依頼書(bilet de trimitere pentru analize / recomandare de investigații)

医師が患者を **検査に回す** とき、**検査依頼書** が発行される。呼び名は文脈により **bilet de trimitere pentru analize**(検査のための紹介票)、**recomandare de investigații**(検査の推奨・依頼)などがある。⚠ ここで注意すべきは、前章の **rezultatele analizelor(検査結果票)** とは **別物** だという点である——依頼書は検査を **これから受けるための「依頼」**、結果票は検査を **受けた「結果」** である(読解上の区別)。本章は **言語の型** のみを扱う。

検査依頼書に現れる語彙。

| 語 | IPA/発音 | 読解上の意味 |
|---|---|---|
| **recomandare de investigații** | /re.ko.manˈda.re de in.ves.tiˈɡa.t͡sij/ | 検査の依頼・推奨 |
| **analize solicitate** | /aˈna.li.ze so.li.t͡ʃiˈta.te/ | 依頼された検査(項目の列挙) |
| **cu / à jeun** | / a ʒø/ | 空腹時に(検査前の条件の定型として現れうる) |
| **diagnostic prezumtiv** | /di.aɡˈnos.tik pre.zumˈtiv/ | 推定診断(依頼の根拠。紹介状と共通) |
| **unitatea recomandată** | /u.ni.ˈta.te̯a re.ko.manˈda.tə/ | 推奨される検査機関 |
| **valabil până la...** | /vaˈla.bil ˈpɨ.nə la/ | 〜まで有効(有効期限) |

読解の要点は、検査依頼書が **「どんな検査を(analize solicitate)・なぜ(diagnostic prezumtiv)・どこで・いつまでに(valabil până la)受けるか」** を指示する文書だと掴むことである。⚠ **依頼された検査の項目名(hemoglobină, glicemie...)を語として読むのは前章[医療文書](../05-c1/05-medical-documents-c1.md)で既習** であり、本章はその項目が **依頼書という別の文書に現れる** ことを扱う。**検査の内容・必要性・準備条件(空腹時か等)の医学的判断は、必ず医師・検査機関に委ねる。**

**この文書の「どこを最初に読むか」**:検査依頼書はまず **analize solicitate(何の検査を依頼されているか)** と、あれば **準備条件(空腹時など)・valabil până la(有効期限)** を読む。何を・いつまでに受けるかが掴めれば、次のアクションが分かる。⚠ 検査の医学的意味・準備の実行は医師・検査機関に委ねる。

## ⚠ 医療・行政文書の誤読リスク(本章で新たに要注意の語)

前章[医療文書(C1)](../05-c1/05-medical-documents-c1.md)の ⚠(acut/cronic, benign/malign, diagnostic/prognostic, a se administra, buletin de analize, recomandări)とは **重複させず**、本章の文書群で新たに要注意となる語に絞る。**いずれも語の区別を示すにとどまり、医学的・行政的判断はしない。**

- **⚠ concediu(休暇)と concediu medical(病欠)** ― **concediu** /konˈt͡ʃe.di.u/ 単独は日常語で「休暇・バカンス」(concediu de odihnă=年次有給休暇)を意味するが、**concediu medical** は「**病欠**(疾病による休業)」という制度的に別の文書・意味になる。⚠ 日本語話者は concediu を「休暇」の一択で読みがちだが、medical が付くと「(楽しい)休暇」ではなく「病気による休業」である。読解上「concediu medical = 病欠証明」と掴む。休業日数・給付の内容は制度の領域で判断しない。

- **⚠ certificat(証明書)と certificare / a certifica** ― **certificat** /t͡ʃer.ti.fiˈkat/ は「証明書」(名詞。certificat medical=診断書、certificat de naștere=出生証明)を指す。⚠ 英語 certificate の連想で「資格・免許」と読みがちだが、医療の certificat medical は「診断を証明する書類」であって資格証ではない。**a certifica**(証明する)/ **se certifică**(証明される)は証明の動詞・定型。読解上「certificat = (事実を)証明する書類」と掴む。証明の効力は行政・法律の領域で判断しない。

- **⚠ trimitere(紹介・送付)と rețetă(処方箋)〔前章・overview 再確認〕** ― [overview](./overview.md) と前章で既習の別物性を、本章の紹介状・検査依頼書の文脈で再確認する。**trimitere** は「専門医・検査へ **送る**」書類、**rețetă** は「薬局で薬を受け取る」書類。⚠ 本章の bilet de trimitere(紹介状)・bilet de trimitere pentru analize(検査依頼書)はいずれも **trimitere 系(送る)** であり、rețetă(処方箋)とは持って行く先が違う。取り違えると、薬局に紹介状を持って行くような誤りが起きる。

- **⚠ prezumtiv(推定の)を「確定」と読まない** ― **diagnostic prezumtiv**(推定診断)の **prezumtiv** /pre.zumˈtiv/ は「**推定の・仮の・疑いの段階の**」を意味し、**確定診断ではない**。⚠ 日本語話者は「診断」の語に引かれて確定的に読みがちだが、prezumtiv が付けば「紹介元の医師が **疑っている段階** の、送り先で確かめてもらう診断」である。読解上「prezumtiv = まだ確定していない」と掴む。⚠⚠ **その推定の当否・確定は必ず医師に委ねる**(本章は語の区別を示すにとどまる)。

- **⚠ eliberat(発行された)と liber(自由な)** ― **a elibera** /a e.li.beˈra/ は「(書類を)**発行する**・交付する」で、**eliberat la data de...**(〜に発行)のように文書の発行を示す。⚠ 語幹が liber(自由な)に近いため「解放された・自由な」と読みたくなるが、文書文脈の eliberat は「発行された・交付された」である。読解上「document eliberat = 発行された書類」と掴む。

- **⚠ solicitat(要請された)を軽く読まない** ― **investigații / analize solicitate**(依頼された検査)の **solicitat** /so.li.t͡ʃiˈtat/(a solicita=要請する・依頼する)は「(正式に)要請された」であり、単なる「希望」より重い。⚠ 読解上「solicitate = (医師により)依頼された(受けるべき)検査」と受け取る。ただし検査の要否・実行は医師・検査機関に委ねる。

## モデル文書 ― 2本(架空・読解用)

**以下2本の文書は語学学習用の架空の例示** である。患者名・医師名・機関名・診断名・日付・番号はすべて架空で、実在の個人・医療データとは一切関係がない。**言語の型を読解する練習用であり、医学的・行政的な参照・判断の材料にしてはならない。** 内容の解釈は必ず専門家に委ねること。

### モデル文書A ― 紹介状(bilet de trimitere・抜粋)

---

**BILET DE TRIMITERE (extras — date fictive)**

**Medic trimițător:** Dr. [nume fictiv], medic de familie
**Către:** Serviciul de **Cardiologie**, în atenția dr. [nume fictiv]

**Pacient:** [nume fictiv] — CNP: [număr fictiv]

**Motivul trimiterii:** Pacientul se prezintă pentru evaluare de specialitate.

**Diagnostic prezumtiv:** [diagnostic fictiv, în scop lingvistic]

**Investigații solicitate:** [listă fictivă, fără interpretare]

**Eliberat la data de:** [dată fictivă]

**Semnătura și parafa:** [fictiv]

---

**対訳(言語の型のみ)**

> **紹介状(抜粋 — 架空データ)**
> **紹介元医師:** [架空氏名]医師、家庭医
> **宛先:** **循環器科**、[架空氏名]医師 御中
> **患者:** [架空氏名] — 個人番号:[架空番号]
> **紹介理由:** 患者は専門的評価のために受診する。
> **推定診断:** [語学目的の架空の診断名]
> **依頼する検査:** [架空の一覧・解釈なし]
> **発行日:** [架空の日付]
> **署名と医師印:** [架空]

**読解解説(言語の型のみ)**

- **Medic trimițător / Către** ― 送る側(家庭医)/送られる先(循環器科・専門医)。⚠ **紹介状はこの二点をまず読む。**
- **Motivul trimiterii** ― 紹介理由(属格 trimiterii)。「なぜ送るか」の欄。
- **Diagnostic prezumtiv** ― ⚠ 推定診断(prezumtiv=まだ確定していない)。**確定と読まない。** 内容の当否は医師へ。
- **Investigații solicitate** ― ⚠ 依頼された検査(solicitat=要請された)。項目の医学的意味は判断しない。
- **Eliberat la data de** ― ⚠ 発行日(eliberat=発行された。liber と別)。

### モデル文書B ― 病欠証明(concediu medical・抜粋)

---

**CERTIFICAT DE CONCEDIU MEDICAL (extras — date fictive)**

**Seria și numărul:** [fictiv]

**Se certifică prin prezenta că** pacientul [nume fictiv] se află în **incapacitate temporară de muncă**.

**Perioada:** de la [dată fictivă] până la [dată fictivă].

**Cod de indemnizație:** [cod fictiv]

**Eliberat de:** dr. [nume fictiv]. **Ștampila și semnătura:** [fictiv].

---

**対訳(言語の型のみ)**

> **病欠証明書(抜粋 — 架空データ)**
> **記号と番号:** [架空]
> **本書により、患者[架空氏名]が一時的労働不能の状態にあることを証明する。**
> **期間:** [架空日付]から[架空日付]まで。
> **給付コード:** [架空コード]
> **発行者:** [架空氏名]医師。**印章と署名:** [架空]。

**読解解説(言語の型のみ)**

- **Se certifică prin prezenta că...** ― 「本書により〜を証明する」。se 受動(B2)の硬い証明定型。⚠ certificat=証明書(資格証ではない)。
- **incapacitate temporară de muncă** ― 一時的労働不能。病欠の根拠となる状態区分。⚠ 状態区分・給付の妥当性は制度の領域で判断しない。
- **Perioada: de la ... până la ...** ― ⚠ **病欠証明はこの期間をまず読む。** 期間が証明の核。
- **Cod de indemnizație** ― 給付コード。⚠ 分類・給付の内容は行政に委ねる。
- **⚠ concediu medical = 病欠**(concediu 単独の「休暇」ではない)。

## 重要ポイント

- **本章は前章[医療文書(C1)](../05-c1/05-medical-documents-c1.md)の深化・拡張。既習の退院サマリー・検査結果票・処方箋とは重複させず、紹介状・診断書・病欠証明・接種記録・検査依頼書を扱う**
- **紹介状(bilet de trimitere)** は「送る側(medic trimițător)→ 送られる先(către)→ なぜ(motivul trimiterii)」で読む。**⚠ diagnostic prezumtiv=推定診断(確定ではない)**
- **診断書(certificat medical)** は「誰が・何を(se certifică că)・何のために(pentru a servi la)証明するか」で読む
- **病欠証明(concediu medical)** は **期間(de la ... până la)** をまず読む。**⚠ concediu 単独は「休暇」、concediu medical は「病欠」**
- **接種記録(schema de vaccinare)** は vaccin / doză / rapel / data administrării を **語として** 読む(接種内容・要否の判断はしない)
- **検査依頼書** は analize solicitate と有効期限をまず読む。⚠ 検査「結果票」(前章)とは別物
- ⚠ 新規の誤読リスク:**concediu(休暇/病欠)、certificat(証明書/≠資格)、prezumtiv(推定/≠確定)、eliberat(発行/≠自由)、solicitat(依頼)、trimitere/rețetă**
- **各文書の医学的・行政的内容の判断は一切しない。必ず医師・薬剤師・行政窓口・有資格の医療通訳者に委ねる**

## 文法メモ ― 本章の文書に頻出する形

| 形 | 意味 | 例 | どこで読むか |
|---|---|---|---|
| **[名詞] + 形容詞(-tor / -tiv / -at)** | 動作主・性質・受動を示す派生形 | trimițător(送る側)/ prezumtiv(推定の)/ solicitat(依頼された) | 紹介状・依頼書 |
| **către + 名詞** | 〜宛・〜へ向けて | Către Serviciul de Cardiologie | 紹介先の明示 |
| **se certifică / se atestă că...** | 〜と証明される(非人称受動の書式) | Se certifică prin prezenta că... | 証明文書の定型 |
| **de la ... până la ...** | 〜から〜まで | de la [X] până la [Y] | 期間の明示 |
| **属格 -ării / -tării / -ului** | 〜の | motivul trimiterii, data administrării | 理由・日付の明示 |
| **pentru a + 不定詞** | 〜するため(目的) | pentru a servi la... | 用途・目的の明示 |

**派生語尾の読み分け** が本章の要である。**-tor**(trimițător=送る側=動作主)、**-tiv**(prezumtiv=推定的な=性質)、**-at**(solicitat / eliberat=依頼された/発行された=受動)の語尾は、語の役割を示す手がかりになる。⚠ ただし、語尾から役割の見当をつけても、**内容の医学的・行政的意味は判断せず**、確信が持てなければ専門家・辞書に確認する(前章と共通の鉄則)。

| **検査依頼書**と**検査結果票**を同じと読む | 依頼書=これから受ける / 結果票=受けた結果(前章) | 依頼(recomandare)と結果(rezultate)は別物 ⚠ |
| 接種記録の doza / rapel を見て自分で「次に何を受けるか」判断する | **語として読むにとどめ、接種の要否・時期は医師・公的案内へ** | 接種スケジュールの医学的判断は本章の範囲外(免責) |

> **⚠ 本章の新規誤読リスク語(集約)**:上の「よくある誤用」で扱った **concediu(休暇)/ concediu medical(病欠)・certificat(証明書、≠資格証)・prezumtiv(推定、≠確定)・eliberat(発行、≠自由)・trimitere/rețetă(送付/薬局)・solicitat(正式な依頼)** は、いずれも語の区別を示すにとどまり、医学的・行政的判断はしない。全カテゴリ横断の一覧は [日羅医療用語集](./japanese-romanian-medical-glossary.md) の ⚠ 早見表を参照。

## ネイティブらしい理解のコツ

| つまずき | 読解のコツ |
|---|---|
| 派生語尾(-tor/-tiv/-at)で役割が読めない | -tor=動作主(trimițător)、-tiv=性質(prezumtiv)、-at=受動(solicitat/eliberat)を手がかりに |
| 文書種別が多くて混乱する | **trimitere(送る)系**(紹介状・検査依頼書)と **certificat(証明する)系**(診断書・病欠)で二分すると整理しやすい |
| どこから読めばいいか分からない | 各文書の「最初に読む箇所」(紹介状=către+motivul / 病欠=期間 / 依頼書=analize solicitate)から入る |
| concediu / certificat / prezumtiv で意味を取り違える | 語形の近い日常語・英語に引かれず、医療・行政文脈の語義で読む |
| 接種記録の語を見て次を判断したくなる | **語として読むにとどめ、接種の要否は医師・公的案内へ**(本章の原則) |
| 意味に確信が持てない | **推測で確定せず、医師・薬剤師・行政窓口・有資格の医療通訳者・辞書に確認する** |

最後のコツが最も重要である。医療・行政文書は、**読み違いが人の健康・権利・手続きに直結する** 領域である。前章[医療文書(C1)](../05-c1/05-medical-documents-c1.md)で「分からなければ医師・薬剤師・有資格の医療通訳者に確認する」と学んだのと同じ姿勢を、本章の文書群にも徹底する。本章はその確認を **語学の側から支える** ものであって、確認を **省くための** ものではない。

## 文化的注意点

- 一般に、ルーマニアの医療・行政文書は、**ラテン語系の硬い書式語(se certifică prin prezenta, subsemnatul, eliberat)と定型構造** を持つ傾向があるとされる(あくまで一般的傾向で、文書・機関・電子化の程度による差はある)。電子化(接種記録の電子化、電子紹介など)で書式は変わりうるため、本章は特定の様式を規定するものではない。
- 文書に **CNAS / asigurat / card de sănătate / CNP** といった語が現れたら「保険・行政に関わる文書だ」と **言語的に認識** し、制度の実務(要否・手続き・効力)は一次情報・専門家に委ねる([overview](./overview.md) の骨格を参照)。
- 医療文書の内容の解釈・実行(診断の理解、病欠期間の扱い、接種の要否、検査の実行)は、**必ず資格を持つ医療専門家・行政窓口に委ねる。** 家族への橋渡しが必要な重大な場面では、有資格の医療通訳者を介することが望ましい(通訳の技術・原則は [医療通訳の言語運用](./medical-interpretation.md))。**本章は言語(構造・定型・略号・語彙)の読解を扱うものであり、医学的・行政的判断を代替するものではない**(強化免責の再確認)。

## 練習問題(読解のみ・医学的/行政的判断を求めない)

### 問題1: 文書種別を機能で分類せよ

次の文書を「**送る(trimitere)系**」と「**証明する(certificat)系**」に分けよ。
1. bilet de trimitere 2. certificat medical 3. concediu medical 4. recomandare de investigații

### 問題2: 紹介状の「最初に読む箇所」

紹介状(bilet de trimitere)で、文書の骨組みを掴むためにまず読むべき二つの欄を答えよ。

### 問題3: ⚠ 誤読リスク ― 正しい読みを選べ

1. **concediu medical** は(休暇 / 病欠)。
2. **diagnostic prezumtiv** は(確定診断 / 推定診断)。
3. **certificat medical** は(医療資格証 / 診断書)。
4. **eliberat la data de** の eliberat は(自由な / 発行された)。

### 問題4: 病欠証明の核

病欠証明(concediu medical)で、証明の核として **まず読むべき** 箇所は何か。ルーマニア語の定型で答えよ。

### 問題5: 接種記録の語彙

次の語の読解上の対応語を答えよ(接種の要否・時期は判断しない)。
1. **vaccin** 2. **doză** 3. **rapel** 4. **data administrării**

### 問題6: 検査依頼書と結果票の区別

**bilet de trimitere pentru analize(検査依頼書)** と、前章で学んだ **rezultatele analizelor(検査結果票)** の違いを、読解上の位置づけで1文で説明せよ。

## 解答

### 問題1
**送る(trimitere)系:** 1. bilet de trimitere(紹介状)/ 4. recomandare de investigații(検査依頼書)。
**証明する(certificat)系:** 2. certificat medical(診断書)/ 3. concediu medical(病欠証明)。
— trimitere(送る)と certificat(証明)の二機能で文書群を整理できる。

### 問題2
**Către(誰宛か=紹介先の診療科・医師)** と **Motivul trimiterii(紹介理由=なぜ送るか)**。
— この二点で「何のための誰への紹介か」が掴める。診断名・検査は枠組みを掴んだ後に読む。⚠ 内容の医学的意味は医師へ。

### 問題3
1. **病欠**(concediu 単独は休暇だが、medical が付くと病欠) 2. **推定診断**(prezumtiv=まだ確定していない ⚠) 3. **診断書**(certificat=証明書。資格証ではない ⚠) 4. **発行された**(a elibera=発行する。liber と別 ⚠)。
— すべて本章 ⚠ の要点。語の区別を示すにとどまり、内容の判断はしない。

### 問題4
**Perioada:(de la ... până la ...)= 有効期間(休業の期間)**。
— 病欠証明は「いつからいつまでの休業か」がまさに証明の核。⚠ 期間・給付の妥当性は行政・制度に委ねる。

### 問題5
1. ワクチン 2. 接種回・用量(prima doză=1回目) 3. 追加接種・ブースター 4. 接種日。
— いずれも **語として** 読むにとどめる。⚠ どの接種をいつ受けるべきか・記録が完全かは本章では判断しない(免責。医師・公的案内へ)。

### 問題6
**検査依頼書(recomandare / bilet de trimitere pentru analize)は「これから受ける検査を依頼する」文書、検査結果票(rezultatele analizelor、前章)は「受けた検査の結果」を示す文書で、依頼と結果という別の段階の文書である。**
— ⚠ analize solicitate(依頼された検査)と rezultate(結果)を取り違えない。検査の実行・結果の解釈は医師・検査機関へ。

## 次章への橋渡し

前章[医療文書(C1)](../05-c1/05-medical-documents-c1.md)の退院サマリー・検査結果票・処方箋に続き、本章では **紹介状・診断書・病欠証明・接種記録・検査依頼書** という、受診の流れを動かし事実を証明する文書を、**言語の型に厳格に限定して** 読解した。C1・C2 の高リスク領域を通じて貫いた原則——**言語の型は読み解くが、内容の判断(医学的・行政的)は専門家に委ねる**——は、本章でも背骨であり続けた。医療セクションの締めくくりとして、次の[日羅医療用語集](./japanese-romanian-medical-glossary.md)では、本書の医療系各章に登場した語彙を **カテゴリ別に集約** し、各章で確立した ⚠ 誤訳リスクを一覧として参照できる形にまとめる。個々の章で学んだ語と警告を、一つの参照表として手元に置けるようにするのが、その用語集の役割である。
