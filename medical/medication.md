# 薬局での会話と処方の読解【CEFR B1】

> **免責**: 本章は語学学習を目的とした教材であり、医学的・法的助言ではない。
> 実務で使用する場合は、必ず資格を持つ専門家およびネイティブ話者の確認を経ること。
> 症状の自己判断・自己治療の指針として本章を用いてはならない。**薬の選択・用法・用量・服薬の可否は、必ず薬剤師(farmacist)・医師の判断に委ねること。**本章は用法用量の「文字を読む」語学技能を扱うにとどまり、その用量が適切かどうかは一切判断しない。体調不良や緊急時には、必ず医療機関・救急サービス(112)に連絡すること。

## この章の目標

この章を終えると、次のことができるようになる。

- 薬の**形状**(comprimat / capsulă / sirop / unguent / picături)を語彙として言い分けられる
- 処方箋・箱の**用法用量表記**(de X ori pe zi / înainte de masă / după masă / câte X)を**読解**できる
- **処方薬**(cu rețetă)と**市販薬**(fără rețetă)の区別を尋ね、聞き取れる
- 薬剤師に**副作用**の有無(Are efecte secundare?)や飲み合わせ・保管を尋ねられる
- 薬局での相談を、A2 の「頭痛に効くものはありますか」から一歩進めた**能動的なやり取り**にできる
- 誤訳リスクの高い語(a lua と a administra、o dată と odată、comprimat と compres など)を区別できる

- [医療の全体像(overview)](./overview.md) — 受診と薬局の位置づけ
- 数(numerale)・時間表現・前置詞 de / înainte de / după(本編 A2〜B1)。用法用量は数と前置詞の塊で書かれる

A2 では薬局で「頭痛に効くものはありますか(Aveți ceva pentru durere de cap?)」と**尋ねる**ところまでを学んだ。B1 では、薬剤師に**副作用を確認し・用法を聞き返し・箱や処方箋の表記を読み解く**——つまり受け身の購入から、能動的に情報を確かめる相談へと比重を移す。薬の用法用量は誤読が実害に直結する高リスク領域であり、本章では ⚠ を多めに置く。ただし繰り返すが、**本章が扱うのは「表記を読む」語学技能だけ**であり、その薬・用量が適切かは薬剤師・医師に委ねる。

## 本文解説

### 薬の形状(forme de prezentare)

同じ薬効成分でも、剤形(薬の形)によって呼び名と用法が変わる。薬局で「どの形のものか」を言えると相談が正確になる。以下は日常的に出会う主要な剤形である。

- **comprimat** /kom.priˈmat/(名中、複 comprimate)= 錠剤(押し固めた固形)。**pastilă** /pasˈti.lə/(名女)= 錠剤・タブレットのやや口語的な言い方。⚠ comprimat と pastilă はほぼ同義で使われるが、処方・箱の表記では **comprimat** が標準。
- **capsulă** /kapˈsu.lə/(名女、複 capsule)= カプセル。
- **sirop** /siˈrop/(名中)= シロップ(液状。小児や咳止めに多い)。**Un sirop de tuse.**(咳止めシロップ)。
- **unguent** /unˈɡwent/(名中)= 軟膏(塗り薬)。**cremă** /ˈkre.mə/(名女)= クリーム、**gel** /d͡ʒel/ = ジェル。⚠ 塗り薬は「飲む」薬ではない(後述の誤用)。
- **picături** /pi.kəˈturʲ/(名女複)= 点滴薬・滴剤。**picături de ochi / de nas**(目薬/点鼻薬)。単数 picătură は「一滴」。
- **supozitor** /su.po.ziˈtor/(名中)= 坐薬。**plasture** /ˈplas.tu.re/(名男)= 貼り薬・絆創膏。**spray** /sprej/ = スプレー。
- **fiolă** /fiˈo.lə/(名女)= アンプル、**injectabil** /in.ʒekˈta.bil/ = 注射剤(形容詞的にも)。⚠ 注射剤は医療者が扱うもので、薬局での相談語彙としては「そういう剤形がある」ことを識別できれば十分。

剤形を尋ねる・言う型:

- **Este comprimat sau sirop?** ― 錠剤ですか、シロップですか?
- **Aveți varianta sirop?** ― シロップのタイプはありますか?(variantă = 種類・タイプ)
- **Pentru copii, aveți sirop?** ― 子ども用にシロップはありますか?

### 用法用量の読解(posologie)― 頻度・タイミング・配分

薬の箱・処方箋・薬剤師の説明には、**いつ・何回・いくつ**飲むかが決まった型で書かれる。この型を**読める**ことが本章の核である。⚠ 読めることと、その量が自分に適切かは別問題であり、後者は薬剤師・医師に確認する。

**① 頻度 ― de X ori pe zi(1日X回)**

- **o dată pe zi** /o ˈda.tə pe zi/ = 1日1回(o dată = 1回)⚠ **odată**(かつて・いったん)と綴りを区別(後述)
- **de două ori pe zi** = 1日2回
- **de trei ori pe zi** = 1日3回
- **la fiecare 8 ore** /la fieˈka.re opt ˈo.re/ = 8時間ごとに(la fiecare + 時間 = 〜ごとに)
- **la nevoie** /la neˈvo.je/ = 必要時に(頓服)⚠ 「必要なとき」で、定時服用ではない

**② タイミング ― 食事との関係**

- **înainte de masă** /ɨ.naˈin.te de ˈma.sə/ = 食前(înainte de = 〜の前)
- **după masă** /ˈdu.pə ˈma.sə/ = 食後(după = 〜の後)⚠ după masă は「昼食後/午後」の意もあり文脈依存
- **în timpul mesei** /ɨn ˈtim.pul ˈme.sei/ = 食事中に(mesei = masă の属格)
- **pe stomacul gol** /pe stoˈma.kul ɡol/ = 空腹時に(直訳「空の胃で」)
- **dimineața / seara / la culcare** = 朝/夜/就寝時に(la culcare = 寝る前に)

**③ 配分 ― câte(1回あたりいくつ)**

**câte** /ˈkɨ.te/ は「1回・1人・1個あたり」の**配分**を表す語で、用量表記の要である。日本語の助数詞的な「〜ずつ」に近い。

- **câte un comprimat de trei ori pe zi** = 1回1錠を1日3回(câte un = 1つずつ)
- **câte două capsule dimineața** = 朝2カプセルずつ
- **câte 10 ml de două ori pe zi** = 1回10 mlを1日2回(シロップ等の液量)
- **câte o linguriță** /ˈkɨ.te o lin.ɡuˈri.t͡sə/ = 小さじ1杯ずつ(linguriță = 小さじ。lingură = 大さじ)

これらを組み合わせた完全な表記の例(架空):**Câte un comprimat de două ori pe zi, după masă.**(1回1錠を1日2回、食後に)。読解では **câte(1回量)+ de X ori pe zi(頻度)+ タイミング** の3ブロックに分解して読む。

**④ 期間・その他**

- **timp de 7 zile** = 7日間(timp de + 期間)
- **cură de 10 zile** = 10日間の服用コース(cură = 一連の治療・服用期間)
- **a se agita înainte de utilizare** = 使用前によく振ること(シロップ等。se 受動の非人称指示)
- **a nu se depăși doza** = 用量を超えないこと ⚠ doza = 用量。この否定形の非人称指示は箱に頻出

### 処方薬と市販薬(cu rețetă / fără rețetă)

- **medicament cu rețetă** = 処方箋が必要な薬(cu = 〜付きで)。**cu prescripție medicală** とも(prescripție = 処方)。
- **medicament fără rețetă**(OTC)= 処方箋なしで買える市販薬(fără = 〜なしで)。**eliberat fără prescripție** = 処方箋なしで交付される、と箱・案内に書かれることがある。
- **compensat** /kom.penˈsat/ = (公的保険で)一部負担・補助される。⚠ 「補償」の一般語だが、薬の文脈では「保険で価格が補助される」の意。**Este compensat?**(保険適用ですか)と尋ねられるが、**補助の要件・割合は制度の領域**であり本章では扱わない。

尋ねる型:

- **Se eliberează fără rețetă?** ― 処方箋なしで買えますか?(a se elibera = 交付される)
- **Am nevoie de rețetă pentru asta?** ― これには処方箋が要りますか?
- **Aveți un echivalent fără rețetă?** ― 処方箋なしの同等品はありますか?(echivalent = 同等品)⚠ 代替薬の可否は薬剤師の判断

### 副作用・注意事項を確認する(efecte secundare)

B1 の相談の核は、買う前に**確認する**ことである。副作用・飲み合わせ・保管を尋ねる定型を持っておく。

- **Are efecte secundare?** /ˈa.re eˈfek.te se.kunˈda.re/ ― 副作用はありますか?(efecte secundare = 副作用、直訳「二次的効果」)⚠ efecte adverse(有害事象)ともいうが、相談では secundare が一般的
- **Provoacă somnolență?** /proˈvo̯a.kə som.noˈlen.t͡sə/ ― 眠気が出ますか?(a provoca = 引き起こす、somnolență = 眠気)
- **Pot să iau asta cu...?** ― これを〜と一緒に飲んでも大丈夫ですか?(飲み合わせ)
- **Se poate lua cu alcool?** ― アルコールと一緒に飲めますか?(⚠ 実際の可否は必ず薬剤師へ)
- **Cum se păstrează?** /kum se pəsˈtre̯a.zə/ ― どう保管しますか?(a se păstra = 保管される)。**la frigider**(冷蔵)/ **la loc uscat**(乾燥した場所)。
- **Este contraindicat în sarcină?** /kon.tra.in.diˈkat ɨn ˈsar.t͡ʃi.nə/ ― 妊娠中は禁忌ですか?(contraindicat = 禁忌の、sarcină = 妊娠)⚠ 判断は必ず医師へ

薬剤師の説明を聞き取る型(受信):

- **Se ia câte un comprimat la 8 ore.** ― 8時間ごとに1錠ずつ服用します。
- **Nu se ia pe stomacul gol.** ― 空腹時には飲みません。
- **Dacă apar reacții, întrerupeți și consultați medicul.** ― 反応が出たら中止して医師に相談してください(a întrerupe = 中断する)。

### 薬剤師への相談定型(まとめのチャンク)

薬局での相談を一続きにする定型を、丸ごと覚える**チャンク**として持つ。

- **Ce îmi recomandați pentru...?** ― 〜には何をお勧めしますか?(recomandați = a recomanda 敬称)
- **Cum se administrează?** /kum se ad.mi.nisˈtre̯a.zə/ ― どのように使用/服用しますか?⚠ a administra = 「投与する」。飲み薬にも塗り薬にも使える広い語(後述の誤用)
- **De câte ori pe zi?** ― 1日何回ですか?
- **Câte odată?**(= câte o dată)― 1回にいくつですか?⚠ ここは「1回あたり」の意
- **Îmi puteți explica cum se ia?** ― 飲み方を説明してもらえますか?

## 重要ポイント

- 剤形:**comprimat**(錠)/ **capsulă**(カプセル)/ **sirop**(シロップ)/ **unguent**(軟膏)/ **picături**(点滴薬)。塗り薬・坐薬は「飲む」ものではない
- 頻度:**de X ori pe zi**(1日X回)/ **la fiecare X ore**(X時間ごと)/ **la nevoie**(頓服)
- タイミング:**înainte de masă**(食前)/ **după masă**(食後)/ **pe stomacul gol**(空腹時)
- 配分:**câte + 数**(1回量。câte un comprimat = 1回1錠)。linguriță(小さじ)/ lingură(大さじ)
- **cu rețetă**(処方薬)⇔ **fără rețetă**(市販薬)。**Se eliberează fără rețetă?** で確認
- 確認:**Are efecte secundare? / Provoacă somnolență? / Cum se păstrează?**
- ⚠ 読めることと用量が適切かは別。用法・用量・可否は必ず薬剤師・医師へ

## 文法解説 ― 用法指示の se(非人称の se)

薬の指示文は、**「誰が」を言わずに「〜される・〜すること」**という非人称の形で書かれることが多い。これは既習の **se 受動/非人称の se** の応用である。

- **Se ia câte un comprimat.**（1錠ずつ服用する)― 直訳「1錠ずつ飲まれる」。主語(飲む人)を明示せず、指示だけを述べる。
- **Se administrează de două ori pe zi.**（1日2回投与する)― a administra の se 形。
- **A nu se depăși doza recomandată.**（推奨用量を超えないこと)― **a nu se + 不定詞** は箱の注意書きに頻出する否定の非人称指示。命令より硬く、印刷指示の定型。
- **A se agita înainte de utilizare.**（使用前に振ること)― 同じ **a se + 不定詞** の肯定形。

読解の要点:指示文の **se** は「自分で〜する」の再帰ではなく、**「一般に〜される/〜すべし」**の非人称である。⚠ **Se ia...** を「自分を飲む」と読まない。主語のない指示として読む。この点は誤訳リスク語の項でも扱う。

## よくある誤用

| ❌ 誤り | ✅ 正しく | 説明 |
|---|---|---|
| Iau un unguent.(軟膏を「飲む」) | **Aplic un unguent.** / **Folosesc un unguent.** | unguent は塗り薬。a lua(飲む)ではなく a aplica(塗る)/ a folosi(使う)⚠ |
| Se ia câte un comprimat.(「自分を飲む」と解釈) | (指示として)**1錠ずつ服用する** | se は非人称の指示。「自分を飲む」ではない ⚠ |
| o dată を odată と書く | **o dată pe zi**(1日1回) | o dată=1回(頻度)/ odată=かつて・いったん。綴りで意味が変わる ⚠ |
| de doi ori pe zi | **de două ori pe zi** | ori(回)は女性扱い。数詞は女性形 două |
| Am nevoie de o pastilă?(処方箋の要否を尋ねたい) | **Am nevoie de rețetă?** | pastilă=錠剤。処方箋は rețetă。A2 の ⚠ を継続 |
| Cât de multe comprimate?(1回量を尋ねたい) | **Câte comprimate odată?** | 配分は câte。cât de mult は「どのくらい多い」で用法の型に合わない |

## ⚠ 誤訳・混同に注意する語

薬の語は取り違えると用法・剤形の誤りに直結する。B1 で特に注意すべき点を挙げる。いずれも**確信を持って区別できる**ものに絞る。

- **⚠ a lua(飲む・服用する)と a administra(投与する・使用する)** ― **a lua un medicament** は「薬を(自分で)飲む・服用する」、**a administra** は「(薬を)投与する・使用する」で、飲み薬にも塗り薬にも注射にも使える広い語である。薬剤師の説明 **Cum se administrează?**(どう使うか)は剤形を問わず使えるが、患者が自分の服用を言うときは **Când îl iau?**(いつ飲むか)のように a lua が自然。塗り薬を a lua で言うと「飲む」ことになってしまう(上の誤用参照)。剤形に応じて a lua(飲む)/ a aplica(塗る)/ a administra(広く投与・使用)を使い分ける。
- **⚠ o dată(1回)と odată(かつて・いったん)** ― 分かち書きの **o dată** は「1回」(o dată pe zi=1日1回)という**頻度**、続けて書く **odată** は「かつて・いったん・同時に」という**時の副詞**である。綴りが1字の空白だけの違いで意味が全く変わる。用法用量では必ず分かち書きの **o dată**(1回)を使う。**odată pe zi** と書くと語として崩れる。読解でも、頻度の文脈なら o dată と読む。
- **⚠ comprimat(錠剤)と compresă / a comprima(湿布・圧迫する)** ― **comprimat** /kom.priˈmat/ は名詞「錠剤」、**compresă** /komˈpre.sə/ は「湿布・パッド」(冷湿布 compresă rece など)、動詞 **a comprima** は「圧縮する・圧迫する」である。語源は同じ(押し固める)だが指すものが違う。「錠剤」を compresă と言うと湿布になってしまう。薬局では剤形の **comprimat** を正確に。
- **⚠ doză(用量)と doză(缶)** ― **doză** /ˈdo.zə/ は薬の文脈では「用量・投与量」(doza zilnică=1日量)だが、同じ綴りで「缶」(o doză de bere=ビール1缶)も意味する。薬局・処方の文脈では常に「用量」と読む。**a nu se depăși doza**(用量を超えないこと)を「缶を超えない」と読まない。文脈で確実に判別できるが、初学者は面食らいやすい。
- **⚠ efecte secundare(副作用)を「二の次の効果」と軽く読まない** ― 直訳は「二次的効果」だが、意味は**副作用**であり、軽視してよいという含意ではない。**Are efecte secundare?**(副作用はありますか)は重要な確認である。「secundare だから大したことない」と読み違えないこと。副作用の内容・程度の判断は必ず薬剤師・医師へ。
- **⚠ recomandat(推奨)と obligatoriu(必須)の混同** ― 箱の **doza recomandată** は「推奨用量」で、**自己判断で増やしてよい**という意味では**ない**。「推奨=目安だから加減してよい」と誤読すると過量服用の危険がある。recomandat の語感に引きずられず、用量は指示どおりに、疑問は薬剤師・医師に確認する(内容判断は本章の外)。

## ネイティブらしい言い換え

| 教科書的 | ネイティブが実際によく言う形 | 注 |
|---|---|---|
| Aveți un medicament pentru durere de cap? | **Aveți ceva de cap?** / **Ceva pentru cap?** | 短く「頭に効くもの、何かある?」が口語的。A2 のチャンクのくだけた形 |
| De câte ori pe zi trebuie să iau? | **Cum îl iau?** | 「これ、どう飲むの?」の一言で用法全体を尋ねられる |
| Care sunt efectele secundare? | **Are ceva reacții?** | reacții(反応)で副作用を軽く尋ねる口語 |
| Acest medicament provoacă somnolență. | **Te ia somnul de la el.** | 「これで眠くなる」の口語的な言い方(a lua somnul=眠気に襲われる) |
| Trebuie luat înainte de masă. | **Se ia pe stomacul gol.** | 空腹時=食前を、実感のある「空の胃で」で言う |

## フォーマル/カジュアルの違い

薬局では基本 **丁寧(dumneavoastră 系)**。家族・友人に薬の話をするときは tu 系になる。

- **薬剤師に(丁寧)**: Bună ziua. Ce îmi recomandați pentru durere de gât? Are efecte secundare? Se eliberează fără rețetă?
- **友人に(くだけた)**: Ai ceva de cap? Mă doare rău.(頭に効くもの持ってない?すごく痛い)/ Ia o pastilă, dar nu pe stomacul gol.(薬飲みなよ、でも空腹時はダメ)
- 薬剤師の説明は丁寧形の指示: **Se ia câte un comprimat...** / **Nu depășiți doza.**(用量を超えないでください、敬称命令)

迷ったら丁寧形。ただし用法用量の**中身**(câte / de X ori / după masă)は、レジスターにかかわらず正確に聞き取ることが最優先である。分からなければ **Îmi puteți repeta cum se ia?**(飲み方をもう一度お願いできますか)と確認する。

## 文化的注意点

- **薬局は最初の相談窓口** ― A2 で見たとおり、ルーマニアでは軽い不調はまず薬局(farmacie)で薬剤師に相談するのが一般的で、市販薬(fără rețetă)をその場で求められることが多い。24時間営業(farmacie non-stop)もある。B1 の相談力は、この習慣の中で「買う」から「確かめて買う」へ踏み込むためのものである。ただし本章の免責のとおり、薬・用量の最終判断は薬剤師・医師に委ねる。
- **保険補助(compensat)は制度の領域** ― 一部の処方薬は公的保険で価格が補助される(compensat)仕組みがあるとされるが、対象・割合・要件は制度として複雑で変化もするため、本章では扱わない(制度面は `overview.md` 参照)。薬局で **Este compensat?**(保険適用ですか)と尋ねる言語は学べるが、その適否の判断は制度・専門家に委ねる。
- **用量表記は必ず読み合わせる** ― 箱・処方箋の用法用量は、桁や単位(mg / ml)、頻度(de X ori)を取り違えると危険である。分からなければ薬剤師に **Îmi confirmați doza?**(用量を確認してもらえますか)と読み合わせる習慣が、高リスク領域では大切である(数値確認の厳密な言語運用は C2 第6章で深める)。

## 練習問題

### 問題1: 剤形を答えよ(ルーマニア語で)

1. 錠剤 2. カプセル 3. シロップ 4. 軟膏(塗り薬) 5. 点滴薬・滴剤

### 問題2: 用法用量を読解せよ(日本語に)

1. **Câte un comprimat de două ori pe zi, după masă.**
2. **Câte 10 ml de trei ori pe zi, timp de 5 zile.**
3. **Un comprimat la nevoie, dar nu mai mult de doza recomandată.**
4. **Se ia pe stomacul gol, dimineața.**

### 問題3: ⚠ 区別 ― 正しい語を選べ

1. 1日1回(頻度):(o dată / odată)pe zi。
2. 軟膏を肌に「塗る」:(Iau / Aplic)un unguent。
3. 錠剤:(comprimat / compresă)。
4. 「これはどう使いますか」(薬剤師へ):Cum se(administrează / mănâncă)?

### 問題4: 薬剤師への相談 ― ルーマニア語で書け

1. 「のどの痛みには何をお勧めしますか?」
2. 「副作用はありますか?」
3. 「処方箋なしで買えますか?」
4. 「1日何回、何錠ずつ飲みますか?」

### 問題5: 対話を完成させよ(客と薬剤師)

客: Bună ziua. (「せきに効くものはありますか、シロップで」と尋ねる)
薬剤師: Da, avem. Se ia câte o linguriță de trei ori pe zi.
客: (「副作用はありますか?眠気は出ますか?」と確認する)

## 解答

### 問題1
1. **comprimat**(または pastilă) 2. **capsulă** 3. **sirop** 4. **unguent**(または cremă / gel) 5. **picături**。
― 剤形は用法と結びつく。塗り薬(unguent)は「飲む」ものではない点に注意(⚠)。

### 問題2
1. **1回1錠を1日2回、食後に。** 2. **1回10 mlを1日3回、5日間。** 3. **必要時に1錠、ただし推奨用量を超えないこと。** 4. **空腹時に、朝服用する。**
― câte(1回量)+ de X ori pe zi(頻度)+ タイミング/期間の3〜4ブロックに分解して読む。la nevoie=頓服、pe stomacul gol=空腹時。⚠ 実際の服用可否は薬剤師・医師へ。

### 問題3
1. **o dată**(1回=頻度。odată は「かつて」) 2. **Aplic**(軟膏は塗る。a lua=飲む) 3. **comprimat**(錠剤。compresă は湿布) 4. **administrează**(a se administra=どう使うか。a mânca=食べる、は不適)。
― すべて ⚠ 混同注意そのもの。o dată/odată、a lua/a aplica、comprimat/compresă を分ける。

### 問題4
1. **Ce îmi recomandați pentru durere de gât?** 2. **Are efecte secundare?** 3. **Se eliberează fără rețetă?**(または Am nevoie de rețetă?) 4. **De câte ori pe zi și câte comprimate odată?**
― 相談は「何を勧めるか→副作用→処方箋の要否→用法」の順で組める。odată=「1回に」の意(=o dată)。

### 問題5
客(1回目): **Aveți ceva pentru tuse, sirop?**(せきに効くもの、シロップはありますか)
客(2回目): **Are efecte secundare? Provoacă somnolență?**(副作用はありますか?眠気は出ますか?)
― 剤形(sirop)を指定して尋ね、買う前に副作用を確認するのが B1 の相談の型。linguriță=小さじ。

- [ ] 剤形(comprimat / capsulă / sirop / unguent / picături)を言い分けられる
- [ ] 用法用量(câte + de X ori pe zi + înainte/după masă)を読解できる
- [ ] cu rețetă(処方薬)と fără rețetă(市販薬)を区別し、Se eliberează fără rețetă? で尋ねられる
