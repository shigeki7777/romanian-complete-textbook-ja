# 緊急時の対応 ― 112 と救急外来【CEFR B1〜B2】

> **免責**: 本章は語学学習を目的とした教材であり、医学的・法的助言ではない。
> 実務で使用する場合は、必ず資格を持つ専門家およびネイティブ話者の確認を経ること。
> **本章は緊急時に「状況を言葉で伝える」語学技能のみを扱い、応急手当・救命処置などの医学的な手当ての指示は一切含まない。**手当ての方法は、通報時に **112 のオペレーターの指示** に従い、資格を持つ救急・医療従事者に委ねること。体調不良・緊急時には迷わず **112** に連絡すること。

## この章の目標

この章を終えると、次のことができるようになる。

- 緊急通報 **112** で、最初に言うべき情報を**正しい順序**(場所→状態→人数)で組み立てられる
- オペレーターの定番質問(**Ce s-a întâmplat? / Unde vă aflați? / Câți răniți sunt?**)を聞き取り、答えられる
- 救急外来 **UPU**(Unitatea de Primiri Urgențe)まわりの語彙(triaj / brancardă / ambulanță 等)を識別できる
- 緊急を要する症状表現(意識・呼吸・出血・胸痛など)を、和文と対照して言える
- **付き添い者**として、本人に代わって状況・既往・服薬を説明できる
- 薬・アレルギー・持病を**即答するための携帯カード**を、定型文で用意できる

## 前提知識

- [A2 第7章:医療の基本場面](../02-a2/07-basic-medical-situations.md) — **112 / Chemați o ambulanță! / Am nevoie de ajutor! / Este o urgență.** は既習。本章はこの緊急チャンクを、**通報の構造**と**救急外来の場面**へ広げる
- [症状の表現(symptoms)](./symptoms.md) — 症状を描写する B1 語彙。緊急時はこれを**最短で・最重要から**伝える技能に組み替える
- [薬局での会話(medication)](./medication.md) — 服薬・アレルギー情報。携帯カードの中身になる
- [身体部位・臓器の語彙(anatomy)](./anatomy.md) — 「どこが」を言う部位語彙
- 場所表現・命令形(敬称命令)・複合過去(本編 A2〜B1)。通報は「どこで・何が起きたか」を言う

A2 では緊急チャンク(Chemați o ambulanță!)を**丸暗記**した。B1〜B2 では、それを実際の通報の流れ——**オペレーターが何を聞き、こちらが何をどの順で答えるか**——として運用し、さらに救急外来での語彙、付き添い者としての説明、携帯カードの作成まで広げる。緊急時の言語は、正確さと**速さ・優先順位**が命に関わる高リスク領域であり、⚠ を多めに置く。ただし繰り返すが、**手当ての方法は本章では扱わない**。伝えることに徹する。

## 本文解説

### 緊急番号 112 ― 基本の確認

ルーマニアの緊急通報番号は **112** である。112 は EU 共通の単一緊急番号で、EU 域内の固定・携帯電話から**無料**でかけられ、警察・救急・消防につながる。ルーマニアは 112 を**唯一の国内緊急番号**として採用している(出典: European Commission, digital-strategy.ec.europa.eu/en/policies/112, 2026年時点)。モルドバ共和国も **112** を国内統一の緊急番号(救急・警察・消防)として採用している(出典: Serviciul Național Unic 112 公式 112.md, 参照 2026-07)。旅行者・在住者ともに、覚える番号はこの1つでよい。

- 番号は **unu, unu, doi**(ウヌ ウヌ ドイ)と**1桁ずつ**言うのが最も誤解がない(A2 既習)。
- **Sunați la 112!** /suˈnat͡sʲ la o su.nə do.ˈis.pre.ze.t͡ʃe/ ― 112に電話して!
- 通報は落ち着いて、**はっきり・ゆっくり**話す。⚠ 焦って早口になると住所が伝わらない。

### 通報の構造 ― 何を最初に言うか(場所→状態→人数)

緊急通報で最も大切なのは、**情報の順序**である。救助を送るために、オペレーターはまず**どこか**を知る必要がある。そこで本章では、伝える順序を **①場所 → ②何が起きたか(状態)→ ③人数** の骨格として覚える。⚠ この順序は「救助隊を向かわせる」ために場所を最優先する実務的な組み立てであり、症状を語る前に**まず場所**を言えることが要である。

**① 場所(Unde ― 最優先)**

- **Sunt pe strada [名] numărul [数], în [都市].** ― [都市]の[通り名][番地]にいます。
- **Sunt lângă [目印].** /ˈlɨn.ɡə/ ― [目印]の近くにいます(lângă = 〜のそばに)。
- **Nu știu exact adresa, dar sunt vizavi de [目印].** ― 正確な住所は分かりませんが、[目印]の向かいにいます(vizavi de = 〜の向かい)。
- **Este la etajul [数].** ― [数]階です(場所の補足)。

**② 何が起きたか(状態)**

- **A leșinat o persoană.** /a leˈʃi.nat/ ― 人が意識を失いました(a leșina = 気を失う)。
- **Cineva nu respiră.** ― 誰かが呼吸していません(a respira = 呼吸する)。
- **Este un accident de mașină.** ― 自動車事故です。
- **O persoană are dureri mari în piept.** ― 胸を強く痛がっている人がいます。

**③ 人数(Câți ― 負傷者の数)**

- **Este o persoană rănită.** /rəˈni.tə/ ― 負傷者が1人います(rănit = 負傷した)。
- **Sunt doi răniți.** ― 負傷者が2人います。
- **Sunt mai mulți răniți.** ― 複数の負傷者がいます。

通報の骨格を一続きにした例(架空):**Sunt pe strada Florilor numărul 10, în București. O persoană a leșinat și nu respiră. Este o singură persoană rănită.**(ブカレスト、フロリロル通り10番地にいます。人が意識を失って呼吸していません。負傷者は1人です)。⚠ **最初に場所**、次に状態、次に人数。この順で言えるよう練習する。

さらに重要:**電話を先に切らない**。オペレーターの指示を待つのが原則である。

- **Ce trebuie să fac până vine ambulanța?** ― 救急車が来るまで何をすべきですか?
- **Rămân la telefon.** ― 電話を切らずにいます(rămân = とどまる)。

⚠ **手当ての具体的な方法は、オペレーターの指示に従う。**本章は「何をすべきか尋ねる/指示を聞き取る」言語だけを扱い、手当ての内容は書かない。

### オペレーターの質問を聞き取り、答える

112 のオペレーターは決まった質問をする。**聞き取れる**うえで、**応答例とセット**で持っておく。

- **Ce s-a întâmplat?** /t͡ʃe sa ɨn.tɨmˈplat/ ― 何がありましたか?(a se întâmpla = 起こる)
  - 応答: **A leșinat o persoană și nu răspunde.**(人が意識を失って反応しません)
- **Unde vă aflați? / Care este adresa?** /ˈun.de və aˈfla.t͡sʲ/ ― どこにいますか?/住所は?(a se afla = いる・所在する)
  - 応答: **Sunt pe strada..., numărul..., în...**
- **Câți răniți sunt?** /kɨt͡sʲ rəˈni.t͡sʲ sunt/ ― 負傷者は何人ですか?
  - 応答: **O persoană. / Doi.**
- **Persoana este conștientă?** /konʃti.ˈen.tə/ ― その人は意識がありますか?(conștient = 意識がある)
  - 応答: **Da, este conștientă. / Nu, este inconștientă.**(はい/いいえ、意識がありません)
- **Respiră?** ― 呼吸していますか?
  - 応答: **Da, respiră. / Nu sunt sigur(ă).**(はい/分かりません)
- **Câți ani are?** ― 何歳ですか?
  - 応答: **Are cam 60 de ani. / Este un copil.**(60歳くらいです/子どもです)

⚠ 分からないことは、**推測せず正直に言う**。**Nu știu.**(分かりません)/ **Nu sunt sigur(ă).**(確信がありません)は、誤った情報を伝えるより安全である。緊急時こそ「察して埋める」を避け、事実だけを言う(この誠実さは C2 の高リスク言語章で徹底される)。

### 救急外来 UPU の語彙(Unitatea de Primiri Urgențe)

病院に着いてからの場面も語彙で押さえる。**UPU**(Unitatea de Primiri Urgențe)は**救急外来・救急受け入れ部門**である。読み方は文字どおり「ウー・ペー・ウー」。

- **UPU** = 救急外来(Unitatea de Primiri Urgențe)。⚠ 略語のまま案内表示に出る。
- **camera de gardă** /ˈka.me.ra de ˈɡar.də/ = 当直室・救急受付(小規模施設の救急窓口を指すことも)。
- **triaj** /triˈaʒ/ = トリアージ(緊急度による優先順位づけの受付)。⚠ フランス語由来。「選別」の語感を持つ。
- **ambulanță** /am.buˈlan.t͡sə/ = 救急車(A2 既習)。**SMURD** /smurd/ = 高度救急の緊急介入部門の略称(案内で見かける)。
- **brancardă** /branˈkar.də/ = 担架(ストレッチャー)。**targă** /ˈtar.ɡə/ も担架。
- **rănit / rănită** = 負傷者。**victimă** /ˈvik.ti.mə/ = 被害者・負傷者。
- **a interna** /a in.terˈna/ = 入院させる。**Pacientul a fost internat.**(患者は入院した)。
- **a externa** = 退院させる(⇔ interna)。
- **însoțitor / însoțitoare** /ɨn.so.t͡siˈtor/ = 付き添い者。

窓口で使う型:

- **Avem o urgență.** ― 緊急です(A2 の Este o urgență の複数主体版)。
- **Unde este triajul?** ― トリアージ受付はどこですか?
- **A venit cu ambulanța.** ― 救急車で来ました。

### 緊急を要する症状表現(和文対照)

緊急時は、症状を**最重要のものから短く**言う。以下は「重大」を示唆する表現を、和文と対照して並べる。⚠ **どれが緊急かの医学的判断は本章の役目ではない**。伝える言語だけを扱う。迷えば 112 に伝え、オペレーターの判断を仰ぐ。

| ルーマニア語 | 和文 | 注 |
|---|---|---|
| **Nu respiră.** | 呼吸していない | a respira の否定。最重要級 ⚠ |
| **Și-a pierdut cunoștința. / A leșinat.** | 意識を失った | a-și pierde cunoștința / a leșina |
| **Nu răspunde. / Nu reacționează.** | 反応がない | a răspunde(答える)/ a reacționa(反応する) |
| **Are dureri mari în piept.** | 胸を強く痛がっている | piept=胸。⚠ 「圧迫感」は apăsare(symptoms 既習) |
| **Sângerează puternic.** | 激しく出血している | a sângera=出血する。sânge=血 |
| **Are o hemoragie.** | 出血がある | hemoragie=出血(医学語) |
| **A căzut și nu se poate mișca.** | 倒れて動けない | a cădea=倒れる/ a se mișca=動く |
| **Are convulsii.** | けいれんを起こしている | convulsii=けいれん |
| **S-a înecat. / Se sufocă.** | 溺れた/窒息している | a se îneca / a se sufoca ⚠ 区別(後述) |
| **Are o reacție alergică severă.** | 重いアレルギー反応が出ている | symptoms の alergic を緊急語に接続 |
| **A luat o supradoză. / S-a intoxicat.** | 過量摂取した/中毒を起こした | supradoză=過量、a se intoxica=中毒になる |

これらは「**最重要から1文で**」言うのが原則。**Nu respiră și nu răspunde.**(呼吸も反応もありません)のように、命に関わる情報を先に束ねる。

### 付き添い者としての説明

本人が話せない・意識がない場合、**付き添い者(însoțitor)**が代わりに説明する。ここでは症状の申告に加え、**本人の情報**(年齢・持病・服薬・アレルギー)を伝える語彙が要る。

- **Sunt însoțitorul/însoțitoarea lui/ei.** ― 私は彼/彼女の付き添いです。
- **El/Ea are [数] ani.** ― 彼/彼女は[数]歳です。
- **Suferă de diabet / de hipertensiune / de inimă.** /ˈsu.fe.rə de/ ― 糖尿病/高血圧/心臓病を患っています(a suferi de = 〜を患う。A2 の ⚠ を応用)。
- **Ia medicamente pentru inimă.** ― 心臓の薬を飲んでいます。
- **Este alergic(ă) la penicilină.** ― ペニシリンにアレルギーがあります(symptoms 既習)。
- **A mai avut un episod la fel.** ― 前にも同じ発作がありました(episod=発作の一回)。
- **Nu știu ce boli are. / Nu sunt rudă, doar am fost de față.** ― 持病は分かりません/家族ではなく、居合わせただけです。

⚠ 付き添いとして話すときも、**分からないことは Nu știu と正直に**。想像で持病や薬を答えると、かえって危険である。本人の情報が分かる書類(次項の携帯カード)があれば、それを見せて読み上げるのが確実。

### 薬・アレルギーの即答準備カード(携帯用文例)

緊急時、本人が話せないときに命を守るのが、**あらかじめ用意した情報カード**である。ルーマニア語で書いておけば、意識がなくても救急隊・医療者に伝わる。以下は**携帯カードの定型フォーマット**(架空の記入例)である。⚠ 記入内容(病名・薬・用量)は必ず**医師の確認のもと**で書くこと。本章はカードの**言語の型**を示すにとどまり、医学的な中身は判断しない。

```
FIȘĂ MEDICALĂ DE URGENȚĂ    [緊急医療カード]

Nume: [氏名]                 [氏名]
Data nașterii: [生年月日]     [生年月日]
Grupa sanguină: [血液型]      [血液型 ― 分かる場合のみ・医師確認]

Alergii: [例: penicilină]     [アレルギー]
Boli cronice: [例: diabet]    [持病]
Medicamente: [例: ...]        [常用薬]

Persoană de contact: [氏名, 電話] [緊急連絡先]

În caz de urgență, sunați la 112.  [緊急時は112へ]
```

カードや口頭で使う即答文例(チャンク):

- **Sunt alergic(ă) la [薬].** ― [薬]にアレルギーがあります。
- **Sufăr de [病名] și iau [薬].** ― [病名]を患い、[薬]を飲んでいます。
- **Grupa mea sanguină este [型].** ― 私の血液型は[型]です(⚠ 不確かなら言わない)。
- **Am o fișă medicală în portofel.** ― 財布に医療カードがあります(portofel=財布)。
- **Contactați-o pe [氏名] la numărul...** ― [氏名]に…番へ連絡してください。

⚠ **fișă**(カード・用紙)と **fisă**(コイン・トークン)は綴りが近い別語(後述)。医療カードは **fișă medicală**。

## 重要ポイント

- 緊急番号は **112**(EU 共通・無料。ルーマニア/モルドバ共通)。**unu, unu, doi** と1桁ずつ
- 通報の順序:**①場所 → ②状態(何が起きたか)→ ③人数**。まず場所を言う ⚠
- オペレーター質問 **Ce s-a întâmplat? / Unde vă aflați? / Câți răniți sunt? / Este conștientă? / Respiră?** を聞き取り応答
- 分からないことは **Nu știu / Nu sunt sigur(ă)** と正直に。推測で埋めない ⚠
- 救急外来 = **UPU**(Unitatea de Primiri Urgențe)。triaj / brancardă / a interna / însoțitor
- 緊急症状は最重要から1文で:**Nu respiră și nu răspunde. / Sângerează puternic.**
- 付き添いは本人情報(ani / suferă de / ia medicamente / alergic la)を伝える。分からなければ Nu știu
- 携帯カード(**fișă medicală de urgență**)に Alergii / Boli cronice / Medicamente を医師確認のうえ記入

## 文法解説 ― 緊急の言語がとる形

緊急表現は、平時の丁寧な文とは形が変わる。読解・産出の要点を整理する。

- **敬称命令(命令形)**:**Chemați!**(呼んでください)/ **Sunați!**(電話して)/ **Rămâneți la telefon!**(電話を切らないで)。相手に即座に行動を促すため、命令形が中心になる。A2 の Chemați o ambulanță! と同じ形。
- **現在形で「今の状態」を言う**:**Nu respiră.**(呼吸していない)/ **Sângerează.**(出血している)。進行中の危機は**現在形**で端的に述べる。日本語の「〜している」に対応。
- **複合過去で「今起きたこと」を言う**:**A leșinat.**(意識を失った)/ **A căzut.**(倒れた)。直前に起きた出来事は複合過去(symptoms・A2 既習)。
- **a se + 動詞(再帰)の緊急語**:**a se îneca**(溺れる/のどを詰まらせる)/ **a se sufoca**(窒息する)/ **a se intoxica**(中毒になる)。⚠ 再帰形が緊急語彙に多い。
- **語順は「最重要を先に」**:文法的な整いより、**命に関わる情報を先頭に**置く。**Nu respiră!** を最初に叫ぶのが正しい。緊急時は「察してもらう」余地を残さず、事実を短く前に出す。

## よくある誤用

| ❌ 誤り | ✅ 正しく | 説明 |
|---|---|---|
| 症状を先に、場所を後に言う | まず **Sunt pe strada...** で場所 → 状態 → 人数 | 救助を送るため場所が最優先。順序が命 ⚠ |
| Este o urgent.(形容詞で) | **Este o urgență.**(名詞) | 名詞 urgență=緊急。urgent は形容詞「緊急の」 |
| Este inconștient.(意識のある女性に) | **Este inconștientă.**(女性形) | conștient/inconștient は性で一致 |
| a leșina を「疲れる」の意で使う | **a leșina = 意識を失う** | 気絶。単なる疲労(a obosi)とは別 ⚠ |
| S-a sufocat と S-a înecat を同義に | 窒息 = a se sufoca / 溺れ・のど詰まり = a se îneca | 近いが別。原因が違う ⚠ |
| fișă medicală を fisă と書く | **fișă**(カード)。fisă=コイン | ș の1字で別語 ⚠ |
| 分からない持病を推測で答える | **Nu știu.** と正直に | 想像で持病・薬を言うのは危険。正直が安全 ⚠ |

## ⚠ 誤訳・混同に注意する語

緊急語は取り違えると状況が誤って伝わる。B1〜B2 で特に注意すべき点を挙げる。いずれも**確信を持って区別できる**ものに絞る。

- **⚠ a se îneca(溺れる/のどを詰まらせる)と a se sufoca(窒息する)** ― **a se îneca** は「水で溺れる」または「食べ物・液体でのどを詰まらせる」、**a se sufoca** は「(空気が吸えず)窒息する」を指す。日本語では両方「息ができない」と言えてしまうが、原因が異なる。**S-a înecat cu apă.**(水で溺れた/むせた)と **Se sufocă.**(窒息している)は別の状況を伝える。⚠ ただし**どう対処するかは本章では扱わない**。通報で状況を正しく言い分け、手当てはオペレーター・救急隊に委ねる。
- **⚠ urgență(緊急・救急)と urgent(緊急の)** ― **urgență** /urˈd͡ʒen.t͡sə/ は名詞「緊急・救急(事態)」(Este o urgență=緊急です、UPU の U)、**urgent** /urˈd͡ʒent/ は形容詞・副詞「緊急の・急いで」(Vino urgent!=急いで来て)。**Este o urgent.** は誤り。名詞が要る位置では urgență を使う。UPU の Urgențe(複数)も名詞。
- **⚠ conștient(意識がある)と conștiincios(几帳面な)** ― **conștient** /konʃti.ˈent/ は「意識がある・自覚している」(Este conștient=意識がある)、**conștiincios** /konʃti.in.t͡ʃiˈos/ は「良心的な・几帳面な」で全く別語である。綴りが似るため、緊急の意識確認 **Este conștient(ă)?**(意識がありますか)を取り違えないこと。否定は **inconștient**(意識がない)。
- **⚠ a interna(入院させる)と intern(内部の/インターン)** ― **a interna** は「入院させる」(a fost internat=入院した)。形容詞 **intern** は「内部の」、名詞 **intern** は「研修医・インターン」でもある(medic intern)。⚠ 「入院」を言うのは動詞 a interna。**A fost internat la spital.**(病院に入院した)。medicină internă(内科)の intern とは文脈で区別する。
- **⚠ fișă(カード・用紙)と fisă(コイン・トークン)** ― **fișă** /ˈfi.ʃə/ は「カード・記録票・用紙」(fișă medicală=医療カード)、**fisă** /ˈfi.sə/ は「コイン・トークン(旧公衆電話やロッカーの)」である。ș(/ʃ/)と s(/s/)の1字違いで別物。緊急医療カードは必ず **fișă medicală**。fisă medicală は意味をなさない。
- **⚠ leșin(失神)と lețin は存在しない/a leșina を「休む」と誤解** ― **a leșina** /a leˈʃi.na/ は「気絶する・意識を失う」であり、「横になって休む」ではない。**A leșinat** は緊急の意識消失を意味する。「疲れて休んだ」を a leșina で言うと「気を失った」と伝わり、不要な緊急対応を招く。休息は a se odihni。
- **⚠ rană(傷)と rănit(負傷した)と durere(痛み)** ― **rană** /ˈra.nə/(名女)は「傷・創」、**rănit**(形)は「負傷した(人)」、**durere** は「痛み」(symptoms 既習)。**Are o rană.**(傷がある)と **Are dureri.**(痛みがある)は別の情報である。出血を伴う「傷」は rană / hemoragie で、単なる「痛み」は durere。緊急通報では、**傷・出血か・痛みか**を区別して伝えると状況が正確になる。

## フォーマル/カジュアルの違い

緊急は基本 **丁寧(dumneavoastră 系)**でオペレーターに話すが、切迫時は形より**速さと明瞭さ**が優先される。

- **オペレーターに(丁寧)**: Bună ziua. Am o urgență. Sunt pe strada..., o persoană a leșinat și nu respiră.
- **周囲の人に助けを求める(切迫)**: Ajutor! Sunați la 112!(助けて!112に電話して!)/ Cineva să cheme o ambulanță!(誰か救急車を!)
- 家族へ状況を知らせる(くだけた): Sună repede la 112, tata a căzut și nu se ridică!(早く112に、父さんが倒れて起き上がれない!)

迷ったら丁寧形だが、**命に関わる一言(Nu respiră! / Ajutor!)は形を気にせず最優先で出す**。正確さと速さが命の場面である。

## 文化的注意点

- **112 は EU 共通** ― 112 は EU 域内のどの国からでも同じ番号で通じ、ルーマニア・モルドバでも緊急通報番号として機能する(出典: European Commission, digital-strategy.ec.europa.eu/en/policies/112, 2026年時点)。旅行中に別の番号を覚える必要はない。オペレーターは複数言語に対応することがあるが、**ルーマニア語で場所と状況を言えること**が確実な備えになる。
- **位置情報** ― スマートフォンからの 112 通報では位置情報が自動送信される仕組み(AML 等)が EU で普及しているとされるが、**依存せず口頭で住所・目印を言えるようにしておく**のが安全である。制度・技術の詳細は変化するため本章では深入りしない。
- **救急外来 UPU の混雑とトリアージ** ― 救急外来(UPU)では、来院順ではなく**緊急度(triaj)**で診療順が決まる。命に関わる状態が優先されるため、軽症では待つことがある。これは制度の一般的な運用に関する説明であり、**個々の緊急度の判断は医療者が行う**。本章は語彙(triaj 等)の識別にとどめ、緊急度の判断そのものには踏み込まない。

## 練習問題

### 問題1: 通報の順序に並べ替えよ

次の3文を、緊急通報の正しい順序(場所→状態→人数)に並べよ。
a. **Este o persoană rănită.** b. **Sunt pe strada Mihai Viteazul, numărul 5.** c. **A avut loc un accident.**

### 問題2: オペレーターの質問に答えよ(ルーマニア語で)

1. **Ce s-a întâmplat?**(「人が意識を失って呼吸していません」)
2. **Unde vă aflați?**(「正確には分かりませんが、薬局の向かいです」)
3. **Persoana este conștientă?**(「いいえ、意識がありません」女性について)
4. **Câți răniți sunt?**(「1人です」)

### 問題3: ⚠ 区別 ― 正しい語を選べ

1. 窒息している(空気が吸えない):Se(sufocă / îneca)。
2. 医療カード:fișă / fisă medicală。
3. 意識がありますか:Este(conștient / conștiincios)?
4. 緊急です(名詞):Este o(urgent / urgență)。

### 問題4: 付き添い者として説明せよ(ルーマニア語で)

1. 「私は彼の付き添いです。」
2. 「糖尿病を患い、心臓の薬を飲んでいます。」
3. 「ペニシリンにアレルギーがあります。」
4. 「持病は分かりません。」

### 問題5: 携帯カードを作れ

次の(架空の)情報を、本章の fișă medicală de urgență フォーマットで書け。
- 氏名:Ana Popescu / 生年月日:1980年5月12日 / アレルギー:penicilină / 持病:diabet / 緊急連絡先:Ion Popescu

## 解答

### 問題1
**b(場所)→ c(状態)→ a(人数)**:Sunt pe strada Mihai Viteazul, numărul 5. A avut loc un accident. Este o persoană rănită.
― まず場所(救助を送るため)、次に何が起きたか、最後に人数。順序が命(⚠)。

### 問題2
1. **A leșinat o persoană și nu respiră.**(意識喪失+呼吸なしを最重要から) 2. **Nu știu exact, sunt vizavi de o farmacie.**(不明でも目印。Nu știu を正直に) 3. **Nu, este inconștientă.**(女性形 inconștientă) 4. **Este o persoană.**(または Una.)。
― 質問を聞き取り、最重要情報から短く答える。分からなければ Nu știu(⚠)。

### 問題3
1. **sufocă**(窒息=a se sufoca。溺れ・のど詰まりは a se îneca) 2. **fișă**(カード。fisă はコイン) 3. **conștient**(意識がある。conștiincios は几帳面) 4. **urgență**(名詞。urgent は形容詞)。
― すべて ⚠ 混同注意そのもの。sufoca/îneca、fișă/fisă、conștient/conștiincios、urgență/urgent を分ける。

### 問題4
1. **Sunt însoțitorul lui.**(女性なら însoțitoarea) 2. **Suferă de diabet și ia medicamente pentru inimă.** 3. **Este alergic la penicilină.**(女性なら alergică) 4. **Nu știu ce boli are.**
― 付き添いは 関係→持病(a suferi de)→服薬→アレルギー の順で。分からなければ Nu știu(⚠)。

### 問題5
```
FIȘĂ MEDICALĂ DE URGENȚĂ
Nume: Ana Popescu
Data nașterii: 12 mai 1980
Alergii: penicilină
Boli cronice: diabet
Persoană de contact: Ion Popescu, [電話番号]
În caz de urgență, sunați la 112.
```
― fișă(カード)を正しく綴る。日付は「日+月名+年」で(12 mai 1980)。⚠ 記入内容は医師確認のもとで。

## 次章への橋渡し

緊急という、速さと正確さが命に直結する場面の言語を、伝えることに徹して扱った。ここまでで、症状(symptoms)・薬(medication)・緊急(本章)という、話し言葉での高リスク場面がそろった。次は、**書かれた高リスク文書**を読む段階に上がる。次章 [`informed-consent.md`](./informed-consent.md)【C1】は、インフォームドコンセント(consimțământ informat)の**文書構造の読解**を扱う。本章で用意した「同意・撤回・拒否」の言語(次章で本格化)や、「分からなければ確認する」という緊急で貫いた誠実さは、同意文書を読むときにも背骨となる。ただし次章は**文書の言語構造の教材**であり、同意判断そのものは必ず医療者との対話で行うべき領域であることを、あらかじめ心に留めてほしい。
