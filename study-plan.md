# 学習プラン

学習ペース別のモデルプラン。CEFR 公式の目安では、1 レベル上がるのに必要な学習時間はおよそ 180〜200 時間とされる(レベルが上がるほど増える傾向がある)。ルーマニア語は日本語話者にとって文字の負担が小さく(ラテン文字)、文法は屈折が多く負担が大きい、という特徴を踏まえて配分している。

## 総時間の目安

| 到達レベル | 累積学習時間の目安 |
|---|---|
| A1 | 80〜120 時間 |
| A2 | 200〜260 時間 |
| B1 | 400〜500 時間 |
| B2 | 650〜800 時間 |
| C1 | 950〜1,200 時間 |
| C2 | 1,500 時間〜(実使用経験を含む) |

> この数値は計画のための目安であり、個人差が大きい。特に C1→C2 は教材学習だけでは到達できず、実際の使用(仕事・生活・大量の読書)が必須になる。

## 週 5 時間プラン(働きながら)

| 期間 | 内容 |
|---|---|
| 1〜4か月 | A1 本編(週2章ペース)+ 毎日10分の発音ドリル |
| 5〜10か月 | A2 本編 + `vocabulary/core-500.md` の定着 |
| 11〜22か月 | B1 本編 + `reading/graded-readers-b1.md` 並行 |
| 2〜3年目 | B2 本編 + 専門編([medical/](./medical/index.md) or [business/](./business/index.md))の必要章 |
| 3年目以降 | C1〜C2 + ネイティブ素材(ニュース・文学)中心へ移行 |

## 週 15 時間プラン(集中学習)

| 期間 | 内容 |
|---|---|
| 1〜6週 | A1 本編を完走 + チェックポイントテスト |
| 2〜4か月 | A2 完走 + core-1000 着手 |
| 5〜9か月 | B1 完走 + 専門編の基礎章 |
| 10〜15か月 | B2 完走 + ニュース読解を日課化 |
| 16〜24か月 | C1 + 現地生活/実務での使用を強く推奨 |

## 毎日やること(全レベル共通)

1. **音声化** — 新しい例文は必ず音読する(発音の詳細は [00-orientation/02-pronunciation-overview.md](./00-orientation/02-pronunciation-overview.md))
2. **復習優先** — 新しい章に進む前に、前章の「復習チェックリスト」を確認
3. **語彙の間隔反復** — 語彙表は Anki 等の SRS に入れることを推奨
4. **チェックポイントテスト** — 各レベル末のテストで 80% 未満なら、該当章に戻る

## レベル別の学習重点

- **A1**: 発音を最優先。ここで音の土台を作ると以後の全てが楽になる
- **A2**: 複合過去(perfect compus)と代名詞。日常会話の 8 割はこのレベルの文法で回る
- **B1**: 接続法(conjunctiv)。ルーマニア語の「不定詞の代わりに să + 接続法」への慣れが中級の壁
- **B2**: 文体差とニュース語彙。読む量を一気に増やす時期
- **C1**: 公式文書・専門分野。書き言葉の精度を上げる
- **C2**: ニュアンス・皮肉・文化的含意。教材の外(文学・映画・実会話)が主戦場
