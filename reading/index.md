# 読解

このセクションは、CEFR レベルに応じた段階的読解教材(graded readers)と、実際のルーマニア語メディア・文学に触れるための読解教材をまとめた参照集である。本編(`01-a1/` 〜 `06-c2/`)にも各レベルに短い読解パートが含まれるが、それは1レベルあたり1〜2本程度にとどまる。読解力は読んだ総語数にほぼ比例して伸びるため、本編の読解パートだけでは量が絶対的に不足する。このセクションはその不足を補う「多読(extensive reading)」のための教材プールとして機能し、本編と並行して独立に消化していくことを想定する。

`graded-readers-a1.md` から `graded-readers-b2.md` までは、[cefr-map.md](../cefr-map.md) の「テキスト長の目安」(A1=50〜80語、A2=100〜150語、B1=200〜300語、B2=350〜500語)に沿って統制された語彙・文法で書き下ろすグレーデッドリーダーである。各レベルで使ってよい文法・語彙は、そのレベルまでの本編と [grammar/](../grammar/index.md)・[vocabulary/](../vocabulary/index.md) で導入済みのものに限定し、未習事項を読解教材の中で不意打ち的に使わないという STYLE_GUIDE の CEFR 整合原則を厳格に適用する。C1〜C2 では統制を外し、`advanced-articles-c1.md` と `native-level-texts-c2.md` で実際のルーマニア語話者向けに書かれた文章(の難度を踏まえて選定・編集したもの)に橋渡しする。

`news-reading.md` と `literary-reading.md` はジャンル別読解であり、レベル横断で使用する。ニュース読解は時事語彙が多く [vocabulary/media-and-politics.md](../vocabulary/index.md) との連携が強く、文学読解は比喩・詩的表現・歴史的背景を伴うことが多いため [culture/](../culture/index.md)・[history/](../history/index.md) への参照を伴う。出典が実在の記事・作品である場合は SOURCE_POLICY に従って出典を明記する方針である。

## 学習の目安

- **A1〜A2 の学習者**: 本編の読解パートを終えたら `graded-readers-a1.md`・`graded-readers-a2.md` で同レベルの多読量を確保する。
- **B1〜B2 の学習者**: `graded-readers-b1.md`・`graded-readers-b2.md` で物語・意見文に慣れつつ、`news-reading.md` で時事的な話題への抵抗感をなくしていく。
- **C1 の学習者**: `advanced-articles-c1.md` で統制のない実践的な文章に移行し、`literary-reading.md` で文学的表現にも触れ始める。
- **C2 の学習者**: `native-level-texts-c2.md` を中心に据え、[idioms/irony-and-humor.md](../idioms/index.md) と組み合わせて含意・皮肉まで読み取れる読解力を仕上げる。
