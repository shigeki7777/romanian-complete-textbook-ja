# 横断解答集

## このセクションの目的と使い方

このセクションは、`exercises/` に収録された横断練習問題の解答を提供することを目的とする。解答を練習問題本体から分離しているのは、学習者が答えを先に見てしまうことを避け、自力で解く過程そのものを学習の中心に据えるためである。単に正解を示すだけでなく、STYLE_GUIDE の方針に従い、**「なぜそうなるか」の短い解説を必ず付す**。文法問題であれば該当する規則やレベル別本編の参照章を示し、翻訳問題であれば直訳では通じない理由や自然な言い換えのポイントを補足する。これにより、解答集そのものが復習教材として機能する。

構成は `exercises/` の各ファイルとレベル・技能の両面で対応しており、学習者は本編の該当レベルを終えた後、練習問題を解き、その直後に対応する解答ファイルで答え合わせをする、という往復を繰り返すことを想定している。誤答が多かった場合は、解答の解説から本編の該当章に戻って復習する導線を意識している。

翻訳問題の解答(`translation-model-answers.md`)については、ルーマニア語には一つの正解が存在しない場合が多いため、模範解答に加えて許容される別解や、レジスター(丁寧度)による表現の違いも併記する方針とする。読解問題の解答(`reading-answers.md`)では、設問の根拠となる本文箇所を明示し、読解のプロセス自体を学べるようにする。

## 学習の目安

練習問題を解いた直後に対応する解答ファイルで答え合わせを行い、誤答の傾向を把握する。同じレベルの解答集で誤答が多い分野(文法か語彙か翻訳か)が見えてきたら、次のレベルに進む前にその分野の問題を重点的に再演習することを推奨する。
