# 横断練習問題

## このセクションの目的と使い方

このセクションは、レベル別本編(`01-a1/` 〜 `06-c2/`)の各章末に組み込まれた練習問題とは別に、技能別・レベル横断で反復練習を行うための問題集を提供することを目的とする。本編の練習問題が「その章で学んだ項目の直後の定着確認」であるのに対し、本セクションの練習問題は「複数の章にまたがる知識を組み合わせて使う」「時間を置いてから再演習し、忘却を防ぐ」という役割を担う。文法・語彙・翻訳・読解・聴解・会話・作文という技能ごとに独立したファイルを用意し、学習者が自分の弱点に応じて必要な技能だけを集中的に鍛えられるようにしている。

構成はレベル別本編と密接に連動する。`grammar-drills.md` と `vocabulary-drills.md` は各 CEFR レベルごとに問題が区分されており、本編でそのレベルの該当章を終えるたびに対応する問題を解く使い方を想定している。たとえば A2 で複合過去(perfect compus)を学んだ直後に `grammar-drills.md` の A2 セクションを解く、B1 で接続法(conjunctiv)を学んだ後に該当セクションを解く、といった具合である。`translation-ja-ro.md`(日本語→ルーマニア語)と `translation-ro-ja.md`(ルーマニア語→日本語)は、双方向の翻訳力を独立して鍛えるために分けている。`reading-comprehension.md` と `listening-comprehension.md` は本編の読解・聴解章と同様にレベル別の段階的なテキスト長・速度で構成し、`speaking-prompts.md` と `writing-prompts.md` は自由度の高い発話・作文課題を通じて、覚えた表現を実際に運用する力を養う。最後に `mock-tests-a1-c2.md` では各レベルの模擬チェックポイントテストを提供し、本編のレベル末チェックポイントテストと合わせて到達度を客観的に測れるようにする。

本セクションの解答は `answer-key/` に分離して収録している。これは、学習者が答えを見る前に自力で解く習慣を保ちやすくするための構成であり、`answer-key/index.md` に両セクションの対応表を掲載している。

## 学習の目安

本編の各レベルを終えるごとに、対応するレベル区分の練習問題を一通り解くことを推奨する。特に `grammar-drills.md` と `vocabulary-drills.md` は間隔反復(数日〜数週間後の再演習)で使うことで定着率が上がる。`mock-tests-a1-c2.md` は本編のレベル末チェックポイントテストで合格点(目安 80%)に達した後の総仕上げとして使い、次のレベルに進む前の最終確認とする。
