# 第12章:B2 への橋渡し【CEFR B1】

## この章の目標

この章を終えると、次のことができるようになる。

- B1 で自分が「何をできるようになったか」を具体的に言葉にできる
- B1(考えを述べる)と B2(議論し、文体を操る)の違いを説明できる
- B2 で最初に出会う新しい仕組み(大過去・単純過去の認識・受動態・文体の使い分け・ニュース読解)を事前に知り、心の準備ができる
- B1 の知識を維持する復習習慣と、多読の本格化・ニュースの読み方を持って B2 に進める

この章は問題集ではなく**読み物**である。肩の力を抜いて読んでほしい。B1 の締めくくりとして、これまでの道のりを振り返り、次のレベルへの地図を手に入れる章である。

## 本文解説

### まず、あなたができるようになったこと(棚卸し)

A2 を終えたとき、あなたは「時間軸の上で動く一人の生活者」として話せるようになっていた。B1 を終えたいま、あなたは**自分の考えを組み立て、社会の中で大人として人と関わる**ことができるようになっている。具体的に、B1 を終えたあなたは次のことができる。

**時制と物語り**
- [ ] 半過去(imperfect)を作れる(eram / aveam / voiam / lucram / mergeam)
- [ ] 半過去と複合過去を「状態・習慣・背景=半過去 / 完結した出来事=複合過去」で使い分けられる
- [ ] 背景(半過去)+ 割り込む出来事(複合過去)の型で、思い出も物語も語れる
- [ ] 談話標識(deodată/apoi=複合過去、în timp ce/pe când/pe atunci=半過去)を使い分けられる

**接続法と条件法**
- [ ] 接続法(să + 活用)を「動詞が2つ続くときの接着剤」として運用できる(vreau să / pot să / trebuie să / îmi place să)
- [ ] 3人称の交替(merge→meargă)と主語交替(Vreau să plec ⇔ Vreau să pleci)を作れる
- [ ] 勧誘(Să mergem!)・依頼(Vrei să...?)・目的(ca să)を使い分けられる
- [ ] 条件法(aș/ai/ar + 不定詞)で、丁寧な依頼(Ați putea să...?)・控えめな希望(Mi-ar plăcea să...)・非現実の仮定(Dacă aș avea..., aș...)・助言(Ar trebui să...)を言える

**意見と場面**
- [ ] 自分の意見を確信度に応じて切り出せる(Mi se pare că / Cred că / După părerea mea / Din punctul meu de vedere)
- [ ] 理由(pentru că / deoarece)と対比(totuși / pe de o parte...)で主張を組み立て、緩衝表現で目上に穏やかに反対できる
- [ ] 面接に応答し、役所の窓口で用件を切り出せる(Aș dori să depun o cerere. / Ce documente îmi trebuie?)
- [ ] 医療で症状の経過を語り(Aveam febră și mă durea capul.)、電話予約を取り、医師の指示を聞き取れる
- [ ] 件名付きのビジネスメールを、dumneavoastră 系で一貫して書ける(問い合わせ・日程調整・資料送付・催促)

**読解**
- [ ] 200 語超の随筆・物語を辞書なしで通読し、大意をつかめる
- [ ] 未知語を、ラテン語系国際語(-ție ↔ -tion)と接頭辞 ne-/re- から推測できる

これだけのことが、できるようになった。ルーマニア語で昔の思い出を Când eram copil, mergeam la mare. と物語り、自分の意見を După părerea mea... と述べ、役所で Aș dori să depun o cerere. と用件を切り出し、非現実の仮定を Dacă aș avea timp, aș călători. と組み立てられる——**「場面をこなす」段階を超えて、「自分の考えを言葉にする」段階に立っている**。まず、そのことを誇ってほしい。A2 修了時に「不安だ」と感じていた接続法・半過去・条件法を、あなたはもう自分の道具にしたのだ。

### B1 と B2 の違い — 「考えを述べる」から「議論し、文体を操る」へ

B1 と B2 の違いを一言でいうと、こうなる。

> **B1 = 「考えを述べる」。B2 = 「議論し、文体を操る」。**

B1 であなたが身につけたのは、自分の意見・理由・希望・仮定を**組み立てて表明する**力である。Cred că... で考えを言い、pentru că... で理由を添え、pe de o parte... pe de altă parte... で両面を並べられるようになった。

B2 では、その一段上——**相手と議論を交わし、場面に応じて文体(register)を操る**段階に進む。単に意見を言うだけでなく、根拠を示して相手を説得し、譲歩し、妥協案を出す。そして同じ内容を、フォーマルな文書・ニュース記事・くだけた会話で**まったく違う語彙と構文**に着替えさせる。日本語の敬語のように、ルーマニア語にも「文体の着替え」があり、B2 はそれを本格的に学ぶ段階である。

| | B1(考えを述べる) | B2(議論し、文体を操る) |
|---|---|---|
| 話の中心 | 自分の意見・理由・希望 | 論点の構造・説得・妥協・文体選択 |
| 時制・法 | 半過去+接続法+条件法 | +大過去・単純過去(認識)+受動態+複文の複雑な組み合わせ |
| 文の形 | 従属節で文を重ねる(că / să / dacă) | 逆接・譲歩・因果を明示する接続表現で論理を編む(totuși / în ciuda faptului că) |
| 典型フレーズ | Cred că… / Aș dori să… | Articolul susține că… / Pe de altă parte, trebuie să luăm în considerare… |
| 読む素材 | 200 語の随筆・物語 | ニュース記事・論説文 |

急に別の言語になるわけではない。B1 で作った土台——半過去・複合過去の使い分け、接続法・条件法、意見表明の型——は、B2 でもそのまま毎日使う。B2 はその上に「議論の技術」と「文体の技術」を積み増していく段階である。

### B2 の主役予告 — これから出会う新しい仕組み

B2 で新しく前面に出てくる主役を、あらかじめ紹介しておく。「聞いたことがある」だけで、B2 に入ったときの負担がぐっと減る。

**① 高度な時制 — 大過去(mai mult ca perfectul)と単純過去(認識)**

B1 で過去は「半過去(背景)+ 複合過去(出来事)」の2枚になった。B2 では、さらに2つの過去が視野に入る。

- **大過去(mai mult ca perfectul)** = 「過去のある時点より、さらに前に完了していたこと」。日本語の「〜してしまっていた」にあたる。第11章の読解に出てきた **întrebările pe care le pregătise**(準備していた質問)の pregătise がこれである。「過去の中の過去」を1語で表せるようになる。
- **単純過去(perfectul simplu)** = 主に文学・地方(とくにオルテニア地方)で使われる、書き言葉の完了過去。会話ではまず使わないが、**小説を読むときに「これは複合過去と同じ意味だ」と認識できる**ことが B2 の目標になる。話せるようになる必要はなく、読んで分かればよい。

半過去と複合過去の使い分けが定着していれば、大過去は「もう1段前の過去」として素直に乗る。B1 の時制の土台が、そのまま B2 の入り口になる。

**② 受動態(diateza pasivă)**

「AがBをする」(能動)を、「BがAによってされる」(受動)に言い換える形である。ニュースや報告書で頻出する:**Proiectul a fost aprobat.**(プロジェクトは承認された)。B1 では非人称構文 **Se eliberează un act.**(証明書が発行される)を第5章で少し見たが、B2 ではこの「動作の受け手を主語にする」表現を体系的に扱う。a fi(である)+ 過去分詞という組み立てなので、複合過去(a avea + 分詞)や条件法過去(aș fi + 分詞)を知っているあなたには、部品はすでにおなじみである。

**③ 文体・レジスターの本格的な使い分け**

B1 でも「友人にはくだけて、目上には丁寧に」というレジスターの階段は使ってきた(第2章の緩衝表現、第7章のビジネスメール)。B2 では、これが**語彙選択と構文の複雑さの両方**に広がる。同じ「〜しなければならない」でも、会話の trebuie să、報告書の este necesar să / se impune、ニュースの urmează să と、場面で語彙そのものが変わる。日本語の敬語という文体差の感覚を応用しつつ、「ルーマニア語の文体差は語彙と構文に現れる」と意識すると理解が進む。

**④ ニュース・論説文の読解**

B1 の第8章では 200 語の随筆・物語を読んだ。B2 では、**ニュース記事や論説文**を読み、筆者の立場や論点の構造を把握する段階に進む。抽象語彙(社会・経済・政治)が増え、受動態や複文が多用されるが、読み方の原則は B1 と同じ——「まず大意、次に細部、未知語は国際語と接頭辞から推測」である。次の項で、その入り口として「見出しから始める方法」を紹介する。

### B2 の章構成

B2 の全体像は [B2 概要](../04-b2/00-b2-overview.md) にまとまっている。B2 を始める前に必ず一度読んでほしい。章構成はおおまかに次の流れである。

- **高度な時制と相**(文法)— 大過去・単純過去(認識)・時制の一致
- **議論と主張**(会話)— 根拠を示して議論し、譲歩・妥協する
- **ニュースと論説文読解**(読解)— 200 語の随筆から、報道・論説へ
- **会議と交渉**(会話)— B1 の意見表明を交渉レベルへ
- **医療の説明**(会話)— B1 の症状申告を、より専門的な説明へ
- **文体とレジスター**(文法)— フォーマル/インフォーマルの構文・語彙選択
- **レポート作成**(作文)— B1 のビジネスメールを、まとまった報告書へ
- **復習・チェックポイントテスト・C1 への橋渡し**で締めくくる

B1 と同じく「文法 → 会話 → 作文 → 読解」を織り交ぜて進む構成であり、進め方のコツも変わらない。**声に出す・書いてみる・間違えたら該当章に戻る**。

### B1 の知識を維持する習慣

新しいことを学ぶと、古いことは自然に薄れていく。対策は「思い出す機会を意図的に作る」ことに尽きる。おすすめの復習スケジュールを挙げる。

| タイミング | やること |
|---|---|
| **1 週間後** | [第9章:総復習練習](./09-b1-review-exercises.md) をもう一度解く(答えを覚えていても、手が動くか確認) |
| **1 か月後** | [第11章:チェックポイントテスト](./11-b1-checkpoint-test.md) を再挑戦し、前回との得点差を見る |
| **毎日 5 分** | 半過去の a fi(eram…)・a avea(aveam…)、接続法の頻出3人称(să fie / să aibă / să meargă / să vină)、条件法(aș/ai/ar…)を口で唱える。「状態は半過去、出来事は複合過去」を身の回りの出来事で1文ずつ言ってみる |
| **B2 の各章の前** | 対応する B1 の章を 5 分だけ読み返す(例: B2 の受動態の前に [第5章](./05-work-and-administration.md) の非人称 Se eliberează... を、B2 の議論の前に [第2章](./02-expressing-opinions.md) の意見表明を復習) |

特に**半過去と複合過去の使い分け**と**接続法・条件法**は、B2 でも毎日使う「筋肉」である。大過去も受動態も、この土台の上に載る。使わないと衰えるので、短くてよいから毎日触れてほしい。

### 多読の本格化とニュースの読み方 — 教科書の外へ一歩

B2 からは、教科書だけでなく**生のルーマニア語を読む量**が、伸びを決定的に左右する。A2 修了時に始めた「やさしい素材を毎日10分」を、B1 修了のいまは**本格化**させる番である。

**多読の本格化**

1. **知らない語が「1ページに数個」の素材を選ぶ** — これは A2 のときと同じ原則だが、B2 では素材のレベルを一段上げる。学習者向けの記事、やさしい短編、子ども向けニュースサイトなどが適正レベルである。難しい素材で3割分かるより、やさしい素材で9割分かるほうが力がつく、は変わらない。
2. **辞書を引く回数を意図的に減らす** — B1 で学んだ推測の武器(国際語 -ție ↔ -tion、接頭辞 ne-/re-)を総動員し、「分からない語も文脈で流す」訓練をする。1ページに2〜3回だけ辞書を引くと決めると、読む速度と推測力が同時に育つ。
3. **同じ素材を2回読む** — 1回目は辞書なしで大意、2回目で細部を確認。B1 のモデルテキスト(第8章の「都会か田舎か」「ブカレストの仕事の一日」)を再読の種にするのもよい。

**ニュースは「見出しから始める」**

いきなりニュース記事の全文に挑むと挫折する。B1 修了時点の現実的な入り方は、**見出し(titlu)だけを読む**ことである。

1. **まず見出しを10本、意味だけ取る** — 見出しは短く、国際語(economie, guvern, proiect, decizie)が多く、動詞も少ない。1本10秒で「何についての記事か」だけつかむ練習を毎日する。
2. **興味を持った見出しの記事だけ、最初の1段落を読む** — ニュースは最初の段落(リード)に「誰が・何を・いつ」の骨子が凝縮されている。1段落読めれば要点はつかめる。全文を読まなくてよい。
3. **受動態と過去時制に目を慣らす** — ニュースには **a fost aprobat**(承認された)、**a declarat**(発言した)のような受動態・過去が頻出する。B2 の受動態の章の前に、見出しと1段落で「こういう形が多い」と体で知っておくと、章に入ったとき腑に落ちる。
4. **毎日 10 本の見出し + 1 本のリード、をやめないこと** — 週末に長文1本より、毎日少しずつ。ニュース読解は「その形式に慣れる」ことが8割なので、続けられる形に小さくする。

## 重要ポイント

- B1 = 「考えを述べる」、B2 = 「議論し、文体を操る」(説得・譲歩・妥協・文体選択)
- B2 の主役予告:①高度な時制(大過去=過去の中の過去、単純過去=読んで認識できればよい) ②受動態(a fi + 分詞) ③文体・レジスターの本格的使い分け(語彙+構文) ④ニュース・論説文読解
- B2 の新文法は、B1 の土台(半過去/複合過去・接続法・条件法・非人称構文)の上に積み増す。ゼロからではない
- 維持の習慣: 1 週間後に第9章、1 か月後に第11章、毎日 5 分(半過去・接続法3人称・条件法)
- 多読は「やさしい素材を2回読み・辞書は1ページ2〜3回」に本格化。ニュースは「毎日 見出し10本+リード1本」から

## 学習相談 Q&A

これまで多くの学習者が B1 修了時に抱いてきた悩みに、具体的な助言で答える。

### Q1. 接続法(să)が、話すときに反射的に出てきません。毎回「あ、să が要る」と後から気づきます。

全員がそうなる。原因は、日本語にもロマンス語未習の人にも「動詞が2つ続いたら間に接着剤を挟む」という習慣がないからで、理解の問題ではなく**運動の問題**である。処方箋は「考えて挟む」のをやめて、**動詞の組み合わせをかたまりで口癖にする**こと。Vreau să..., Pot să..., Trebuie să..., Îmi place să..., Vrei să...? ——この5つを、後ろに来る動詞を意識せず言えるまで毎日唱える。次に後ろの動詞だけ入れ替える(Vreau să dorm → Vreau să plec → Vreau să mănânc)。「動詞が2つ見えたら să」が反射になるまで、量をこなすのが唯一の道である。B2 でも接続法は毎日出てくるので、この投資は必ず回収できる。

### Q2. 半過去の使いどきが、まだ曖昧です。物語を書くと、つい全部複合過去にしてしまいます。

これは日本語話者の宿命的なつまずきである。日本語の「た」が半過去と複合過去を区別しないので、放っておくとどちらか一方(たいてい複合過去)に寄る。対策は2つ。**第一に**、書く前に文を「これは動く出来事か、止まった背景か」で仕分ける癖をつける。「電話が鳴った」(動く=複合過去)、「外は雨だった」(止まった背景=半過去)。**第二に**、半過去を呼ぶ合図の語を先に置く。**în fiecare zi**(毎日)、**de obicei**(たいてい)、**pe atunci**(当時)、**în timp ce**(〜する間)が付く文は、ほぼ半過去である。逆に **deodată**(突然)、**apoi**(それから)、**ieri**(昨日)が付けば複合過去。合図の語を先に書いてから動詞を選ぶ、を2週間続けると反射になる。困ったら [第1章](./01-telling-stories-in-the-past.md) のモデルテキスト「O amintire de vară」を、時制の色分けごと音読するのが効く。

### Q3. 条件法(aș)と接続法(să)が、複合パターンで混ざると混乱します。Ai putea să... の中で、どちらがどちらか分からなくなります。

混乱するのは、この2つが**正反対の作り**だからである。整理しよう。**条件法 = 助動詞 aș/ai/ar + 不定詞(後ろは活用しない)**。**接続法 = să + 活用形(後ろは活用する)**。Ai putea să lucrezi(働いていただけますか)を分解すると、前半 **ai putea** は条件法(助動詞 ai + 不定詞 putea)、後半 **să lucrezi** は接続法(să + 活用 lucrezi)である。役割も分かれている:条件法が「もしよろしければ」という丁寧のクッション、接続法が依頼の中身。この「クッション(条件法)+ 中身(接続法)」の役割分担さえ覚えれば、Ați putea să..., Aș vrea să..., Mi-ar plăcea să... はすべて同じ構造だと見える。詳しくは [第4章](./04-conditional-introduction.md) の文法解説「助動詞のあとは必ず不定詞」に戻るとよい。

### Q4. ニュースや映画がまだほとんど聞き取れません。B2 でついていけるか不安です。

B1 修了時点で生のニュース・映画が聞き取れないのは**正常**である。ネイティブの速度では縮約が連続し、抽象語彙も多いからだ。焦らず段階を踏もう。**第1段階(いま)**: 本文で正体を知っている音源=自分の音読を使う。B1 のモデルテキスト(第1章の思い出話、第8章の2本、第11章の Radu の話)を毎日音読し、半過去・複合過去・接続法の形を口で言えるようにする。自分の口で言える音は、耳でも拾えるようになる。**第2段階(B2 と並行)**: 前項の「ニュースは見出しから」を実践し、まず**読んで**慣れる。読める形は聞けるようになる。**第3段階(B2 後半)**: 字幕付きの短い動画で「知っているチャンクが聞こえた回数」を数えるゲームにする。全部聞き取ろうとせず、成功体験を積むのが継続の燃料になる。聞き取りは B2 を通してゆっくり伸びるもので、いま完璧でなくてよい。

### Q5. B2 は難しくなりすぎませんか? 大過去・受動態・文体差と聞くだけで身構えてしまいます。

不安になるのは、真剣に取り組んできた証拠である。安心してほしい理由が3つある。**第一に**、B2 の主役たちの部品を、あなたはもう持っている。大過去は「もう1段前の過去」で、半過去・複合過去の使い分けの延長。受動態は a fi + 過去分詞で、複合過去や条件法過去で分詞はおなじみ。文体差は、第2章の緩衝表現や第7章のメールのレジスターで、もう半歩踏み込んでいる。**第二に**、本書は B2 でも「まず大意・チャンクで慣れる → 後で仕組みを理解する」という A1 からの同じやさしい順序で進む。**第三に**、詰まったらいつでも B1 に戻れる——各章に相互リンクがあり、第10章の復習マップも使える。A2 の終わりにも「接続法が不安だ」と感じたはずだ。そして、あなたはいまその接続法を自分の道具にして、ここに立っている。一歩ずつで大丈夫である。
