# 第4章:契約書と行政文書【CEFR C1】

> **免責(強化)**: 本章は **語学学習を目的とした教材であり、法的助言では一切ない。**
> 本章が扱うのは、契約書・行政文書に現れる **言語の型(定型表現・語彙・構文)の読解** だけである。
> **法制度の内容(契約の効力・要件・手続き・権利義務の実際)には踏み込まない。**契約条項の意味・効力・有効性、行政手続きの要件・期限・必要書類は、制度として複雑で変化もするため、本章では扱わない。
> 実務で契約書を締結・署名する、行政文書を提出する、といった場面では、**必ず資格を持つ弁護士(avocat)・公証人(notar)、および契約相手の言語のネイティブ話者の確認を経ること。**
> 本章の理解だけで契約に署名したり、法的判断を下したりしてはならない。
> 本章に登場する契約・申請・当事者・金額・日付・住所はすべて **架空の設例** であり、実在の契約・個人・団体とは一切関係がない。

## この章の目標

この章を終えると、次のことができるようになる。

- 契約書の骨格語彙(**părțile contractante / obiectul contractului / durata / obligațiile părților / clauze / reziliere / forța majoră**)を **読んで識別できる**
- 契約書の定型句(**se obligă să... / în conformitate cu... / sub sancțiunea... / prezentul contract intră în vigoare...**)を **読解できる**
- 行政文書の型(**cerere / adeverință / împuternicire・procură**)の言語的特徴を把握できる
- 法律用語の誤読リスク(termen の多義、a rezilia≠a anula、parte の多義など)を認識し、慎重に読める
- 架空のモデル文書(賃貸契約の抜粋・勤務証明の申請)を、言語の型として読み解ける
- **本章が言語の型の読解にとどまり、法的内容判断には踏み込まないこと** を自覚して運用できる

本章は C1 の読解技能のなかでも、最も専門的で高リスクな領域を扱う。ただし、繰り返し強調するとおり、**本章の目的は「契約書・行政文書に何が書いてあるかを言語として読み解く」ことに限られ、「その契約がどういう効力を持つか」「その手続きに何が要るか」という法的・制度的内容には立ち入らない。** 日本語の契約書・公文書も硬く長い文体を持つ。その読解姿勢(主語と述語を丁寧に追う、条項の見出しで構造を掴む)をルーマニア語に応用しつつ、契約特有の定型語彙をまとめて覚えるのが本章の学び方である。

## 本文解説

### なぜ「読解のみ」なのか ― 本章のスコープ

契約書(contract)と行政文書は、書かれている言葉が **法的・制度的な効果** を持つ文書である。したがって、その内容を正しく理解し、判断し、産出するには、語学力だけでなく、法律・制度の専門知識が不可欠である。本章はその専門知識を提供するものではない。

そこで本章は、スコープを厳格に **言語の型の読解** に限定する。具体的には——

- **扱う**:契約書・行政文書に現れる定型語彙・定型句・構文パターンを、言語として識別・読解する
- **扱わない**:各条項が何を意味し・どんな効力を持つか、契約が有効か、手続きに何が必要か、といった**法的・制度的内容の判断**

たとえば **clauză de reziliere**(解除条項)という語句を見たとき、本章が教えるのは「これは『解除に関する条項』という意味の語である」ことだけである。「その解除条項が有効か、どんな条件で解除できるか」は法律の領域であり、本章は答えない。この線引きを常に意識してほしい。

### 契約書の構造 ― 骨格を成す見出し語彙

ルーマニア語の契約書は、決まった順序の見出し(条項の区切り)で構成されることが多い。まずこの骨格語彙を **読んで識別できる** ようにする。各語の意味は「読解のための語義」であり、法的定義ではない。

| 契約書の見出し語彙 | IPA/発音 | 読解上の意味 |
|---|---|---|
| **părțile contractante** | /ˈpər.t͡si.le kon.trakˈtan.te/ | 契約当事者(直訳「契約する諸当事者」) |
| **obiectul contractului** | /oˈbjek.tul kon.ˈtrak.tu.lui/ | 契約の目的・対象(直訳「契約の対象物」) |
| **durata (contractului)** | /duˈra.ta/ | 契約期間 |
| **obligațiile părților** | /o.bli.ˈɡa.t͡si.i.le ˈpər.t͡si.lor/ | 当事者の義務(直訳「諸当事者の諸義務」) |
| **drepturile părților** | /ˈdrep.tu.ri.le ˈpər.t͡si.lor/ | 当事者の権利 |
| **prețul / valoarea contractului** | /ˈpre.t͡sul/ | 価格・契約金額 |
| **clauze** | /ˈkla.u.ze/(単 **clauză**) | 条項(直訳「節・条」) |
| **reziliere** | /re.zi.liˈe.re/ | (契約の)解除・解約 |
| **forța majoră** | /ˈfor.t͡sa maˈd͡ʒo.rə/ | 不可抗力 |
| **litigii** | /liˈti.d͡ʒij/ | 紛争(直訳「争訟」) |
| **dispoziții finale** | /dis.po.ˈzi.t͡sij fiˈna.le/ | 最終規定・末尾条項 |

これらは契約書を上から下へ読むときの「地図」になる。日本語の契約書が「第1条(目的)」「第2条(契約期間)」「第X条(解除)」「第X条(不可抗力)」と条見出しで進むのと同じ構造である。**見出し語彙を識別できれば、どこに何が書いてあるかの当たりがつく** ——これが本章の読解の第一歩である。

**párțile contractante の書き方**(架空):契約書の冒頭では、当事者が定型で列挙される。

> **Între:**
> **1. S.C. Exemplu Imobiliare S.R.L., cu sediul în ..., reprezentată prin ..., denumită în continuare „Locator",**
> **și**
> **2. Doamna Maria Vasile, domiciliată în ..., denumită în continuare „Locatar",**
> **s-a încheiat prezentul contract...**

**denumită în continuare „..."**(以下「……」という)は、当事者に略称を与える定型句である。日本語の契約書の「(以下「甲」という)」「(以下「乙」という)」にあたる。**Locator**(賃貸人)/ **Locatar**(賃借人)のように、役割名で以降を指す。⚠ この二語は綴りが1文字違いで紛らわしい(後述の誤読リスク)。

### 契約書の定型句 ― 読解のための頻出パターン

契約書には、条項を成立させる定型句が繰り返し現れる。ここでは **読んで意味が取れる** ことを目標にする。産出(自分で契約を書く)は本章の範囲外であり、実務は専門家に委ねる。

- **se obligă să...** /se oˈbli.ɡə sə/ = 〜する義務を負う・〜することを約する(a se obliga = 義務を負う + să 節)。**Locatarul se obligă să plătească chiria...**（賃借人は賃料を支払う義務を負う……)。契約書で「当事者が何をなすべきか」を述べる中核定型。
- **în conformitate cu...** /ɨn kon.for.miˈta.te ku/ = 〜に従い・準拠して(前章既習)。**în conformitate cu prevederile prezentului contract**（本契約の規定に従い)。何に依拠するかを示す。
- **sub sancțiunea...** /sub sank.t͡siˈu.ne̯a/ = 〜の制裁のもとに・〜を条件として(読解上「〜しないと……という帰結になる」を導く言い回し)。⚠ sancțiune = 制裁・罰則。**sub sancțiunea rezilierii**（解除の制裁のもとに=違反すれば解除されうる、という定型)。
- **prezentul contract intră în vigoare...** /preˈzen.tul konˈtrakt ˈin.trə ɨn viˈɡo̯a.re/ = 本契約は……に発効する(a intra în vigoare = 効力を生じる)。**...la data semnării**（署名日に)などが続く。契約の効力発生を述べる定型。
- **prezentul contract se încheie pe o durată de...** = 本契約は……の期間で締結される。
- **părțile convin...** /ˈpər.t͡si.le konˈvin/ = 当事者は……を合意する(a conveni = 合意する)。
- **prezentul contract a fost încheiat în două exemplare** = 本契約は2部作成された(exemplar = 部・通)。

これらは **読解のチャンク** として覚える。文中で **se obligă să** を見たら「〜する義務の規定だ」、**sub sancțiunea** を見たら「違反時の帰結を述べている」、**intră în vigoare** を見たら「効力の発生を述べている」と、構文の役割を即座に読み取れるのが目標である。繰り返すが、その義務・制裁・効力が **実際にどう働くか** は法律の領域であり、本章は判断しない。

### 行政文書の型

行政文書は、前章で書いた公式書面と地続きだが、いくつかの定型的な文書「種別」がある。ここでは各種別の **言語的な型** を把握する(制度上の効力・要件は扱わない)。

**① cerere(申請書)の標準フォーマット**

**cerere**(申請書)は、行政・機関に何かを求める書面である。読解・識別できるべき定型構造は次のとおり。

```
Domnule Director,   [宛名:役職への呼びかけ]

Subsemnatul/Subsemnata, [氏名], [身元情報],
vă rog să binevoiți a-mi aproba...   [本文:願い出]
[または: solicit... / vă rog să...]

[理由・付記]

Data              Semnătura
[日付]            [署名]
```

- **Domnule Director,** = 部長殿(役職への呼びかけ)。宛名を役職で呼ぶのが cerere の型。**Domnule Primar,**（市長殿)、**Doamnă Director,**（女性の部長殿)なども同型。
- **Subsemnatul/Subsemnata, ...,** = 下名の者(前章の非人称化)。
- **Vă rog să binevoiți a-mi aproba...** /və roɡ sə bi.ne.voˈitsʲ/ = 何とぞ……をご承認くださいますようお願い申し上げます。**a binevoi** = 快く〜する・恩恵的に〜する(直訳「善く意する」)。**Vă rog să binevoiți** は行政申請の非常に硬い定型で、日本語の「ご高配を賜りたく」「何とぞよろしくお取り計らいください」に近い格式。⚠ ただし相当に古風・儀礼的な言い回しで、現代では **solicit** / **vă rog să** で足りる場面も多い(読解では出会うが、産出では過剰になりうる)。

**② adeverință(証明書)**

**adeverință** /a.de.veˈrin.t͡sə/ = 証明書(事実を証する書面)。在職(**adeverință de salariat**)、在学(**adeverință de student**)など、「ある事実が真である」ことを機関が証する文書。読解上の特徴は、**「〜であることを証する」という証言定型** が現れる点である。

- **Prin prezenta, se adeverește că...** = 本書をもって、……であることを証する(a adeveri = 真であると証する)。
- **...este angajat/angajată în cadrul ... începând cu data de ...** = ……は……に……年……月……日より在籍している。
- **Prezenta adeverință a fost eliberată pentru a servi la...** = 本証明書は……に供するために発行された(a servi la = 〜に役立つ・供する)。

⚠ 証明書は「機関が事実を証する」文書なので、主語は個人ではなく **se adeverește**(証される)という非人称・受動形が典型である。前章の subsemnatul(自分を主語にする申請)とは主語の立て方が逆になる点に注意。

**③ împuternicire / procură(委任状)― 概念の紹介**

**împuternicire** /ɨm.pu.ter.ni.ˈt͡ʃi.re/ / **procură** /proˈku.rə/ = 委任状(ある者が別の者に、自分に代わって何かを行う権限を与える書面)。**a împuternici** = 権限を与える・委任する。読解上出会う定型は——

- **Subsemnatul, ..., împuternicesc pe domnul/doamna ... să mă reprezinte în...** = 下名の者は、……氏に、……において私を代理する権限を委任する。
- **...să efectueze în numele meu următoarele...** = ……私の名において以下を行うこと(în numele meu = 私の名において)。

⚠ **procură は多くの場合、公証人(notar)の関与を要する種類の文書として言及される** が、その要否・効力・作成手続きは制度の領域であり、本章では扱わない。本章が示すのは「împuternicire / procură = 委任状という文書種別で、a împuternici / a reprezenta / în numele meu といった語彙が現れる」という **言語的な把握** だけである。実際の委任状の作成・使用は、必ず公証人・専門家に確認すること(免責)。

## ⚠ 法律用語の誤読リスク

契約・行政の語は、日常語と綴りや語源が近いために、意味を取り違えると文書の理解が根本からずれる。**確信の持てる、頻出かつ重要な誤読リスクを挙げる。** いずれも「日本語話者が特に取り違えやすい」ものに絞る。

- **⚠ termen(期限 / 用語)の多義** ― **termen** /ˈter.men/ は、契約・行政文脈で二つの重要な意味を持つ。①**期限・期間**(**termen de plată**=支払期限、**termen de livrare**=納期、**termenul contractului**=契約期間)、②**用語・術語**(**termen juridic**=法律用語、**termeni și condiții**=規約・利用条件、直訳「用語と条件」)。同じ termen が「期限」と「用語」の両方を指すため、**termen de garanție**(保証期間)を「保証の用語」と読むと意味が崩れる。文脈で「時間の締め切り/期間」か「ことば・術語」かを判別する。複数形にも注意:**termene**(期限・期間の複数)と **termeni**(用語・条件の複数)で複数形が分かれる。⚠ **termeni și condiții** は「用語と条件」ではなく「規約・条件」の定型句として読む。

- **⚠ a rezilia ≠ a anula(解除 ≠ 無効化)** ― どちらも日本語で「(契約を)なくす」と訳したくなるが、法的な意味合いが異なる語である。**a rezilia** /a re.zi.liˈa/ は「(継続的な契約を将来に向かって)解除・解約する」、**a anula** /a a.nuˈla/ は「(予約・注文・契約などを)取り消す・無効にする」を指す傾向がある。**読解上、この二語を機械的に同じ「解除」と一括りにしない** ことが重要である。**rezilierea contractului**(契約の解除)と **anularea comenzii**(注文の取り消し)は別の場面の語である。ただし、**両語の法的な区別・効果の詳細は法律の領域であり、本章は判断しない。** ここで押さえるのは「rezilia と anula は別語であり、安易に同義扱いしない」という読解上の注意にとどめる。

- **⚠ parte(当事者 / 部分)の多義** ― **parte** /ˈpar.te/ は日常語では「部分・一部」だが、契約文脈では **当事者(契約の一方)** を指す。**părțile contractante**（契約当事者）、**ambele părți**（両当事者）、**partea vânzătoare**（売主側）は「部分」ではなく「当事者・側」である。日本語話者は parte を「部分」の連想で読みがちだが、契約書の parte は原則「当事者」と読む。⚠ 一方、**în parte**(個別に・それぞれ)、**pe de altă parte**(他方で)のような句では「部分・側面」の意が残るので、句として区別する。

- **⚠ dispoziție(規定 / 気分・意向 / 処分)の多義** ― **dispoziție** /dis.po.ˈzi.t͡si.e/ は文脈で意味が大きく振れる語である。契約・行政では **規定・条項**(**dispoziții finale**=最終規定、**dispoziții legale**=法規定)を指すが、日常語では **気分・機嫌**(**bună dispoziție**=上機嫌)、また **la dispoziția dumneavoastră**(あなたの意のままに=「いつでも対応します」)という別の慣用もある。契約・行政の見出しにある dispoziții は「規定・条項」と読む。「気分」や「意のまま」の意で読むと文書の性格を取り違える。

- **⚠ act(証書・文書 / 幕・行為)** ― **act** /akt/ は行政・法律文脈では **公的な書類・証書**(**act de identitate**=身分証、**act adițional**=追加契約書・付属文書、**acte**=書類一式)を指す。演劇の「幕」や一般語の「行為」の意もあるが、契約・行政では「証書・文書」と読む。⚠ 特に **act adițional**(追加契約書・変更契約書、直訳「追加の証書」)は契約の修正文書を指す頻出語で、「追加の行為」と読むと意味がずれる。

- **⚠ sub sancțiunea(制裁のもとに)を軽く読まない** ― 前述のとおり **sancțiune** = 制裁・罰則である。**sub sancțiunea nulității / rezilierii** のような句は「(それをしないと)無効/解除という帰結になる」という重い条件を述べている。日本語話者は sancțiune を英語 sanction(承認・許可の意もある)の連想で「認可」と誤読することがあるが、ルーマニア語の法文脈では **制裁・罰則** の意で読む。ただし、その制裁が具体的にどう働くかは法律の領域であり、本章は判断しない。

## モデル文書1 ― 賃貸契約の抜粋(架空・読解用)

架空の設例:架空の会社「**S.C. Exemplu Imobiliare S.R.L.**」(賃貸人)と架空の個人「**Maria Vasile**」(賃借人)の住居賃貸契約の **抜粋**。**当事者・金額・期間・条項はすべて架空** であり、実在の契約・物件・取引とは一切関係がない。**この文書は言語の型を読解する練習用であり、契約のひな形として使用してはならない。** 実際の契約は必ず弁護士・公証人・ネイティブの確認を経ること。

---

**CONTRACT DE ÎNCHIRIERE (extras)**

**I. Părțile contractante**
Între **S.C. Exemplu Imobiliare S.R.L.**, denumită în continuare „Locator", și doamna **Maria Vasile**, denumită în continuare „Locatar", s-a încheiat prezentul contract.

**II. Obiectul contractului**
Obiectul prezentului contract îl constituie închirierea apartamentului situat în [adresă fictivă], în scopul utilizării ca locuință.

**III. Durata contractului**
Prezentul contract se încheie pe o durată de 12 luni și intră în vigoare la data semnării.

**IV. Obligațiile părților**
Locatarul se obligă să achite chiria lunară la termenul stabilit și să folosească imobilul cu diligența unui bun proprietar. Locatorul se obligă să asigure folosința liniștită a imobilului.

**V. Rezilierea**
Nerespectarea obligațiilor asumate poate atrage rezilierea prezentului contract, în conformitate cu clauzele convenite.

**VI. Forța majoră**
Forța majoră exonerează de răspundere partea care o invocă, în condițiile legii.

Prezentul contract a fost încheiat în două exemplare, câte unul pentru fiecare parte.

---

### 対訳

> **賃貸契約(抜粋)**
>
> **I. 契約当事者**
> エグゼンプル・イモビリアーレ有限会社(以下「賃貸人」という)と、マリア・ヴァシレ女史(以下「賃借人」という)との間で、本契約が締結された。
>
> **II. 契約の目的**
> 本契約の目的は、[架空の住所]に所在するアパートを、住居として利用する目的で賃貸することにある。
>
> **III. 契約期間**
> 本契約は12か月の期間で締結され、署名の日に発効する。
>
> **IV. 当事者の義務**
> 賃借人は、定められた期限に月額賃料を支払い、善良な所有者の注意をもって当該不動産を使用する義務を負う。賃貸人は、当該不動産の平穏な使用を保証する義務を負う。
>
> **V. 解除**
> 引き受けた義務の不履行は、合意された条項に従い、本契約の解除を招きうる。
>
> **VI. 不可抗力**
> 不可抗力は、それを援用する当事者を、法の定める条件のもとで責任から免れさせる。
>
> 本契約は、各当事者に1部ずつ、2部作成された。

### 読解解説(言語の型のみ)

- **denumită în continuare „Locator" / „Locatar"** ― 「以下『賃貸人』/『賃借人』という」。当事者に略称を与える定型。⚠ Locator と Locatar は1字違いで役割が逆(誤読リスク後述)。
- **Obiectul ... îl constituie închirierea...** ― 「目的は……を構成する」。a constitui(構成する)+ 名詞化 închirierea(賃貸)。契約の目的を述べる定型。
- **se încheie pe o durată de ... și intră în vigoare la data semnării** ― 期間と効力発生の定型。se încheie(締結される=受動)、intră în vigoare(発効する)、semnării(署名の=名詞化の属格)。
- **se obligă să achite ... și să folosească ...** ― 義務の定型 se obligă să + 接続法の連結。「支払い」と「使用」の二義務を să で並べる。
- **cu diligența unui bun proprietar** ― 「善良な所有者の注意をもって」。契約書に現れる定型的な注意義務の言い回し(読解語彙として識別できればよい)。
- **poate atrage rezilierea** ― 「解除を招きうる」。a atrage(引き起こす)+ 名詞化 rezilierea(解除)。⚠ rezilierea は anularea と区別する(誤読リスク)。
- **exonerează de răspundere ... în condițiile legii** ― 「……法の定める条件のもとで責任を免れさせる」。**în condițiile legii**(法の条件下で)は「詳細は法による」を示す定型で、まさに **本章が踏み込まない法的内容** を指し示している。読解では「ここは法の領域だ」と認識して先へ進む。

**この抜粋の読み方**:見出し(Părțile / Obiectul / Durata / Obligațiile / Rezilierea / Forța majoră)で構造を掴み、各条項の定型句(se obligă să / intră în vigoare / poate atrage rezilierea)で「何を述べている条項か」を識別する。**各条項が実際にどんな効力・条件で働くかは判断せず、言語の型の読解にとどめる。** 実際の契約を読む・署名する場面では、必ず弁護士・公証人・ネイティブに確認すること。

## モデル文書2 ― 勤務証明の申請(cerere)(架空・読解用)

架空の設例:架空の社員「**Elena Dumitru**」が、勤務先の人事部長に在職証明書の発行を申請する **cerere**(申請書)。**人名・部署・日付はすべて架空** である。前章のモデル要望文と同じ場面を、ここでは古典的な cerere の型で示し、読解する。

---

**Domnule Director,**

Subsemnata, **Elena Dumitru**, angajată în cadrul Departamentului Financiar începând cu data de 1 martie 2024, vă rog să binevoiți a-mi aproba eliberarea unei adeverințe de salariat.

Menționez că adeverința îmi este necesară pentru un demers personal la o instituție bancară.

Vă mulțumesc pentru solicitudine.

Data: 15 septembrie 2026            Semnătura,
                                    Elena Dumitru

---

### 対訳

> **部長殿**
>
> 下名の者、エレナ・ドゥミトゥル(2024年3月1日より財務部に在籍)は、在職証明書の発行を何とぞご承認くださいますようお願い申し上げます。
> 当該証明書は、金融機関での個人的な手続きのために必要である旨を申し添えます。
> ご高配に御礼申し上げます。
>
> 日付:2026年9月15日            署名、
>                              エレナ・ドゥミトゥル

### 読解解説(言語の型のみ)

- **Domnule Director,** ― 役職への呼びかけ。cerere は宛名を役職で始める型。
- **Subsemnata, ..., angajată ... începând cu data de ...** ― 非人称化(前章)+ 在籍情報の同格挿入。angajată(女性形)が主語 subsemnata に一致。
- **vă rog să binevoiți a-mi aproba eliberarea...** ― ⚠ 古典的・硬い申請定型。**a binevoi**(快く〜する)+ 不定詞 **a-mi aproba**(私に承認する)+ 名詞化 **eliberarea**(発行)。現代では **solicit eliberarea** / **vă rog să îmi aprobați** で足りるが、公式の cerere ではこの古風な言い回しに出会う。**読解できることが目標で、産出でここまで古風にする必要はない。**
- **Menționez că ... îmi este necesară pentru un demers...** ― 理由の付記。a menționa(言及する・付言する)。demers = 手続き・働きかけ。
- **pentru solicitudine** ― 「ご高配に(対して)」。solicitudine = 配慮・親身な対応(硬い語)。
- 末尾の **Data / Semnătura** ― 日付と署名の欄。cerere は本文の後にこの二欄を置く型。

**この cerere の読み方**:前章の要望文(solicit を使う現代的な書面)と、この cerere(vă rog să binevoiți a- を使う古典的な書面)を読み比べると、**同じ「証明書の申請」でも文体の格・古風さに幅がある** ことが分かる。読解では両方に対応できるように、産出では相手・場面に合わせて過度に古風にしないのが賢明である(古風な定型は誤って使うと不自然になりうる。実際の提出はネイティブ確認)。

## 重要ポイント

- **本章は言語の型の読解のみ。契約の効力・要件・手続きなど法的・制度的内容には踏み込まない。実務は必ず弁護士・公証人・ネイティブに確認する**
- 契約書の骨格見出し:**părțile contractante / obiectul contractului / durata / obligațiile părților / clauze / reziliere / forța majoră**
- 契約書の読解定型:**se obligă să... / în conformitate cu... / sub sancțiunea... / prezentul contract intră în vigoare... / denumit în continuare „..."**
- 行政文書の型:**cerere**(Domnule Director / Subsemnatul / Vă rog să binevoiți)、**adeverință**(se adeverește că...)、**împuternicire・procură**(a împuternici / în numele meu)
- ⚠ 誤読リスク:**termen(期限/用語)、a rezilia≠a anula、parte(当事者/部分)、dispoziție(規定/気分)、act(証書/行為)、sub sancțiunea(制裁のもと)**
- 古典的な cerere の **vă rog să binevoiți a-...** は読解目標。産出では solicit / vă rog să で足りることが多い

## よくある誤用(読解上の取り違え)

| ❌ 誤読 | ✅ 正しい読み | 説明 |
|---|---|---|
| **termen de garanție** を「保証の用語」と読む | 「保証**期間**」 | termen は「期限/期間」の意もある。文脈で判別 ⚠ |
| **termeni și condiții** を「用語と条件」と直訳 | 「規約・利用条件」 | 定型句として「規約」と読む ⚠ |
| **a rezilia** と **a anula** を同じ「解除」と一括り | 別語として区別(rezilia=解除/anula=取消・無効) | 安易に同義扱いしない。効力の違いは法の領域(本章は判断せず) ⚠ |
| **părțile** を「部分」と読む | 契約文脈では「当事者」 | parte は「当事者/部分」の多義。契約では当事者 ⚠ |
| **dispoziții finale** を「最後の気分」と読む | 「最終規定・末尾条項」 | dispoziție は「規定/気分/意のまま」の多義。契約見出しでは規定 ⚠ |
| **act adițional** を「追加の行為」と読む | 「追加契約書・変更文書」 | act は法文脈では「証書・文書」 ⚠ |
| **Locator** と **Locatar** を取り違える | Locator=賃貸人(貸す側)/Locatar=賃借人(借りる側) | 1字違いで役割が逆。文書内の denumită în continuare で確認 ⚠ |
| 本章の読解だけで契約内容を判断する | 言語の型を読むにとどめ、内容判断は専門家に | 本章は法的助言ではない(免責) |

## ⚠ 誤読・混同に注意する語(まとめ)

上の「よくある誤用」と重複するが、確信の持てる誤読リスクを、判別の指針とともに再掲・集約する。

- **⚠ termen** ― 時間の締め切り・期間なら「期限/期間」(termen de plată/livrare/garanție)、ことば・術語なら「用語」(termen juridic, termeni și condiții)。複数形が termene(期限)/ termeni(用語)で割れる。
- **⚠ a rezilia / a anula** ― 別語。rezilia=(継続的契約の)解除、anula=(予約・注文等の)取消・無効。同義扱いしない。効力の差は法の領域(本章は判断しない)。
- **⚠ parte** ― 契約の主体なら「当事者」(părțile contractante)、句なら「部分・側」(în parte, pe de altă parte)。
- **⚠ dispoziție** ― 契約・行政の見出しなら「規定」(dispoziții finale/legale)、慣用なら「意のまま」(la dispoziția dumneavoastră)・「気分」(bună dispoziție)。
- **⚠ act** ― 法・行政文脈なら「証書・文書」(act de identitate, act adițional, acte)。
- **⚠ sancțiune / sub sancțiunea** ― 「制裁・罰則」。英 sanction の「承認」の意で読まない。sub sancțiunea = 「〜という制裁の帰結を条件として」。

## ネイティブらしい理解のコツ

契約・行政文書は、ネイティブでも一読で意味を確定しにくい高密度な文体である。読解の実務的なコツを挙げる。

| つまずき | 読解のコツ |
|---|---|
| 一文が長すぎて主語が見えない | 見出し(clauza)ごとに「誰が(どの parte が)何をする」を先に特定する |
| 名詞化が連なって動作が見えない | 名詞化(rezilierea, eliberarea)を動詞(a rezilia, a elibera)に戻して意味を取る |
| se 受動で行為者が不明 | 契約では「当事者のいずれか/法」が行為者。誰かは条項の主語で確認 |
| 古風な定型(vă rog să binevoiți a-) | 定型として「〜してくださいますよう」と受け取り、逐語分解に固執しない |
| 意味に確信が持てない条項 | **推測で確定せず、専門家・ネイティブに確認する**(本章の原則) |

最後のコツが最も重要である。契約・行政は、**読み違いが実害に直結する** 領域である。医療の章で「分からなければ医師に確認する」と学んだのと同じ姿勢で、契約・行政では「分からなければ弁護士・公証人・ネイティブに確認する」を徹底する。本章はその確認を **語学の側から支える** ものであって、確認を **省くための** ものではない。

## 文化的注意点

- 一般に、ルーマニアの契約書・公文書は、**定型性が高く、古典的・儀礼的な言い回し(vă rog să binevoiți a-)が残る** 傾向があるとされる。現代的な文書ほど簡潔になる傾向もあるが、公的機関・公証文書では硬い定型に出会うことが多いとされる(あくまで一般的傾向で、文書・機関による差はある)。
- 一般に、重要な契約・委任状(procură)は、**公証人(notar)の関与を伴う種類の文書として言及される** ことがあるが、その要否・手続き・効力は制度の領域であり、本章では扱わない。文書に notar / autentificare(認証)といった語が現れたら「これは公証・認証に関わる文書だ」と **言語的に認識** し、実務は専門家に委ねる。
- 契約書・行政文書の書式・様式・提出方法は、機関・契約の種類ごとに大きく異なりうる。本章は言語(定型・語彙・構文)の読解を扱うものであり、特定の書式・要件を示すものではない。**実際の契約締結・文書提出は、必ず資格を持つ専門家(弁護士・公証人)とネイティブ話者の確認を経ること**(強化免責の再確認)。

## 練習問題(読解のみ)

### 問題1: 見出し語彙を識別せよ

次の契約書見出しの読解上の意味を答えよ。
1. **Obiectul contractului** 2. **Durata contractului** 3. **Obligațiile părților** 4. **Forța majoră** 5. **Dispoziții finale**

### 問題2: 定型句の役割

次の定型句が契約書で「何を述べている」かを答えよ(効力の判断は不要)。
1. **Locatarul se obligă să achite chiria.**
2. **Prezentul contract intră în vigoare la data semnării.**
3. **Nerespectarea obligațiilor poate atrage rezilierea contractului.**

### 問題3: ⚠ 誤読リスク ― 正しい読みを選べ

1. **termen de plată** の termen は(期限 / 用語)。
2. **termeni și condiții** は(用語と条件 / 規約)。
3. 契約書の **părțile contractante** の parte は(部分 / 当事者)。
4. **dispoziții finale** の dispoziție は(気分 / 規定)。
5. **act adițional** の act は(行為 / 証書・文書)。

### 問題4: rezilia か anula か(読解上の区別)

読解上、より適切な語を選べ(効力の判断ではなく語の区別として)。
1. 継続的な賃貸契約を将来に向けて終わらせる文脈 →（a rezilia / a anula)
2. 出したばかりの注文を取り消す文脈 →（a rezilia / a anula)

### 問題5: cerere の型を読む

次の cerere 抜粋を読み、①宛名の呼びかけ ②非人称化された自己指示 ③申請本体の定型 を、それぞれ本文から抜き出せ。
> Domnule Primar,
> Subsemnatul, Andrei Marinescu, domiciliat în [adresă fictivă], vă rog să binevoiți a-mi aproba eliberarea unui certificat.

## 解答

### 問題1
1. 契約の目的・対象 2. 契約期間 3. 当事者の義務 4. 不可抗力 5. 最終規定・末尾条項。
— いずれも契約書の構造を示す見出し語彙。意味は読解上のものであり、法的定義ではない。

### 問題2
1. **賃借人の義務を述べている**(se obligă să=義務の規定)。 2. **契約の効力発生(発効)を述べている**(intră în vigoare)。 3. **義務違反の帰結(解除を招きうること)を述べている**(poate atrage rezilierea)。
— それぞれ「義務/効力発生/違反の帰結」を述べる条項だと識別できればよい。その効力が実際どう働くかは法の領域(本章は判断しない)。

### 問題3
1. **期限**(termen de plată=支払期限) 2. **規約**(termeni și condiții=規約・利用条件) 3. **当事者**(părțile contractante=契約当事者) 4. **規定**(dispoziții finale=最終規定) 5. **証書・文書**(act adițional=追加契約書)。
— すべて本章 ⚠ の要点。termen/parte/dispoziție/act の多義を、契約・行政の文脈で正しい側に読む。

### 問題4
1. **a rezilia**(継続的契約を将来に向けて解除) 2. **a anula**(注文の取り消し)。
— 読解上、rezilia=(継続的契約の)解除、anula=(予約・注文等の)取消・無効として区別する。両語を同義扱いしない。効力の詳細は法の領域(本章は判断しない)。

### 問題5
① **Domnule Primar,**(市長殿=役職への呼びかけ) ② **Subsemnatul, Andrei Marinescu, domiciliat în...**(下名の者=非人称化された自己指示) ③ **vă rog să binevoiți a-mi aproba eliberarea unui certificat.**(何とぞ証明書の発行をご承認くださいますよう=申請本体の古典的定型)。
— cerere の型:役職宛名 → subsemnatul の自己指示 → vă rog să binevoiți a- の申請定型。産出では solicit eliberarea でも足りる。

## 次章への橋渡し

契約書・行政文書という、言語の型が最も硬く結晶した文書を、読解の目的で扱った。ここで貫いた原則——**言語の型は読み解くが、内容の判断は専門家に委ねる**——は、次の[第5章:医療文書](./05-medical-documents-c1.md)にそのまま引き継がれる。医療文書もまた、退院サマリー・検査結果・処方箋という高リスクな読解対象であり、略号や専門構造の **言語的な型** は学べても、**数値や診断の医学的解釈は医療専門家に委ねる** べき領域である。本章の「型は読む、判断は委ねる」という姿勢を、次章の医療文書でも同じ緊張感で保ってほしい。
