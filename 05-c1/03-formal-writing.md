# 第3章:フォーマルな文章【CEFR C1】

> **免責**: 本章は語学学習を目的とした教材であり、法的・行政的助言ではない。
> 苦情・要望・推薦などの公式文書を実務で提出する場合は、必ずネイティブ話者、および必要に応じて資格を持つ専門家の確認を経ること。
> 本章に登場する人名・会社名・機関名・日付・住所・事案はすべて **架空の設例** であり、実在の個人・団体・事件とは一切関係がない。

## この章の目標

この章を終えると、次のことができるようになる。

- 公式文書の非人称化(**subsemnatul / subsemnata** による自己言及、一人称の抑制)を運用できる
- 遂行動詞(**solicit / declar / mă angajez / vă aduc la cunoștință**)を使い、書面上で「行為そのもの」を成立させる文を書ける
- 公式手紙の完全な型(宛名 **În atenția...** / 件名 **Ref:** / 導入定型 **Prin prezenta...** / 結び **Cu deosebită considerație,**)を組み立てられる
- 苦情文(**reclamație**)・要望文(**solicitare**)・推薦状(**scrisoare de recomandare**)を、目的に応じた構成で書ける
- 受動+名詞化の連鎖、長文の構造管理(関係節・ジェルンジウの入れ子)を、破綻させずに扱える
- 日本語話者の癖(「恐れ入りますが」的な過剰緩衝の直訳)を避け、簡潔で正式な依頼文を書ける

## 前提知識

- [第7章:レポートを書く](../04-b2/07-writing-reports.md)(B2) — 客観的文体の二大道具(**se + 動詞** / 名詞化)、書き言葉の接続表現、結論先行。本章はこの「読み手一般に向けた客観文」を、**特定の宛先(機関・担当者)に向けた公式書面** へと精密化する
- [ビジネスメール](../business/emails.md)(B1〜B2) — 書き出し・結びのマトリクス、条件法(aș dori)による丁寧な依頼、定型表現のチャンク運用。本章のフォーマル書面はこの延長で、より硬い定型と非人称化を加える
- [第5章:仕事と行政](../03-b1/05-work-and-administration.md)(B1) — **cerere / a depune / a solicita** など行政語彙の基礎。本章はこの語彙を書面の産出に使う
- [第2章:ニュアンスと含意](./02-nuance-and-implication.md)(C1・並行学習) — 婉曲・緩衝の言語手段。本章の「過剰緩衝を避ける」議論はこの章と対をなす

本章は C1 の作文技能の中核である。B2 のレポートは「読み手一般に事実を提示する」文書だった。C1 のフォーマルな文章で新しいのは、**特定の宛先(役所・企業・上長)に対して、書面という形式そのものが持つ効力を意識して書く** ことである。苦情・要望・宣言・推薦は、いずれも「読ませる」だけでなく「相手に何かを行わせる/自分が何かを引き受ける」ための書面であり、そこには定型と非人称化という、日本語の公文書とも通じる硬質な文体が要求される。

## 本文解説

### 公式文書の文体規範 ― 三つの柱

ルーマニア語の公式文書(document oficial)を特徴づける文体規範は、大きく三つある。

1. **非人称化(depersonalizare)** ― 「私(eu)」という生身の主語を前面に出さず、**subsemnatul / subsemnata**(下名の者)という定型的な自己指示や、se 構文・受動態で書き手を客観化する。
2. **遂行性(performativitate)** ― **solicit**(私は要請する)、**declar**(私は宣言する)のように、その語を書くこと自体が行為の遂行になる動詞を用いる。
3. **定型性(formulare-tip)** ― 導入・結びに決まった言い回し(**Prin prezenta... / Cu deosebită considerație,**)を使い、個人的な色を抑える。

この三本柱は、日本語の申請書・陳情書・始末書が持つ硬さ(「下記の通り」「〜致したく申し出ます」「上記に相違ありません」)と発想が近い。日本語話者は、この「役所言葉・公文書言葉」の感覚をルーマニア語に移せば、C1 の公式文体に早く馴染める。

### 非人称化 ― subsemnatul / subsemnata

公式文書、とりわけ役所への申請・宣言では、書き手は自分を **eu**(私)ではなく **subsemnatul**(男性)/ **subsemnata**(女性)と呼ぶ。直訳は「下(に署名した)者」で、日本語の「下名(かめい)の者」「下記署名人」にあたる硬い自己指示である。

- **subsemnatul** /sub.semˈna.tul/(男性)= 下名の者(男)。語源は sub-(下に)+ semnat(署名された)。
- **subsemnata** /sub.semˈna.ta/(女性)= 下名の者(女)。

典型的な書き出しは、この語の後に **氏名・身元情報** を同格で並べ、そこから遂行動詞へつなぐ形である。

> **Subsemnatul, Andrei Marinescu, ..., solicit prin prezenta...**
> (下名の者、アンドレイ・マリネスク、……は、本書をもって……を要請する)

ここで注意すべきは、**subsemnatul は文の主語であり、後続の動詞は一人称単数で活用する** 点である。「下名の者は要請する」を、ルーマニア語は **Subsemnatul ... solicit**(subsemnatul + 一人称 solicit)と、三人称の名詞を主語にしながら動詞を一人称で受ける独特の形で書く。日本語話者は「subsemnatul(彼)なら solicită(三人称)では?」と迷いがちだが、公式文書ではこの一人称受けが定型である。

### 遂行動詞(verbe performative)

遂行動詞とは、**それを一人称・現在形で述べること自体が、その行為の遂行になる** 動詞である。「私は約束する」と言うことが約束すること自体であるように、公式文書では次の動詞が「書くこと=行うこと」として機能する。

- **a solicita** /a so.li.t͡ʃiˈta/(要請する・申請する)→ **solicit**(私は要請する)。cere より硬い、書面向きの語。
- **a declara** /a de.klaˈra/(宣言する・申告する)→ **declar**(私は宣言する)。**declar pe propria răspundere**(自己の責任において宣言する)は宣誓供述の定型。
- **a se angaja** /a se an.ɡaˈʒa/(約束する・引き受ける)→ **mă angajez**(私は誓約する)。就職の「雇われる」とは別義で、ここは「義務を引き受ける」。
- **a aduce la cunoștință** /a aˈdu.t͡ʃe la ku.noʃˈtin.t͡sə/(通知する・お知らせする)→ **vă aduc la cunoștință**(貴殿にお知らせする)。
- **a confirma**(確認する)→ **confirm** / **a certifica**(証明する)→ **certific**(推薦状などで)。

これらは、日本語の「申請いたします」「宣誓します」「誓約いたします」「ご通知申し上げます」に対応する、書面の背骨となる動詞である。ビジネスメールの **aș dori să vă rog**(〜をお願いしたいのですが)が条件法で相手に選択の余地を残す柔らかい依頼であるのに対し、**solicit**(私は要請する)は直説法・現在で断定的に行為を成立させる。**公式の申請・宣言では、条件法でぼかさず、遂行動詞の直説法で言い切る** のが規範である——ここは、丁寧なメールの感覚とあえて逆になる点に注意する。

### 公式手紙の完全な型

公式手紙(scrisoare oficială)は、上から下へ決まった順序で要素が並ぶ。架空の設例で全体像を示す。

```
[差出人ブロック:氏名・住所・連絡先]
Andrei Marinescu
Str. Exemplu nr. 10, București
Tel: 07XX XXX XXX

[日付・場所]
București, 15 septembrie 2026

În atenția: Departamentului Relații cu Clienții
[宛先機関・部署]
S.C. Exemplu Comercial S.R.L.

Ref: Reclamație privind comanda nr. 12345
[件名・参照番号]

Stimate domn / Stimată doamnă,

Prin prezenta, vă aduc la cunoștință următoarele...
[導入定型 → 本文]

...

Vă mulțumesc pentru atenția acordată și rămân în așteptarea
unui răspuns.

Cu deosebită considerație,
[結び]

Andrei Marinescu
[署名]
```

各要素の役割と定型を確認する。

- **În atenția...** /ɨn aˈten.t͡si.a/ = 〜宛(直訳「〜の注意に」)。後ろは属格。**În atenția domnului director**(部長殿宛)、**În atenția Departamentului...**(〜部宛)。日本語の「〜御中/〜殿」に近い。
- **Ref:** /ref/ = 参照・件名(referință / referitor la の略)。**Ref: Solicitare de adeverință**(件名:証明書の申請)。案件番号があれば添える。
- **Prin prezenta** /prin preˈzen.ta/ = 本書をもって(直訳「この(手紙)によって」。prezenta = scrisoarea prezentă「この手紙」の省略)。公式書面の導入定型の筆頭。**Prin prezenta, vă aduc la cunoștință...**(本書をもって、〜をお知らせいたします)、**Prin prezenta, solicit...**(本書をもって、〜を申請いたします)。
- 結びの定型は丁寧度で段階がある(下表)。

| 結び | IPA/発音 | 日本語 | 丁寧度 |
|---|---|---|---|
| **Cu deosebită considerație,** | /ku de.o.seˈbi.tə kon.si.deˈra.t͡si.e/ | 深い敬意をもって | 最上級(公式・目上・機関宛) |
| **Cu stimă,** | /ku ˈsti.mə/ | 敬具 | 標準的なフォーマル |
| **Cu respect,** | /ku resˈpekt/ | 敬具(敬意をもって) | フォーマル |
| **Cu considerație,** | /ku kon.si.deˈra.t͡si.e/ | 敬意をもって | フォーマル |

**Cu deosebită considerație**(深い敬意をもって)は、公式手紙の結びとして最も改まった形で、機関・要人宛の申請・苦情・推薦にふさわしい。ビジネスメールで学んだ **Cu stimă / Cu respect** より一段上の格式である。

### 導入定型のバリエーション

**Prin prezenta** に続ける遂行動詞を、目的別に持っておく。

| 目的 | 導入定型 | 日本語 |
|---|---|---|
| 通知する | **Prin prezenta, vă aduc la cunoștință faptul că...** | 本書をもって、〜の事実をお知らせいたします |
| 申請する | **Prin prezenta, solicit...** / **Prin prezenta, vă rog să...** | 本書をもって、〜を申請いたします |
| 宣言・申告する | **Subsemnatul, ..., declar pe propria răspundere că...** | 下名の者……は、自己の責任において〜を申告いたします |
| 苦情を述べる | **Prin prezenta, doresc să vă semnalez o problemă privind...** | 本書をもって、〜に関する問題を申し立てたく存じます |
| 推薦する | **Prin prezenta, îl/o recomand cu căldură pe...** | 本書をもって、〜を心より推薦いたします |

## モデル文書1 ― 苦情文(reclamație)

架空の設例:顧客 **Andrei Marinescu**(架空)が、架空の会社「**S.C. Exemplu Comercial S.R.L.**」に、届いた商品の不備について苦情を述べる書面。**人名・会社名・注文番号・日付はすべて架空** であり、実在の企業・取引とは無関係である。

---

**În atenția: Departamentului Relații cu Clienții**
**S.C. Exemplu Comercial S.R.L.**

**Ref: Reclamație privind comanda nr. 12345 din 3 septembrie 2026**

Stimată doamnă / Stimate domn,

Prin prezenta, doresc să vă semnalez o problemă privind comanda cu numărul 12345, plasată pe data de 3 septembrie 2026.

Produsul livrat nu corespunde descrierii de pe site: în locul modelului comandat, am primit un articol diferit. De asemenea, ambalajul prezenta deteriorări vizibile la momentul livrării.

În consecință, solicit fie înlocuirea produsului cu cel comandat inițial, fie restituirea sumei achitate, în conformitate cu politica de returnare afișată.

Anexez, în copie, factura și fotografiile produsului primit.

Vă mulțumesc pentru atenția acordată și rămân în așteptarea unui răspuns în cel mai scurt timp.

Cu deosebită considerație,
Andrei Marinescu

---

### 対訳

> **宛先:顧客対応部**
> **エグゼンプル・コメルシャル有限会社**
>
> **件名:2026年9月3日付 注文番号12345 に関する苦情**
>
> 拝啓
> 本書をもって、2026年9月3日に発注いたしました注文番号12345に関する問題を申し立てたく存じます。
> 納品された商品は、ウェブサイトの記載と一致しておりません。注文した型番の代わりに、異なる品が届きました。また、納品時点で梱包に明らかな損傷が見られました。
> つきましては、掲示されている返品ポリシーに従い、当初注文した商品との交換、または支払済み金額の返金のいずれかを要請いたします。
> 請求書および届いた商品の写真を、写しにて添付いたします。
> ご対応いただきますことに御礼申し上げ、できるだけ早いお返事をお待ちしております。
> 深い敬意をもって、アンドレイ・マリネスク

### 解説

- **Prin prezenta, doresc să vă semnalez o problemă...** ― 導入定型。a semnala(申し立てる・注意を促す)は苦情文の中核動詞。
- **nu corespunde descrierii** ― a corespunde(一致する)+ 与格 descrierii(記載に)。「記載と合わない」の定型。
- **În consecință, solicit fie ... fie ...** ― 「したがって、AかBのいずれかを要請する」。遂行動詞 **solicit** で要求を明示し、**fie A, fie B**(AかBか)で二つの救済を並べる。
- **în conformitate cu** ― 「〜に従って」。公式文書の頻出前置詞句(後述)。
- **Anexez ... în copie** ― a anexa(添付する)。書面では **Anexez / Atașez**、メールでは Atașat găsiți。
- 結びは最上級の **Cu deosebită considerație,**。苦情でも礼を失わない格式を保つのが公式文書の作法である。

## モデル文書2 ― 要望文(solicitare)

架空の設例:社員 **Elena Dumitru**(架空)が、勤務先の人事部に対して、証明書の発行を申請する書面。**人名・部署名・日付はすべて架空** である。

---

**În atenția: Departamentului de Resurse Umane**

**Ref: Solicitare de adeverință de salariat**

Stimată doamnă / Stimate domn,

Subsemnata, Elena Dumitru, angajată în cadrul companiei începând cu data de 1 martie 2024, vă adresez prezenta solicitare.

Prin prezenta, solicit eliberarea unei adeverințe care să ateste calitatea mea de salariat, necesară pentru un demers personal.

Vă rog să îmi comunicați termenul în care documentul poate fi eliberat.

Vă mulțumesc anticipat pentru amabilitate și pentru sprijinul acordat.

Cu stimă,
Elena Dumitru

---

### 対訳

> **宛先:人事部**
> **件名:在職証明書の申請**
>
> 拝啓
> 下名の者、エレナ・ドゥミトゥル(2024年3月1日より当社に在籍)は、本申請を差し上げます。
> 本書をもって、私が社員であることを証明する在職証明書の発行を申請いたします。個人的な手続きのために必要としております。
> 当該書類を発行いただける期限をお知らせいただけますようお願い申し上げます。
> ご親切とご支援に、あらかじめ御礼申し上げます。
> 敬具、エレナ・ドゥミトゥル

### 解説

- **Subsemnata, Elena Dumitru, ..., vă adresez...** ― **subsemnata**(女性)を主語に立て、氏名・在籍情報を同格で挟み、動詞は一人称 **vă adresez**(あなたに差し上げる)。非人称化された自己指示の典型。
- **solicit eliberarea unei adeverințe care să ateste...** ― 遂行動詞 solicit + 名詞化 **eliberarea**(発行)+ 関係節 **care să ateste**(〜を証明するような)。関係節に接続法 să を使い、「証明する目的の書類」という機能を表す(接続法関係節は後述)。
- **adeverință** /a.de.veˈrin.t͡sə/ = 証明書(在職・在学など事実を証する書面)。行政文書の型は次章で詳しく扱う。
- 依頼の丁寧さは **Vă rog să îmi comunicați...**(お知らせいただけますよう)で保ちつつ、申請本体は solicit で言い切る。この「申請=言い切り/付随の依頼=丁寧形」の二層構造が要望文の型である。
- 結びは **Cu stimă,**(社内・部署宛の標準的フォーマル)。苦情文の Cu deosebită considerație より一段軽い格でも自然。

## モデル文書3 ― 推薦状(scrisoare de recomandare)

架空の設例:元上司 **Mihai Ionescu**(架空)が、元部下 **Ana Popa**(架空)を推薦する書面。**人名・会社名・職位はすべて架空** であり、実在の人物・評価とは無関係である。

---

**Scrisoare de recomandare**

Subsemnatul, Mihai Ionescu, în calitate de fost coordonator de proiect la S.C. Exemplu Tehnologii S.R.L., recomand cu deosebită considerație pe doamna Ana Popa.

Am colaborat cu doamna Popa timp de trei ani, perioadă în care a demonstrat seriozitate, capacitate de organizare și o remarcabilă atenție la detalii. Proiectele care i-au fost încredințate au fost finalizate la termen și la un standard ridicat.

Consider că doamna Popa reprezintă un candidat valoros pentru orice echipă, motiv pentru care o recomand fără rezerve.

Rămân la dispoziție pentru eventuale clarificări.

Cu deosebită considerație,
Mihai Ionescu

---

### 対訳

> **推薦状**
>
> 下名の者、ミハイ・イオネスク(エグゼンプル・テクノロジー有限会社の元プロジェクト・コーディネーター)は、アナ・ポパ女史を深い敬意をもって推薦いたします。
> 私はポパ女史と3年間にわたり協働してまいりました。その間、彼女は誠実さ、組織力、そして細部への並外れた注意力を示しました。彼女に委ねられたプロジェクトは、いずれも期限内に、高い水準で完遂されました。
> ポパ女史は、いかなるチームにとっても価値ある人材であると考えます。それゆえ、私は留保なく彼女を推薦いたします。
> ご不明な点があれば、いつでも対応いたします。
> 深い敬意をもって、ミハイ・イオネスク

### 解説

- **Subsemnatul, ..., în calitate de ..., recomand ...** ― subsemnatul + **în calitate de**(〜の資格で・〜として)+ 遂行動詞 **recomand**(私は推薦する)。推薦者の立場を明示するのが推薦状の型。
- **recomand ... pe doamna Ana Popa** ― 人を目的語に取るとき前置詞 **pe** を付す(対格の pe)。**o recomand**(彼女を推薦する)と代名詞化もされる。
- **perioadă în care...** ― 関係節「〜した期間」。関係詞 **în care**(その中で)で先行詞 perioadă を受ける(関係節の入れ子は後述)。
- **Proiectele care i-au fost încredințate au fost finalizate...** ― 受動態の連鎖:**i-au fost încredințate**(彼女に委ねられた)+ **au fost finalizate**(完遂された)。受動+関係節で、行為者(誰が委ねたか)をぼかしつつ事実だけを述べる公式文体。
- **motiv pentru care...** ― 「それゆえ(その理由のために)」。前文を受けて結論へつなぐ関係節的接続。
- 推薦状では **fără rezerve**(留保なく)、**valoros**(価値ある)など、明確に肯定する評価語を用いる。ただし、事実に基づかない誇張は避け、具体的な根拠(3年間の協働・期限内完遂)を添えるのが誠実な推薦状の作法である(正直さの原則)。

## 文レベルの技法

### 受動+名詞化の連鎖

公式文体の密度は、**受動態と名詞化を鎖のようにつなぐ** ことで生まれる。B2 で学んだ名詞化(îmbunătățire, evitare)と受動(se recomandă / a fost finalizat)を、一文の中で連結する。

- (口語的)**Am trimis cererea și așteptăm ca ei să o aprobe.**(申請書を送って、彼らが承認するのを待っている)
- (公式・受動+名詞化)**Cererea a fost depusă, iar aprobarea acesteia este în curs de procesare.**(申請書は提出済みであり、その承認は処理中である)

ここでは、**a fost depusă**(提出された=受動)+ **aprobarea**(承認=名詞化)+ **procesare**(処理=名詞化)が連なり、「誰が」を一切言わずに手続きの状態だけを述べている。日本語の「申請書は提出済みであり、その承認は目下処理中である」という公文書調と発想が同じである。

**⚠ 過度な名詞化の注意**:名詞化は硬さを与えるが、連ねすぎると意味が取りにくくなる(いわゆる「名詞病」)。**Efectuarea verificării realizării obiectivelor**(目標達成の検証の実施……)のように名詞を数珠つなぎにすると、ネイティブでも読みにくい。**一文につき名詞化は要点だけ** にとどめ、動詞で処理できる部分は動詞で書くのが、読める公式文である。

### 長文の構造管理 ― 関係節とジェルンジウの入れ子

公式文書は一文が長い。長文を破綻させないための二つの入れ子構造を押さえる。

**① 関係節(propoziții relative)の入れ子**

関係代名詞 **care**(〜する/した)を軸に、先行詞へ節をぶら下げる。前置詞と結ぶときは **前置詞 + care**(în care「その中で」, pentru care「そのために」, la care「それに」)の形になる。

- **documentul care atestă calitatea de salariat** ― 社員資格を証明する書類(care = 主格)
- **perioada în care am colaborat** ― 私が協働した期間(în care = 前置詞 în + care)
- **motiv pentru care recomand** ― 私が推薦する理由(pentru care)

⚠ 関係節を二重・三重に入れ子にすると、どの先行詞にかかるかが曖昧になりやすい。**一つの先行詞に一つの関係節** を基本とし、必要なら文を切る。日本語の「〜するところの〜」を機械的に care で連鎖させると、かかり受けが崩れる。

**② ジェルンジウ(gerunziu)の入れ子**

ジェルンジウ(現在分詞・-ând/-ind 形、日本語の「〜しつつ・〜して」)は、主節に状況・理由・付帯を添える。公式文体では、二つの動作を一文にまとめる圧縮手段として使う。

- **Ținând cont de cele menționate, solicit...** ― 上記を考慮し、〜を要請する(a ține cont de = 考慮する)
- **Având în vedere situația descrisă, vă rog...** ― 述べた状況に鑑み、〜をお願いする(a avea în vedere = 勘案する)
- **Menționând că documentul este necesar urgent, ...** ― 当該書類が至急必要である旨を付言しつつ、〜

⚠ **ジェルンジウの主語は、原則として主節の主語と一致する。** **Solicitând adeverința, a fost aprobată cererea**(証明書を申請しつつ、申請が承認された)のように主語がずれると非文的になる。「申請したのは私」なのに承認されたのが「申請」では、ジェルンジウの主体と主節主語が食い違う。日本語話者は「〜して」を主語不問で付けがちなので、ここは意識して主語を揃える。

### 公式文体の頻出前置詞句

一文を硬く長く保つ接着剤として、次の前置詞句を覚える。いずれも書面の格を上げる定型である。

| 前置詞句 | IPA/発音 | 日本語 | 用法 |
|---|---|---|---|
| **în conformitate cu** | /ɨn kon.for.miˈta.te ku/ | 〜に従って・準拠して | 規定・方針に依拠する |
| **în conformitate cu prevederile...** | /... pre.veˈde.ri.le/ | 〜の規定に従い | 「規定(prevederi)」を明示 |
| **având în vedere** | /aˈvɨnd ɨn veˈde.re/ | 〜に鑑み・を勘案し | 前提・理由の提示 |
| **ținând cont de** | /t͡siˈnɨnd kont de/ | 〜を考慮して | 前提・理由の提示 |
| **în consecință** | /ɨn kon.seˈkin.t͡sə/ | したがって・その結果 | 帰結の導入 |
| **în ceea ce privește** | /ɨn ˈt͡ʃe̯a t͡ʃe priˈveʃ.te/ | 〜に関しては | 話題の提示(B2 既習) |
| **cu privire la** | /ku priˈvi.re la/ | 〜について・に関して | 話題の提示 |

## 日本語話者の癖 ― 過剰緩衝の直訳を避ける

日本語のフォーマル文は、依頼の前に緩衝表現を厚く重ねる。「**恐れ入りますが**」「**お忙しいところ誠に恐縮ですが**」「**ご多用の折**」「**差し支えなければ**」——これらは日本語の敬語文化では礼儀だが、**そのままルーマニア語に直訳すると、卑屈・冗長・不自然に響く** ことが多い。

- ❌(過剰緩衝の直訳)**Îmi cer scuze foarte mult că vă deranjez în acest moment atât de aglomerat, dar dacă nu vă supărați și dacă aveți amabilitatea, aș îndrăzni să vă rog...**（このように混み合った時期にお邪魔して大変申し訳ございません、もしご気分を害されず、ご親切がおありでしたら、あえてお願い申し上げたく……)
- ⭕(公式で簡潔)**Prin prezenta, vă solicit...** / **Vă rog să...**（本書をもって、〜を申請いたします / 〜をお願いいたします)

ルーマニア語の公式文書では、**敬意は「へりくだり」ではなく「定型」と「明晰さ」で示す**。**Prin prezenta**(本書をもって)という定型、**Cu deosebită considerație**(深い敬意をもって)という結び、**Vă mulțumesc anticipat**(あらかじめ御礼申し上げます)という一言——これらで礼は十分に尽くされる。用件は遂行動詞で明確に言い切る方が、むしろ相手の時間を尊重する誠実な書面になる。

一方、日本語の「恐れ入りますが」に **機能的に** 対応する、適量の緩衝表現は存在する。過不足なく使うなら次の程度でよい。

| 日本語の緩衝 | ルーマニア語の適量表現 | 注 |
|---|---|---|
| 恐れ入りますが | **Vă rog...** / **V-aș ruga...**（条件法で一段柔らかく) | 一文に一つで十分 |
| お手数をおかけしますが | **Vă mulțumesc anticipat pentru sprijin.** | 依頼の後に感謝で受ける |
| ご多用のところ | (基本は省略。必要なら)**Înțeleg că sunteți ocupați, însă...** | 催促・フォローアップ時のみ |
| 差し支えなければ | **Dacă este posibil, ...** | 相手に裁量を残す一言 |

要は、**緩衝は「重ねる」ものではなく「一つ選んで置く」もの**、という発想の転換である。日本語の丁寧さ=緩衝の厚み、という感覚を、ルーマニア語では定型と明晰さに置き換える。

## 重要ポイント

- 公式文体の三本柱:**非人称化(subsemnatul/subsemnata)/ 遂行動詞(solicit, declar, mă angajez)/ 定型性(Prin prezenta, Cu deosebită considerație)**
- **subsemnatul** を主語に立てても、動詞は **一人称**(Subsemnatul ... solicit)で受ける
- 公式手紙の型:**În atenția...(宛)→ Ref:(件名)→ Prin prezenta...(導入)→ 本文 → Cu deosebită considerație,(結び)**
- 申請・宣言は条件法でぼかさず、**遂行動詞の直説法** で言い切る(丁寧メールとは逆)。付随する依頼だけ Vă rog / 条件法で和らげる
- 文レベルの技法:**受動+名詞化の連鎖**(過度な名詞病は避ける)、**関係節(前置詞+care)とジェルンジウ(主節主語と一致)の入れ子**
- 前置詞句:**în conformitate cu / având în vedere / ținând cont de / în consecință / cu privire la**
- 日本語話者の癖:**過剰緩衝(「恐れ入りますが」の重ね)の直訳を避け、定型と明晰さで敬意を示す**

## よくある誤用

| ❌ 誤り | ✅ 正しく | 説明 |
|---|---|---|
| **Subsemnatul ... solicită**(subsemnatul を三人称で受ける) | **Subsemnatul ... solicit** | subsemnatul は主語だが動詞は一人称で受けるのが公式定型 |
| 男性が **Subsemnata** と書く / 女性が **Subsemnatul** | 男 **Subsemnatul** / 女 **Subsemnata** | 性で語尾が変わる(-ul / -a)。自分の性に一致させる |
| 申請本体を条件法でぼかす:**Aș solicita...**（〜を申請したいのですが) | **Solicit...**（申請いたします) | 公式の申請・宣言は遂行動詞の直説法で言い切る。ぼかすと効力が弱く見える |
| **Prin prezenta scrisoare prezentă**（prezenta を重複) | **Prin prezenta,**（これだけで「この手紙により」) | prezenta 自体が「この手紙」を指すので scrisoare を重ねない |
| 「恐れ入りますが」を **Îmi cer scuze foarte mult că...** と厚く直訳 | **Vă rog...** / **Prin prezenta, solicit...** | 過剰な謝罪・緩衝の重ねは不自然。定型と明晰さで礼を示す |
| ジェルンジウの主語が主節とずれる:**Solicitând adeverința, a fost aprobată cererea.** | **Solicitând adeverința, aștept aprobarea.**（主語を私に揃える) | ジェルンジウの主体は主節主語と一致させる |
| 名詞化の数珠つなぎ:**Efectuarea verificării realizării...** | 要点だけ名詞化し、残りは動詞で:**Se verifică dacă obiectivele au fost realizate.** | 名詞病を避け、読める密度に保つ |

## ネイティブらしい言い換え

| 教科書的(直訳調・過剰) | 公式文書で実際に使われる形 | 注 |
|---|---|---|
| Eu vreau să vă cer să... | **Prin prezenta, vă solicit să...** / **Solicit...** | 一人称 eu vreau を遂行動詞の定型に |
| Vă scriu această scrisoare pentru a vă spune că... | **Prin prezenta, vă aduc la cunoștință că...** | 「この手紙を書く」を Prin prezenta に圧縮 |
| Sunt foarte foarte supărat de produsul stricat. | **Produsul livrat prezintă defecte, motiv pentru care solicit...** | 感情の強調より、事実+要請の構造で |
| Vă mulțumesc din suflet foarte mult. | **Vă mulțumesc pentru atenția acordată.** | 過度な感情語を、対象を明示した定型に |
| La sfârșit vreau să spun mulțumesc și la revedere. | **Cu deosebită considerație,** | 結びは定型で。口語的な締めを避ける |

## フォーマル/カジュアルの違い

同じ「商品の不備について対応してほしい」という内容を、文書の格で並べる。

- **チャット・SMS(口語)**: Salut, produsul a venit stricat, ce fac?（やあ、商品壊れて届いた、どうすれば?)
- **メール(標準〜フォーマル)**: Bună ziua, produsul comandat a sosit deteriorat. Vă rog să îmi comunicați cum putem rezolva.（こんにちは、注文した商品が破損して届きました。解決方法をお知らせください)
- **公式書面(reclamație・最上級フォーマル)**: Prin prezenta, vă semnalez că produsul livrat prezintă deteriorări. În consecință, solicit înlocuirea sau restituirea sumei, în conformitate cu politica de returnare.（本書をもって、納品された商品に損傷がある旨を申し立てます。つきましては、返品ポリシーに従い、交換または返金を要請いたします)

右に行くほど、①一人称の生の主語が消え(ce fac → solicit)、②口語語彙(stricat)が正式語彙(deteriorat / deteriorări)になり、③遂行動詞と前置詞句(solicit / în conformitate cu)で行為と根拠を明示する。公式書面は「困っている」を伝えるのではなく、「何を要請するか・何に基づくか」を成立させる文書である——この差が、メールと公式書面の本質的な違いである。

## 文化的注意点

- 一般に、ルーマニアの公式書面では、**簡潔さ・明晰さ・定型の遵守** が評価される傾向があるとされる。長い感情的な訴えより、事実・要請・根拠を分けて明示する構成が実務的とされる(あくまで一般的傾向で、宛先・事案による差はある)。
- 一般に、苦情・要望であっても、結びの敬意表現(**Cu deosebită considerație**)や導入の定型(**Prin prezenta**)を省かないことが、書面の格を保つうえで期待されるとされる。感情の強さを敬意の欠如で表さないのが公式文書の作法とされる。
- 書式(余白・フォント・提出方法・押印/署名の要否)は宛先機関ごとに規定が異なりうる。本章は言語(文体・定型・構造)を扱うものであり、特定の書式規定を示すものではない。実際に提出する書面は、宛先の様式とネイティブ話者の確認に従うこと(免責)。

## 練習問題

### 問題1: 遂行動詞を選べ

次の目的に合う遂行動詞(一人称・現在形)を選べ。
1. 証明書の発行を「申請する」 →（solicit / declar / recomand）
2. 自己の責任において事実を「申告する」 →（solicit / declar / mă angajez）
3. 元部下を「推薦する」 →（solicit / recomand / confirm）
4. 期限内の納品を「誓約する」 →（declar / mă angajez / semnalez）

### 問題2: subsemnatul / subsemnata と一人称受け

次の書き出しを、性と動詞の活用に注意して完成させよ。
1. (女性が申請)___, Elena Dumitru, ___ eliberarea unei adeverințe.（solicit を活用)
2. (男性が申告)___, Andrei Marinescu, ___ pe propria răspundere că...（declar を活用)

### 問題3: 導入定型を入れよ

次の日本語に対応する導入定型を書け。
1.「本書をもって、〜の事実をお知らせいたします」
2.「本書をもって、〜を申請いたします」
3. 結びの最上級「深い敬意をもって」

### 問題4: 過剰緩衝を簡潔に

次の過剰緩衝の直訳を、公式で簡潔な依頼に書き換えよ。
> Îmi cer scuze foarte mult că vă deranjez, dar dacă nu vă supărați și dacă aveți amabilitatea, aș îndrăzni să vă rog să îmi trimiteți adeverința.

### 問題5: 短い要望文を書く

架空の設例で、次の内容を、**subsemnatul/subsemnata + 遂行動詞 + Prin prezenta + 適切な結び** を使った公式書面として書け(3〜5文)。
> 内容:自分(架空の氏名・在籍社員)が、人事部に、在職証明書(adeverință de salariat)の発行を申請する。個人的手続きに必要。発行期限を知りたい。
>（人名・部署はすべて架空とすること)

## 解答

### 問題1
1. **solicit**（申請する) 2. **declar**（申告する。declar pe propria răspundere) 3. **recomand**（推薦する) 4. **mă angajez**（誓約する)。
— それぞれ日本語の「申請/申告/推薦/誓約」に対応する遂行動詞。solicit=公式の要請、declar=宣言・申告、recomand=推薦、mă angajez=義務の引き受け。

### 問題2
1. **Subsemnata**, Elena Dumitru, **solicit** eliberarea unei adeverințe.（女性なので subsemnata、動詞は一人称 solicit)
2. **Subsemnatul**, Andrei Marinescu, **declar** pe propria răspundere că...（男性なので subsemnatul、動詞は一人称 declar)。
— subsemnatul/subsemnata は主語だが動詞は一人称で受ける。性の一致(-a / -ul)に注意。

### 問題3
1. **Prin prezenta, vă aduc la cunoștință faptul că...** 2. **Prin prezenta, solicit...** 3. **Cu deosebită considerație,**。
— Prin prezenta が導入定型の筆頭。通知は vă aduc la cunoștință、申請は solicit。結びの最上級は Cu deosebită considerație。

### 問題4(モデル解答)
> **Prin prezenta, vă rog să îmi eliberați adeverința.** / **Prin prezenta, solicit eliberarea adeverinței. Vă mulțumesc anticipat.**
> (本書をもって、証明書の発行をお願いいたします。あらかじめ御礼申し上げます)
— 過剰な謝罪(Îmi cer scuze foarte mult)・条件・緩衝の重ねを削り、Prin prezenta + 遂行動詞/Vă rog + 一言の感謝に圧縮する。敬意は定型と明晰さで示す。

### 問題5(モデル解答)
> **În atenția: Departamentului de Resurse Umane**
> **Ref: Solicitare de adeverință de salariat**
>
> Stimată doamnă / Stimate domn,
>
> Subsemnata, Maria Vasile, angajată în cadrul companiei, vă adresez prezenta solicitare. Prin prezenta, solicit eliberarea unei adeverințe de salariat, necesară pentru un demers personal. Vă rog să îmi comunicați termenul în care documentul poate fi eliberat.
>
> Cu stimă,
> Maria Vasile
—
採点ポイント:①非人称化(Subsemnata + 一人称 vă adresez / solicit)②遂行動詞 solicit で申請本体を言い切る ③導入定型 Prin prezenta ④件名 Ref: と宛先 În atenția: ⑤結び Cu stimă,(部署宛の標準的フォーマル)。付随の依頼(Vă rog să îmi comunicați)は丁寧形で和らげる二層構造。人名・部署は架空。

## 復習チェックリスト

- [ ] 非人称化(subsemnatul/subsemnata)を、性と一人称受けを守って使える
- [ ] 遂行動詞(solicit / declar / mă angajez / vă aduc la cunoștință / recomand)を書面の背骨として使える
- [ ] 公式手紙の型(În atenția / Ref: / Prin prezenta / Cu deosebită considerație)を組める
- [ ] 苦情文・要望文・推薦状を、目的に応じた構成で書ける
- [ ] 申請・宣言は遂行動詞で言い切り、付随する依頼だけ条件法/Vă rog で和らげられる
- [ ] 受動+名詞化の連鎖を、名詞病に陥らず適度な密度で書ける

## 次章への橋渡し

公式手紙の型と非人称化・遂行動詞を身につけたことで、「書面で行為を成立させる」文体の基礎ができた。次の[第4章:契約書と行政文書](./04-contracts-and-administration.md)では、この文体が最も硬く結晶した領域——契約書と行政文書——を、**読解の目的で** 扱う。本章で書いた solicit / în conformitate cu / subsemnatul は、契約書・申請書(cerere)・委任状(procură)の中で、定型言語の一部として繰り返し現れる。次章は「自分で書く」から「正確に読み解く」へ軸足を移し、法律用語の誤読リスクにも踏み込む。ただし、契約・行政は高リスク領域であり、次章はあくまで**言語の型の読解**にとどめ、法的な内容判断には踏み込まないことを、先に心に留めておこう。
