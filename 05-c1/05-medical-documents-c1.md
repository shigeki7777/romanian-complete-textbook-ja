# 第5章:医療文書【CEFR C1】

> **免責(強化)**: 本章は **語学学習を目的とした教材であり、医学的助言では一切ない。**
> 本章が扱うのは、医療文書に現れる **言語の型(構造・定型表現・略号・語彙)の読解** だけである。
> **検査値・診断名・治療内容の医学的な解釈には一切踏み込まない。**数値が正常か異常か、診断が何を意味するか、処方がどう作用するかは、本章では判断しない。
> 医療文書の内容の解釈は、**必ず資格を持つ医師・薬剤師などの医療専門家、および必要に応じて有資格の医療通訳者に委ねること。**
> 本章の理解だけで自己判断・自己治療をしてはならない。体調不良や緊急時には、必ず医療機関・救急サービス(112)に連絡すること。
> 本章に登場する患者名・医師名・機関名・診断名・検査値・日付・処方はすべて **語学学習用の架空の例示** であり、実在の個人・医療データとは一切関係がない。

## この章の目標

この章を終えると、次のことができるようになる。

- 医療文書の主要な種別(**scrisoare medicală / bilet de externare / rezultatele analizelor / rețetă**)の構造を **言語として読み分けられる**
- 文書に頻出する見出し構造(**diagnostic la internare/externare / antecedente personale / antecedente heredocolaterale / recomandări la externare**)を **識別できる**
- 検査結果票の項目名(**hemoglobină, glicemie** など)を **語として読める**(数値の医学的解釈はしない)
- 処方箋の略号(**Rp. / D.S.** など)が「存在する」ことを知り、読解の手がかりにできる
- 医療語の誤読リスク(**acut/cronic、benign/malign、a se administra の指示文脈** など)を認識し、慎重に読める
- **本章が言語の型の読解にとどまり、医学的判断には踏み込まないこと** を自覚して運用できる

- [医療の全体像](../medical/overview.md)(A2〜B1) — 受診の流れ(programare → consultație → rețetă / trimitere → farmacie)、家庭医(medic de familie)制度、**trimitere(紹介状)と rețetă(処方箋)の別物性**。本章の文書はこの流れの中で発行される
- [身体部位・臓器の語彙](../medical/anatomy.md)(A2〜B1) — 体の部位・臓器名。退院サマリー・検査票を読むときの土台になる

本章は C1 の読解技能のなかでも、契約・行政と並んで最も高リスクな領域を扱う。B2 では、患者は **医師の口頭の説明を理解し、質問で確定する** 側に立った。C1 の医療文書で新しいのは、その説明が **紙の文書** として、高密度の専門構造・略号・ラテン語系語彙で結晶した形を **読む** ことである。ただし——前章の契約と同じく——**本章の目的は「医療文書に何が書いてあるかを言語として読み解く」ことに厳格に限られ、「その数値が何を意味するか」「その診断・処方が正しいか」という医学的内容には立ち入らない。** 数値の解釈も、診断の判断も、本章はしない。それは医療専門家の領域である。

## 本文解説

### なぜ「読解のみ・解釈なし」なのか ― 本章のスコープ

医療文書は、書かれている一語一語が **人の健康・安全に直結する** 文書である。検査値の一つ、診断名の一語、処方の略号の一つを取り違えれば、重大な結果を招きうる。したがって、その内容を正しく **解釈** するには、語学力だけでは決定的に足りず、医学の専門知識が不可欠である。本章はその専門知識を提供しない。

そこで本章は、スコープを厳格に **言語の型の読解** に限定する。

- **扱う**:医療文書の構造(どこに何が書かれるか)、見出しの定型語彙、頻出略号の存在、項目名の読み(語として)
- **扱わない**:検査値が正常か異常か、診断名が何を意味するか、処方がどう作用するか、といった **医学的解釈・判断**

たとえば検査票に **Glicemie: [値]** とあるとき、本章が教えるのは「glicemie は『血糖(値)』という項目名の語である」ことだけである。その値が高いか低いか、何を意味するかは **一切判断しない**。B2 で学んだ **în limite normale**(正常範囲内)/ **ușor crescut**(やや高い)は「医師がそう述べた」ことの **聞き取り** だが、本章は **数値そのものの医学的判断** には踏み込まない。両者は別物である。

### 医療文書の主要種別 ― 構造で読み分ける

ルーマニア語圏で出会う代表的な医療文書を、**言語的な種別** として押さえる。まず「何という種類の文書か」を識別できることが読解の第一歩である。各語の意味は読解上の語義であり、その文書の医学的効力・正確さを保証するものではない。

| 文書種別 | IPA/発音 | 読解上の意味 |
|---|---|---|
| **scrisoare medicală** | /skriˈso̯a.re me.diˈka.lə/ | 診療情報提供書(直訳「医療の手紙」。医師間の情報伝達) |
| **bilet de externare** | /biˈlet de eks.terˈna.re/ | 退院サマリー(直訳「退院の紙片」) |
| **bilet de trimitere** | /biˈlet de triˈmi.te.re/ | 紹介状(既習 trimitere。専門医・検査へ送る) |
| **rezultatele analizelor** | /re.zulˈta.te.le a.na.liˈze.lor/ | 検査結果(直訳「分析の結果」。既習 analize) |
| **buletin de analize** | /buleˈtin de aˈna.li.ze/ | 検査結果票(直訳「分析の会報・票」) |
| **rețetă (medicală)** | /reˈt͡se.tə/ | 処方箋(既習。⚠「レシピ」との多義は B2 で扱った) |
| **fișă medicală** | /ˈfi.ʃə me.diˈka.lə/ | 診療録・カルテ(直訳「医療のカード・票」) |

この表は、医療文書を手にしたときの「地図」になる。**bilet**(紙片・票)という語が、externare(退院)や trimitere(紹介)と結んで種別を作る点に注目してほしい。**bilet de externare**(退院の紙 = 退院サマリー)、**bilet de trimitere**(紹介の紙 = 紹介状)のように、bilet + de + 名詞 で文書種別が組み立てられる。⚠ **buletin**(票・会報)は B1 で学んだ **buletin = 身分証の口語名** とは別語脈である。医療の **buletin de analize** は「検査結果票」であり、身分証ではない(後述の誤読リスク)。

### 診療情報提供書・退院サマリーの構造 ― 頻出の見出し

**scrisoare medicală**(診療情報提供書)や **bilet de externare**(退院サマリー)は、決まった見出しで構成されることが多い。この骨格を **読んで識別できる** ようにする。各見出しの意味は読解上のものであり、その内容の医学的判断はしない。

| 見出し語彙 | IPA/発音 | 読解上の意味 |
|---|---|---|
| **diagnostic la internare** | /di.aɡˈnos.tik la in.terˈna.re/ | 入院時診断(既習 a interna/internare) |
| **diagnostic la externare** | /di.aɡˈnos.tik la eks.terˈna.re/ | 退院時診断 |
| **motivul internării** | /moˈti.vul in.terˈnə.rij/ | 入院理由(直訳「入院の理由」) |
| **antecedente personale** | /an.te.t͡ʃeˈden.te per.soˈna.le/ | 既往歴(直訳「個人の先行事項」) |
| **antecedente heredocolaterale** | /... he.re.do.ko.la.teˈra.le/ | 家族歴(直訳「遺伝・傍系の先行事項」) |
| **istoricul bolii** | /isˈto.ri.kul ˈbo.lij/ | 現病歴(直訳「病の履歴」) |
| **examen clinic** | /eɡˈza.men ˈkli.nik/ | 臨床所見・診察所見 |
| **evoluție** | /e.voˈlu.t͡si.e/ | 経過(直訳「進展」) |
| **tratament (efectuat)** | /tra.taˈment/ | (実施した)治療。既習 tratament |
| **recomandări la externare** | /re.ko.manˈdərʲ la eks.terˈna.re/ | 退院時の指示・推奨(既習 a recomanda) |

これらは退院サマリーを上から下へ読むときの構造マーカーである。日本語の退院サマリーが「入院時診断」「既往歴」「現病歴」「入院後経過」「退院時処方・指導」と項目で進むのと同じ発想である。**見出しを識別できれば、どこに何が書いてあるかの当たりがつく** ——これが本章の読解の第一歩である。各項目の **内容の医学的意味は判断しない**。

**antecedente(既往・先行事項)の二種**は特に重要である。**antecedente personale**(個人の先行 = 既往歴、その人自身の過去の病歴)と、**antecedente heredocolaterale**(遺伝・傍系の先行 = 家族歴、血縁者の病歴)は、対で現れる定型見出しである。**heredocolateral** は hered-(遺伝)+ colateral(傍系の)という硬いラテン語系の複合語で、日常会話にはまず出ないが、医療文書では家族歴の見出しとして定型化している。読解では「antecedente personale = 本人の既往、antecedente heredocolaterale = 家族の病歴」と **かたまりで** 識別できればよい。⚠ しばしば **A.P.**(antecedente personale)、**A.H.C.**(antecedente heredocolaterale)と略記されることがある点も、読解の手がかりとして知っておく。

### 検査結果票を読む ― 項目名は語として読み、数値は解釈しない

**buletin de analize / rezultatele analizelor**(検査結果票)には、項目名・数値・単位・参照範囲が表形式で並ぶことが多い。本章が扱うのは **項目名を語として読める** ことだけである。**数値・単位・参照範囲の医学的な意味は一切判断しない。**

頻出する検査項目名を、**読み(語として)** に絞って挙げる。**これらの値が何を意味するか、正常か異常かは本章では一切扱わない**(免責)。

| 項目名(語) | IPA/発音 | 読解上の対応語 |
|---|---|---|
| **hemoglobină** | /he.mo.ɡloˈbi.nə/ | ヘモグロビン(しばしば **Hb** と略記) |
| **glicemie** | /ɡli.t͡ʃeˈmi.e/ | 血糖(値) |
| **colesterol** | /ko.les.teˈrol/ | コレステロール |
| **leucocite** | /le.u.koˈt͡ʃi.te/ | 白血球 |
| **trombocite** | /trom.boˈt͡ʃi.te/ | 血小板 |
| **creatinină** | /kre.a.tiˈni.nə/ | クレアチニン |
| **valori de referință** | /vaˈlorʲ de re.feˈrin.t͡sə/ | 参照範囲・基準値(直訳「参照の値」) |
| **unitate de măsură** | /u.niˈta.te de məˈsu.rə/ | 測定単位 |

読解の要点は、検査票の **列の構造** を言語として把握することである。典型的には「**項目名 | 結果(rezultat)| 単位(U.M. = unitate de măsură)| 参照範囲(valori de referință)**」の並びになる。この構造を読めれば「どの列が測定値で、どの列が基準値か」という **レイアウトの理解** はできる。しかし——**測定値が基準範囲に入っているか、それが健康上何を意味するかは、本章は判断しない。** 項目名を語として読み、判断は医師に委ねる——これが検査票読解の原則である。

### 処方箋の略号 ― 「存在の紹介」にとどめる

**rețetă**(処方箋)には、ラテン語由来の略号が用いられることがある。本章は、これらの略号が **存在すること** を紹介するにとどめ、**具体的な用法・用量の指示内容には踏み込まない。** 用法・用量は、必ず医師・薬剤師の指示と、処方箋そのものの記載に従うこと(免責)。

読解の手がかりとして、次のような略号が **存在しうる** ことを知っておく。ただし、**これらの解釈・実行は必ず薬剤師・医師に委ねる。**

| 略号(存在の紹介) | 由来の目安 | 読解上の位置づけ |
|---|---|---|
| **Rp.** | ラテン語 recipe(「取れ」) | 処方の開始を示す伝統的な記号として言及される |
| **D.S.** | ラテン語 detur, signetur 系 | 服用の指示部分に関わる記号として言及される |

⚠ **重要**:上記は「そういう略号が文書に現れることがある」という **存在の紹介** に過ぎない。**その略号に続く具体的な指示(何を・いつ・どれだけ)の解釈は、本章では一切扱わない。** 略号を見て自己判断で服薬してはならない。処方の内容は **必ず処方した医師・調剤する薬剤師に確認する。** 本章が教えるのは「Rp. や D.S. のような略号が処方箋に存在する」という言語的事実のみであり、それ以上ではない。

現代の処方箋の多くは電子的に発行され、用法が平文(**a se administra ...**「〜のように投与する」など)で書かれることも多い。この **a se administra** の読解には後述の ⚠ 注意がある(指示文脈での読み方)。

## ⚠ 医療語の誤読リスク

医療語は、綴りや語源が日常語・別語に近いために、取り違えると文書の理解が根本からずれ、実害に直結しうる。**確信の持てる、頻出かつ重要な誤読リスクを挙げる。** B2 で扱った ⚠(analiză の多義 / a interna・internare / efecte secundare / rețetă の多義)とは **重複させず**、本章の文書読解で新たに要注意となる語に絞る。

- **⚠ acut と cronic(急性 / 慢性)** ― **acut** /aˈkut/ は「急性の・急激な」、**cronic** /ˈkro.nik/ は「慢性の・長期の」を指す語で、経過の時間的性格を表す対語である。医療文書では **afecțiune acută**(急性の疾患)/ **afecțiune cronică**(慢性の疾患)のように現れる。⚠ 日本語話者は **acut** を英語 acute の「鋭い・激しい」の連想で「重症」と、**cronic** を「軽い・慢性=たいしたことない」と **重症度の語** に読み替えがちだが、これらは本来 **時間的経過(急性/慢性)** を表す語であって、重症度そのものを述べる語ではない。**読解上、acut/cronic は「経過が急か長期か」を示す語** として読む。ただし、**その疾患が実際にどの程度重いか・どう対応すべきかは医学的判断であり、本章は一切扱わない。**

- **⚠ benign と malign(良性 / 悪性)** ― **benign** /beˈniɡn/ は「良性の」、**malign** /maˈliɡn/ は「悪性の」を指す語である。**formațiune benignă**(良性の腫瘤)/ **formațiune malignă**(悪性の腫瘤)のように現れる。⚠ この二語は綴りが近く(benign / malign、語尾 -ign が共通)、**取り違えると意味が正反対** になる、極めて誤読リスクの高い対語である。日本語話者は見慣れない語形のため両者を混同しやすい。読解では **benign = 良性 / malign = 悪性** と、対で・確実に区別する。⚠⚠ **ただし、この語が文書に現れた場合、その意味の重大性ゆえに、内容の理解・説明は必ず医師に委ねること。素人が文書だけを見て判断・説明してはならない**(免責。本章は語の区別を示すにとどまる)。

- **⚠ a se administra の指示文脈(「投与する」の受動・非人称)** ― **a se administra** /a se ad.mi.nisˈtra/ は B2 で「投与される」(se administrează)として触れたが、処方・指示の文脈では **不定詞 a se administra** の形で「〜のように投与すること/投与すべし」という **指示** を表す。**a se administra de două ori pe zi**(1日2回投与のこと)のように、用法の指示部分に現れる。⚠ ここで注意すべきは、これは **一般的・非人称の指示表現** であって、「誰が飲むか」を言っていない点である。日本語話者は se を反射的に「自分で(自らに)」と読みがちだが、この se は **非人称・受動の se**(一般に〜される/すべし)である。**読解上「a se administra = (この薬は)〜のように投与すること、という指示だ」と受け取る。** ただし——繰り返すが——**その具体的な用量・回数・タイミングの解釈と実行は、必ず医師・薬剤師に確認する。** 本章は「a se administra が指示の定型である」という言語的事実を示すにとどめ、指示内容そのものは判断しない。

- **⚠ recomandări(推奨・指示)を「任意のおすすめ」と軽く読まない** ― **recomandări** /re.ko.manˈdərʲ/(単 recomandare)は「推奨・指示」で、**recomandări la externare**(退院時の指示)のように退院サマリーの重要な見出しになる。⚠ 日本語話者は recomandare を「(してもしなくてもよい)おすすめ」程度に軽く読みがちだが、医療文書の recomandări は **退院後に従うべき指示** として書かれていることが多い。**読解上「recomandări = 従うべき指示・推奨事項」と重く受け取る。** ただし、その指示の内容・実行方法は必ず医師・薬剤師に確認すること(本章は語の重みを示すにとどめ、指示内容は判断しない)。

- **⚠ diagnostic(診断 / 診断名)と prognostic(予後)** ― **diagnostic** /di.aɡˈnos.tik/ は「診断(名)」、**prognostic** /pro.ɡˈnos.tik/ は「予後(病気の今後の見通し)」を指す別語である。綴りが近い(diagnostic / prognostic、語尾 -gnostic が共通)ため混同しやすい。⚠ **diagnostic** は「今どういう状態か(診断名)」、**prognostic** は「今後どうなる見通しか(予後)」で、指すものが異なる。文書で **diagnostic la externare** とあれば「退院時診断(名)」、**prognostic** とあれば「予後(見通し)」と読み分ける。両語の医学的内容の判断は本章では扱わない。

- **⚠ pacient(患者)と pancreas / 医療の似た語形に引きずられない(綴りの近さへの一般的注意)** ― 医療文書はラテン語系の専門語が密集し、日本語話者には見慣れない語形が並ぶ。**acut/cronic、benign/malign、diagnostic/prognostic** のように、**綴りが近く意味が正反対・別物** の対語が多いのが医療語の特徴である。⚠ **一般的な読解姿勢として、語形が似ているというだけで意味を推測せず、対語・別語の可能性を常に疑う。** 意味に確信が持てない語は、推測で確定せず、必ず医療専門家・医療通訳者・辞書で確認する——これが医療文書読解の鉄則である(本章の原則)。

## モデル文書 ― 退院サマリーの抜粋(架空・読解用)

架空の設例:架空の患者「**Ion Radu**」(架空)が、架空の医療機関から受け取る **bilet de externare**(退院サマリー)の **抜粋**。**患者名・医師名・機関名・診断名・検査値・日付・処方はすべて語学学習用の架空の例示** であり、実在の個人・医療データとは一切関係がない。**この文書は言語の型を読解する練習用であり、医学的な参照・判断の材料にしてはならない。** 内容の解釈は必ず医療専門家に委ねること。

### 読解解説(言語の型のみ)

- **Diagnostic la internare / la externare** ― 入院時/退院時の診断(名)の見出し。既習 a interna/internare(入院)+ externare(退院)。⚠ diagnostic は prognostic(予後)と読み分ける(誤読リスク)。**診断名の内容は架空であり、医学的意味は持たない。**
- **Motivul internării** ― 入院理由(直訳「入院の理由」)。属格 internării(第4章・B1 の属格の応用)。
- **Antecedente personale (A.P.) / heredocolaterale (A.H.C.)** ― 既往歴/家族歴の対の見出し。略号 A.P. / A.H.C. の存在も読解の手がかり。⚠ heredocolateral は硬いラテン語系複合語。
- **Pe parcursul internării, evoluția a fost favorabilă** ― 「入院期間を通じて経過は良好だった」。pe parcursul(〜の間ずっと)+ evoluție(経過)+ 複合過去 a fost favorabilă。**経過の記述の定型を読解するにとどめ、医学的評価はしない。**
- **Rezultatele analizelor** ― 検査結果。項目名(hemoglobină / glicemie / colesterol)は **語として読める**。⚠ **値はすべて架空で、正常か異常かは一切判断しない**(免責)。
- **Se recomandă control periodic ... Tratamentul se administrează conform indicațiilor medicului** ― se 受動(B2 既習)+ recomandări(⚠ 従うべき指示として重く読む)+ **a se administra**(⚠ 非人称の投与指示)。**conform indicațiilor medicului**(医師の指示に従って)という語句自体が、**まさに本章が踏み込まない「内容の判断は医師へ」を文書内で指し示している。** 読解では「ここは医師の指示に委ねる領域だ」と認識して先へ進む。

**この抜粋の読み方**:見出し(Diagnostic / Motivul / Antecedente / Evoluție / Rezultate / Recomandări)で文書構造を掴み、各項目の定型(se recomandă / a se administra / evoluția a fost favorabilă)で「何を述べている項目か」を識別する。**各項目の医学的内容——診断名の意味、検査値の評価、指示の妥当性——は一切判断せず、言語の型の読解にとどめる。** 実際の退院サマリーを読む場面では、内容の理解・実行は必ず医師・薬剤師・有資格の医療通訳者に委ねること。

## 医療通訳の入口 ― 逐次通訳の心得3点(本格は専門章へ)

日本語話者がルーマニアで生活すると、医療文書や医師の説明を、ルーマニア語が不得手な家族などに **口頭で橋渡し** する場面が起こりうる。B2([第5章:医療の説明を理解する](../04-b2/05-medical-explanations-b2.md))で「日常的な付き添いレベルの橋渡し」を扱ったが、医療文書が絡むと正確さの要求は一段と上がる。ここでは **逐次通訳(interpretariat consecutiv)の入口** として、心得を **3点** だけ示す。**本格的な医療通訳の技術・倫理・中立性・責任は本章の範囲外であり、`medical/medical-interpretation.md` に送る。**

1. **推測で訳さない(不確かなら必ず確認する)** ― 意味に確信の持てない語・数値・略号は、**推測で訳さず、必ず医師・薬剤師に確認してから伝える。** 医療では「たぶんこういう意味」の推測が実害を生む。**Ce înseamnă acest termen? Puteți să confirmați?**(この語はどういう意味ですか。確認していただけますか)を挟む手間を惜しまない。これが医療通訳の入口における最重要の心得である。

2. **足さない・引かない・変えない(中立に、そのまま訳す)** ― 逐次通訳では、**話し手の言ったことを、足さず・引かず・勝手に和らげたり強めたりせず** に訳すのが原則とされる。聞き手が不安がるからと内容を軽く言い換える、専門用語を自己判断で「意訳」する、といったことは避ける。分かりにくければ **話し手に平易化を依頼する**(**Puteți să spuneți mai simplu?**)のであって、通訳者が勝手に補ってはならない。

3. **自分の限界と役割を自覚する(重大な場面は有資格者へ)** ― 手術の同意、投薬量、検査値の解釈、悪性/良性(benign/malign)の説明といった、**誤りが命に直結する場面** では、善意の付き添いではなく **有資格の医療通訳者・医療者を介する。** 「自分は語学の橋渡しはできるが、医学的判断はできない」という役割の線引きを、通訳する側が明確に持つことが、この場面での誠実さである。

⚠ 繰り返す。上記は **逐次通訳の入口** の3点にすぎない。**職業的な医療通訳の技術・倫理・中立性の詳細は本章では扱わず、`medical/medical-interpretation.md` に委ねる。** 本章はあくまで医療文書の **言語的な読解** を扱う教材である。

## 重要ポイント

- **本章は言語の型の読解のみ。検査値・診断・処方の医学的解釈には一切踏み込まない。内容の判断は必ず医療専門家に委ねる**
- 文書種別:**scrisoare medicală(診療情報提供書)/ bilet de externare(退院サマリー)/ rezultatele analizelor・buletin de analize(検査結果票)/ rețetă(処方箋)**。**bilet de + 名詞** で種別が作られる
- 退院サマリーの見出し:**diagnostic la internare/externare / motivul internării / antecedente personale(A.P.)/ antecedente heredocolaterale(A.H.C.)/ evoluție / recomandări la externare**
- 検査票は **項目名を語として読む(hemoglobină, glicemie, colesterol...)。数値・参照範囲の医学的解釈はしない**
- 処方箋の略号(**Rp. / D.S.**)は **存在の紹介** にとどめ、指示内容は医師・薬剤師に委ねる
- ⚠ 誤読リスク:**acut/cronic(急性/慢性・重症度ではない)、benign/malign(良性/悪性・正反対)、a se administra(非人称の投与指示)、recomandări(従うべき指示)、diagnostic/prognostic(診断/予後)**
- 医療通訳の入口3点:**推測で訳さない / 足さない引かない変えない / 限界を自覚し重大場面は有資格者へ**(本格は `medical/medical-interpretation.md`)

## 文法メモ ― 医療文書に頻出する形

| 形 | 意味 | 例 | どこで読むか |
|---|---|---|---|
| **[名詞] la internare / la externare** | 入院時/退院時の〜 | diagnostic la externare | 診断・所見の見出し |
| **antecedente personale / heredocolaterale** | 既往歴/家族歴 | A.P. / A.H.C. | 病歴の見出し |
| **se + 動詞(3人称)** | 一般に〜される(非人称受動・B2 既習) | se recomandă / se administrează | 指示・推奨 |
| **a se + 不定詞** | 〜すること・〜すべし(非人称の指示) | a se administra / a se confirma | 処方・指示 |
| **属格 -ului / -ării** | 〜の | motivul internării, indicațiilor medicului | 理由・出典の明示 |
| **conform + 与格** | 〜に従って | conform indicațiilor | 指示への従属 |

**a se + 不定詞**(a se administra, a se interpreta)は、本章で新しく読解対象になる「非人称の指示」の形である。se 受動(B2)の不定詞版で、「(一般に)〜すること/〜すべし」という指示・注意書きの定型として、処方・検査票・指示欄に現れる。⚠ 主体(誰が)を言わない点は se 受動と同じで、**内容の実行は文書の指示と医療専門家の確認に委ねる。**

| **benign** と **malign** を混同する | benign=良性 / malign=悪性(**正反対**) | 綴りが近く誤読リスク最大。内容説明は必ず医師に委ねる ⚠ |
| **diagnostic** と **prognostic** を同じと読む | diagnostic=診断(名) / prognostic=予後(見通し) | 指すものが別。-gnostic が共通で紛らわしい ⚠ |
| **a se administra** の se を「自分で(自らに)」と読む | 非人称・受動の se(一般に〜すべし・投与のこと) | 「誰が」を言わない指示定型。用量は医師・薬剤師へ ⚠ |
| **buletin de analize** の buletin を「身分証」と読む | 医療の buletin=(検査結果)票 | B1 の buletin=身分証(口語)とは別語脈 ⚠ |
| **recomandări** を「任意のおすすめ」と軽く読む | 従うべき指示・推奨事項 | 退院後に従う指示として重く読む。実行法は医師へ ⚠ |
| 検査票の数値を自分で「正常/異常」と判定する | **項目名を語として読むにとどめ、数値の解釈はしない** | 数値の医学的判断は本章の範囲外。必ず医師へ(免責) |
| 略号 Rp./D.S. の指示内容を自己解釈して服薬する | 略号の**存在を知る**にとどめ、内容は薬剤師・医師に確認 | 用法・用量の自己判断は危険。免責 |

## ⚠ 誤読・混同に注意する語(まとめ)

上の「よくある誤用」と重複するが、確信の持てる誤読リスクを、判別の指針とともに集約する。**いずれも、語の区別を示すにとどまり、医学的内容の判断はしない。**

- **⚠ acut / cronic** ― 急性/慢性(**経過の語**)。重症度の語ではない。重症度・対応は医師の判断。
- **⚠ benign / malign** ― 良性/悪性(**正反対**)。綴りが近く誤読高リスク。⚠⚠ 内容の理解・説明は必ず医師に委ねる。
- **⚠ diagnostic / prognostic** ― 診断(名)/予後(見通し)。-gnostic が共通で紛らわしい。指すものが別。
- **⚠ a se administra** ― 非人称の投与指示(「〜のように投与のこと」)。se を「自分で」と読まない。用量・回数の解釈は医師・薬剤師へ。
- **⚠ buletin de analize** ― (検査結果)票。B1 の buletin=身分証(口語)とは別語脈。
- **⚠ recomandări** ― 従うべき指示・推奨。「任意のおすすめ」と軽く読まない。実行法は医師へ。

## ネイティブらしい理解のコツ

医療文書は、ネイティブでも一読で確定しにくい高密度・専門語密集の文体である。読解の実務的なコツを挙げる。**いずれも「言語として読む」ためのコツであり、医学的判断のためのものではない。**

| つまずき | 読解のコツ |
|---|---|
| 略号(A.P. / A.H.C. / Hb / Rp. / D.S.)が読めない | 略号の**存在と対応**を手がかりにするが、指示内容は医師・薬剤師に確認する |
| ラテン語系の専門語が並んで意味が取れない | 見出し構造(diagnostic / antecedente / evoluție / recomandări)で文書の骨格を先に掴む |
| 検査票の数値に目が行く | **項目名を語として読むにとどめ、数値の評価は医師に委ねる**(本章の原則) |
| 似た語形(benign/malign, acut/cronic, diagnostic/prognostic)で迷う | 語形の近さで推測せず、**対語・別語**として確実に区別。確信がなければ確認 |
| a se + 不定詞の指示が誰宛か分からない | 非人称の指示(一般に〜すべし)と読む。実行は文書の指示と医師の確認に従う |
| 意味に確信が持てない | **推測で確定せず、医師・薬剤師・有資格の医療通訳者・辞書に確認する** |

最後のコツが最も重要である。医療文書は、**読み違いが人の健康・安全に直結する** 領域である。前章の契約で「分からなければ弁護士・公証人・ネイティブに確認する」と学んだのと同じ姿勢で、医療文書では「分からなければ医師・薬剤師・有資格の医療通訳者に確認する」を徹底する。本章はその確認を **語学の側から支える** ものであって、確認を **省くための** ものではない。

## 文化的注意点

- 一般に、ルーマニアの医療文書は、**ラテン語由来の専門語・略号が多用され、定型的な見出し構造(diagnostic / antecedente / evoluție / recomandări)を持つ** 傾向があるとされる(あくまで一般的傾向で、文書・機関・電子化の程度による差はある)。近年は電子処方箋・電子カルテの普及で書式が変わりうるため、本章は特定の書式・様式を規定するものではない。
- 一般に、退院や専門医紹介の際に **bilet de externare(退院サマリー)や trimitere(紹介状)** といった文書が発行される流れが言及されるが、その発行の要否・手続き・効力は制度の領域であり、本章では扱わない([医療の全体像](../medical/overview.md)の骨格を参照)。文書に **CNAS / asigurat / card de sănătate** といった語が現れたら「保険・制度に関わる文書だ」と **言語的に認識** し、制度の実務は一次情報・専門家に委ねる。
- 医療文書の内容の解釈・実行(検査値の評価、診断の理解、処方の実行)は、**必ず資格を持つ医療専門家に委ねる。** 家族への橋渡しが必要な重大な場面では、有資格の医療通訳者を介することが望ましい(職業通訳の技術・倫理は `medical/medical-interpretation.md`)。**本章は言語(構造・定型・略号・語彙)の読解を扱うものであり、医学的判断を代替するものではない**(強化免責の再確認)。

## 練習問題(読解のみ・医学的判断を求めない)

### 問題1: 文書種別を識別せよ

次の文書種別の読解上の意味を答えよ。
1. **bilet de externare** 2. **scrisoare medicală** 3. **buletin de analize** 4. **bilet de trimitere** 5. **rețetă**

### 問題2: 退院サマリーの見出しを識別せよ

次の見出しが「何を書く欄か」を答えよ(内容の医学的判断は不要)。
1. **diagnostic la internare** 2. **antecedente personale** 3. **antecedente heredocolaterale** 4. **evoluție** 5. **recomandări la externare**

### 問題3: ⚠ 誤読リスク ― 正しい読みを選べ

1. **acut** は(重症 / 急性)。**cronic** は(軽症 / 慢性)。
2. **benign** は(良性 / 悪性)。**malign** は(良性 / 悪性)。
3. **diagnostic** は(診断名 / 予後)。**prognostic** は(診断名 / 予後)。
4. **a se administra** の se は(自分で / 非人称の「一般に〜すべし」)。
5. **buletin de analize** の buletin は(身分証 / 検査結果票)。

### 問題4: 検査項目名を語として読め

### 問題5: 略号の位置づけ

2. 処方箋の **Rp. / D.S.** について、本章が教える範囲は何か(1文で)。

### 問題6: 医療通訳の入口

医療文書を家族に口頭で橋渡しするとき、本章が示した逐次通訳の心得3点を挙げよ。

## 解答

### 問題1
1. 退院サマリー 2. 診療情報提供書 3. 検査結果票 4. 紹介状 5. 処方箋。
— **bilet de + 名詞** で文書種別が作られる(bilet de externare / de trimitere)。buletin de analize は「検査結果票」で、B1 の buletin=身分証とは別語脈 ⚠。

### 問題2
1. 入院時診断(名)を書く欄 2. 既往歴(本人の過去の病歴) 3. 家族歴(血縁者の病歴) 4. (入院中の)経過 5. 退院時の指示・推奨。
— いずれも文書の構造を示す見出し。**内容の医学的意味は判断しない**(本章の原則)。A.P./A.H.C. は既往歴/家族歴の略。

### 問題3
1. **急性 / 慢性**(acut=急性、cronic=慢性。**経過**の語で、重症度ではない ⚠) 2. **良性 / 悪性**(benign=良性、malign=悪性。⚠ 正反対・誤読高リスク) 3. **診断名 / 予後**(diagnostic=診断名、prognostic=予後) 4. **非人称の「一般に〜すべし」**(se を「自分で」と読まない) 5. **検査結果票**。
— すべて本章 ⚠ の要点。語の区別を示すにとどまり、**内容の医学的判断はしない**。benign/malign が現れたら内容は必ず医師に委ねる。

### 問題4
1. ヘモグロビン(略 Hb) 2. 血糖(値) 3. コレステロール 4. 参照範囲・基準値。
— **項目名を語として読むにとどめる**。⚠ その値が正常か異常か、何を意味するかは本章では一切判断しない(免責)。

### 問題5
1. **antecedente personale(既往歴)** と **antecedente heredocolaterale(家族歴)** の略。
2. 「**Rp. / D.S. のような略号が処方箋に存在する** という言語的事実を紹介するにとどまり、その指示内容(用法・用量)の解釈は必ず医師・薬剤師に委ねる」。
— 略号は読解の手がかりとしての**存在の紹介**まで。自己判断で服薬しない(免責)。

### 問題6
1. **推測で訳さない**(不確かなら必ず医師・薬剤師に確認する)。
2. **足さない・引かない・変えない**(中立にそのまま訳す。分かりにくければ話し手に平易化を頼む)。
3. **自分の限界と役割を自覚する**(誤りが命に直結する重大場面は有資格の医療通訳者・医療者を介する)。
— 本格的な医療通訳の技術・倫理は本章の範囲外(`medical/medical-interpretation.md` 送り)。

## 次章への橋渡し

契約・行政に続き、医療文書という高リスクな読解対象を、**言語の型に厳格に限定して** 扱った。ここで貫いた原則——**言語の型は読み解くが、内容の判断(法的・医学的)は専門家に委ねる**——は、C1 の読解章に共通する背骨である。次の[第6章:学術・職業的言説](./06-academic-and-professional-discourse.md)では、読解から **産出・対話** へ軸足を移し、専門的な話題を根拠と反論を組み立てながら議論する技能へ進む。本章で読んだ高密度の専門語彙・非人称構文(se 受動・a se + 不定詞)は、学術・職業的な言説を **話す・書く** 場面でも道具として生きる。ただし、医療・法律のような高リスク領域では、専門的に語れるようになっても **内容判断は専門家に委ねる** という本章までの姿勢を、C1 全体を通じて保ってほしい。
