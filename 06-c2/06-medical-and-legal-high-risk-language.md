# 第6章:医療・法律の高リスク言語【CEFR C2】

> **免責(最強度)**: 本章は **語学学習を目的とした教材であり、医学的・法的助言では一切ない。**
> 本章が訓練するのは、高リスク場面で **言語の精度(曖昧さの排除・厳密な言い換え・正確な確認)** を扱う力だけである。
> **診断・治療・投薬・法的効力・権利義務・手続きといった内容の判断には一切踏み込まない。**
> 医療・法律の実務(通訳・翻訳・診療・法律相談・署名・同意)は、**必ず資格を持つ医師・薬剤師・弁護士・公証人、および有資格の医療通訳者・法廷通訳者、ならびにネイティブ話者の確認を経ること。**
> 本章の理解だけで、医学的・法的な判断を下したり、同意・拒否の意思表示を確定したり、他者に説明・助言したりしてはならない。
> 体調不良・緊急時には必ず医療機関・救急サービス(112)に連絡すること。法的な判断が必要な場面では必ず弁護士等に相談すること。
> 本章に登場する患者・当事者・症例・条項・数値・日付・氏名・機関名はすべて **語学学習用の架空の設例** であり、実在の個人・医療データ・契約・法令とは一切関係がない。

## この章の目標

この章を終えると、次のことができるようになる。

- 高リスク場面で **許されない曖昧表現**(cam / vreo / parcă など)を認識し、**精密表現**(exact / în mod precis / conform documentului)に言い換えられる
- 法文に頻出する **義務・許可・禁止** の言語(trebuie să / este obligat să / este interzis să / are dreptul să など)を、強度の階層として **読み分けられる**
- **同意の言語**(consimțământ informat)の言語的構造を把握し、同意・撤回の表現を **正確に読解** できる(意思の代行はしない)
- 数字・単位・日付の **絶対精度** が要る場面で、読み合わせ確認の定型(Permiteți-mi să confirm...)を運用できる
- 高リスク通訳の言語運用原則(追加・省略・推測の禁止、不明時の確認、一人称直訳)を理解する(職業倫理の詳細は専門章へ)
- **本章が「言語の精度」の訓練にとどまり、医学的・法的内容の判断には踏み込まないこと** を自覚して運用できる

## 本文解説

### なぜ「言語の精度」だけなのか ― 本章のスコープ

医療・法律の言語は、一語の曖昧さが誤診・誤投薬・誤った権利放棄に直結しうる。だからこそ、この領域で語学者が担えるのは **言語の精度**——曖昧さを排し、義務や同意を一義に読み取り、数値を正確に確認する——までである。**その内容が医学的・法的に妥当かどうかの判断は、語学の外にある。** 本章はこの線を、これまでのどの章よりも強く引く。

- **扱う**:曖昧表現の排除、義務・許可・禁止・同意を表す言語形式の識別と読解、数値・日付の確認の言い回し、高リスク通訳の言語運用原則。
- **扱わない**:診断・治療・投薬・予後の医学的判断、契約・同意・権利義務の法的効力の判断、意思表示の代行、実務上の助言。

たとえば **este obligat să...**(〜する義務を負う)という形を見たとき、本章が教えるのは「これは **義務** を表す最も強い言語形式の一つだ」という **言語的事実** である。その義務が法的に有効か、どんな条件で発生するかは **法律の領域** であり、本章は答えない。

### 曖昧さの排除 ― 高リスク場面で避けるべき語と、精密表現への言い換え

日常会話では便利な曖昧表現が、高リスク場面では **危険** になる。話者が意味を丸めた語をそのまま通せば、聞き手・読み手が誤解する余地を残す。避けるべき曖昧語と、精密表現への言い換えを対比する。

| ⚠ 避けたい曖昧表現 | IPA/発音 | 曖昧さの内容 | ✅ 精密表現への言い換え |
|---|---|---|---|
| **cam** | /kam/ | 「だいたい・およそ」。量・程度をぼかす | **exact**(正確に)/ 具体的な数値を示す |
| **vreo / vreun** | /ˈvre.o/ | 「〜くらい・およそ(数)」。数量をぼかす | **exact [数]** / **precis [数]**(正確に〜) |
| **parcă** | /ˈpar.kə/ | 「〜のような気がする・たぶん」。話者の不確かさ | **conform documentului**(文書によれば)/ **este confirmat că**(〜と確認されている) |
| **cred că** | /kred kə/ | 「〜と思う」。主観的推測 | **este stabilit că / se precizează că**(〜と定められている/明記されている) |
| **pe undeva / cumva** | /pe unˈde.va/ | 「どこかで・なんとなく」 | 具体的な場所・方法を明示する |
| **un fel de** | /un fel de/ | 「一種の・〜みたいな」。曖昧なぼかし | 正確な名称・定義を示す |
| **și așa mai departe** | /ʃi aˈʃa maj deˈpar.te/ | 「〜など・など等」。列挙をぼかす | 列挙を **省略せず** 明示する |

言い換えの中核となる **精密表現の道具** を押さえておく。

- **exact** /eɡˈzakt/ =「正確に・ちょうど」。**exact 15 mg**(正確に15 mg)。
- **în mod precis** /ɨn mod preˈt͡ʃis/ =「精密に・厳密に」。硬い書き言葉で「曖昧さなく」を示す。
- **conform documentului / conform indicațiilor** /konˈform/ =「文書に従い/指示に従い」。**話者の推測でなく、文書という一次情報に根拠を置く** ことを明示する。前章までで見た conform(+ 与格)の応用。
- **se precizează că / se stabilește că** =「〜と明記されている/定められている」。se 受動(既習)で **主観を排し、文書の記載に帰す**。

⚠ 原則:高リスク場面では、**話者の主観(cred că / parcă)** より **文書・事実への帰属(conform... / se precizează că...)** を優先する。「私はこう思う」ではなく「文書にはこう書かれている」に言い換えることで、責任の所在と情報の出所が明確になる。**ただし、その文書の内容が医学的・法的に妥当かは、本章は判断しない**(免責)。

### 義務・許可・禁止の言語 ― 強度の階層(法文の読解)

法文・規則・指示文には、**しなければならない/してよい/してはならない** を表す言語形式が繰り返し現れる。これらは **強度(拘束の強さ)** に階層がある。読解では、どの形式が **どれくらい強い拘束** を言語的に示しているかを見分ける。**その拘束が法的にどう働くかは法律の領域であり、本章は判断しない。**

| 言語形式 | IPA/発音 | 読解上の強度・意味 |
|---|---|---|
| **este interzis să / este interzisă** | /ˈes.te in.terˈzis sə/ | **禁止**(最も強い否定の拘束)。「〜することは禁じられている」 |
| **este obligat/obligată să** | /ˈes.te o.bliˈɡat sə/ | **義務(人が主語)**。「(その人は)〜する義務を負う」 |
| **este obligatoriu să** | /ˈes.te o.bli.ɡaˈto.ri.u sə/ | **義務(非人称)**。「〜することが義務である/必須である」 |
| **trebuie să** | /ˈtre.bu.je sə/ | **必要・義務**。「〜しなければならない」(広く使う中核形) |
| **urmează să** | /urˈme̯a.ză sə/ | **予定・これから〜することになっている**(義務よりは手順・段取り) |
| **are dreptul să** | /ˈa.re ˈdrep.tul sə/ | **権利**。「〜する権利を有する」(許可より強い、権利の主張) |
| **poate să / i se permite să** | /ˈpo̯a.te sə/ | **許可・可能**。「〜してよい/〜することが許される」 |
| **nu este obligat să** | /nu ˈes.te o.bliˈɡat sə/ | **義務の否定**。「〜する義務はない」(禁止 interzis とは別) |

読解の要点を整理する。

- **禁止(este interzis)と「義務の否定」(nu este obligat)は別物** である。⚠ 前者は「してはならない」、後者は「しなくてよい(してもよい)」。日本語話者はどちらも「〜ない」の連想で混同しがちだが、**禁止** と **義務がないだけ** は正反対に近い。
- **義務の este obligat(人が主語)/ este obligatoriu(非人称)** は、いずれも強い拘束を示す。obligat は「誰が義務を負うか」を主語で示し、obligatoriu は「〜が必須だ」と非人称で述べる。
- **trebuie să** は最も広く使われる義務・必要の形。文脈で「(規則上)しなければならない」から「(必要上)〜すべき」まで幅がある。
- **are dreptul să(権利)** は、単なる許可(poate să)より強く、**権利の主張** を言語化する。**同意・拒否の場面**(次節)で重要になる。

⚠ 繰り返す。これらは **言語形式の強度** の読み分けであって、その義務・禁止・権利が **実際に法的効力を持つか** の判断ではない。法文の効力・適用・例外は法律の領域であり、本章は踏み込まない。

### 同意の言語 ― インフォームドコンセント(consimțământ informat)

**consimțământ informat**(インフォームドコンセント=十分な説明を受けたうえでの同意)は、医療・法律の両方に関わる高リスクの言語場面である。ここで本章が扱うのは **同意・拒否・撤回を表す言語形式の読解** のみで、**その同意が有効か、何に同意しているかの内容判断・意思の代行は一切しない。**

同意の場面に現れる言語構造を、**言語形式として** 押さえる。

| 言語形式 | IPA/発音 | 読解上の意味 |
|---|---|---|
| **consimțământ informat** | /kon.sim.t͡səˈmɨnt in.forˈmat/ | インフォームドコンセント(直訳「知らされた同意」) |
| **Sunteți de acord cu...?** | /sunˈtetsʲ de aˈkord ku/ | 「〜に同意されますか」(同意を問う定型) |
| **Sunt de acord.** | /sunt de aˈkord/ | 「同意します」(肯定の意思表示) |
| **Nu sunt de acord.** | /nu sunt de aˈkord/ | 「同意しません」(拒否の意思表示) |
| **Îmi dau consimțământul.** | /ɨmʲ daw kon.sim.t͡səˈmɨn.tul/ | 「同意を与えます」(硬い・書面的) |
| **Îmi retrag consimțământul.** | /ɨmʲ reˈtraɡ kon.sim.t͡səˈmɨn.tul/ | 「同意を撤回します」(a retrage=撤回する) |
| **Am înțeles explicațiile și îmi asum...** | /am ɨn.t͡ʃeˈles/ | 「説明を理解し、〜を引き受けます」(理解の確認) |
| **Am dreptul să refuz.** | /am ˈdrep.tul sə reˈfuz/ | 「拒否する権利があります」(前節の are dreptul の応用) |

読解の要点。

- **Sunteți de acord cu...?**(〜に同意されますか)への応答は、**Sunt de acord.**(同意する)/ **Nu sunt de acord.**(同意しない)で **一義に** 示す。⚠ 高リスク場面では「まあ、たぶん(parcă / cred că da)」のような **曖昧な肯定** を避け、明確に肯定か否定かを示す言語形式を読み取る/選ぶ。
- **撤回(a retrage)** の言語は特に重要である。**Îmi retrag consimțământul.**(同意を撤回します)は、いったん与えた同意を取り消す明確な意思表示の定型である。⚠ 撤回の可否・効力・手続きは制度・法律の領域であり、本章は判断しない。本章が示すのは「a retrage consimțământul = 同意を撤回するという **言語形式** がある」ことだけである。
- **are dreptul să / am dreptul să**(〜する権利がある)は、同意・拒否の文脈で「拒否する権利」「説明を求める権利」などを主張する言語形式として現れる。

⚠⚠ **最重要**:同意・拒否・撤回は、**本人の意思** に関わる。語学者・通訳者が **代わりに同意したり、意思を推測して埋めたりしてはならない。** 本章は「同意の言語形式を正確に読解する」訓練であって、**同意そのものを行う・代行する訓練ではない。** 実際の同意場面では、必ず有資格の医療者・法律家・有資格通訳者を介し、本人の意思を **本人自身に** 確認すること。

### 数字・単位・日付の絶対精度 ― 読み合わせ確認

高リスク場面で **数字・単位・日付を取り違えることは、致命的** である。投薬量、金額、期限、生年月日——これらは **推測が一切許されない**。ここでの言語運用は「読み合わせて確認する(citirea de confirmare)」ことに尽きる。

確認の中核定型は **Permiteți-mi să confirm: ...**(確認させてください:……)である。数値・日付を **復唱して照合** する。

| ルーマニア語 | IPA/発音 | 使う場面 |
|---|---|---|
| **Permiteți-mi să confirm: ...** | /per.miˈtetsʲ mʲ sə konˈfirm/ | 「確認させてください:〜」(復唱確認の口火) |
| **Am notat exact: [数値/日付]. Este corect?** | /am noˈtat eɡˈzakt ... ˈes.te koˈrekt/ | 「正確に控えました:〜。よろしいですか?」 |
| **Repet, pentru confirmare: ...** | /reˈpet ˈpen.tru kon.firˈma.re/ | 「確認のため復唱します:〜」 |
| **Vă rog să confirmați unitatea de măsură.** | /və roɡ sə kon.firˈmatsʲ u.niˈta.te̯a de məˈsu.rə/ | 「単位をご確認ください」(mg / ml など) |
| **Data este [zi] [lună] [an], nu invers?** | /ˈda.ta ˈes.te.../ | 「日付は〜ですね、逆ではなく?」(日月の順の確認) |

読み合わせの実務的注意。

- **単位を必ず添えて確認する**:数だけでなく **単位(mg, ml, g, %, RON)** を含めて復唱する。「15」ではなく「exact 15 mg」と確認する。
- **小数点のコンマに注意**:ルーマニア語は小数点に **コンマ** を使う(**2,5** = 2.5)。桁区切りの点(**1.000** = 千)と混同しない。数値を口頭で確認するときは、**cifra**(数字)を一つずつ、単位とともに照合する。
- **日付の順序を確認**:**data de 3 aprilie 2026**(2026年4月3日)のように、日→月→年の順を明示的に確認する。数字だけの表記(03/04)は日月順の取り違えを生むため、**月名を語で言う** ことで曖昧さを排す。

⚠ ただし——数値・日付を **正確に確認** することと、その数値が **医学的・法的に妥当か** を **判断** することは別である。本章が扱うのは前者(正確な確認の言語運用)だけであり、**その投薬量が適切か、その期限が妥当かの判断は、必ず有資格の専門家に委ねる。**

## ルーマニア語例文(高リスク精度・すべて架空)

高リスク場面で「精度」を担う定型を、標準フォーマットで示す。**すべて架空の設例** であり、内容の医学的・法的判断はしない。

| ルーマニア語 | IPA/発音 | 自然な日本語訳 | 直訳 | 解説 | 使用場面・レジスター |
|---|---|---|---|---|---|
| Permiteți-mi să confirm: exact 15 mg, nu 50. | /per.miˈtetsʲ mʲ sə konˈfirm eɡˈzakt ... / | 確認させてください:正確に15 mgで、50ではありませんね。 | 許してください 私に 〜する 確認する 正確に | 数値の読み合わせ。exact + 単位 + 誤りうる値との対比 | 高リスク確認・丁寧(架空) |
| Conform documentului, termenul este de 30 de zile. | /konˈform do.ku.ˈmen.tu.lui ˈter.me.nul ˈes.te de ... / | 文書によれば、期限は30日である。 | 従い 文書-に 期限-その である 〜の 30 の 日 | ⚠ 主観(cred)でなく文書に帰属。⚠ termen=期限 | 法文読解・中立(架空) |
| Este interzis să se depășească doza prescrisă. | /ˈes.te in.terˈzis sə se de.pəˈʃe̯as.kə ... / | 処方された用量を超えることは禁じられている。 | である 禁止 〜する 自ら 超える 用量-その 処方された | ⚠ interzis=禁止(最強度)。用量判断は医師へ | 指示文読解・中立(架空) |
| Pacientul are dreptul să refuze tratamentul. | /pa.t͡ʃiˈen.tul ˈa.re ˈdrep.tul sə reˈfu.ze ... / | 患者は治療を拒否する権利を有する。 | 患者-その 持つ 権利-その 〜する 拒否する 治療-その | ⚠ are dreptul=権利(許可より強い)。同意・拒否は本人の意思 | 権利読解・中立(架空) |
| Îmi retrag consimțământul, în scris. | /ɨmʲ reˈtraɡ kon.sim.t͡səˈmɨn.tul ɨn skris/ | 同意を書面で撤回します。 | 私に 撤回する 同意-その 〜で 書面 | ⚠ a retrage consimțământul=撤回の言語形式。効力は法の領域(判断せず) | 意思表示読解・書面的(架空) |
| Vă rog să confirmați: data este 3 aprilie, nu 4 martie. | /və roɡ sə kon.firˈmatsʲ ˈda.ta ˈes.te ... / | ご確認ください:日付は4月3日で、3月4日ではありません。 | あなたに 頼む 〜する 確認する 日付-その である | 日付の読み合わせ。月名を語で言い日月順を排す | 高リスク確認・丁寧(架空) |

## 高リスク対話の型 ― 同意場面の読み合わせ(架空・言語の型のみ)

架空の設例:ある同意場面で、説明者と本人(または有資格通訳を介したやり取り)が、**曖昧さを残さずに** 同意・数値・日付を確定していく **言語の型** を示す。**内容はすべて架空** で、医学的・法的判断は含まない。**この対話は同意手続きのひな形ではなく、言語運用の練習用である。**

> **説明者:** Sunteți de acord cu procedura explicată?（説明した手続きに同意されますか)
> **本人:** Am o întrebare. Ce înseamnă exact acest termen?（一つ質問があります。この語は正確にどういう意味ですか)
> **説明者:** [説明]
> **本人:** Am înțeles. Sunt de acord.（理解しました。同意します)
> **説明者:** Permiteți-mi să confirm: sunteți de acord, iar data de azi este 3 aprilie 2026.（確認させてください:ご同意で、本日は2026年4月3日です)
> **本人:** Corect. Dar rețineți că am dreptul să îmi retrag consimțământul mai târziu.（そのとおりです。ただ、後で同意を撤回する権利があることは留意してください)

### 読解解説(言語の型のみ)

- **Sunteți de acord cu...?** ― 同意を問う定型。応答は **Sunt de acord / Nu sunt de acord** で一義に。
- **Ce înseamnă exact acest termen?** ― ⚠ 不明時の確認。**推測せず、意味を確定してから同意する** 姿勢の言語化。前章の「訳せないとき確認する」の応用。
- **Am înțeles. Sunt de acord.** ― 理解の確認 + 明確な肯定。曖昧な「たぶん(parcă)」を避ける。
- **Permiteți-mi să confirm: ... data de azi este ...** ― 同意 **と** 日付を読み合わせで確定。数値・日付の精度をこの一句に集約。
- **am dreptul să îmi retrag consimțământul** ― ⚠ 撤回する権利の言語形式(are dreptul + a retrage)。⚠ その権利・撤回が実際にどう働くかは法・制度の領域で、本章は判断しない。

⚠ この対話が示すのは **言語の型**(問う→確認する→一義に答える→読み合わせる)だけである。**実際の同意場面では、必ず有資格の医療者・法律家・有資格通訳者を介し、本人の意思を本人自身に確認すること。** 語学者・通訳者が同意を代行・推測してはならない。

## 高リスク通訳の原則(言語運用として)

医療・法律の通訳は、誤りが生命・権利に直結する最高リスクの言語運用である。**職業通訳者の倫理・中立性・守秘・責任の詳細は本章の範囲外** であり、`medical/medical-interpretation.md`(および法廷通訳の専門資料)に委ねる。ここでは前章までに積んだ姿勢を、高リスク通訳の **言語運用原則** として集約する。

1. **追加・省略・推測の禁止(fără adăugări, omisiuni sau presupuneri)** ― 話し手の言ったことを、**足さず・省かず・推測で補わず** に訳す。聞き手が不安がるからと和らげる、専門語を勝手に意訳する、聞き取れなかった箇所を推測で埋める——いずれも高リスク場面では禁じ手である。前章「足さない・引かない・変えない」を、最も厳格な形で守る。
2. **不明時の確認義務(obligația de a clarifica)** ― 意味・数値・語が確定できないときは、**推測で訳さず必ず確認する。** **Permiteți-mi să confirm...**(確認させてください)/ **Ce înseamnă exact acest termen?**(この語は正確にどういう意味か)を挟む。確認を挟む手間は、高リスク通訳では **義務** に近い。
3. **当事者性の排除 ― 一人称直訳(traducerea la persoana întâi)** ― 通訳は原則として **話者の一人称をそのまま訳す。** 患者が「私は痛みがあります」と言えば、通訳は「彼女は痛みがあると言っています」ではなく「**私は痛みがあります**(Am dureri)」と **一人称で** 訳す。通訳者は **自分の意見・立場を差し挟まない**(当事者にならない)。⚠ これは職業通訳の基本規範として広く言及される原則だが、その **倫理・責任・適用の詳細** は専門章に委ねる。本章が示すのは「一人称直訳という **言語運用** の型がある」ことである。

⚠ 繰り返す。上記は **言語運用の原則** であって、職業通訳者の資格・倫理・責任・守秘義務・法的地位を論じるものではない。医療・法律の実際の通訳は、**必ず有資格の医療通訳者・法廷通訳者を介すること。** 善意の付き添いが、高リスク場面で人の生命・権利を損なう危険を常に自覚する。

## ⚠ 総整理表 ― 医療・法律の誤訳リスク語(集約)

本書でこれまで扱った医療・法律の誤訳・誤読リスク語を、一覧に集約する。各語の **確実な区別** のみを示し、**医学的・法的内容の判断はしない。** 詳細は各章へ。

| **a rezilia / a anula** | 同じ「解除」 | rezilia=(継続的契約の)解除 / anula=(注文・予約等の)取消・無効。同義扱いしない |

| **benign / malign** | 混同(正反対) | benign=良性 / malign=悪性。綴りが近く誤読最高リスク。内容説明は必ず医師へ |
| **diagnostic / prognostic** | 同じ | diagnostic=診断(名)/ prognostic=予後(見通し) |

### 本章で加わる高リスク語(義務・同意)

| ⚠ 語 | 取り違えやすい読み | 正しい区別(言語上) |
|---|---|---|
| **este interzis / nu este obligat** | どちらも「〜ない」で混同 | interzis=**禁止**(してはならない)/ nu este obligat=**義務なし**(しなくてよい)。正反対に近い |
| **are dreptul să / poate să** | 同じ「〜できる」 | are dreptul=**権利**(主張)/ poate=**可能・許可**(できる)。権利は許可より強い |
| **a retrage consimțământul** | 見落とし | **同意の撤回**(a retrage=撤回)。撤回の言語形式。効力は法・制度の領域(本章は判断せず) |
| **cam / vreo / parcă** | 高リスクで放置 | 曖昧語。精密表現(exact / conform... / se precizează că)へ言い換える |

⚠ この表は **言語上の区別** の集約であって、**医学的・法的内容の判断ではない。** 各語が実際にどんな効力・意味を持つかは、必ず有資格の医師・弁護士・公証人・有資格通訳者に委ねること(最強度免責の再確認)。

## 練習問題(言語の精度のみ・内容判断を求めない)

### 問題1: 曖昧→精密の書き換え(5問)

次の曖昧な言い方を、高リスク場面にふさわしい精密表現に書き換えよ(架空の文脈。数値は任意の例でよい)。

1. **Trebuie luat cam de două ori pe zi, parcă.**(だいたい1日2回くらい、だと思う)
2. **Documentul zice ceva despre un termen, cred.**(文書に期限みたいなことが書いてある気がする)
3. **Pacientul are vreo 40 de ani.**(患者は40歳くらい)
4. **Suma e pe undeva pe la 5000.**(金額はどこか5000くらい)
5. **Semnătura trebuie pusă cumva până săptămâna asta.**(署名は今週中になんとか)

### 問題2: 義務・許可・禁止の強度を読み分けよ

次の各形式が「禁止/義務/権利/許可/義務の否定」のどれを **言語的に** 示すかを答えよ(法的効力の判断は不要)。

1. **Este interzis să...** 2. **Este obligat să...** 3. **Are dreptul să...** 4. **Poate să...** 5. **Nu este obligat să...**

### 問題3: 同意の言語

次の各表現の読解上の意味を答えよ(意思の代行はしない)。

1. **Sunteți de acord cu...?** 2. **Nu sunt de acord.** 3. **Îmi retrag consimțământul.** 4. **Am dreptul să refuz.**

### 問題4: 数値・日付の読み合わせ

次の場面で使う確認の定型を、本章の表現から選んで書け。

1. 投薬量「15 mg」を復唱確認したい。
2. 日付「2026年4月3日」を、日月の順を取り違えていないか確認したい。

### 問題5: 高リスク通訳の間違い探し(2問)

次の通訳者のふるまいには、本章の原則に照らして問題がある。何が問題かを指摘せよ。

1. 患者が不安がるといけないので、医師の「悪性の可能性がある(posibil malign)」を、通訳者が「たいしたことないでしょう」と和らげて伝えた。
2. 患者が「私は薬を飲んでいません(Nu iau medicamente)」と言ったのを、通訳者が聞き取れなかったため、「たぶん飲んでいると思います」と推測で訳した。

## 解答

### 問題1

1. **Se administrează exact de două ori pe zi, conform indicațiilor.**(正確に1日2回、指示に従い投与する)— cam・parcă を除き、exact + conform に。⚠ 実際の用量は医師・薬剤師へ。
2. **Documentul precizează un termen de [数値] zile.**(文書は〜日の期限を明記している)— cred(推測)を se precizează(明記)に、cevaを具体化。
3. **Pacientul are exact 40 de ani.**(または生年月日を確認する)— vreo を除き exact に。年齢が重要なら生年月日で確認。
4. **Suma este exact [数値] RON.**(金額は正確に〜RON)— pe undeva を除き exact + 単位(RON)。
5. **Semnătura trebuie aplicată până la data de [日付].**(署名は〜日までに要する)— cumva を除き、締め切りを日付で明示。⚠ 署名の効力は法の領域(本章は判断せず)。

いずれも「話者の主観(parcă/cred/cam/vreo/cumva)」を除き、「exact / conform... / se precizează că / 具体的な数値・日付・単位」に置き換えるのが要点。内容の妥当性は判断しない。

### 問題2

1. **禁止**(してはならない) 2. **義務**(〜する義務を負う) 3. **権利**(〜する権利がある) 4. **許可・可能**(〜してよい/できる) 5. **義務の否定**(〜する義務はない=しなくてよい)。
— ⚠ 1(禁止)と 5(義務の否定)は正反対に近い。混同しない。3(権利)は 4(許可)より強い主張。いずれも **言語形式の強度** の読み分けで、法的効力の判断ではない。

### 問題3

1. 「〜に同意されますか」(同意を問う定型) 2. 「同意しません」(拒否の明確な意思表示) 3. 「同意を撤回します」(a retrage=撤回の言語形式) 4. 「拒否する権利があります」(are dreptul の応用)。
— ⚠ 同意・拒否・撤回は **本人の意思**。通訳者・語学者が代行・推測してはならない。撤回の効力・手続きは法・制度の領域(本章は判断せず)。

### 問題4

1. **Permiteți-mi să confirm: exact 15 mg. Este corect?**(または **Am notat exact 15 mg. Este corect?**)— 単位 mg を必ず添えて復唱確認。
2. **Data este 3 aprilie 2026, nu invers?**(または **Permiteți-mi să confirm: data de 3 aprilie 2026.**)— 月名を語で言い、日月順の取り違えを排す。
— 数値・日付の正確な確認は言語運用。その値の妥当性(用量・期限が適切か)は医師・弁護士へ。

### 問題5

1. **問題:追加・省略・改変の禁止に反する。** 「悪性の可能性がある」を「たいしたことない」と **勝手に和らげた**(内容の改変)。高リスク通訳では、聞き手が不安でも **足さず・引かず・変えず** に訳し、必要なら医師に平易化を依頼する。⚠ benign/malign の内容説明は医師の領域で、通訳者が判断・改変してはならない。
2. **問題:推測の禁止・不明時の確認義務に反する。** 聞き取れなかったのに **推測で(しかも意味を反転させて)訳した**。正しくは **Puteți să repetați, vă rog?**(もう一度お願いします)で確認する。推測で埋めるのは高リスク場面で最も危険な過ち。

## 次章への橋渡し

医療・法律という、誤りが人の生命・権利に直結する高リスク領域の言語を、**「言語の精度」に厳格に限定して** 扱った。曖昧さを排し、義務・許可・禁止・同意を一義に読み取り、数値を正確に確認する——この訓練で貫いた原則は、**言語の精度は語学者が担えるが、内容の判断は専門家に委ねる** という、C1 の高リスク読解章から C2 の翻訳・通訳章まで一貫してきた背骨の、最も厳格な到達点である。次の[第7章:C2 総復習](./07-c2-review-exercises.md)では、C2 全体——ネイティブらしい表現、皮肉・言外の意味、文学・政治的文章、高度な交渉、翻訳・通訳、そして本章の高リスク言語——を横断して復習する。C2 の言語運用は、最終的に「**目的と場面に応じて、言葉の含み・強度・精度を意識的に選ぶ**」一つの技能に収束する。その最も緊張度の高い応用が、本章の高リスク言語であった。この精度への意識を、C2 の総仕上げへ持ち込んでほしい。
