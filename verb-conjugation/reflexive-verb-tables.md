# 再帰動詞の完全活用表【対象 CEFR: A2〜B1】

## この参照ページの使い方

ここは**再帰動詞(verbe reflexive)**の活用を、再帰代名詞込みで完全に引くための辞書的ページである。再帰動詞とは、a se spăla(自分を洗う=体を洗う)のように、**動詞に再帰代名詞(se など)が必ず付いて活用する**動詞をいう。動詞本体の活用は [full-conjugation-tables.md](./full-conjugation-tables.md) や [regular-verbs.md](./regular-verbs.md) と同じなので、このページの焦点は**「どの再帰代名詞が、どの位置に、どんな縮約形で付くか」**に絞る。運用面(再帰の意味の広がり)は grammar 側が正であり、ここは形態に徹する。

日本語話者がまず知るべき最重要事項は、**ルーマニア語の再帰動詞には2つの系統がある**ことである。

| 系統 | 再帰代名詞 | 代表動詞 | 意味の型 |
|---|---|---|---|
| **対格再帰(reflexiv la acuzativ)** | mă / te / se / ne / vă / se | a se spăla(体を洗う) | 「自分**を**〜する」 |
| **与格再帰(reflexiv la dativ)** | îmi / îți / își / ne / vă / își | a-și aminti(思い出す) | 「自分**に**〜する/自分のために〜する」 |

この2系統は再帰代名詞の形が違うので、**辞書で `a se ...` か `a-și ...` かを必ず確認する**。`a se` なら対格系、`a-și` なら与格系である。再帰代名詞の格の全体像は [grammar/pronouns.md](../grammar/pronouns.md) §1 が正で、ここはその弱形(clitic)の再帰用法に対応する。

---

## 1. 再帰代名詞の一覧(対格・与格)

再帰代名詞は、3人称だけが専用形(se / își)を持ち、1・2人称は目的語の弱形代名詞([grammar/pronouns.md](../grammar/pronouns.md))と同じ形を借用する。

| 人称 | 対格再帰(自分を) | IPA | 与格再帰(自分に) | IPA |
|---|---|---|---|---|
| eu | **mă** | /mə/ | **îmi** | /ɨmʲ/ |
| tu | **te** | /te/ | **îți** | /ɨt͡sʲ/ |
| el, ea | **se** | /se/ | **își** | /ɨʃʲ/ |
| noi | **ne** | /ne/ | **ne** | /ne/ |
| voi | **vă** | /və/ | **vă** | /və/ |
| ei, ele | **se** | /se/ | **își** | /ɨʃʲ/ |

**要点**: 対格と与格で違うのは 1単(mă / îmi)・2単(te / îți)・3人称(se / își)の3組だけ。noi(ne)と voi(vă)は対格・与格が同形なので、そこでは系統の区別が表面に出ない。

---

## 2. 対格再帰の完全活用 — a se spăla(体を洗う)

第 I 群の a se spăla(自分を洗う)を全時制・全法で示す。再帰代名詞 **mă/te/se/ne/vă/se** が動詞の前に置かれるのが基本形である。

### 2.1 直説法現在

| 人称 | 形 | IPA | 訳 |
|---|---|---|---|
| eu | **mă** spăl | /mə spəl/ | 私は体を洗う |
| tu | **te** speli | /te spelʲ/ | 君は体を洗う |
| el, ea | **se** spală | /se ˈspa.lə/ | 彼(女)は体を洗う |
| noi | **ne** spălăm | /ne spəˈləm/ | 私たちは体を洗う |
| voi | **vă** spălați | /və spəˈlat͡sʲ/ | 君たちは体を洗う |
| ei, ele | **se** spală | /se ˈspa.lə/ | 彼らは体を洗う |

### 2.2 複合過去 — 再帰代名詞は助動詞の前で縮約する

複合過去では、再帰代名詞が a avea 短縮助動詞(am/ai/a/am/ați/au)と結びついて**縮約**する。ここが日本語話者の最大の難所なので、縮約の中身を明示する。

| 人称 | 縮約形 | 分解 | IPA | 訳 |
|---|---|---|---|---|
| eu | **m-am** spălat | mă + am | /mam spəˈlat/ | 私は体を洗った |
| tu | **te-ai** spălat | te + ai | /teaj spəˈlat/ | |
| el, ea | **s-a** spălat | se + a | /sa spəˈlat/ | |
| noi | **ne-am** spălat | ne + am | /neam spəˈlat/ | |
| voi | **v-ați** spălat | vă + ați | /vat͡sʲ spəˈlat/ | |
| ei, ele | **s-au** spălat | se + au | /saw spəˈlat/ | |

規則は「**mă→m-、se→s-、vă→v- に子音だけ残して助動詞にくっつく**(母音を落とす)」「te と ne は te-/ne- のままハイフンで結ぶ」。過去分詞 spălat 自体は不変である。

### 2.3 半過去

| 人称 | 形 | 訳 |
|---|---|---|
| eu | **mă** spălam | 私は体を洗っていた |
| tu | **te** spălai | |
| el, ea | **se** spăla | |
| noi | **ne** spălam | |
| voi | **vă** spălați | |
| ei, ele | **se** spălau | |

### 2.4 接続法現在 — să と再帰代名詞

接続法では **să + 再帰代名詞 + 活用形**の順。3人称のみ動詞語尾が交替(spală→spele)する点は非再帰と同じ。

| 人称 | 形 | 訳 |
|---|---|---|
| eu | să **mă** spăl | (私が)体を洗うこと |
| tu | să **te** speli | |
| el, ea | să **se** spele | (3単で -ă→-e 交替) |
| noi | să **ne** spălăm | |
| voi | să **vă** spălați | |
| ei, ele | să **se** spele | |

速い会話では să の母音が落ち、să mă→**să mă**、să te→**să te** はそのままだが、与格の să-mi / să-ți のように縮約する系統もある(§3.4)。

### 2.5 命令法 — 肯定は後置、否定は前置

再帰動詞の命令法は、**肯定で代名詞が動詞の後ろに付き、否定で前に戻る**。位置が入れ替わるのが要点である。

| | 2人称単数(tu) | 2人称複数(voi) |
|---|---|---|
| **肯定** | **Spală-te!**（体を洗いなさい) | **Spălați-vă!** |
| **否定** | **Nu te spăla!**（否定は nu + te + 不定詞) | **Nu vă spălați!** |

- 肯定2単 **Spală-te!** /ˈspa.lə te/ — 動詞(命令形 spală)+ ハイフン + te。
- 否定2単 **Nu te spăla!** /nu te spəˈla/ — nu + 再帰代名詞 te + 不定詞 spăla(否定命令2単は一律「nu + 不定詞」、[full-conjugation-tables.md](./full-conjugation-tables.md) §4)。
- 肯定2複 **Spălați-vă!** は動詞(spălați)の後ろに vă。

### 2.6 条件法・未来(参考)

| 時制 | eu 形 | 作り方 |
|---|---|---|
| 条件法現在 | **m-aș** spăla | 再帰 + 条件助動詞 aș/ai/ar…(mă+aș→m-aș) |
| 未来(標準) | **mă** voi spăla | 再帰 + voi/vei/va… + 不定詞 |
| 未来(o să) | **o să mă** spăl | o să + 再帰 + 接続法 |

条件法でも m-aș / te-ai / s-ar のように縮約する(mă+aș→m-aș、se+ar→s-ar)。

## 3. 与格再帰の完全活用 — a-și aminti(思い出す)

与格再帰は「自分**に**〜する/自分のために〜する」を表す。代表は a-și aminti(自分に思い出させる=思い出す)。再帰代名詞は **îmi/îți/își/ne/vă/își**。動詞本体 a aminti は第 IV 群 -esc 型なので、非再帰部分の活用は amintesc / amintești / amintește… となる。

### 3.1 直説法現在

| 人称 | 形 | IPA | 訳 |
|---|---|---|---|
| eu | **îmi** amintesc | /ɨmʲ a.minˈtesk/ | 私は思い出す |
| tu | **îți** amintești | /ɨt͡sʲ a.minˈteʃtʲ/ | |
| el, ea | **își** amintește | /ɨʃʲ a.minˈteʃ.te/ | |
| noi | **ne** amintim | /ne a.minˈtim/ | |
| voi | **vă** amintiți | /və a.minˈtit͡sʲ/ | |
| ei, ele | **își** amintesc | /ɨʃʲ a.minˈtesk/ | |

### 3.2 複合過去 — 与格代名詞の縮約(mi-am / ți-a / și-a)

与格再帰の複合過去は、対格とは**縮約形が違う**。ここが2系統を分ける最大の見どころである。

| 人称 | 縮約形 | 分解 | IPA | 訳 |
|---|---|---|---|---|
| eu | **mi-am** amintit | îmi + am | /mjam a.minˈtit/ | 私は思い出した |
| tu | **ți-ai** amintit | îți + ai | /t͡sjaj a.minˈtit/ | |
| el, ea | **și-a** amintit | își + a | /ʃja a.minˈtit/ | |
| noi | **ne-am** amintit | ne + am | /neam a.minˈtit/ | |
| voi | **v-ați** amintit | vă + ați | /vat͡sʲ a.minˈtit/ | |
| ei, ele | **și-au** amintit | și + au | /ʃjaw a.minˈtit/ | |

**対格 m-am spălat と与格 mi-am amintit を必ず対比する**。1単は対格 **m-am**(mă起源)/ 与格 **mi-am**(îmi起源)で形が違う。3単も対格 **s-a** / 与格 **și-a** で違う。noi(ne-am)と voi(v-ați)だけは両系統で同形。

### 3.3 半過去・接続法

| 人称 | 半過去 | 接続法現在 |
|---|---|---|
| eu | **îmi** aminteam | să-**mi** amintesc |
| tu | **îți** aminteai | să-**ți** amintești |
| el, ea | **își** amintea | să-**și** amintească |
| noi | **ne** aminteam | să **ne** amintim |
| voi | **vă** aminteați | să **vă** amintiți |
| ei, ele | **își** aminteau | să-**și** amintească |

### 3.4 接続法での縮約(să-mi / să-ți / să-și)

与格再帰は接続法 să と縮約する:să + îmi → **să-mi**、să + îți → **să-ți**、să + își → **să-și**。

- Vreau **să-mi** amintesc.(思い出したい)
- Trebuie **să-ți** amintești.(君は思い出さねばならない)

### 3.5 命令法(与格再帰)

| | 2人称単数(tu) | 2人称複数(voi) |
|---|---|---|
| **肯定** | **Amintește-ți!**（思い出しなさい) | **Amintiți-vă!** |
| **否定** | **Nu-ți aminti!** | **Nu vă amintiți!** |

肯定は動詞の後ろに -ți(Amintește-**ți**!)、否定は前に nu-**ți**(nu+îți の縮約)+ 不定詞。

## 4. 頻出再帰動詞15語 — 系統・主要形一覧

日常会話・医療・ビジネスで高頻度の再帰動詞を、**系統(対格/与格)・eu 形・過去分詞・訳**で一覧化する。辞書形の `a se` / `a-și` で系統が読み取れる。

| 不定詞 | 系統 | eu 形(現在) | 過去分詞 | 訳 |
|---|---|---|---|---|
| a se spăla | 対格 | mă spăl | spălat | 体を洗う |
| a se trezi | 対格 | mă trezesc | trezit | 目を覚ます |
| a se scula | 対格 | mă scol | sculat | 起床する |
| a se îmbrăca | 対格 | mă îmbrac | îmbrăcat | 服を着る |
| a se odihni | 対格 | mă odihnesc | odihnit | 休む |
| a se simți | 対格 | mă simt | simțit | (体調・気分を)感じる |
| a se uita | 対格 | mă uit | uitat | 見る・眺める |
| a se gândi | 対格 | mă gândesc | gândit | 考える |
| a se grăbi | 対格 | mă grăbesc | grăbit | 急ぐ |
| a se plimba | 対格 | mă plimb | plimbat | 散歩する |
| a se întâlni | 対格 | mă întâlnesc | întâlnit | 会う・落ち合う |
| a se naște | 対格 | mă nasc | născut | 生まれる |
| a-și aminti | 与格 | îmi amintesc | amintit | 思い出す |
| a-și imagina | 与格 | îmi imaginez | imaginat | 想像する |
| a-și dori | 与格 | îmi doresc | dorit | (強く)望む |

**注意すべき対比**: a se uita(対格、mă uit=見る)と a uita(非再帰、uit=忘れる)は別語である。再帰の有無で意味が変わる動詞があるため、再帰代名詞は意味の一部として扱う。同様に、a-și aminti(与格再帰=思い出す)は「思い出す」を再帰で表す典型で、日本語の「思い出す」に再帰の含みがないぶん、日本語話者は își を落としがちなので注意する。

## 5. よくある誤用(❌→✅)

| ❌ 誤り | ✅ 正しく | 説明 |
|---|---|---|
| 再帰代名詞の脱落:Am amintit | **Mi-am** amintit | a-și aminti は与格再帰。își を落とさない |
| 複合過去で縮約せず:**mă am** spălat | **m-am** spălat | mă+am は m-am に縮約(母音脱落) |
| 対格と与格の混同:**m-am** amintit | **mi-am** amintit | a-și aminti は与格系(mi-am) |
| 肯定命令で前置:**Te spală!** | **Spală-te!** | 肯定命令は代名詞を後置 |
| 否定命令で後置:Nu **spală-te**! | **Nu te spăla!** | 否定命令は前置+不定詞 |
| a se uita と a uita の取り違え | mă uit(見る)/ uit(忘れる) | 再帰の有無で別語 |
| 接続法で縮約せず:să **îmi** amintesc | să-**mi** amintesc | 与格再帰は să と縮約(să-mi) |
| 3人称与格を se に:**se-a** amintit | **și-a** amintit | 与格3人称は își(→și-a) |

## 6. ネイティブらしい言い換え・口語形

- **「元気?」は Cum te simți?**(a se simți、対格)。体調・気分の定番。返答は Mă simt bine.(元気だ)。
- **「思い出せない」は Nu-mi amintesc.**(与格 îmi の否定縮約 nu-mi)。ど忘れの常套句。Nu-mi aduc aminte. も同義で頻出。
- **「急いで!」は Grăbește-te!**(a se grăbi の肯定命令)。複数・丁寧には Grăbiți-vă!。
- **「座って」は Ia loc.** が丁寧だが、口語では再帰の a se așeza → Așază-te.(座りなさい)も普通。
- **「〜したい(強い願望)」は Îmi doresc să...**(a-și dori、与格)。単なる Vreau より願望が強く響く。

## 7. 学習の指針

1. 辞書形の **`a se ...`(対格)/ `a-și ...`(与格)**で系統を即判定する。
2. 動詞本体の活用は非再帰と同じ([full-conjugation-tables.md](./full-conjugation-tables.md))。**変わるのは代名詞の形と位置だけ**と割り切る。
3. 複合過去の縮約は、対格(**m-am / s-a / s-au**)と与格(**mi-am / și-a / și-au**)を**対にして音で覚える**。
4. 命令法は「肯定=後置(Spală-te!)/ 否定=前置+不定詞(Nu te spăla!)」の1組で全再帰動詞に通用する。
5. 意味が再帰の有無で変わる動詞(a se uita↔a uita)に注意し、再帰代名詞を意味の一部として扱う。
