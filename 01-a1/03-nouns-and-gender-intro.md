# 第3章:名詞と性の導入【CEFR A1】

## この章の目標

この章を終えると、次のことができるようになる。

- ルーマニア語の名詞に 3 つの性(男性・女性・中性)があることを理解し、「un / doi テスト」で見分けられる
- 語尾から名詞の性をおおよそ推測できる
- 不定冠詞 un / o を正しく付けられる
- 基本的な複数形のパターン(-i, -e, -uri)を知っている
- 身近な名詞 30 語程度を性・複数形とセットで言える

## 本文解説

### 「性」とは何か — 日本語話者のための導入

日本語の名詞には性がない。「机」も「本」も「先生」も、文法上はまったく同じように扱う。一方ルーマニア語では、**すべての名詞が「男性(masculin)」「女性(feminin)」「中性(neutru)」のどれかに属する**。これは生物学的な性別とは別の、純粋に文法的な分類である。「テーブル(masă)」は女性名詞、「椅子(scaun)」は中性名詞、「壁(perete)」は男性名詞 — 意味から論理的に導けるものではない。

なぜ性が重要なのか。**性が決まらないと、冠詞も形容詞も数詞も正しく付けられない**からだ。英語で a/an を使い分けるように、ルーマニア語では un/o を性で使い分ける。さらに「良い」という形容詞も bun/bună と形が変わる。名詞を覚えるときは、意味だけでなく**性と複数形をセットで**覚える — これがルーマニア語学習の基本動作である。

### ルーマニア語の「中性」は特殊 — 単数は男性、複数は女性

他のヨーロッパ言語(ドイツ語・ロシア語)の中性と違い、ルーマニア語の中性名詞は独自の形を持たない。**単数では男性名詞として、複数では女性名詞として振る舞う**。

- scaun(椅子)— 単数: **un** scaun(男性扱い)/ 複数: **două** scaune(女性扱い)

この性質を使ったのが、ネイティブも使う性の判定法「**un / doi テスト**」である。

### un / doi テスト — 確実な性の見分け方

「1つの〜」と「2つの〜」を言ってみて、使う語の組み合わせで性が分かる。

| 性 | 1つ | 2つ | 例 |
|---|---|---|---|
| 男性 | **un** | **doi** | un student, doi studenți(学生) |
| 女性 | **o** | **două** | o casă, două case(家) |
| 中性 | **un** | **două** | un scaun, două scaune(椅子) |

- 「un ○○, doi ○○」と言える → 男性
- 「o ○○, două ○○」と言える → 女性
- 「**un** ○○(単数は男性型), **două** ○○(複数は女性型)」→ 中性

辞書やこの教科書で新しい名詞に出会ったら、必ずこのテストの形で口に出して覚えよう。

### 語尾からの推測 — 8 割は当たる経験則

| 語尾 | 推測 | 例 | 主な例外 |
|---|---|---|---|
| -ă | 女性 | casă(家)、fată(娘)、mamă(母) | tată(父)は男性 |
| -e | 女性が多い | carte(本)、pâine(パン) | perete(壁)、frate(兄弟)は男性 |
| -a / -ea(強勢あり) | 女性 | cafea(コーヒー)、stea(星) | |
| 子音で終わる | 男性か中性 | bărbat(男)= 男、oraș(町)= 中 | 子音終わりは un/doi テストで確定させる |
| -u | 男性か中性 | codru(森)= 男、muzeu(博物館)= 中 | |
| -ie | 女性 | familie(家族)、bucurie(喜び) | |

**実務的な結論**: -ă と -a で終われば女性、それ以外は「男性か中性」— まずこの大づかみで十分。人・動物は自然の性に従うことが多い(bărbat 男性、femeie 女性)。

### 不定冠詞 un / o

「1つの、ある〜」を表す不定冠詞(articol nehotărât)は名詞の**前**に置く。

- **un** + 男性・中性単数: un om(ある人)、un tren(ある電車)
- **o** + 女性単数: o zi(ある日)、o întrebare(ある質問)

> ルーマニア語には英語の the に当たる**定冠詞**もあるが、名詞の**後ろにくっつく**(lup → lup**ul**「その狼」)という大きな特徴がある。定冠詞は [第7章:生活語彙](./07-daily-life-vocabulary.md) 以降で少しずつ登場し、本格的には A2 と参照文法(`grammar/articles.md`)で扱う。この章では un / o に集中しよう。

### 複数形の基本パターン

複数形の作り方は名詞ごとに覚えるのが原則だが、大きな傾向は次のとおり。

| 性 | 主な複数語尾 | 例 |
|---|---|---|
| 男性 | **-i**(前の子音が変化することあり) | student → studenți /stuˈdent͡sʲ/、pom → pomi(木) |
| 女性 | **-e** または **-i** | casă → case、carte → cărți(本) |
| 中性 | **-e** または **-uri** | scaun → scaune、tren → trenuri(電車) |

注意点は 2 つ。

1. **子音交替**: 複数語尾 -i の前で t→ț(student→studenți)、s→ș、c→ci などの音変化が起きる。第1章で学んだ軟音化の仕組みがここで生きる
2. **母音交替**: casă→case のほか、fată→fete(娘)、masă→mese(テーブル)のように語中の a→e 交替を伴う語がある

## 重要ポイント

- すべての名詞が男性・女性・中性のどれかに属する。意味からは決まらない
- 中性 = 単数は男性型・複数は女性型。un / două の組み合わせが中性の証拠
- 名詞は「un/o +単数、doi/două +複数」の形で丸ごと覚える
- -ă で終わる名詞はほぼ女性(tată が最重要例外)
- 不定冠詞: un(男・中)/ o(女)。定冠詞は後ろに付く(詳細は後の章)
- 複数形 -i は語末子音を変化させる(t→ț など)

## よくある誤用

| ❌ 誤り | ✅ 正しく | 説明 |
|---|---|---|
| o tată | **un** tată | tată は -ă で終わるが男性名詞。最頻出の例外 |
| doi case | **două** case | casă は女性。doi は男性専用、女性・中性の「2」は două |
| un carte | **o** carte | carte は女性(-e 終わりの女性名詞) |
| doi scaune | **două** scaune | scaun は中性なので複数は女性型の două |
| studenti(ț を t のまま) | studen**ț**i | 複数語尾 -i の前で t→ț の子音交替が必須 |
| 複数の -i をはっきり「イ」と発音 | /stuˈdent͡sʲ/ | 語末 -i は口蓋化 /ʲ/(第1章参照)。ここが単複の聞き分けの鍵 |

## ネイティブらしい言い換え

- 教科書的「Am un frate și o soră.」→ 実際の会話では「Am un frate și-o soră.」(și o が縮約して și-o /ʃo/)と聞こえることが多い。自分で言うときは縮約しなくてよいが、耳では知っておく
- 「2つください」は Vreau două. と数詞だけで済ませるのが自然(文脈で名詞が明らかな場合)

## フォーマル/カジュアルの違い

- 名詞の性・冠詞にフォーマル差はない(文法なのでどの場面でも同じ)
- ただし「男の人/女の人」を指す語には差がある: bărbat/femeie は中立、domn/doamnă(紳士/淑女)は丁寧。店や役所では Un domn vă caută.(男の方がお訪ねです)のように domn/doamnă を使うほうが丁寧

## 文化的注意点

- ルーマニア語の月曜日 luni、火曜日 marți などの曜日名は女性名詞、月名(ianuarie など)は男性扱いである — 覚え始めたら性を意識する癖をつけよう
- 「性」は人にも文法的に適用される。職業名詞は男女で形が違うことが多い(student/studentă、profesor/profesoară)。近年は職業名の女性形の使い方が社会的話題になることもあるが、A1 では標準的な male/female ペアをそのまま覚えればよい

## 練習問題

### 問題1: un / doi テストで性を判定せよ

例: pom → un pom, doi pomi → 男性

1. casă 2. hotel(ホテル) 3. profesor(先生) 4. cafea(コーヒー) 5. muzeu(博物館)

### 問題2: 正しい不定冠詞(un / o)を付けよ

1. ___ zi 2. ___ oraș 3. ___ femeie 4. ___ tată 5. ___ bilet

### 問題3: 複数形を作れ(子音交替に注意)

1. student 2. casă 3. scaun 4. carte 5. tren

### 問題4: 正しい数詞(doi / două)を選べ

1. (doi / două) copii 2. (doi / două) case 3. (doi / două) bilete 4. (doi / două) bărbați

### 問題5: 次の文の誤りを直せ

1. Am un soră. 2. Vreau doi bilete. 3. O tată e acasă.

## 解答

### 問題1
1. o casă, două case → **女性** 2. un hotel, două hoteluri → **中性** 3. un profesor, doi profesori → **男性** 4. o cafea, două cafele → **女性** 5. un muzeu, două muzee → **中性**

### 問題2
1. **o** zi(女) 2. **un** oraș(中) 3. **o** femeie(女) 4. **un** tată(男 — 例外) 5. **un** bilet(中)

### 問題3
1. studen**ți**(t→ț 交替) 2. cas**e** 3. scaun**e** 4. c**ă**r**ți**(母音交替 a→ă +子音交替 t→ț の二段変化) 5. tren**uri**

### 問題4
1. **doi** copii(男) 2. **două** case(女) 3. **două** bilete(中 — 複数は女性型) 4. **doi** bărbați(男)

### 問題5
1. Am **o** soră.(soră は女性) 2. Vreau **două** bilete.(bilet は中性) 3. **Un** tată e acasă. または自然には Tata e acasă.(tată は男性)
