# 第13章:A2 への橋渡し【CEFR A1】

## この章の目標

この章を終えると、次のことができるようになる。

- A1 で自分が「何をできるようになったか」を具体的に言葉にできる
- A1(存在と属性を言える)と A2(行動と時間を言える)の違いを説明できる
- A2 で最初にぶつかる 4 つの壁を事前に知り、心の準備ができる
- A1 の知識を維持する復習習慣を持って A2 に進める

この章は問題集ではなく**読み物**である。肩の力を抜いて読んでほしい。A1 の締めくくりとして、これまでの道のりを振り返り、次のレベルへの地図を手に入れる章である。

## 前提知識

- [第12章:A1 チェックポイントテスト](./12-a1-checkpoint-test.md) — 80 点以上で A1 修了。まだの人はテストと復習を先に
- A1 全体([第1章](./01-pronunciation-and-alphabet.md)〜[第9章](./09-writing-simple-sentences.md))

## 本文解説

### まず、あなたができるようになったこと(棚卸し)

ゼロから始めて、ここまで来た。かつては「ラテン文字なのに読み方が分からない」だった状態から、いまや辞書なしで短い家族紹介文を読み、自己紹介を書けるようになっている。具体的に、A1 を終えたあなたは次のことができる。

**発音・文字**
- [ ] ルーマニア語のアルファベット 31 文字を読める
- [ ] ă/â/î/ș/ț を区別して発音できる
- [ ] ce/ci・ge/gi・che/chi・ghe/ghi の 4 組を読み分けられる
- [ ] ea・oa の二重母音を 1 拍で発音できる
- [ ] 語末 -i の口蓋化 /ʲ/ で単数・複数を聞き分けられる
- [ ] 初見の単語をほぼ正しく読める

**会話**
- [ ] 時間帯・相手に合わせて挨拶を使い分けられる(Bună ziua / Salut など)
- [ ] tu と dumneavoastră を場面で選べる
- [ ] 名前・出身・居住・職業・年齢を名乗れる
- [ ] 相手に出身・仕事を尋ねられる(De unde ești? / Cu ce vă ocupați?)
- [ ] 聞き取れないとき聞き返せる(Poftim? / Mai încet, vă rog.)

**文法**
- [ ] 名詞の性(男・女・中)を un/doi テストで判定できる
- [ ] 不定冠詞 un/o を正しく付けられる
- [ ] a fi(sunt/ești/este/suntem/sunteți/sunt)を活用できる
- [ ] a avea(am/ai/are/avem/aveți/au)を活用できる
- [ ] 基本語順 SVO と形容詞後置(性数一致)を守れる
- [ ] nu で否定文、イントネーション/疑問詞で疑問文を作れる
- [ ] 数詞 0〜100 と「20 以上は de」の規則を使える

**読解・作文**
- [ ] 60〜80 語の家族紹介文を辞書なしで読める
- [ ] 接続詞 și / dar / sau で文をつなげる
- [ ] 句読点・大文字・ダイアクリティクスを正しく書ける
- [ ] 自己紹介+家族紹介を 8〜10 文で書ける

これだけのことが、ゼロからの出発でできるようになった。ルーマニアの店で Bună ziua! と挨拶し、Sunt din Japonia. と名乗り、Am o întrebare. と質問を切り出す — もう「まったく話せない」状態ではない。ここまで積み上げた土台は、この先ずっとあなたを支える。まず、そのことを誇ってほしい。

### A1 と A2 の違い — 「存在と属性」から「行動と時間」へ

A1 と A2 の違いを一言でいうと、こうなる。

> **A1 = 「存在と属性」を言える。A2 = 「行動と時間」を言える。**

A1 では、あなたが使ってきた動詞は主に **a fi(〜である)** と **a avea(持っている)** の 2 本柱だった。この 2 つで言えるのは「A は B である」「A は B を持っている」という、**存在と属性**の世界である。

- Sunt student.(私は学生**である**)
- Am o soră.(私には姉妹が**いる**)
- Casa e mare.(家は大きい=大きい**という属性を持つ**)

これはとても大切な土台だが、まだ「私は毎朝コーヒーを**飲む**」「昨日、店に**行った**」「明日、電話**する**」のような、**行動**や**時間の流れ**を語ることはできない。なぜなら、A1 では一般動詞(a bea 飲む、a merge 行く など)を **1 人称単数の形(beau, merg)だけ**をチャンクとして使ってきたからである。

A2 は、この「行動と時間」の世界に踏み込む段階である。

| | A1(存在と属性) | A2(行動と時間) |
|---|---|---|
| 中心の動詞 | a fi / a avea | 一般動詞すべて |
| 動詞の活用 | a fi / a avea の 6 形 + 一般動詞の 1 人称だけ | 一般動詞を 6 人称すべて活用 |
| 時制 | 現在のみ | 現在 + **過去**(複合過去)+ **未来** |
| 表現できること | 「〜である」「〜を持つ」 | 「〜する」「〜した」「〜するつもり」 |
| 文の自然さ | 短い文の積み重ね | 代名詞で文をなめらかに接続 |

つまり A2 は、A1 で作った骨組みに「動きと時間」という血を通わせる段階である。急に別の言語になるわけではない。a fi と a avea はむしろ A2 でさらに活躍する — 複合過去は「**a avea の現在形** + 過去分詞」で作るので、A1 で覚えた am/ai/are/… がそのまま助動詞として生き続ける。**A1 の知識は捨てるものではなく、A2 の部品になる**。

### A2 で最初にぶつかる 4 つの壁 — 予告と心構え

A2 に進むと、新しい仕組みにいくつか戸惑うはずである。あらかじめ「ここでつまずくよ」と知っておくだけで、実際にぶつかったときの動揺がずっと小さくなる。主な壁は 4 つである。

**① 一般動詞の現在活用(4 グループ)**
A1 では beau / merg のように 1 人称単数だけを使ってきたが、A2 では一般動詞を 6 人称すべて活用する。ルーマニア語の動詞は語尾のパターンで大きく **4 つのグループ(活用型)** に分かれる。最初は「同じ『飲む』でも、私・君・彼で形が変わる」ことに慣れが必要だが、a fi / a avea を暗記できたあなたなら、パターンをつかめば必ず乗り越えられる。

**② 複合過去(perfectul compus)**
「昨日〜した」を表す過去形である。構造は「**a avea の現在形 + 過去分詞**」— たとえば「私は行った」は am mers(am = a avea の 1 人称 + mers = 「行く」の過去分詞)となる。日本語には「持つ」で過去を作るという発想がないので最初は奇妙に感じるが、A1 で am/ai/are を完璧にしているあなたには、実は**半分できあがっている**。あとは動詞ごとの過去分詞の形を覚えるだけである。

**③ 目的語代名詞**
「私は**それを**見る」「私は**彼に**話す」の「それを・彼に」にあたる短い代名詞(mă, te, îl, o, îmi, îi など)である。これらは動詞の前に置かれ、日本語の「を・に」という助詞とは位置の感覚がかなり違う。焦って全部覚えようとせず、A2 概要も勧めているように、まず **Îmi place.(私は好きだ)** のような定型フレーズごと覚え、後から仕組みを理解するのが挫折しないコツである。

**④ 定冠詞の本格運用**
A1 では定冠詞(-ul / -a / -ua)を「その〜という印」として**認識できる**ところまでにとどめた(第5・7章の先取りメモ)。A2 では、これを自分で**作って使い分ける**段階に入る。trenul(その電車)、cafeaua(そのコーヒー)を場面に応じて選べるようになる。A1 で un/o(不定冠詞)を体に入れたあなたは、その対になる「定」の感覚を足していくことになる。

どの壁も、**A1 の土台があるからこそ登れる**ものである。順番に、ひとつずつ。焦らなくてよい。

### A2 の学習の進め方

A2 の章構成は、A1 と同じく「会話 → 文法 → 会話 → …」と技能を織り交ぜて進む。全体像は [A2 概要](../02-a2/00-a2-overview.md) にまとまっているので、A2 を始める前に必ず一度読んでほしい。おおまかな流れは次のとおりである。

- **日常のルーティン・買い物・レストラン**(会話)で、一般動詞を実際の場面で使い始める
- **過去形の基礎・未来表現の基礎**(文法)で、時間を語る力を得る
- **目的語と代名詞**(文法)で、文をなめらかにつなぐ
- **旅行と交通・医療の基本場面**(会話)で、実生活の場面を広げる
- **メールとメッセージ**(作文)で、書く力を伸ばす
- **復習・チェックポイントテスト・B1 への橋渡し**で締めくくる

進め方のコツは A1 と変わらない。**声に出す・書いてみる・間違えたら該当章に戻る**。この 3 つを続ければ、A2 も必ず越えられる。

### A1 の知識を維持する習慣

新しいことを学ぶと、古いことは自然に薄れていく。これは誰にでも起こる当たり前の現象で、対策は「思い出す機会を意図的に作る」ことに尽きる。おすすめの復習スケジュールを挙げる。

| タイミング | やること |
|---|---|
| **1 週間後** | [第10章:総復習練習](./10-a1-review-exercises.md) をもう一度解く(答えを覚えていても、手が動くか確認) |
| **1 か月後** | [第12章:チェックポイントテスト](./12-a1-checkpoint-test.md) を再挑戦し、前回との得点差を見る |
| **毎日 5 分** | a fi と a avea の 6 活用を口で唱える。数詞 1〜20 を数える |
| **A2 の各章の前** | 対応する A1 の章を 5 分だけ読み返す(例: A2 の過去形の前に、第7章の a avea を復習) |

特に **a fi / a avea の活用**と**数詞・de 規則**は、A2 でも毎日使う「筋肉」である。使わないと衰えるので、短くてよいので毎日触れてほしい。第10章の復習問題は、答えを覚えてしまっても「手が反射的に動くか」を測る道具として何度でも使える。

## 重要ポイント

- A1 = 「存在と属性」(a fi / a avea)、A2 = 「行動と時間」(一般動詞・過去・未来)
- A1 の知識は捨てず、A2 の部品になる(複合過去は a avea + 過去分詞)
- A2 の 4 つの壁: ①一般動詞の現在活用(4 グループ)②複合過去 ③目的語代名詞 ④定冠詞の本格運用
- 壁はどれも A1 の土台があれば越えられる。焦らず順番に
- 維持の習慣: 1 週間後に第10章、1 か月後に第12章、毎日 a fi/a avea と数詞

## 学習相談 Q&A

これまで多くの学習者が抱いてきた悩みに、具体的な助言で答える。

### Q1. 単語がなかなか覚えられません。

まず、**名詞は「un/o + 単数、doi/două + 複数」の形で丸ごと**覚えること(第3章)。casă だけでなく「o casă / două case」とセットにすると、性も複数形も一度に定着する。バラバラの単語カードより、**例文ごと**覚えるほうが記憶に残る(本書の語彙表に例文が付いているのはこのためである)。さらに、覚えた語を「使う」機会を作る — 家の中の物に心の中でルーマニア語のラベルを貼る(masă, scaun, ușă…)、自己紹介を毎朝つぶやく、など。**思い出す回数**が記憶を作る。一度に 100 語より、10 語を 10 回思い出すほうが効く。

### Q2. 発音に自信がありません。特に ă と â/î の区別が難しいです。

これは日本語話者ほぼ全員がぶつかる壁なので、心配いらない。コツは**最小対**で覚えること。râu(/rɨw/ 川)と rău(/rəw/ 悪い)を交互に声に出す(第1章)。**â/î は「ウの口でイと言う」、ă は「口の力を抜いたア」**。鏡を見て口の形を確認し、録音して聞き返すと上達が早い。完璧を目指さなくてよい — ネイティブは文脈で聞き取ってくれる。ただし ă と â/î を「ア」で代用すると別の単語(fată と fata、râu と rău)に聞こえるので、そこだけは意識する。毎日 30 秒、最小対を唱えるだけで確実に変わる。

### Q3. tu と dumneavoastră のどちらを使うか、いつも迷います。

迷ったら **dumneavoastră(敬称)を選べば失礼にはならない**(第2・4章)。これが鉄則である。初対面の大人、目上、店員と客、役所ではまず dumneavoastră。友人・同年代・子供・IT 業界では tu。ルーマニアでは、親しくなると立場が上の側から「tu で話しましょう(Putem să ne tutuim.)」と提案する文化があるので、**相手が tu で来たら合わせる**、来なければ dumneavoastră を続ける、と覚えておけば実用上まず困らない。「敬称で失礼になることはないが、いきなり tu は失礼になりうる」— この非対称を覚えておこう。

### Q4. 文を作ろうとすると、つい日本語の語順(動詞が最後)になってしまいます。

これも日本語話者の定番のクセである。対策は第5章の**語順アルゴリズム**を意識すること: ①動詞を決めて活用 → ②動詞の直後に目的語・補語 → ③形容詞は名詞の後ろ → ④否定 nu は動詞の前 → ⑤場所・時間は文末。とにかく **「動詞を 2 番目に置く」** を口ぐせにする。「私は コーヒーを 飲む」ではなく「私は 飲む コーヒーを(Beau cafea.)」。慣れるまでは、日本語を一度「主語→動詞→目的語」の順に組み替えてから訳すとよい。数をこなせば反射的にできるようになる。

### Q5. A2 に進むのが不安です。難しくなりすぎませんか?

不安になるのは、真剣に取り組んできた証拠である。安心してほしい理由が 3 つある。**第一に**、A2 は A1 の続きであって別物ではない。a fi / a avea は A2 でも主役級に働き続ける(複合過去は a avea で作る)。**第二に**、本書は A2 でも「難しい文法はチャンクとして先に使う → 後で仕組みを理解する」という A1 と同じやさしい順序で進む。**第三に**、詰まったらいつでも A1 に戻れる — 各章に相互リンクがあり、第11章の復習マップも使える。壁は予告済み(上記の 4 つ)なので、「あ、これが例のやつだ」と思えば動揺しない。一歩ずつで大丈夫である。
