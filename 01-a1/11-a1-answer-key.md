# 前半:第10章 総復習練習の解答

## セクション① 発音・文字(問題 1〜5)

### 問題1
1. /t͡ʃ/(ci=/t͡ʃi/) 2. /k/(ca) 3. /k/(che — h が硬さを守る) 4. /t͡ʃ/(du**c**e の ce=/t͡ʃe/) 5. /k/(ban**c**ă の că)
- **なぜ**: c は e・i の前だけ /t͡ʃ/(チェ・チ)になり、それ以外は /k/。h を挟んだ ch は「硬さを守る印」で /k/ に戻る(第1章)。
- **よくある誤答**: 3 の cheie を /t͡ʃ/ と読む誤り。che の h を見落とすと起きる。h があれば必ず硬音 /k/。

### 問題2
1. **râu**(/rɨw/ 川)— rău は /rəw/(悪い)で ă。 2. **pâine**(/ˈpɨj.ne/ パン)— â を含む。pare は /ˈpa.re/ で /ɨ/ なし。 3. **なし**(lume /ˈlu.me/・lună /ˈlu.nə/ とも /ɨ/ を含まない)。
- **なぜ**: /ɨ/ は â か î の綴りでしか現れない。â/î が語中・語頭にあるかを見る(第1章)。
- **よくある誤答**: rău を /ɨ/ と思う誤り。ă=/ə/(脱力のア)と â/î=/ɨ/(舌を奥に引いたイ)は別音。râu と rău は /ɨ/ vs /ə/ の最重要ミニマルペア。

### 問題3
**1・3・4**(frați /frat͡sʲ/、ani /anʲ/、studenți /stuˈdent͡sʲ/)。2 の copiii は -iii なので完全母音を含み /koˈpi.ji/。
- **なぜ**: 子音の後の語末 -i は口蓋化 /ʲ/ になるが、-ii と重なる語(copiii)は完全な /i/ で読む例外(第1章)。
- **よくある誤答**: 4 を「イ」と強く発音する誤り。studenți の -i は口蓋化。ここが単数 student と複数 studenți の聞き分けの鍵。

### 問題4
**1・3・4**(seară=ea、școală=oa、profesoară=oa)。2 masă・5 carte には二重母音なし。
- **なぜ**: ea /e̯a/・oa /o̯a/ は 1 拍で滑らかに発音する二重母音(第1章)。
- **よくある誤答**: școală を「シュ・コ・ア・ラ」と 4 拍で読む誤り。oa は 1 拍。

### 問題5
1. **mulțumesc** 2. **școală** 3. **țară** 4. **câine**
- **なぜ**: ț=/t͡s/(ツ)、ș=/ʃ/(シュ)、â=/ɨ/。音から綴りを逆算する練習(第1章)。câine の「クイネ」に近い音は â=/ɨ/ を含み、綴りは ai ではなく âi。
- **よくある誤答**: țară を「tara」と書く誤り。「ツァ」の子音は ț(コンマ付き U+021B)。セディーユ付きの文字は使わない。

## セクション② 名詞の性と冠詞(問題 6〜13)

### 問題6
1. o casă, două case → **女性** 2. un scaun, două scaune → **中性** 3. un bărbat, doi bărbați → **男性** 4. o cafea, două cafele → **女性** 5. un tren, două trenuri → **中性**
- **なぜ**: 「un…, doi…」なら男性、「o…, două…」なら女性、「un…(単数), două…(複数)」なら中性(第3章)。
- **よくある誤答**: scaun・tren を「子音終わり=男性」と即断する誤り。子音終わりは男性か中性。両方 un だが複数が două(女性型)なら中性。必ず複数まで確かめる。

### 問題7
1. **o** zi(女) 2. **un** oraș(中) 3. **o** femeie(女) 4. **un** tată(男) 5. **un** bilet(中) 6. **o** carte(女)
- **なぜ**: un=男性・中性、o=女性(第3章)。
- **よくある誤答**: 4 を o tată とする誤り。tată は -ă で終わるが**男性**の最重要例外。6 を un carte とする誤り。carte は -e 終わりの女性名詞。

### 問題8
**3. tată**(男性)。他は女性。
- **なぜ**: -ă 終わりはほぼ女性だが、tată(父)だけは男性(第3章)。「男性なのに -ă」を丸暗記する。

### 問題9
1. studen**ți**(t→ț の子音交替) 2. cas**e** 3. scaun**e** 4. c**ă**r**ți**(母音交替 a→ă + 子音交替 t→ț の二段変化) 5. tren**uri** 6. f**e**te(母音交替 a→e)
- **なぜ**: 複数語尾 -i の前で子音が交替(t→ț)、語中の母音も交替する語がある(第3章)。
- **よくある誤答**: studenti(ț を書かない)、carti(ă/ț の二段変化を忘れる)。fată→fata と誤る(fata は定形「その娘」で複数ではない。複数は fete)。

### 問題10
1. **doi** copii(男) 2. **două** case(女) 3. **două** bilete(中は複数で女性型) 4. **doi** frați(男)
- **なぜ**: 「2」は性で doi(男)/ două(女・中)を使い分ける。中性は複数で女性型なので două(第3章)。
- **よくある誤答**: doi case、doi bilete。doi は男性専用。女性・中性の「2」は必ず două。

### 問題11
> 中性名詞は、単数では **男性** 名詞のように、複数では **女性** 名詞のように振る舞う。
- **なぜ**: ルーマニア語の中性は独自の形を持たず、un scaun(単数=男性型)/ două scaune(複数=女性型)のように振る舞う(第3章)。

### 問題12
**1・3**(Trenul、Cafeaua)に定冠詞が付いている。2・4 は不定冠詞(un / o)。
- **なぜ**: 定冠詞は名詞の**後ろ**にくっつく(-ul / -ua など)。un / o は前に置く不定冠詞(第3・5章の先取りメモ)。
- **よくある誤答**: 「un も定冠詞」と混同する誤り。un/o は「ある〜(不定)」、-ul/-ua は「その〜(定)」。

### 問題13
1. Am **o** soră.(soră は女性) 2. Vreau **două** bilete.(bilet は中性、複数の「2」は două) 3. **Un** tată e acasă.(tată は男性。より自然には Tata e acasă.)
- **なぜ**: 冠詞・数詞は名詞の性に合わせる(第3章)。
- **よくある誤答**: soră を男性扱いして un、bilete に doi、tată に o を付ける — いずれも性の取り違え。

## セクション③ a fi / a avea の活用(問題 14〜21)

### 問題14
1. **sunt** 2. **ești** 3. **suntem** 4. **sunt**(ei は 3 人称複数) 5. **sunteți**(敬称は常に 2 複)
- **なぜ**: a fi = sunt / ești / este(e)/ suntem / sunteți / sunt。「私」と「彼ら」が同じ sunt(第4章)。
- **よくある誤答**: dumneavoastră に este を使う誤り。敬称は voi と同じ 2 人称複数 sunteți。

### 問題15
1. **am** 2. **ai** 3. **are** 4. **avem** 5. **aveți**
- **なぜ**: a avea = am / ai / are / avem / aveți / au。are は彼・彼女共通(第7章)。
- **よくある誤答**: a fi と混同して sunt/ești を入れる誤り。「持つ」は a avea。

### 問題16
1. **Nu** sunt ocupat. 2. El **nu** e acasă.(または Nu-i acasă.) 3. **Nu** avem timp.
- **なぜ**: 否定は nu を動詞の直前に置く(第5章)。
- **よくある誤答**: nu を文頭固定や目的語の前に置く誤り。位置は必ず動詞の直前。

### 問題17
1. japonez**ă** 2. obosit**ă** 3. fericit 4. încântat**ă**
- **なぜ**: a fi の補語(国籍・状態)は主語(女性なら話者)の性に一致する(第4章)。3 は主語 el が男性なので男性形のまま。
- **よくある誤答**: Ea este japonez(女性主語に男性形)。日本語に性の一致がないため忘れやすい。

### 問題18
1. **am**(a avea — 年齢は「歳を持つ」) 2. **este / e**(a fi — 職業) 3. **avem**(a avea — 人も所有動詞) 4. **ești**(a fi — 状態)
- **なぜ**: 年齢・所有・家族は a avea、身元・職業・状態は a fi(第4・7章)。
- **よくある誤答**: 年齢に a fi を使う Sunt 25 de ani。年齢は必ず a avea(Am 25 de ani)。

### 問題19
1. **Tu ești domnul Popescu?**(→ 友人相手ならそもそも domnul は使わないが、活用の練習として ești に) 2. **Ai copii?**
- **なぜ**: 敬称の 2 複(sunteți / aveți)を tu の 2 単(ești / ai)に落とす(第4・7章)。
- **よくある誤答**: 動詞だけ変えて vă を残す誤り。tu 系では te / ai を使う。

### 問題20
1. **Sunt student.**(通常は主語 eu を省く) 2. この文は **Eu sunt medic, el este inginer.** のまま代名詞を残すのが正しい。
- **なぜ**: 動詞の形で主語が分かるので代名詞は原則省略。ただし**対比・強調**のときは残す(第4章)。2 は「私は医師、彼はエンジニア」と対比しているので eu / el を残す。
- **よくある誤答**: 対比文でも代名詞を消してしまい、対比のニュアンスが失われる。

### 問題21
> A: Bună ziua! **Sunteți** domnul Ionescu?
> B: Da, eu **sunt**. Dumneavoastră cine **sunteți**?
> A: **Sunt** Tanaka, din Japonia.
- **なぜ**: 初対面なので敬称 2 複 sunteți。B の「eu sunt」は「はい、私です」の強調で代名詞を残す(第4章)。
- **よくある誤答**: B で ești を使う誤り。相手を敬称で呼んでいるので自分の応答は 1 人称 sunt。

## セクション④ 数詞と de 規則(問題 22〜26)

### 問題22
1. **șapte** 2. **doisprezece**(女性名詞相手なら douăsprezece) 3. **șaisprezece**(șase→șai の短縮) 4. **douăzeci** 5. **treizeci și cinci**(21 以上は și でつなぐ)
- **なぜ**: 11〜19 は -sprezece、20〜90 は -zeci、21〜は și で連結(第7章)。
- **よくある誤答**: șaisprezece を șasesprezece と書く(16 は șase→șai に短縮)。35 を treizeci cinci と și を落とす。

### 問題23
1. cinci ani(**不要** — 19 以下) 2. douăzeci **de** ani(20 以上) 3. trei cărți(**不要**) 4. o sută **de** oameni(100 は 20 以上) 5. nouăsprezece ani(**不要** — 19 が境目、まだ de なし)
- **なぜ**: 19 までは de なし、20 からは「数 + de + 名詞」(第7章)。
- **よくある誤答**: 19(nouăsprezece)に de を付ける誤り。境目は 19/20。19 はまだ de なし。

### 問題24
1. **Am optsprezece ani.**(18=de なし) 2. **Am treizeci de ani.**(30=de あり) 3. **Am cincisprezece ani.**(15=de なし) 4. **Am patruzeci de ani.**(40=de あり)
- **なぜ**: 年齢も de 規則に従う。19 以下は de なし、20 以上は de あり(第7章)。
- **よくある誤答**: Am 30 ani(de 抜け)、Am optsprezece de ani(19 以下に de を付ける)。

### 問題25
1. **un** student 2. **o** casă 3. **două** bilete 4. **doi** frați
- **なぜ**: 1・2 だけは性に一致(un/o、doi/două)。中性 bilete は複数で女性型 două(第3・7章)。

### 問題26
1. **vineri** 2. **女性名詞** 3. **lunea**
- **なぜ**: 曜日はすべて女性名詞・小文字で書く。「毎週〜曜」は曜日+定冠詞(luni→lunea)(第7章)。
- **よくある誤答**: Vineri と大文字で書く誤り(曜日・月・言語名・国民名は小文字)。「毎週月曜」を無冠詞 luni で表す誤り(無冠詞 luni は「今度の月曜」)。

## セクション⑤ 語順・否定・疑問(問題 27〜34)

### 問題27
1. **Beau o cafea bună.** 2. **Maria nu este acasă.** 3. **București e un oraș mare.** 4. **Unde ești acum?**
- **なぜ**: 基本語順 SVO、形容詞は名詞の後ろ、nu は動詞の直前、疑問詞は文頭(第5章)。
- **よくある誤答**: o bună cafea(形容詞を前に置く)。cafea bună が中立語順。

### 問題28
1. o zi **frumoasă**(女性形) 2. un hotel **mic** 3. două case **noi**(複数形) 4. un student **japonez**
- **なぜ**: 形容詞は名詞の後ろで性・数に一致(第3・5章)。
- **よくある誤答**: două case nouă(複数なのに単数女性形)。複数は noi。

### 問題29
1. **Nu** vorbesc românește. 2. Trenul **nu** vine acum. 3. Casa **nu** e mare.
- **なぜ**: nu は動詞の直前(第5章)。

### 問題30
1. **Ești student?** 2. **Ai un frate?** 3. **Casa e mare?**
- **なぜ**: Yes/No 疑問は語順を変えず、文末に ?(話すときは上昇イントネーション)(第5章)。
- **よくある誤答**: 英語式に倒置(Este casa mare?)を必須と思う誤り。倒置は可能だが不要。

### 問題31
1. **Unde** este gara? 2. **Ce** bei? 3. **Cine** e acolo? 4. **Câți** ani ai?
- **なぜ**: 疑問詞は文頭。unde=どこ、ce=何、cine=誰、câți=いくつ(数)(第5・7章)。

### 問題32
1. **Eu beau cafea.**(または Beau cafea.) 2. **Nu beau cafea.** 3. **Unde este gara?**
- **なぜ**: 1 は日本語 SOV を持ち込んだ誤り(動詞を 2 番目へ)。2 は nu の位置ミス(動詞前へ)。3 は疑問詞を文頭へ(第5章)。

### 問題33
1. **Sunt la birou acum.**(場所・時間は文末) 2. **Mâine suntem în București.**(時間を文頭で話題化)
- **なぜ**: 場所・時間は文末が中立だが、話題化したいときは文頭に置ける(第5章)。

### 問題34
1. **Da, beau.** 2. **Da, sunt.**
- **なぜ**: ルーマニア語では「はい」だけより、動詞を繰り返して答えるのが自然(英語 Yes, I do. と同じ仕組み)(第5章)。

## セクション⑥ 語彙(問題 35〜44)

### 問題35
a-2(soră=姉妹) / b-1(apă=水) / c-4(bani=お金) / d-3(scaun=椅子) / e-5(bunică=祖母)
- **なぜ**: 第3・7章の基本語彙。

### 問題36
a-3(medic=医師) / b-4(inginer=エンジニア) / c-1(profesoară=教師) / d-2(student=学生)
- **なぜ**: 第4・6・8章の職業語彙。

### 問題37
1. **Bună dimineața!** 2. **Bună seara!** 3. **Noapte bună!** 4. **Bună ziua!**
- **なぜ**: 時間帯別の挨拶。Noapte bună は就寝・別れ専用(第2章)。
- **よくある誤答**: 夜の出会いに Noapte bună を使う誤り(夜の出会いは Bună seara)。

### 問題38
1. **Mulțumesc** 2. Cu **plăcere** 3. **Vă** rog 4. **Te** rog
- **なぜ**: お礼 Mulțumesc、応答 Cu plăcere、依頼は丁寧 Vă rog / くだけた Te rog(第2章)。

### 問題39
1. **tată** 2. **mamă** 3. **frate** 4. **bunică** 5. **copil**
- **なぜ**: 第7章の家族語彙。

### 問題40
1. mare ↔ **mic**(小さい) 2. bun ↔ **rău**(悪い) 3. **範囲内になし**(nou「新しい」の反対 vechi は未習)
- **なぜ**: mic は第3・5章、rău は第1章(râu との対比)で登場済み。nou の反対語 vechi は A1 の範囲に出てこない(第5章では nou のみ)。
- **よくある誤答**: 3 に vechi と書く誤り。習っていない語は出題範囲外。範囲内で確実なのは mare↔mic。

### 問題41
1. **Puteți repeta, vă rog?** 2. **Mai încet, vă rog.** 3. **Poftim?**
- **なぜ**: 「反復依頼」「速度依頼」「即時の聞き返し」で役割が違う(第6章)。

### 問題42
1. Sunt **din** Japonia. 2. Locuiesc **în** București.
- **なぜ**: din=出身、în=居住(第4・6章)。
- **よくある誤答**: Locuiesc din București(「住む」に din を使う誤り)。居住は în。

### 問題43
**Mă** numesc Yuki. Sunt **din** Japonia. Sunt **studentă**(女性なので -ă). Îmi pare **bine**.
- **なぜ**: Mă numesc はチャンク、din は出身、職業は無冠詞・性一致(第6章)。

### 問題44
1. **carte**(他は飲食物、carte は本) 2. **masă**(他は曜日、masă はテーブル) 3. **scaun**(他は家族、scaun は椅子)
- **なぜ**: カテゴリー分類。第3・7章の語彙知識。

## セクション⑦ ミニ会話完成(問題 45〜48)

### 問題45
> A: **Bună ziua! Ce mai faceți?**
> B: **Bine, mulțumesc. La revedere.**
- **なぜ**: 丁寧度は「挨拶語の格上げ(Salut→Bună ziua)」「動詞の敬称形(faci→faceți)」「別れの格上げ(Pa→La revedere)」の 3 点で作る(第2・6章)。

### 問題46
> A: Bună ziua! **Sunteți** domnul Ionescu?
> B: Da, eu sunt. Dumneavoastră cine **sunteți**?
> A: Mă **numesc** Tanaka. Îmi pare **bine** de cunoștință.
- **なぜ**: 初対面は敬称 2 複 sunteți。Mă numesc は名乗りのチャンク(第6章)。

### 問題47
> Maria: Salut! Eu sunt Maria. Tu cine **ești**?
> Ken: Salut! **Sunt** Ken. Îmi pare bine!
> Maria: Și mie! De unde **ești**?
> Ken: Sunt **din** Japonia.
- **なぜ**: 学生同士は tu。ești(2 単)、名乗りは Sunt、出身は din(第4・6章)。

### 問題48
1-b(お礼には Cu plăcere) / 2-a(「元気?」には Bine, mulțumesc. Dar tu?) / 3-c(挨拶には同じ挨拶を返す)
- **なぜ**: 挨拶・お礼・調子尋ねの定型応答(第2章)。

## セクション⑧ 短文和文羅訳(問題 49〜54)

### 問題49
**Sunt studentă.**
- **なぜ**: 話者が女性なので -ă。職業は無冠詞(第4・9章)。
- **よくある誤答**: Sunt student(女性なのに男性形)、Sunt o studentă(職業に冠詞)。

### 問題50
**Am un frate, dar nu am o soră.**
- **なぜ**: a avea + un/o、逆接 dar の前にコンマ、否定 nu am(第7・9章)。
- **よくある誤答**: dar の前のコンマを落とす。nu の位置ミス。

### 問題51
**Tata are cincizeci de ani și este inginer.**
- **なぜ**: 年齢は a avea、50 は 20 以上なので de、職業は無冠詞、și で接続(第7・9章)。
- **よくある誤答**: cincizeci ani(de 抜け)、este un inginer(冠詞を付ける)。

### 問題52
**Unde este un restaurant bun?**
- **なぜ**: 疑問詞文頭、形容詞後置。会話では Unde-i un restaurant bun? も可(第5章)。

### 問題53
**Sunt din Japonia și locuiesc în Osaka.**
- **なぜ**: din=出身、în=居住、și で接続。în の î を省かない(第6・9章)。

### 問題54
**Familia mea este mică, dar fericită.**
- **なぜ**: 主語は familia(女性名詞)なので、話者の性にかかわらず mică / fericită は女性形。dar の前にコンマ(第9章)。
- **よくある誤答**: 話者が男性だと fericit にしてしまう誤り。一致するのは主語「家族」であって話者ではない。

# 後半:第12章 チェックポイントテストの解答

ここからは [第12章:A1 チェックポイントテスト](./12-a1-checkpoint-test.md) の解答である。**先にテストを解いてから**採点すること。配点は第12章に明記したとおり(合計 100 点)。

## セクション A:音読課題(問題 1〜5 / 各 2 点)

音読は自己評価なので「正答」はないが、各問の**採点ポイントの模範**を示す。録音して次と照合し、できていれば 2 点、一部なら 1 点、ほとんどできなければ 0 点。

1. **Mulțumesc mult!** /mul.t͡suˈmesk mult/ — ț=/t͡s/、強勢 mesc。
2. **Bună ziua! Ce mai faceți?** /ˈbu.nə ˈzi.wa t͡ʃe maj ˈfa.t͡ʃet͡sʲ/ — ă=/ə/、ea=/e̯a/、-ți=/t͡sʲ/。
3. **Sunt din Japonia și locuiesc în București.** /sunt din ʒaˈpo.ni.a ʃi lo.kuˈjesk ɨn bu.kuˈreʃtʲ/ — î=/ɨ/、j=/ʒ/、-ști=/ʃtʲ/。
4. **Am douăzeci de ani.** /am ˈdo.wə.zet͡ʃʲ de anʲ/ — ă=/ə/、două の oa 連なり、ani の -i=/ʲ/。
5. **Câinele are o carte, dar fata are o cheie.** /ˈkɨj.ne.le ˈa.re o ˈkar.te dar ˈfa.ta ˈa.re o ˈke.je/ — câinele の â=/ɨ/、carte の ca=/k/、cheie の che=/k/。
- **自己評価のコツ**: 語頭の e(este 型はここにはないが)や語末 -i、ă/â の区別を重点的に聞く。カタカナ発音になっていないか確認する。

## セクション B:文字と発音(問題 6〜10 / 各 2 点)

### 問題6(各 0.5 点)
1. /t͡ʃ/ 2. /k/ 3. /k/(ch) 4. /t͡ʃ/(du**c**e)
- **なぜ**: c は e・i の前で /t͡ʃ/、ch は硬音 /k/(第1章)。

### 問題7
**1・3**(râu、pâine)。2 rău は ă=/ə/、4 carte に /ɨ/ なし。
- **よくある誤答**: rău を /ɨ/ とする(ă と â/î の混同)。

### 問題8
**1・3・4**(frați、ani、bani)。2 copiii は完全母音。
- **なぜ**: 語末 -i は口蓋化 /ʲ/。-iii は例外で完全母音(第1章)。

### 問題9
**1・2・4**(seară=ea、școală=oa、profesoară=oa)。3 masă は二重母音なし。

### 問題10(各 1 点)
1. **țară** 2. **școală**
- **よくある誤答**: tara(ț を書かない)、scoala(ș/ă を省く)。

### 問題40

- **なぜ**: テキストに o familie mare とある。sau で並べられた 2 択から正しい方を選ぶ。
