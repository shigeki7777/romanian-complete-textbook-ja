# 第8章:読解:私の家族【CEFR A1】

## この章の目標

この章を終えると、次のことができるようになる。

- 第1〜7章の文法だけで書かれた 60〜80 語の読解文をひとまとまりとして読める
- 家族紹介の文章から、人物・人数・年齢・職業・住まいの情報を読み取れる
- 読解文の中で a fi / a avea / un・o / 数詞がどう連動しているかを説明できる
- 内容確認の質問(ルーマニア語)に短く答えられる
- 正しい強勢位置で音読できる

## 本文解説

### この章の読み方

これまでは単語や1文ずつを扱ってきたが、この章では初めて**まとまった文章**に取り組む。読解のコツは「1語ずつ和訳しようとしない」ことである。まず全体を通して読み、知っている語(第7章までの家族語彙・数詞・a fi/a avea)を手がかりに大意をつかむ。そのうえで対訳と語彙表で細部を確認する、という順で進めよう。

この読解文はすべて**第1〜7章の範囲内**の文法で書かれている。過去形・接続法・目的語代名詞は一切使っていないので、既習の知識だけで完全に読めるはずである。

## 読解文:Familia mea(私の家族)

> Bună! Mă numesc Ken. Sunt din Japonia. Am o familie mică. Avem o casă în Osaka.
>
> Tata are cincizeci de ani. El este inginer. Mama are patruzeci și opt de ani. Ea este profesoară. Mama și tata sunt amabili.
>
> Am un frate și o soră. Fratele meu are douăzeci de ani. El este student. Sora mea are cincisprezece ani. Ea este elevă.
>
> Avem un câine mic. Este foarte drăguț. Eu sunt fericit. Familia mea este frumoasă.

**語数**: 約 76 語。

## 発音の難所(5〜8か所)

全文の IPA は付けない(この段階では規則から自力で読むのが目標)。つまずきやすい箇所だけを取り上げる。

1. **Mă numesc** /mə nuˈmesk/ — mă の ă は /ə/(あいまい母音)。強勢は numesc の後半 /-ˈmesk/
2. **cincizeci** /t͡ʃint͡ʃʲˈzet͡ʃʲ/ — ci=/t͡ʃ/ が3回。語末 -i はすべて口蓋化 /ʲ/。「チンチゼチ」に近いが子音の口蓋化を効かせる
3. **patruzeci și opt** /pa.truˈzet͡ʃʲ ʃi opt/ — și=/ʃi/。zeci の -i と și の音が続くので区切りを意識
4. **inginer** /in.d͡ʒiˈner/ — gi=/d͡ʒ/(英語 gin の g)。「インジネル」に近い。強勢は最後 /-ˈner/
5. **profesoară** /pro.feˈso̯a.rə/ — oa=/o̯a/ の二重母音、語末 -ă=/ə/。強勢は /-ˈso̯a-/
6. **câine** /ˈkɨj.ne/ — â=/ɨ/(中舌母音)。「クイネ」ではなく喉の奥で /ɨ/。ai の綴りでなく âi
7. **drăguț** /drəˈɡut͡s/ — ă=/ə/、ț=/t͡s/。語末 ț は「ツ」に近い破擦音。強勢は /-ˈɡut͡s/
8. **fericit** /fe.riˈt͡ʃit/ — ci=/t͡ʃi/。語末 -t は口蓋化しない(-i で終わっていない)。強勢は最後

## 文法ポイント解説(テキスト中の既習文法の実例)

読解文から、これまで学んだ文法がどう働いているかを 5 か所ピックアップする。

1. **a avea による年齢**(第7章): Tata **are** cincizeci **de** ani.(父は50歳)。年齢は a avea、50 は 20 以上なので **de** が入る。一方 Sora mea are cincisprezece ani. は 15(19 以下)なので **de なし**。この対比が第7章の de 規則の実例である。

2. **a fi による職業**(第4章): El **este** inginer. / Ea **este** profesoară.。職業名には冠詞を付けない(× este un inginer)。主語の性で inginer(男)/ profesoară(女)と形が変わる。

3. **形容詞後置と性一致**(第3・5章): o familie **mică**(女性名詞に女性形)、un câine **mic**(男性名詞に男性形)、Mama și tata sunt **amabili**(複数男性形)。名詞の後ろに置き、性・数を一致させている。

4. **un / o の使い分け**(第3章): **un** frate(男性)/ **o** soră(女性)/ **un** câine(男性)/ **o** casă(女性)/ **o** familie(女性)。不定冠詞が性で選ばれている。

5. **主語代名詞の省略と復活**(第4章): Sunt din Japonia. / Avem o casă. では主語を省略。一方 **El** este inginer. / **Ea** este profesoară. では、直前で話題にした人物を受け直すために代名詞を明示している。

## 読解問題:内容確認(5 問)

ルーマニア語で答えよ(日本語の補助付き)。短い文でよい。

1. **De unde este Ken?**(ケンはどこの出身?)
2. **Câți ani are tata?**(父は何歳?)
3. **Ce este mama?**(母の職業は?)
4. **Câți frați și surori are Ken?**(ケンには兄弟姉妹が何人?)
5. **Familia lui Ken este mare sau mică?**(ケンの家族は大きい?小さい? — sau=「または」)

## 解答

1. **Ken este din Japonia.**(ケンは日本出身。第1段落 Sunt din Japonia. より。主語を Ken に戻して答える)
2. **Tata are cincizeci de ani.**(父は50歳。50 は 20 以上なので de を忘れない)
3. **Mama este profesoară.**(母は教師。職業なので冠詞なし・女性形 profesoară)
4. **Ken are un frate și o soră.**(兄弟1人と姉妹1人。un frate / o soră の性に注意)
5. **Familia lui Ken este mică.**(小さい。第1段落 Am o familie mică. より。答えでは mică を選ぶ。lui Ken=「ケンの」はチャンクとして受け取ればよい)

## 音読練習の指示

次の強勢マーク付きガイドで、段落ごとに 3 回音読しよう。**太字の音節に強勢**を置く。

- Bú-nă! Mă nu-**mésc** Ken. Sunt din Ja-**pó**-ni-a.
- Am o fa-**mí**-li-e **mí**-că. A-**vém** o **cá**-să în **Ó**-sa-ka.
- **Tá**-ta **á**-re cin-ci-**zécʲ** de anʲ. El **jés**-te in-d͡ʒi-**nér**.
- **Má**-ma **á**-re pa-tru-**zécʲ** și opt de anʲ. Ea **jés**-te pro-fe-**só̯a**-ră.
- Am un **frá**-te și o **só**-ră. **Fra**-te-le **méw** **á**-re **dó**-wə-zecʲ de anʲ.
- A-**vém** un **kɨj**-ne mic. **Jés**-te **fo̯ár**-te drə-**gút͡s**. Eu sunt fe-ri-**t͡ʃít**.

音読のポイント:

- 語末の -i(anʲ, zecʲ, câini)は「イ」と強く言わず、直前の子音を口蓋化させて軽く添える程度
- 語頭の e(este, eu, el, ea)は /je/。「エステ」ではなく「イェステ」
- 二重母音 oa(profesoară, foarte)、ea(ea)はなめらかに一息で

## 要約練習

次の 3 点だけを含む 1 文で、この文章を要約してみよう(モデルは解答に)。

- ①名前 ②家族の人数(または規模)③住んでいる場所

**要約モデル**: Ken este din Japonia și are o familie mică în Osaka.(ケンは日本出身で、大阪に小さな家族がいる。)— și で 2 つの情報をつないでいる(接続詞 și の使い方は第9章で詳しく学ぶ)。

## 自分の家族に置き換えて書くタスク(第9章への布石)

読解文 Familia mea をひな型にして、**自分の家族**について 4〜6 文を書いてみよう。次の枠を埋めるだけでよい。

1. Mă numesc _______.(自分の名前)
2. Sunt din _______.(出身)
3. Am _______.(家族構成 — 例: un frate, o soră, o familie mare)
4. Tata are _______ de ani. El este _______.(父の年齢・職業。年齢が 20 以上なら de を忘れずに)
5. Mama are _______ de ani. Ea este _______.(母の年齢・職業)

> このタスクで書いた文は、次の [第9章:作文:短い文を書く](./09-writing-simple-sentences.md) で「文をつなぐ」「句読点を整える」練習の素材として使う。うまく書けなくても心配しなくてよい。第9章で作文の型を一から組み立てる。
