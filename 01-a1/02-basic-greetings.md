# 第2章:基本の挨拶【CEFR A1】

## この章の目標

この章を終えると、次のことができるようになる。

- 時間帯(朝・日中・夕方以降・就寝時)に合わせた挨拶を使い分けられる
- 相手との距離に応じてカジュアルな挨拶(Salut, Bună, Pa)と丁寧な挨拶(Bună ziua, La revedere)を選べる
- 「元気ですか?」と尋ね、それに答えられる(tu 系 / dumneavoastră 系の両方)
- お礼・お願い・謝罪の基本表現を、丁寧度を意識して使える
- 握手や Sărut mâna など、挨拶にまつわるルーマニアの習慣の勘所を知り、無理なく振る舞える

## 前提知識

- [第1章:発音とアルファベット](./01-pronunciation-and-alphabet.md) — 特に ă=/ə/、ț=/t͡s/、ș=/ʃ/、ea・oa の二重母音、語末 -i の口蓋化 /ʲ/。挨拶はすべて発音の練習台でもある
- 敬称 dumneavoastră と tu の存在は [第4章](./04-present-tense-of-a-fi.md) で本格的に扱うが、本章では「丁寧度の軸」として先に感覚をつかむ

## 本文解説

### 挨拶は「丸ごと覚える」— チャンク方式

この章の表現の多くは、まだ習っていない文法(動詞の活用、疑問文の作り方、目的格の代名詞など)を含んでいる。しかしこれらは**日常で最も頻繁に使う「決まり文句」**であり、文法を分析してから使うのでは遅い。そこで本章では、Ce mai faci?(元気?)や Îmi pare rău(残念です)といった表現を、**中を分解せずひとかたまり(チャンク)として丸ごと覚える**方針を取る。「なぜこの語形なのか」は後の章で必ず回収されるので、今は**音の形と使う場面をセットで暗記**してほしい。これは母語話者が幼児期に挨拶を覚えるのと同じ、最も自然で確実な入り口である。

以下、本章で「チャンクとして覚えてよい」と明示した表現は、A1 前半の文法(発音 / 名詞の性 / a fi 現在形 / nu 否定 / SVO 語順)の範囲外でも安心して使ってよい。

### 丁寧度の 2 本の軸 — tu 系と dumneavoastră 系

ルーマニア語の対人表現は、大きく **tu 系(親しい・くだけた)** と **dumneavoastră 系(丁寧・敬称)** の 2 本の軸に分かれる。日本語の「タメ口」と「敬語」に近いが、体系はずっと単純で、基本は**この二択**である。挨拶にもこの軸が貫かれており、同じ「元気?」でも相手によって Ce mai faci?(tu)と Ce mai faceți?(敬称)を使い分ける。本章では各表現に必ず **[tu 系]** か **[dvs. 系]** か **[中立]**(どちらでも使える)のラベルを付ける。迷ったら **dumneavoastră 系を選べば失礼にはならない**。

### 時間帯別の挨拶

ルーマニア語の丁寧な挨拶は、日本語と同じく**時間帯で言い分ける**。

- **Bună dimineața** /ˈbu.nə di.miˈne̯a.t͡sa/ — 朝。dimineața =「朝(に)」。おおむね午前中〜正午前まで
- **Bună ziua** /ˈbu.nə ˈzi.wa/ — 日中の標準的な「こんにちは」。ziua =「(その)日」。時間帯に迷ったらこれが最も安全
- **Bună seara** /ˈbu.nə ˈse̯a.ra/ — 夕方以降の「こんばんは」。seara =「夕方(に)」。日が傾いてきたら切り替える
- **Noapte bună** /ˈno̯ap.te ˈbu.nə/ — 「おやすみなさい」。就寝時や夜に別れるときの言葉であって、**出会いの挨拶ではない**点に注意(日本語の「おやすみ」と同じ)

いずれも bun(良い)の女性形 bună と名詞の組み合わせで、直訳は「良い朝を」「良い日を」である。語順は「Bună + 名詞」だが、Noapte bună だけ「名詞 + bună」と**順序が逆**になる慣用がある。理屈で覚えず、この 1 語だけ例外として音で覚えるとよい。

### カジュアルな挨拶

友人・同年代・若者同士では、短いくだけた挨拶を使う。

- **Salut!** /saˈlut/ — 「やあ / じゃあ」。出会いにも別れにも使える万能のカジュアル挨拶 [tu 系]
- **Bună!** /ˈbu.nə/ — 「やあ / こんちは」。Bună ziua を短くした、非常によく使う軽い挨拶 [tu 系]
- **Pa!** /pa/ — 「バイバイ」。別れのときのみ。Pa-pa! と重ねるとさらに親しげ [tu 系]
- **Servus!** /ˈser.vus/ — 「やあ」。**トランシルヴァニア地方(ルーマニア中部・北西部)でよく聞く**、出会いにも別れにも使う挨拶。ハンガリー語・ドイツ語圏と共有する語で、地域色が強い。全国で通じるが、南部(ブカレスト等)ではあまり使わない [tu 系・地域差]

### 別れの挨拶

- **La revedere** /la re.veˈde.re/ — 「さようなら」。最も標準的で丁寧な別れの言葉。直訳は「再会へ」 [中立〜dvs. 系]
- **Pe curând** /pe kuˈrɨnd/ — 「またね / 近いうちに」。curând =「まもなく」。近く再会する含み [中立]
- **O zi bună!** /o zi ˈbu.nə/ — 「良い一日を!」。別れ際に添える一言。店員や窓口でよく交わす [中立]

### 調子を尋ねる — 「元気?」とその返答

出会いの挨拶に続けて相手の様子を尋ねるのが自然である。ここで tu / dumneavoastră の軸がはっきり出る。

- **Ce mai faci?** /t͡ʃe maj ˈfat͡ʃʲ/ — 「元気?/ どう?」[tu 系]
- **Ce mai faceți?** /t͡ʃe maj ˈfa.t͡ʃet͡sʲ/ — 「お元気ですか?」[dvs. 系]

直訳はどちらも「あなたはまだ何をしているか」に近い慣用表現で、faci / faceți は動詞 a face(する)の活用形だが、**今はチャンクとして丸ごと覚える**。faci の語末 -i は口蓋化 /ʲ/、faceți の語末 -ți は /t͡sʲ/ である(第1章)。

返答の型:

- **Bine, mulțumesc.** /ˈbi.ne mul.t͡suˈmesk/ — 「元気です、ありがとう」。最も標準的な返し
- **Foarte bine!** /ˈfo̯ar.te ˈbi.ne/ — 「とても元気!」
- **Așa și așa.** /aˈʃa ʃi aˈʃa/ — 「まあまあ」。直訳「そのように、そして、そのように」
- 聞き返すなら **Dar tu?** /dar tu/(君は?)[tu 系] または **Dar dumneavoastră?** /dar dum.ne̯aˈvo̯as.trə/(あなたは?)[dvs. 系]

### お礼と応答

- **Mulțumesc** /mul.t͡suˈmesk/ — 「ありがとう(ございます)」。標準形で、どの場面でも使える [中立]
- **Mersi** /ˈmer.si/ — 「ありがとう」。フランス語 merci 由来のカジュアルな口語 [tu 系寄り]
- **Mulțumesc mult** /mul.t͡suˈmesk mult/ — 「どうもありがとう」。mult(たくさん)を添えて感謝を強める [中立]

応答(「どういたしまして」):

- **Cu plăcere** /ku pləˈt͡ʃe.re/ — 「どういたしまして」。直訳「喜びをもって」。最も一般的で上品 [中立]
- **Pentru puțin** /ˈpen.tru puˈt͡sin/ — 「どういたしまして(たいしたことじゃない)」。直訳「わずかなことのために」 [中立]

### お願いと謝罪

**お願いの「どうぞ / お願いします」**は丁寧度で語が変わる。

- **Vă rog** /və roɡ/ — 「(あなたに)お願いします」[dvs. 系]
- **Te rog** /te roɡ/ — 「(君に)お願い」[tu 系]

vă / te は相手を指す代名詞、rog は a ruga(頼む)の活用形だが、**チャンクとして覚える**。文頭にも文末にも置ける(Vă rog, apă. / Apă, vă rog. =「お水をお願いします」)。

**謝罪:**

- **Scuzați-mă** /sku.ˈza.t͡si.mə/ — 「すみません / 失礼します」。人に声をかけるとき、通してもらうときにも使う [dvs. 系]
- **Scuze** /ˈsku.ze/ — 「ごめん」。軽いカジュアルな謝罪 [tu 系]
- **Îmi pare rău** /ɨmʲ ˈpa.re rəw/ — 「残念です / 申し訳ありません」。直訳「私には悪く思われる」。**相手への同情やお悔やみ**(「お気の毒に」)にも使える重要チャンク [中立]

Scuzați-mă と Îmi pare rău の違いに注意してほしい。Scuzați-mă は**軽い声かけ・非礼のわび**(足を踏んだ、道を尋ねる)、Îmi pare rău は**心からの遺憾・共感**(悪い知らせ、相手の不幸)に使う。日本語の「すみません」は両方をカバーするので、日本語話者は混同しやすい。

### 呼びかけ — 「〜さん」

姓や肩書きの前に付ける敬称。

- **domnul** /ˈdom.nul/ — 「〜さん(男性)」。英語の Mr. に相当。domnul Popescu(ポペスクさん)
- **doamna** /ˈdo̯am.na/ — 「〜さん(女性)」。既婚・年配を問わず女性への標準的な敬称。英語の Mrs./Ms.
- **domnișoara** /dom.niˈʃo̯a.ra/ — 「〜さん(未婚女性)」。英語の Miss に相当する**旧来の呼称**。年齢や婚姻状況で女性を区別する言い方は近年**使用が減少傾向**にあり、現在は相手を問わず doamna を使うのが無難である

姓を知らないときは domnul / doamna だけでも「もし / お客様」のように呼びかけに使える。

## 重要ポイント

- 時間帯: Bună dimineața(朝)→ Bună ziua(日中)→ Bună seara(夕方以降)→ Noapte bună(就寝時・別れ)
- 迷ったら Bună ziua と La revedere(丁寧・全場面で安全)
- カジュアル: Salut / Bună(出会い)、Pa(別れ)。Servus はトランシルヴァニアの地域色
- 「元気?」は tu なら Ce mai faci?、敬称なら Ce mai faceți?。返しは Bine, mulțumesc.
- お願い: Vă rog(丁寧)/ Te rog(くだけた)。謝罪: Scuzați-mă(声かけ)/ Îmi pare rău(遺憾・共感)
- domnișoara(未婚女性)は使用減。女性へは doamna が無難
- 本章の表現は文法未習でも「チャンク」として丸ごと使ってよい

## フォーマル/カジュアルの違い

同じ「こんにちは、お元気ですか?」でも相手で変わる。

- **フォーマル(初対面・目上・客対応)**: Bună ziua! Ce mai faceți? — 敬称の faceți、時間帯挨拶をきちんと使う
- **カジュアル(友人・同年代)**: Salut! / Bună! Ce mai faci? — 短い挨拶 + tu の faci
- 別れも同様: **フォーマル** La revedere, o zi bună! / **カジュアル** Pa! Pe curând!

丁寧度は主に **①挨拶語の選択(Bună ziua vs Salut)** と **②動詞の相手形(faceți vs faci)** の 2 点に表れる。この 2 点を意識するだけで、レジスターは自然に整う。

## 文化的注意点

- **握手の習慣**: ルーマニアでは、初対面やビジネスの場で**しっかりと握手**を交わすのが一般的である。男性同士はもちろん、男女間・女性同士でも握手する。目を見て握るのが礼儀とされる。親しい間柄では頬への軽いキス(左右)を交わすこともあるが、これは相手との関係が近い場合に限られる
- **Sărut mâna** /səˈrut ˈmɨ.na/ — 直訳「手にキスを(します)」。**女性や年長者への伝統的で非常に丁寧な挨拶**で、特に年配の女性・祖父母世代・聖職者などに対して使われる。実際に手にキスをする場合もあれば、言葉だけの場合もある。**地域差・世代差が大きく**、都市部の若者はあまり使わない。学習者は**無理に使う必要はない**——聞いたときに意味が分かれば十分で、使うなら相手とTPOを慎重に選ぶ
- **店に入るときの挨拶**: 小さな店・パン屋・薬局・カフェなどに入るとき、店員に **Bună ziua** と声をかけるのが礼儀とされる。黙って入って商品だけ見るのは、やや無愛想と受け取られることがある。出るときも La revedere / O zi bună! と一言添えると印象が良い。日本の「いらっしゃいませ」は店員側が言うが、ルーマニアでは**客の側から挨拶する**点が異なる

## 練習問題

### 問題1: 時間帯に合う挨拶を選べ

1. 午前9時、職場で同僚に。 2. 午後8時、レストランで店員に。 3. 就寝前、家族に。 4. 正午、時間帯に迷ったとき無難な一言。

### 問題2: tu 系か dumneavoastră 系か答えよ

1. Ce mai faci? 2. Te rog. 3. Vă rog. 4. Ce mai faceți? 5. Salut!

### 問題3: 適切な応答を結べ

1. Mulțumesc mult! 2. Ce mai faci? 3. Bună seara!

a. Bine, mulțumesc. Dar tu?  b. Cu plăcere.  c. Bună seara!

### 問題4: 場面に合う表現を選べ(Scuzați-mă / Îmi pare rău)

1. 人混みで前を通してほしいとき。 2. 友人が試験に落ちたと聞いたとき。 3. 道を尋ねるため見知らぬ人に声をかけるとき。

### 問題5: 次の対話を丁寧(dumneavoastră 系)に整えよ

A: Salut! Ce mai faci?
B: Bine. Pa!
(A・B とも初対面・目上の相手だと仮定して書き換える)

## 解答

### 問題1
1. **Bună dimineața!**(午前中の朝の挨拶) 2. **Bună seara!**(夕方以降) 3. **Noapte bună!**(就寝時の別れ) 4. **Bună ziua!**(日中で最も安全)

### 問題2
1. **tu 系**(faci は 2 人称単数) 2. **tu 系**(te = 君) 3. **dumneavoastră 系**(vă = あなた) 4. **dumneavoastră 系**(faceți は 2 複/敬称) 5. **tu 系**(Salut はカジュアル)

### 問題3
1–b(お礼には Cu plăcere) / 2–a(「元気?」には Bine, mulțumesc. Dar tu?) / 3–c(挨拶には同じ挨拶を返す)。
— 挨拶に挨拶を返すのはどの言語でも自然な型である。

### 問題4
1. **Scuzați-mă.**(通してもらう声かけ) 2. **Îmi pare rău.**(相手の不幸への共感) 3. **Scuzați-mă.**(見知らぬ人への丁寧な声かけ)。
— Îmi pare rău は「遺憾・お気の毒」、Scuzați-mă は「失礼・声かけ」と役割が違う。

### 問題5
A: **Bună ziua! Ce mai faceți?**(時間帯挨拶 + 敬称 faceți)
B: **Bine, mulțumesc. La revedere.**(丁寧な別れは Pa ではなく La revedere)
— 丁寧度は「挨拶語の格上げ」と「動詞を敬称形にする」の 2 点で作る。

## 復習チェックリスト

- [ ] 時間帯別の 4 挨拶(dimineața / ziua / seara / Noapte bună)を言い分けられる
- [ ] Noapte bună が「出会い」でなく「別れ・就寝」の語だと分かる
- [ ] Ce mai faci?(tu)と Ce mai faceți?(敬称)を使い分けられる
- [ ] お礼(Mulțumesc)と応答(Cu plăcere / Pentru puțin)がセットで言える
- [ ] Vă rog / Te rog、Scuzați-mă / Îmi pare rău の丁寧度と役割を区別できる
- [ ] 店に入るときは Bună ziua と挨拶する習慣を知っている
- [ ] Sărut mâna は「無理に使わなくてよい伝統挨拶」と理解している

## 次章への橋渡し

挨拶で会話の入り口に立てるようになった。次に必要なのは、話題になる「モノ・人」を指す言葉——つまり**名詞**である。[第3章:名詞と性の導入](./03-nouns-and-gender-intro.md) では、ルーマニア語の名詞が持つ「性(男性・女性・中性)」という、日本語にない仕組みと、un / o(不定冠詞)の使い分けを学ぶ。domnul(男性名詞)と doamna(女性名詞)の語尾に、その手がかりがすでに隠れている。
