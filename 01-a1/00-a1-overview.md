# A1 レベルの概要【CEFR A1】

## このレベルで到達すること

A1 は本書全体の「玄関」であり、ルーマニア語という言語の輪郭をゼロから作る段階である。このレベルを終えると、次のことができるようになる。

- ルーマニア語のアルファベットとダイアクリティクス(ă, â, î, ș, ț)を認識し、規則的な発音ルールに従って単語を読める
- あいさつ・自己紹介・簡単な別れの言葉を交わせる
- 名前・出身・職業・年齢など、自分自身に関する基本情報を短い文で伝えられる
- 基本的な名詞と、その性(男性・女性・中性)の見分け方の初歩を理解している
- `a fi`(〜である)と `a avea`(持っている)を現在形で正しく活用できる
- 主語・動詞・目的語からなる基本語順で短文を組み立てられる
- 日常生活(家族・数・時間・曜日など)の基礎語彙を使って簡単なやり取りができる
- ゆっくり話された簡単な会話や短い読解文の要点を理解できる

## 前提

A1 は前提知識ゼロから始められるレベルである。ラテン文字が読めれば十分であり、ルーマニア語の学習経験は一切不要である。本書は `00-orientation/` の使い方ガイドを読んだ後、そのままこの章から学習を始める設計になっている。

## このレベルの学び方

日本語話者にとって、ルーマニア語の入り口で最初に感じる壁は「文字は読めるのに音がわからない」という感覚である。ラテン文字を使うためローマ字読みができそうに見えるが、`ce`・`ci`・`che`・`chi` のような組み合わせ規則や、`ș`・`ț` のような日本語に存在しない子音、`ă`・`â`・`î` のような曖昧母音・中舌母音が最初のつまずきになりやすい。この段階では「文字と音の対応表」を丸暗記するのではなく、実際に単語を声に出しながら規則性を体に入れていくことが近道である。

もう一つのつまずきは「性」の概念である。日本語には名詞の性という文法カテゴリーが存在しないため、なぜ机が男性名詞で本が女性名詞なのか、という問いに論理的な答えを探そうとして立ち止まりがちである。A1 の段階では性を「暗記すべきラベル」として割り切り、名詞は必ず冠詞とセットで覚える習慣をつけるとよい。この土台が A2 以降の格変化・形容詞の一致で効いてくる。

最後に、動詞 `a fi` と `a avea` は不規則活用でありながら会話の中核をなすため、例外として先に丸暗記してしまうのが効率的である。以下はこのレベルで扱う代表的な表現である。

| ルーマニア語 | IPA | 日本語訳 |
|---|---|---|
| Bună ziua! | /ˈbu.nə ˈzi.wa/ | こんにちは(日中のあいさつ) |
| Mă numesc Ana. | /mə nuˈmesk ˈa.na/ | 私はアナと申します |
| Eu sunt din Japonia. | /jew sunt din ʒaˈpo.ni.a/ | 私は日本出身です |
| Ce mai faci? | /t͡ʃe maj fat͡ʃʲ/ | 元気ですか(くだけた場面) |
| Am o carte. | /am o ˈkar.te/ | 私は本を1冊持っています |

## このレベルで身につく文法

- アルファベットとダイアクリティクスの発音規則
- 名詞の性(男性・女性・中性)の基礎と不定冠詞・定冠詞の初歩
- `a fi`(〜である)の現在形の完全活用
- `a avea`(持っている)の現在形の完全活用
- 基本語順(主語+動詞+目的語)と否定文の作り方(`nu` の位置)
- 疑問文の作り方(イントネーションと基本疑問詞 `ce`, `cine`, `unde`)

## 語彙目標

基礎 500 語(あいさつ・数・曜日・家族・職業・日常物品・基本動詞を中心とする)。
