# 第5章:基本語順【CEFR A1】

## この章の目標

この章を終えると、次のことができるようになる。

- ルーマニア語の基本語順が SVO(主語−動詞−目的語)であることを理解し、日本語の SOV から組み替えられる
- **形容詞を名詞の後ろに置く**というルーマニア語の大原則を使える
- 否定語 nu、疑問詞、場所・時間の表現を正しい位置に置ける
- 語順が比較的自由である理由(語形変化が情報を担う)を説明できる
- ここまでの 4 章の知識を組み合わせて、正しい語順の文を組み立てられる

## 前提知識

- [第3章:名詞と性の導入](./03-nouns-and-gender-intro.md) — 性の一致
- [第4章:a fi の現在形](./04-present-tense-of-a-fi.md) — 動詞の人称変化、否定・疑問

## 本文解説

### 基本は SVO — 日本語との最大の違い

日本語は「私は コーヒーを 飲む」(SOV)だが、ルーマニア語は英語と同じ **SVO**:

> **Eu beau cafea.**(私は 飲む コーヒーを)
> /jew be̯aw kaˈfe̯a/

動詞が主語のすぐ後に来る。日本語話者は「動詞を文末に置きたくなる」癖が抜けにくいので、A1 のうちは**「主語(省略可)→ 動詞 → その他」**という順番を体に叩き込むこと。

- Sunt student.(です 学生 = 学生です)
- Maria bea ceai.(マリアは お茶を 飲む)
- Avem două bilete.(私たちは 2枚の チケットを 持っている)

### 形容詞は名詞の後ろ — ルーマニア語の景色を決める規則

日本語では「大きい家」、英語でも a big house と形容詞が前に来るが、ルーマニア語では**形容詞は名詞の後ろ**が基本である。

| ルーマニア語 | 直訳 | 意味 |
|---|---|---|
| o casă **mare** | 1軒の 家 大きい | 大きい家 |
| un oraș **frumos** | 1つの 町 美しい | 美しい町 |
| o cafea **bună** | 1杯の コーヒー 良い | おいしいコーヒー |
| un student **japonez** | 1人の 学生 日本人の | 日本人学生 |

そして第3章で学んだとおり、**形容詞は名詞の性・数に一致**する: un om bun(良い人・男)/ o zi bună(良い日・女)/ doi oameni buni(良い人々)。

> **先取りメモ(定冠詞)**: これまでの例文に出てきた cafea**ua**(そのコーヒー)、tren**ul**(その電車)、mas**a**(そのテーブル)のような語尾は、名詞の後ろにくっついた**定冠詞**(英語の the に相当)である。「語尾に -ul / -a が付いていたら『その〜』という意味」とだけ今は分かっていればよい。体系的には A2 と参照文法 `grammar/articles.md` で学ぶ。

形容詞を名詞の前に置く語順(o mare problemă「大問題」)も存在するが、これは強調・文語的な効果を持つ応用形である。**A1 では「形容詞は後ろ」だけを徹底**しよう。

### 否定 nu の位置 — 動詞の直前

- Nu beau cafea.(コーヒーは飲みません)
- Maria **nu** e acasă.(マリアは家にいない)
- Nu sunt din China, sunt din Japonia.(中国出身ではなく、日本出身です)

### 疑問文の語順

第4章で見たとおり、Yes/No 疑問文は語順を変えずイントネーションを上げるだけでよい。

- Bei cafea? ↗(コーヒー飲む?)
- Casa e mare? ↗(家は大きい?)

疑問詞(cine 誰 / ce 何 / unde どこ / când いつ / cum どのように / de ce なぜ)は**文頭**に置く。

- **Unde** e gara?(駅はどこですか?)
- **Ce** bei?(何を飲む?)
- **De ce** ești obosit?(なぜ疲れているの?)

### 場所・時間の表現の位置

場所や時間を表す語句は**文末**が最も自然(強調したいときは文頭にも置ける)。

- Sunt la birou **acum**.(今、オフィスにいます)
- Maria e **în parc**.(マリアは公園にいる)
- **Mâine** sunt acasă.(明日は家にいます)— 時間を文頭に置いて「明日は」と話題化

### なぜ語順が「わりと自由」なのか

ルーマニア語は動詞が人称で変化し、名詞・形容詞も性数で変化する。**語の形自体が「誰が・何が」の情報を運ぶ**ため、英語ほど語順が固定的でなくてもよい。たとえば強調のために Vine trenul.(来たよ、電車が)のように動詞を先に言うことも自然にある。

ただし A1 の段階では、まず**中立的な語順(SVO・形容詞後置・nu は動詞前・疑問詞文頭)**を確実にすること。自由な語順は「基本形からのずらし」として B1 以降(`grammar/word-order.md` 執筆予定)で体系的に学ぶ。

### 文を組み立てる手順(日本語話者向けアルゴリズム)

日本語の文からルーマニア語を作るときの思考手順:

1. 動詞を決め、主語の人称で活用する(主語代名詞は原則省略)
2. 動詞の直後に目的語・補語を置く
3. 形容詞があれば名詞の**後ろ**に、性数を一致させて置く
4. 否定なら nu を動詞の直前に挿入
5. 場所・時間は文末(話題化したければ文頭)

例:「私は今、大きいホテルにいます」
→ ①sunt ②(補語なし)③ hotel mare ④(肯定)⑤ la un hotel mare + acum
→ **Sunt la un hotel mare acum.**

## 重要ポイント

- 基本語順は SVO。動詞を文末に置かない
- **形容詞は名詞の後ろ** + 性数一致
- nu は動詞の直前
- Yes/No 疑問は語順そのまま+上昇イントネーション、疑問詞は文頭
- 場所・時間は文末が中立
- 語形変化が情報を担うため語順は柔軟だが、A1 では中立語順を徹底する

## ルーマニア語例文

| ルーマニア語 | IPA/発音 | 自然な日本語訳 | 直訳 | 解説 | 使用場面・レジスター |
|---|---|---|---|---|---|
| Beau o cafea bună. | /be̯aw o kaˈfe̯a ˈbu.nə/ | おいしいコーヒーを飲んでいます | 飲む(1単) 1杯の コーヒー 良い(女) | V+O+形容詞後置の最小モデル文 | 日常・中立 |
| Locuiesc într-un oraș mic. | /lo.kuˈjesk ɨntr-un oˈraʃ mik/ | 小さな町に住んでいます | 住む(1単) 中に-1つの 町 小さい | într-un = în + un の縮約。場所句は文末 | 自己紹介・中立 |
| Nu vorbesc bine românește. | /nu vorˈbesk ˈbi.ne ro.mɨˈneʃ.te/ | ルーマニア語はあまり上手に話せません | ない 話す(1単) 良く ルーマニア語で | nu +動詞+副詞。学習者の必須文 | 会話全般・中立 |
| Unde este un restaurant bun? | /ˈun.de ˈjes.te un res.ta.uˈrant bun/ | 良いレストランはどこにありますか? | どこ ある 1つの レストラン 良い | 疑問詞文頭+形容詞後置 | 旅行・丁寧 |

| Trenul vine acum. | /ˈtre.nul ˈvi.ne aˈkum/ | 電車が今来ます | 電車-その 来る(3単) 今 | trenul = tren+定冠詞 ul(先取りメモ参照) | 駅・中立 |

## よくある誤用

| ❌ 誤り | ✅ 正しく | 説明 |
|---|---|---|
| Eu cafea beau. | Eu beau cafea. / Beau cafea. | 日本語の SOV を持ち込まない。動詞は 2 番目へ |
| o mare casă(A1 段階で) | o casă **mare** | 形容詞は後ろ。前置は強調・文語の応用形 |
| Beau nu cafea. | **Nu** beau cafea. | nu は動詞の直前。目的語の前ではない |
| Este unde gara? | **Unde** este gara? | 疑問詞は必ず文頭 |
| o casă mare**ă** のような過剰一致 | o casă mare | mare は男女同形の形容詞。すべての形容詞が -ă を持つわけではない |
| Acum sunt la birou acum.(時間句の重複) | Sunt la birou acum. | 文頭か文末、どちらか一方に置く |

## ネイティブらしい言い換え

| 教科書的 | ネイティブが実際によく言う形 | 注 |
|---|---|---|
| Unde este gara? | Unde**-i** gara? | este → -i の縮約(前の語に付く)。会話で頻出 |
| Vorbesc puțin românește. | Vorbesc **doar** puțin românește. / **Mai** învăț. | doar「ほんの」を足す、「まだ勉強中です」と添えるのがこなれた返し |
| Trenul vine acum. | **Uite**, vine trenul! | uite「ほら」+動詞先行語順。実況的な場面の自然な形 |

## フォーマル/カジュアルの違い

- 語順そのものにフォーマル差はほぼない。差が出るのは**縮約**(unde-i, nu-i)で、フォーマルな書き言葉では縮約を避けて unde este, nu este と書く
- ビジネスメールなど書き言葉では、時間・場所句を文頭に置いて情報を整理する傾向がある(Mâine, la ora 10, suntem la birou. のようにコンマで区切る)

## 文化的注意点

- ルーマニア語の会話は日本語よりも**動詞(結論)が早く出てくる**。聞き取りでは文頭の動詞に集中すると理解が速くなる
- 「はい/いいえ」だけの返答は素っ気なく響くことがある。Da, beau.(うん、飲むよ)のように**動詞を繰り返して答える**のが自然(英語の Yes, I do. に相当する仕組み)

## 練習問題

### 問題1: 語を正しい順に並べよ

1. [cafea / beau / o / bună]
2. [nu / acasă / este / Maria]
3. [mare / un / e / oraș / București]
4. [ești / unde / acum]

### 問題2: 形容詞を正しい位置と形で入れよ

1. o zi (frumos) 2. un hotel (mic) 3. două case (nou) 4. un student (japonez)

### 問題3: 否定文にせよ

1. Vorbesc românește. 2. Trenul vine acum. 3. Casa e mare.

### 問題4: 日本語からルーマニア語へ(語順アルゴリズムを使って)

1. 私は小さい町に住んでいます。
2. マリアは今オフィスにいます。
3. 良いレストランはどこですか?
4. 私はコーヒーを飲みません、お茶を飲みます。

### 問題5: 次の文の「先取りメモ」該当箇所(定冠詞)を指摘せよ

1. Trenul vine. 2. Cafeaua e bună. 3. Un tren vine.

## 解答

### 問題1
1. Beau o cafea bună.
2. Maria nu este acasă.(Nu este Maria acasă. は「家にいるのはマリアではない」という対比の意味合いになる)
3. București e un oraș mare.
4. Unde ești acum?

### 問題2
1. o zi frumoas**ă**(女性形) 2. un hotel mic 3. două case no**i**(複数形) 4. un student japonez

### 問題3
1. **Nu** vorbesc românește. 2. Trenul **nu** vine acum. 3. Casa **nu** e mare.

### 問題4
1. Locuiesc într-un oraș mic.
2. Maria e la birou acum.(este でも可)
3. Unde este un restaurant bun?(会話では Unde-i un restaurant bun?)
4. Nu beau cafea, beau ceai.

### 問題5
1. Tren**ul**(その電車)が定冠詞付き。 2. Cafea**ua**(そのコーヒー)。 3. 定冠詞なし — un tren は不定冠詞(ある電車が)。
