# 第1章:発音とアルファベット【CEFR A1】

## この章の目標

この章を終えると、次のことができるようになる。

- ルーマニア語のアルファベット 31 文字を読み、それぞれの音を発音できる
- 特殊文字 ă・â・î・ș・ț の音を区別して発音できる
- 綴りを見て、初見の単語をほぼ正しく読める(ルーマニア語は綴りと発音の対応が非常に規則的)
- 日本語話者が特に間違えやすい音(ă/ɨ、語末の -i、子音連結、l/r)を意識して練習できる
- この教科書の IPA 表記の読み方が分かる

## 前提知識

なし。ここがルーマニア語学習の出発点である。本書全体の使い方は [本書の使い方](../00-orientation/00-how-to-use-this-textbook.md)、発音の鳥瞰図は [発音の全体像](../00-orientation/02-pronunciation-overview.md) を参照。

## 本文解説

### なぜ発音から始めるのか

ルーマニア語は「書いてあるとおりに読む」言語である。英語のように綴りと音が乖離していないため、**最初にルールを覚えてしまえば、辞書の見出し語を見ただけでほぼ正確に発音できる**。逆に言えば、ここで手を抜くと、以後すべての語彙学習で誤った音を覚え続けることになる。A1 で最も投資対効果が高いのがこの章である。

### アルファベット(31 文字)

ルーマニア語のアルファベットは、基本のラテン文字 26 に、5 つの特殊文字(ă, â, î, ș, ț)を加えた 31 文字である。

### 母音 — 7 つの音

ルーマニア語の母音は 7 つ。日本語の「アエイオウ」5 つに、**ă /ə/ と â・î /ɨ/ の 2 つを追加するだけ**である。これは日本語話者にとって大きな朗報だ。

| 母音 | IPA | 出し方 |
|---|---|---|
| a | /a/ | 日本語の「ア」でよい |
| e | /e/ | 日本語の「エ」でよい |
| i | /i/ | 日本語の「イ」でよい |
| o | /o/ | 「オ」より唇を丸める |
| u | /u/ | 「ウ」より唇を丸めて突き出す(日本語の「ウ」は唇が平らなので注意) |
| ă | /ə/ | **口の力を抜いて出す曖昧な「ア」**。「ア」と「オ」の中間で、口を半開きにして喉から軽く声を出す。英語 about の a と同じ |
| â / î | /ɨ/ | **「ウ」の口の形のまま「イ」と言う**。舌を「イ」の位置より奥に引く。日本語にない音なので最優先で練習する |

â と î は**同じ音**である。使い分けは綴りの規則で決まる: **語頭・語末では î**(în「〜の中に」、a coborî「降りる」)、**語中では â**(român、cântec)。複合語では元の語頭の î が語中に来ても保たれる(neîmplinit)。

> **ă と /ɨ/ の区別練習**: măr /mər/(リンゴ)と mâr…のような最小対比はのちの章で扱うが、まず「ă = 力を抜いたア」「â/î = 舌を奥に引いたイ」と体で覚えること。**ă を「ア」、â を「ウ」で代用すると別の単語に聞こえる**ことが多い(例: fata「その娘」と fată「娘」、râu「川」と rău「悪い」の母音の違いは意味の違いである)。

### 二重母音 — ea と oa が最重要

母音が滑らかに連続する二重母音(diftong)が非常に多い。まず頻度の高い 2 つを完璧にする。

| 綴り | IPA | 出し方 | 例 |
|---|---|---|---|
| ea | /e̯a/ | 「エ」から「ア」へ一気に滑る(「エァ」を1拍で) | seară /ˈse̯a.rə/(夕方) |
| oa | /o̯a/ | 「オ」から「ア」へ一気に滑る(「オァ」を1拍で) | școală /ˈʃko̯a.lə/(学校) |
| ia | /ja/ | 「ヤ」 | iarnă /ˈjar.nə/(冬) |
| ie | /je/ | 「イェ」 | ieri /jerʲ/(昨日) |
| iu | /ju/ | 「ユ」 | iubire /juˈbi.re/(愛) |
| ai | /aj/ | 「アイ」を1拍で | mai /maj/(5月) |
| au | /aw/ | 「アウ」を1拍で | sau /saw/(または) |
| ei | /ej/ | 「エイ」を1拍で | lei /lej/(レイ=通貨) |
| oi | /oj/ | 「オイ」を1拍で | noi /noj/(私たち) |
| ou | /ow/ | 「オウ」を1拍で | ou /ow/(卵) |
| ui | /uj/ | 「ウイ」を1拍で | pui /puj/(ひな鳥・チキン) |

**日本語話者の注意**: 日本語では「エア」「オア」は 2 拍だが、ルーマニア語の ea・oa は**必ず 1 拍**である。「セ・ア・ラ」ではなく「スェァラ」のように一息で発音する。

### c と g の軟音化 — 4 つの組み合わせを暗記する

イタリア語と同じ仕組みで、c と g は後ろの文字によって音が変わる。これが読みの規則で唯一「暗記が必要」な部分である。

| 綴り | IPA | 目安 | 例 |
|---|---|---|---|
| ce / ci | /t͡ʃe/ /t͡ʃi/ | チェ/チ | cine /ˈt͡ʃi.ne/(誰)、ceai /t͡ʃaj/(お茶) |
| ge / gi | /d͡ʒe/ /d͡ʒi/ | ヂェ/ヂ | ger /d͡ʒer/(厳寒)、girafă /d͡ʒiˈra.fə/ |
| che / chi | /ke/ /ki/ | ケ/キ | chelner /ˈkel.ner/(ウェイター)、chibrit /kiˈbrit/(マッチ) |
| ghe / ghi | /ɡe/ /ɡi/ | ゲ/ギ | gheață /ˈɡe̯a.t͡sə/(氷)、ghid /ɡid/(ガイド) |

つまり **h は「硬い音を保つ印」**である。「チェにしたくなければ h を挟む」と覚える。

### 語末の -i — ルーマニア語らしさの核心

子音の後の語末 -i は、多くの場合、完全な「イ」ではなく**直前の子音を口蓋化させる音 /ʲ/** になる。「イ」と言い切らず、子音の最後に「ィ」の気配だけを残すイメージである。

| 単語 | IPA | 意味 |
|---|---|---|
| lup /lup/ | 狼(単数) | |
| lupi /lupʲ/ | 狼(複数)—「ルピ」ではなく「ルプィ」 | |
| ani /anʲ/ | 年(複数) | |
| București /bu.kuˈreʃtʲ/ | ブカレスト | |

ただし例外的に完全な /i/ で読む場合がある: (1) -ii と 2 つ重なるとき(copiii「その子供たち」)、(2) 子音+ l/r +i のとき(membri /ˈmem.bri/「メンバーたち」)、(3) 強勢があるとき(a citi /a t͡ʃiˈti/「読む」)。

**この /ʲ/ は単数と複数を聞き分ける鍵**になるため、A1 のうちから意識してほしい。

### そのほかの重要規則

- **語頭の e**: 人称代名詞と a fi の活用形では /je/ と読む — eu /jew/(私)、el /jel/(彼)、este /ˈjes.te/(〜である)。それ以外の一般語では普通の /e/(elev /eˈlev/)
- **x**: 通常 /ks/(taxi)、母音に挟まれると /ɡz/ になる語がある(examen /eɡˈza.men/)
- **母音の弱化がない**: 英語と違い、強勢のない母音もはっきり発音する。ルーマニア語の o は常に「オ」であり、曖昧化しない
- **長母音・短母音の区別がない**: 日本語の「おばさん/おばあさん」のような長さの対立はない

### 強勢(アクセント)

強勢の位置は語ごとに決まっており、**規則では完全に予測できない**。最後から 2 番目の音節に来ることが最も多いが、例外も多い。本書では例文の IPA に必ず強勢記号 /ˈ/ を付けるので、単語は音の形ごと覚えること。強勢の位置だけで意味が変わる語もある(copii /koˈpij/「子供たち」と copii /ˈko.pij/…ではなく、正しくは a copia の名詞形 copie /ˈko.pi.e/「コピー」と copii /koˈpij/「子供たち」の対比などがのちの章で登場する)。

## 重要ポイント

- アルファベットは 31 文字 = ラテン 26 + ă â î ș ț
- 新しい母音は 2 つだけ: ă /ə/(脱力のア)と â・î /ɨ/(舌を奥に引いたイ)
- â と î は同じ音。語頭・語末 = î、語中 = â
- ce/ci=チェ・チ、ge/gi=ヂェ・ヂ、che/chi=ケ・キ、ghe/ghi=ゲ・ギ
- ea・oa の二重母音は 1 拍で滑らかに
- 子音のあとの語末 -i は /ʲ/(子音の口蓋化)— 複数形の合図
- 綴りと発音の対応は極めて規則的。この章のルールで辞書の単語がすべて読める

## ネイティブらしい言い換え

発音章のため言い換えの代わりに、**音の印象**について 1 点。ルーマニア語は母音をはっきり発音し、リズムは強勢拍である。日本語のように全音節を等長で読むと「機械的」に聞こえる。強勢のある音節を長く強く、それ以外を短く、を意識するだけで一気に自然になる。

## フォーマル/カジュアルの違い

発音そのものにフォーマル差はほぼないが、速い口語では次の縮約が起きる(聞き取りのために知っておく。自分では A1 のうちは丁寧に全部発音してよい)。

- nu am → n-am(〜を持っていない/していない)
- nu este → nu-i / nu e(〜ではない)

## 文化的注意点

- ルーマニア語は歴史的にキリル文字で書かれ、19 世紀にラテン文字へ移行した(詳細は `history/language-history.md` で扱う)。現在のルーマニア・モルドバ共和国ともにラテン文字を使う
- ダイアクリティクスを省いた表記(sa ← să など)が SNS やチャットでは見られるが、学習者は必ず正しい文字で書くこと。省略綴りの読解は B2 以降で扱う

## 練習問題

### 問題1: 次の単語を IPA で書き、声に出して読め

1. mamă(母) 2. înțeleg(分かる) 3. cireșe(サクランボ) 4. unchi(おじ) 5. georgian(ジョージアの)

### 問題2: 下線の文字の音は /k/ か /t͡ʃ/ か

1. **c**ine 2. **c**asă 3. **ch**elner 4. du**c**e 5. ban**c**ă

### 問題3: 次の各組で â/î の音 /ɨ/ を含む単語はどちらか

1. rău / râu 2. fata / fântână 3. inima / început

### 問題4: 語末の -i が /ʲ/(口蓋化)になるものを選べ

1. pomi(木々) 2. copiii(その子供たち) 3. metri(メートル複数) 4. ani(年々)

### 問題5(ディクテーション準備): 音の対応を思い出しながら、カタカナ近似から綴りを推測せよ

1. 「ムルツメスク」(ありがとう) 2. 「シュコアラ」(学校) 3. 「ブクレシュティ」(首都名)

## 解答

### 問題1
1. /ˈma.mə/ — 語末 ă は脱力の「ア」
2. /ɨn.t͡seˈleɡ/ — 語頭 î=/ɨ/、ț=/t͡s/
3. /t͡ʃiˈre.ʃe/ — ci=/t͡ʃi/、ș=/ʃ/
4. /unkʲ/ — chi の i はここでは口蓋化のみ(1音節)
5. /d͡ʒe.or.d͡ʒiˈan/ — ge, gi はどちらも /d͡ʒ/

### 問題2
1. /t͡ʃ/(ci) 2. /k/(ca) 3. /k/(che — h が硬さを守る) 4. /t͡ʃ/(ce) 5. /k/(că)

### 問題3
1. râu(/rɨw/ 川)— rău は /rəw/(悪い)で ă。この 2 語の対比は /ə/ vs /ɨ/ の最重要ミニマルペア
2. fântână(/fɨnˈtɨ.nə/ 泉)— â が 2 回
3. început(/ɨn.t͡ʃeˈput/ 始まり)— 語頭は î と綴る

### 問題4
1・4 が /ʲ/(pomi /pomʲ/、ani /anʲ/)。2 は -iii で完全母音を含む /koˈpi.ji/。3 は子音+r+i なので完全な /i/(/ˈme.tri/)。

### 問題5
1. mulțumesc 2. școală 3. București

## 復習チェックリスト

- [ ] ă と â/î の音の違いを口で再現できる(rău / râu)
- [ ] ce/ci・ge/gi・che/chi・ghe/ghi の 4 組を即答できる
- [ ] ea・oa を 1 拍で発音できる(seară, școală)
- [ ] 語末 -i の口蓋化(lupi /lupʲ/)を再現できる
- [ ] 初見の単語(例: dimineață)を自力で読める
- [ ] î と â の綴り分け規則(語頭語末=î、語中=â)を言える

## 次章への橋渡し

文字と音が読めるようになったので、次はいよいよ**実際に使う言葉** — 挨拶から始める。[第2章:基本の挨拶](./02-basic-greetings.md) では、この章で練習した Bună ziua や Mulțumesc がどんな場面で誰に使えるのか、丁寧さの使い分けとともに学ぶ。挨拶はすべて音の練習台でもある。声に出しながら進もう。
