# 第10章:A1 総復習練習【CEFR A1】

## この章の目標

この章を終えると、次のことができるようになる。

- 第1〜9章で学んだ発音・文法・語彙を、章の順番ではなく**技能別に横断**して確認できる
- 自分の弱点分野(発音でつまずくのか、性でつまずくのか、数詞でつまずくのか)を具体的に特定できる
- A1 チェックポイントテスト(第12章)に向けて、抜けている知識を洗い出せる

この章は**問題集**である。新しい文法や語彙は一切登場しない。すべての問題は第1〜9章の範囲内で作られている。ここで解けなかった問題は、あなたが「まだ定着していない箇所」を教えてくれる貴重な手がかりである。

## この章の使い方

1. **時間を計らず、辞書を使わずに解く**。総復習の目的は「今の自分が何を覚えていて、何を忘れているか」を正直に知ることである。分からない問題は飛ばさず、まず「たぶんこうだろう」と書いてみてほしい。間違いこそが復習の地図になる。
2. **8 つの技能セクションを順に解く**。①発音 → ②名詞の性と冠詞 → ③a fi / a avea → ④数詞と de → ⑤語順・否定・疑問 → ⑥語彙 → ⑦ミニ会話 → ⑧和文羅訳、の順である。問題は通し番号(1〜54)になっている。
3. **解き終わったら [第11章:解答集](./11-a1-answer-key.md) で答え合わせをする**。この章には解答を載せていない。第11章は単なる正答一覧ではなく、全問に「なぜそうなるか」の解説を付けているので、合っていた問題も解説を読むと理解が深まる。
4. **間違えた分野の章に戻る**。たとえば①でよく間違えたら [第1章](./01-pronunciation-and-alphabet.md) へ、④でつまずいたら [第7章](./07-daily-life-vocabulary.md) へ。分野と章の対応は第11章末尾の「復習マップ」にまとめてある。

> 表記の約束: ș と ț は**コンマ付き**(ș=U+0219, ț=U+021B)で書く。答えを書くときも ă/â/î/ș/ț を省かないこと(第9章)。

## セクション① 発音・文字(問題 1〜5)

参照: [第1章:発音とアルファベット](./01-pronunciation-and-alphabet.md)

### 問題1
次の下線の文字の音は /k/ か /t͡ʃ/ か答えよ。

1. **c**ine(誰) 2. **c**asă(家) 3. **ch**eie(鍵) 4. dul**c**e(甘い→ここでは du**c**e「導く」で判定) 5. bă**c**ă(→ ban**c**ă「銀行」で判定)

### 問題2
次の各組で、â/î の音 /ɨ/ を含む単語はどちらか選べ。

1. rău / râu 2. pare / pâine 3. lume / lună(→ どちらでもなければ「なし」と書く)

### 問題3
語末の -i が口蓋化 /ʲ/(完全な「イ」ではない)になるものをすべて選べ。

1. frați(兄弟たち) 2. copiii(その子供たち) 3. ani(年々) 4. studenți(学生たち)

### 問題4
二重母音 ea・oa は**1 拍**で発音する。次の語のうち ea または oa の二重母音を含むものをすべて選べ。

1. seară(夕方) 2. masă(テーブル) 3. școală(学校) 4. profesoară(女性教師) 5. carte(本)

### 問題5
音の対応を思い出しながら、カタカナ近似から正しい綴りを書け(ダイアクリティクスを省かないこと)。

1. 「ムルツメスク」(ありがとう) 2. 「シュコアラ」(学校) 3. 「ツァラ」(国) 4. 「クイネ」に近い音で「犬」

## セクション② 名詞の性と冠詞(問題 6〜13)

参照: [第3章:名詞と性の導入](./03-nouns-and-gender-intro.md)

### 問題6
「un / doi テスト」で次の名詞の性(男性・女性・中性)を判定せよ。

1. casă 2. scaun 3. bărbat 4. cafea 5. tren

### 問題7
正しい不定冠詞(un / o)を付けよ。

1. ___ zi 2. ___ oraș 3. ___ femeie 4. ___ tată 5. ___ bilet 6. ___ carte

### 問題8
次の名詞は -ă で終わるが、性が異なるものが 1 つある。それはどれか。またその性を答えよ。

1. mamă 2. casă 3. tată 4. fată

### 問題9
複数形を作れ(子音交替・母音交替に注意)。

1. student 2. casă 3. scaun 4. carte 5. tren 6. fată

### 問題10
正しい数詞(doi / două)を選べ。

1. (doi / două) copii 2. (doi / două) case 3. (doi / două) bilete 4. (doi / două) frați

### 問題11
ルーマニア語の「中性名詞」の振る舞いを、次の空所を埋めて説明せよ。

> 中性名詞は、単数では ___ 名詞のように(un をとる)、複数では ___ 名詞のように(două をとる)振る舞う。

### 問題12
下線部は名詞の後ろに付いた**定冠詞**(「その〜」)である。定冠詞が付いている語をすべて選べ(第5章の先取りメモ)。

1. Tren**ul** vine. 2. Un tren vine. 3. Cafea**ua** e bună. 4. O cafea, vă rog.

### 問題13
次の文の誤りを直せ。

1. Am un soră. 2. Vreau doi bilete. 3. O tată e acasă.

## セクション③ a fi / a avea の活用(問題 14〜21)

参照: [第4章:a fi の現在形](./04-present-tense-of-a-fi.md) / [第7章:生活語彙と a avea](./07-daily-life-vocabulary.md)

### 問題14
a fi を正しい形にせよ。

1. Eu ___ din Japonia. 2. Tu ___ obosit? 3. Noi ___ la hotel. 4. Ei ___ studenți. 5. Dumneavoastră ___ medic?

### 問題15
a avea を正しい形にせよ。

1. Eu ___ o carte. 2. Tu ___ un frate? 3. Ea ___ o soră. 4. Noi ___ o casă mare. 5. Voi ___ timp acum?

### 問題16
次の文を否定文にせよ(nu の位置に注意)。

1. Sunt ocupat. 2. El e acasă. 3. Avem timp.

### 問題17
主語の性に合わせて、かっこ内の補語を正しい形に直せ。

1. Ea este (japonez). 2. Maria e (obosit). 3. El este (fericit). 4. (女性の話者が)Sunt (încântat).

### 問題18
a fi と a avea のどちらを使うか選び、正しい形にせよ。

1. Eu ___ 25 de ani.(私は25歳です) 2. El ___ medic.(彼は医師です) 3. Noi ___ doi copii.(私たちには子供が2人います) 4. Tu ___ obosit?(疲れてる?)

### 問題19
次の敬称(dumneavoastră)の文を、友人向け(tu)に書き換えよ。

1. Dumneavoastră sunteți domnul Popescu? 2. Aveți copii?

### 問題20
主語代名詞を省略できる文はどれか。省略した形を書け。

1. Eu sunt student. 2. Eu sunt medic, el este inginer.(この文はどう扱うべきか説明も)

### 問題21
次の対話の空所に a fi の正しい形を入れよ。

> A: Bună ziua! ___ domnul Ionescu?
> B: Da, eu ___. Dumneavoastră cine ___?
> A: ___ Tanaka, din Japonia.

## セクション⑤ 語順・否定・疑問(問題 27〜34)

参照: [第5章:基本語順](./05-basic-word-order.md)

### 問題27
語を正しい順に並べて 1 文にせよ(句読点も付ける)。

1. [cafea / beau / o / bună] 2. [nu / acasă / este / Maria] 3. [mare / un / e / oraș / București] 4. [ești / unde / acum]

### 問題28
形容詞を正しい位置と形で入れよ(後置・性数一致)。

1. o zi (frumos) 2. un hotel (mic) 3. două case (nou) 4. un student (japonez)

### 問題29
次の文を否定文にせよ。

1. Vorbesc românește. 2. Trenul vine acum. 3. Casa e mare.

### 問題30
次の平叙文を Yes/No 疑問文にせよ(語順は変えない)。

1. Ești student. 2. Ai un frate. 3. Casa e mare.

### 問題31
適切な疑問詞(cine / ce / unde / câți)を入れよ。

1. ___ este gara?(駅はどこ?) 2. ___ bei?(何を飲む?) 3. ___ e acolo?(あそこにいるのは誰?) 4. ___ ani ai?(何歳?)

### 問題32
次の文の誤りを直せ(語順・位置の誤り)。

1. Eu cafea beau. 2. Beau nu cafea. 3. Este unde gara?

### 問題33
場所・時間の表現の位置に注意して、日本語の意味になるよう並べ替えよ。

1. [acum / la birou / sunt](今、オフィスにいます) 2. [suntem / mâine / în București](明日、私たちはブカレストにいます — 時間を文頭に)

### 問題34
「はい」と答えるとき、ルーマニア語では動詞を繰り返すのが自然である。次の問いに「はい」で、動詞を繰り返して答えよ。

1. Bei cafea?(→ うん、飲むよ) 2. Ești student?(→ はい、そうです)

参照: [第2章](./02-basic-greetings.md) / [第3章](./03-nouns-and-gender-intro.md) / [第6章](./06-self-introduction-dialogue.md) / [第7章](./07-daily-life-vocabulary.md) / [第8章](./08-short-reading-my-family.md)

ルーマニア語(a〜e)と日本語(1〜5)を結べ。

| ルーマニア語 | | 日本語 |
|---|---|---|
| a. soră | | 1. 水 |
| b. apă | | 2. 姉妹 |
| c. bani | | 3. 椅子 |
| d. scaun | | 4. お金 |
| e. bunică | | 5. 祖母 |

### 問題36(マッチング)
ルーマニア語の職業(a〜d)と日本語(1〜4)を結べ。

| ルーマニア語 | | 日本語 |
|---|---|---|
| a. medic | | 1. 教師 |
| b. inginer | | 2. 学生 |
| c. profesoară | | 3. 医師 |
| d. student | | 4. エンジニア |

### 問題37(穴埋め・挨拶)
場面に合う挨拶を下の語群から選んで入れよ。

語群: Bună dimineața / Bună seara / Noapte bună / Bună ziua

1. 午前9時、職場で。 2. 午後8時、レストランで。 3. 就寝前、家族に。 4. 正午、時間帯に迷ったとき無難な一言。

### 問題38(穴埋め・お礼と応答)
空所を埋めよ。

1. お礼を言う: ___(標準形) 2. 「どういたしまして」: Cu ___ 3. 丁寧に「お願いします」: ___ rog 4. くだけて「お願い」: ___ rog

### 問題39(穴埋め・家族)
日本語に合う家族語を入れよ。

1. 父 = ___ 2. 母 = ___ 3. 兄・弟 = ___ 4. 祖母 = ___ 5. 子供 = ___

### 問題40(穴埋め・反対の形容詞)
空所に反対の意味の形容詞を入れよ。

1. mare ↔ ___(小さい) 2. bun ↔ ___(→ この章の範囲では「悪い」rău を使う) 3. nou ↔ ___(→ 範囲外なら「-」)

> 注: 2・3 で範囲内に反対語がなければ「範囲内になし」と書いてよい。范囲内で確実なのは 1(mare ↔ mic)である。

### 問題41(穴埋め・聞き返し)
場面に合う聞き返し表現を入れよ。

語群: Poftim? / Mai încet, vă rog / Puteți repeta, vă rog?

1. 一言も聞き取れず、丁寧にもう一度頼みたい。 2. 相手が早口で、全体的にゆっくり話してほしい。 3. とっさに「え?」と短く聞き返したい。

### 問題42(穴埋め・出身と居住)
din と în を正しく入れよ。

1. Sunt ___ Japonia.(日本出身です) 2. Locuiesc ___ București.(ブカレストに住んでいます)

### 問題43(穴埋め・自己紹介)
初心者版の自己紹介を完成させよ。

> Bună ziua! ___ numesc Yuki. Sunt ___ Japonia. Sunt ___(学生・女性). Îmi pare ___.

### 問題44(語彙・仲間はずれ)
各組で仲間はずれ(カテゴリーが違う語)を 1 つ選べ。

1. pâine / apă / cafea / carte 2. luni / marți / vineri / masă 3. frate / soră / scaun / mamă
