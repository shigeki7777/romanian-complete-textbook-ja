# 第11章:C1 への橋渡し【CEFR B2】

## この章の目標

この章を終えると、次のことができるようになる。

- B2 で自分が「何をできるようになったか」を具体的に言葉にできる
- B2(議論し、文体を操る)と C1(ニュアンスと含意を運用する)の違いを説明できる
- C1 で最初に出会う新しい仕組み(ジェルンジウ・縮約関係節・含意と婉曲・公式文書・学術的談話)を事前に知り、心の準備ができる
- B2 の知識を維持する復習習慣と、ネイティブ素材(ポッドキャスト・文学入門・映画)への本格移行の道筋を持って C1 に進める

この章は問題集ではなく**読み物**である。肩の力を抜いて読んでほしい。B2 の締めくくりとして、これまでの道のりを振り返り、次のレベルへの地図を手に入れる章である。

## 前提知識

- B2 全体([第1章](./01-advanced-tenses-and-aspect.md)〜[第7章](./07-writing-reports.md))

## 本文解説

### まず、あなたができるようになったこと(棚卸し)

B1 を終えたとき、あなたは「自分の考えを組み立て、社会の中で大人として人と関わる」ことができるようになっていた。B2 を終えたいま、あなたは**相手と議論を交わし、場面に応じて文体を着替えられる**ようになっている。具体的に、B2 を終えたあなたは次のことができる。

**高度な時制とアスペクト**
- [ ] 大過去(mai mult ca perfectul)を作り、「過去のある時点よりさらに前に完了していたこと」を表せる(Când am ajuns, trenul plecase deja.)
- [ ] 文学・地方の口語に出る単純過去(perfectul simplu = zise, făcu)を「これは過去だ」と読んで認識できる(自分で作る必要はないと割り切れる)
- [ ] tocmai(ちょうど〜したところ)、a fi pe cale să(〜しかけている)などアスペクトの表現を使い分けられる

**議論と文体**
- [ ] 論証を「序論 → 論点 → 結論」の骨格で組み、接続表現(în primul rând / pe de altă parte / prin urmare / în concluzie)で編める
- [ ] 譲歩(deși / cu toate că / chiar dacă)と切り返し(totuși / cu toate acestea)で、相手を全否定せずに反論できる
- [ ] ヘッジ(s-ar putea să / probabil că / se pare că)で確信度を調整できる
- [ ] フォーマル/中立/口語の語彙対(a achiziționa/a cumpăra、a deceda/a muri)を使い分け、同じ内容を3つのレジスターで書き分けられる

**読解と実務**
- [ ] ニュース記事の3層構造(見出し / リード / 本文)を意識して効率よく読み、報道文体の距離化の道具(se 受動・伝達動詞・条件法の推定報道)を見分けられる
- [ ] 会議を進行し(Trecem la punctul următor.)、丁寧に割り込み(Aș dori să intervin.)、交渉を「提案 → 対案 → 妥協」で組み立てられる
- [ ] 医師の説明を**理解する側**に立ち、検査・結果の言い回し(în limite normale, ușor crescut)を聞き取って質問できる
- [ ] レポートを三部構造(introducere / cuprins / concluzii)+ 客観文体(se 構文・名詞化)で書ける

これだけのことが、できるようになった。ルーマニア語で `Când am ajuns, trenul plecase deja.` と過去の中の過去を語り、`Înțeleg punctul tău de vedere, dar pe de altă parte...` と相手を立てつつ反論し、ニュースの見出しから要点を拾い、`Dacă acceptați X, putem oferi Y.` と交渉の条件を出せる——**「自分の考えを言葉にする」段階を超えて、「議論し、文体を操る」段階に立っている**。まず、そのことを誇ってほしい。B1 修了時に「文体差って何だろう」と感じていたレジスターを、あなたはもう自分の道具にしたのだ。

### B2 と C1 の違い — 「議論し、文体を操る」から「ニュアンスと含意を運用する」へ

B2 と C1 の違いを一言でいうと、こうなる。

> **B2 = 「議論し、文体を操る」。C1 = 「ニュアンスと含意を運用する」。**

B2 であなたが身につけたのは、根拠を示して**議論を組み立て**、同じ内容をフォーマル・報道・口語に**着替えさせる**力である。totuși で切り返し、se 受動で距離を取り、名詞化で書き言葉を締められるようになった。

C1 では、その一段上——**言葉の裏にある意図まで運用する**段階に進む。単に正しく議論するだけでなく、断定を和らげ、遠回しに匂わせ、皮肉を読み取り、公式文書の入り組んだ長文を解きほぐす。B2 が「何を、どの文体で言うか」だったのに対し、C1 は「同じ文を、どんな態度・含みで言うか」を扱う。文法的な正しさは前提となり、評価軸は「意図を読めるか・運用できるか」に移る。

| | B2(議論し、文体を操る) | C1(ニュアンスと含意を運用する) |
|---|---|---|
| 話の中心 | 論点の構造・説得・妥協・文体選択 | 含意・婉曲・緩和・皮肉の読み取りと運用 |
| 構文 | 大過去・受動態・接続表現で論理を編む | +ジェルンジウ・縮約関係節・無人称構文で文を畳み込む |
| 文の形 | 従属節を接続表現でつなぐ | 分詞・副詞的分詞で1文に圧縮し、語順で焦点を作る |
| 典型フレーズ | Pe de altă parte... / Se estimează că... | Neavând timp, ... / Nu prea cred. / Mă mai gândesc. |
| 読む素材 | ニュース記事・論説文 | 契約書・行政文書・学術論文・文学 |

急に別の言語になるわけではない。B2 で作った土台——大過去・受動態、議論の接続表現、レジスターの着替え——は、C1 でもそのまま毎日使う。C1 はその上に「圧縮の統語」と「含意の語用論」を積み増していく段階である。

### C1 の主役予告 — これから出会う新しい仕組み

C1 で新しく前面に出てくる主役を、あらかじめ紹介しておく。「聞いたことがある」だけで、C1 に入ったときの負担がぐっと減る。

**① 高度な構文 — ジェルンジウと縮約関係節**

C1 の第1章は、文を**圧縮する統語**から始まる。会話は短い文を接続詞でつなぐが、論説・報道・行政文書は従属動詞を分詞に「畳んで」1文に押し込む。

- **ジェルンジウ(gerunziu)** = 語幹 + **-ând / -ind** で作る不変化の副詞的分詞。日本語の「〜しながら」「〜して」「〜なので」にあたる(Mergând spre casă, m-am gândit la tine. = 家へ歩きながら、君のことを考えた)。とくに **fiind**(〜なので)・**având**(〜を持って)が高頻度で、理由や状況を1語に畳める。
- **縮約関係節** = 過去分詞の形容詞的用法(cartea **citită** de mine = 私が読んだ本)。B2 で学んだ受動態(a fost citită)を、分詞1語に縮めた形である。

どちらも B2 の受動態・名詞化の延長にあり、日本語の「〜して」「〜された〜」という連用・連体の畳み込みに対応するので、発想としては馴染みやすい。

**② 含意・婉曲・緩和 — 言葉の陰影**

C1 の第2章は、文に乗る**話者の態度**を扱う。断定を和らげる緩和辞(**cam / mai degrabă**)、死や解雇の**婉曲**(a se stinge din viață / a fi disponibilizat)、位置で意味が変わる小辞(**chiar / tocmai / doar**)、そして「いいえ」と言わない断り(**Mă mai gândesc.**)——B2 でヘッジや指小辞に半歩踏み込んだ「匂わせる技術」が、ここで本格的な語用論になる。日本語話者は「行間を読む」素地を持つため、この領域には強い足がかりがある。

**③ 公式文書 — 契約書・行政文書**

C1 では、契約書・行政文書といった**専門文体の長文**を高い精度で読解する。一文が長く、従属節が入れ子になり、名詞化が多用される。B2 のレポート作成で身につけた「客観文体・名詞化・se 構文」が、ここで公式文書特有の定型句(prezentul contract, subsemnatul, în conformitate cu)と結びつく。日本語の契約書・公文書を読む姿勢(主語と述語を丁寧に追う)がそのまま応用できる。

**④ 学術的談話**

C1 の後半では、根拠と反論を組み立てる**学術的・専門的な言説**を扱う。B2 の「議論と論証」を、より抽象度の高い、引用と論理構成を伴うレベルへ引き上げる段階である。B2 の接続表現(prin urmare / pe de altă parte)が、そのまま学術文の論理の骨格になる。

### C1 の章構成

C1 の全体像は [C1 概要](../05-c1/00-c1-overview.md) にまとまっている。C1 を始める前に必ず一度読んでほしい。章構成はおおまかに次の流れである。

- **高度な統語構造**(文法)— ジェルンジウ・縮約関係節・焦点化・無人称構文
- **ニュアンスと含意**(読解)— 婉曲・緩和・小辞の語用論・皮肉の入口
- **フォーマルな作文**(作文)— B2 のレポートを、説得力ある構成へ
- **契約と行政文書 / 医療文書**(読解)— 公式文体の長文を高精度で読む
- **学術・専門的言説**(会話)— 根拠と反論を組み立てる議論へ
- **文化・歴史的読解**(読解)— 文学・随筆・歴史記述へ
- **復習・チェックポイントテスト・C2 への橋渡し**で締めくくる

B2 と同じく「文法 → 会話 → 作文 → 読解」を織り交ぜて進む構成であり、進め方のコツも変わらない。**声に出す・書いてみる・間違えたら該当章に戻る**。

### B2 の知識を維持する習慣

新しいことを学ぶと、古いことは自然に薄れていく。対策は「思い出す機会を意図的に作る」ことに尽きる。おすすめの復習スケジュールを挙げる。

| タイミング | やること |
|---|---|
| **1 週間後** | [第8章:B2 復習練習](./08-b2-review-exercises.md) をもう一度解く(答えを覚えていても、手が動くか確認) |
| **1 か月後** | [第10章:チェックポイントテスト](./10-b2-checkpoint-test.md) を再挑戦し、前回との得点差を見る |
| **毎日 5 分** | 大過去の語尾(plecase / terminaserăm)、議論の接続表現(pe de altă parte / prin urmare / totuși)、レジスターの語彙対(a cumpăra ⇔ a achiziționa)を口で唱える。ニュースの見出しを1本、声に出して訳す |
| **C1 の各章の前** | 対応する B2 の章を 5 分だけ読み返す(例: C1 の縮約関係節の前に [第1章](./01-advanced-tenses-and-aspect.md) の受動態を、C1 の婉曲の前に [第6章](./06-style-and-register.md) の指小辞・婉曲語彙を、C1 のフォーマル作文の前に [第7章](./07-writing-reports.md) の客観文体を復習) |

特に**大過去と受動態**、**議論の接続表現**、**レジスターの着替え**は、C1 でも毎日使う「筋肉」である。ジェルンジウも縮約関係節も、この土台の上に載る。使わないと衰えるので、短くてよいから毎日触れてほしい。

### ネイティブ素材への本格移行 — 教科書の外へ、もう一歩

B1 修了時に「多読の本格化・ニュースは見出しから」を始めた。C1 に向けては、それを**生のネイティブ素材**へと一段引き上げる番である。B2 まででニュースと論説を読めるようになったいま、素材を「学習者向け」から「ネイティブが実際に消費するもの」へ移していく。

**① ポッドキャスト — 耳をネイティブ速度に慣らす**

ニュースの見出しと本文が読めるようになった耳を、次は**音声**で鍛える。いきなり全部聞き取ろうとせず、段階を踏む。

1. **興味のある分野のポッドキャストを1本選ぶ** — 内容を知っている分野(仕事・趣味)なら、未知語があっても文脈で補える。
2. **1回目は「何割聞き取れたか」を数えるゲームにする** — 全部でなく、知っているチャンク(pe de altă parte, se pare că)が聞こえた回数を数える。成功体験を積むのが継続の燃料になる。
3. **書き起こしがあれば「聞く → 読む → もう一度聞く」** — 読んで正体を知った音は、耳でも拾えるようになる。B2 のニュース読解と同じ「読める形は聞ける」の原則である。

**② 文学入門 — やさしい短編・現代小説から**

C1 では単純過去(perfectul simplu)を「読んで認識する」段階に本格的に入る。文学の地の文はまさにこの単純過去の宝庫である。

1. **短編・現代小説から始める** — いきなり古典に挑まない。現代の平明な散文なら、B2 の語彙で大半が読める。
2. **単純過去を「複合過去と同じ意味だ」と流して読む** — zise(言った)、făcu(した)、merse(行った)を見たら、a spus / a făcut / a mers と同じだと即座に置き換える。B2 第1章で仕込んだ「認識だけでよい」がここで生きる。
3. **1ページに辞書は2〜3回まで** — 多読の原則は変わらない。分からない語は文脈で流し、読む速度と推測力を同時に育てる。

**③ 映画・ドラマ — 字幕を階段にする**

1. **ルーマニア語音声 + ルーマニア語字幕** から始める(日本語字幕ではなく)。音と綴りが結びつき、口語の縮約(nu-i, c-am)が「文字」として見えるようになる。
2. **同じ場面を2回** — 1回目は字幕で筋を追い、2回目は音に集中する。全編でなく、好きな1シーンを繰り返すのが効く。
3. **口語の談話詞に耳を貼る** — păi, hai, mă rog, deci など、B2 第6章で学んだ口語マーカーが、実際の会話でどう挟まれるかを体で知る。教科書の口語が「生きた会話」に接続する瞬間である。

ネイティブ素材は「難しい素材で3割分かる」より「やさしい素材で9割分かる」ほうが力がつく、という原則は B1 から一貫している。C1 は素材のレベルを上げる段階だが、**背伸びしすぎず、9割分かる素材を毎日**という基本は変えない。

## 重要ポイント

- B2 = 「議論し、文体を操る」、C1 = 「ニュアンスと含意を運用する」(婉曲・緩和・皮肉・公式文書・学術的談話)
- C1 の主役予告:①高度な構文(ジェルンジウ -ând/-ind・縮約関係節=過去分詞) ②含意・婉曲・緩和(cam / a se stinge / chiar / Mă mai gândesc.) ③公式文書(契約・行政の長文と定型句) ④学術的談話
- C1 の新文法は、B2 の土台(大過去・受動態・名詞化・議論の接続表現・レジスター)の上に積み増す。ゼロからではない
- 維持の習慣: 1 週間後に第8章、1 か月後に第10章、毎日 5 分(大過去・接続表現・レジスターの語彙対)
- ネイティブ素材へ本格移行:ポッドキャスト(知っている分野・チャンクを数える)、文学入門(短編・単純過去は認識で流す)、映画(ルーマニア語字幕を階段に)

## 学習相談 Q&A

これまで多くの学習者が B2 修了時に抱いてきた悩みに、具体的な助言で答える。

### Q1. 報道文の se 構文が、読むときにまだ引っかかります。「誰がしたのか」を探してしまって、そこで止まります。

「誰が」を探して止まるのは、日本語話者の自然な反応であり、間違いではない。ただし報道文の se 受動(se anunță / se estimează)は、**まさに「誰が」を言わないためにこそ選ばれている**構文だと発想を切り替えるとよい。se estimează că prețurile vor crește は「(誰かによって)価格が上がると見積もられている」で、動作主は意図的に伏せられている。処方箋は「動作主を探すのをやめて、se + 動詞を『〜されている/〜という見方だ』という一つの塊で読む」ことである。se anunță(発表されている)、se pare(思われる)、se estimează(見積もられている)——この頻出パターンを塊で覚えれば、目が止まらなくなる。C1 の無人称構文(se impune să = 〜が必要とされる)も同じ発想の延長なので、いまこの読み方を固めておくと後が楽になる。困ったら [第3章](./03-news-and-opinion-reading.md) の報道文体の項に戻るとよい。

### Q2. レジスターの切り替えが遅いです。書くときに、この語はフォーマルすぎないか、口語すぎないかと毎回迷って、手が止まります。

全員が通る段階である。切り替えが「遅い」のは、語彙対を**知識としては知っているが、まだ反射になっていない**からで、理解の問題ではなく運動の問題である。処方箋は2つ。**第一に**、迷う頻出の語彙対を10組だけ選んで、フォーマル/中立/口語を横に並べて口で唱える(a achiziționa / a cumpăra / a lua、a deceda / a muri / a se stinge…)。3語を一息で言えるまで繰り返すと、書くときに「引き出し」から瞬時に選べるようになる。**第二に**、書き始める前に「これは誰に向けた、どの改まりの文か」を1秒で決める癖をつける。宛先が決まれば語彙は自動的に絞られる。会議の議事録なら中立〜フォーマル、友人へのチャットなら口語、と先に枠を決めてから語を選ぶ。迷いは「枠を決めていない」ことから生まれることが多い。速さは量でしか育たないので、B2 第6章の語彙対を毎日少しずつ唱えてほしい。

### Q3. ニュースの本文はなんとか読めますが、映画やドラマがまだほとんど聞き取れません。C1 でついていけるか不安です。

B2 修了時点で生の映画・ドラマが聞き取れないのは**正常**である。ネイティブの速度では口語の縮約(nu-i = nu e、c-am = că am)が連続し、談話詞が挟まり、字幕なしでは字面も見えないからだ。焦らず段階を踏もう。第一に、本文で述べた「ルーマニア語音声 + ルーマニア語字幕」を使い、まず**文字で見えている音**から慣らす。字幕という「読める形」を階段にすれば、読解の力を聴解に橋渡しできる。第二に、全部を聞き取ろうとせず、好きな1シーンを繰り返し、知っているチャンクが聞こえた回数を数えるゲームにする。聴解は C1 を通してゆっくり伸びるもので、いま完璧である必要はない。読めるものは、時間をかければ必ず聞けるようになる。

### Q4. C1 は難しくなりすぎませんか? ジェルンジウ・縮約関係節・契約書と聞くだけで身構えてしまいます。

不安になるのは、真剣に取り組んできた証拠である。安心してほしい理由が3つある。**第一に**、C1 の主役たちの部品を、あなたはもう持っている。ジェルンジウは「〜しながら・〜なので」で、B2 の半過去・受動態の延長。縮約関係節は過去分詞1語で、B2 の受動態(a fost citită)を縮めただけ。婉曲・緩和は、第2章のヘッジや第6章の指小辞・婉曲語彙で、もう半歩踏み込んでいる。契約書の客観文体は、第7章のレポート作成で名詞化と se 構文をすでに使った。**第二に**、本書は C1 でも「まず大意・チャンクで慣れる → 後で仕組みを理解する」という A1 からの同じやさしい順序で進む。**第三に**、詰まったらいつでも B2 に戻れる——各章に相互リンクがあり、C1 各章の前提知識欄が「どの B2 章に戻ればよいか」を示している。B1 の終わりにも「文体差って何だろう」と不安だったはずだ。そして、あなたはいまそのレジスターを自分の道具にして、ここに立っている。一歩ずつで大丈夫である。

### Q5. 皮肉や遠回しの言い方を、いつか自分でも使えるようになりたいです。C1 でそこまで到達できますか。

意欲は素晴らしいが、順序を正しく持ってほしい。C1 の到達目標は、皮肉や婉曲を**「読み取れる・疑える」こと**であって、自分から仕掛けることではない。皮肉(ironie)の発信は C2 の領域である。理由は明快で、**受信を誤ると失礼になるだけだが、発信を誤ると関係を壊す**からだ。字義に取られて滑ったり、こちらの皮肉が相手に伝わらず険悪になったりする危険が、発信側には常につきまとう。だから C1 では、まず `Frumos!`(結構なことで=文脈で非難)や `Mă mai gândesc.`(考えておきます=やんわり断り)を**正しく読み取る感度**を磨くことに集中する。緩和辞(cam / nu prea)による「角の取り方」は C1 でも産出レベルで運用できるようになるので、そこから始めれば十分実用的である。皮肉の発信は、関係が深まり、明確に冗談と分かる相手・場面に限る——この分別自体が、C1 で身につけるべき見識である。
