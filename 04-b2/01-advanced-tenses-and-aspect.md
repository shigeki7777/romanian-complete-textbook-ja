# 第1章:高度な時制とアスペクト【CEFR B2】

## この章の目標

この章を終えると、次のことができるようになる。

- 大過去(mai-mult-ca-perfectul)を「語幹 + -sem/-seși/-se/-serăm/-serăți/-seră」で作り、「過去のある時点よりさらに前に完了していたこと」を表せる(Când am ajuns, trenul plecase deja.)
- 文学の地の文や地方の口語で出会う単純過去(perfectul simplu)を「見て、これは過去だ」と認識できる(zise, făcu, merse)——自分で作る必要はないと割り切れる
- 未来完了(viitorul anterior)voi fi terminat を読んで理解できる
- アスペクト(相)を表す表現——tocmai(ちょうど〜したところ)、încă / deja / de mult、de + 期間による継続、a fi pe cale să(〜しかけている)、a începe / a continua / a termina de——を使い分けられる
- A1〜B2 で学んだすべての時制を1枚のマップに並べ、「どの時制をいつ使うか」を自分の言葉で説明できる

## 前提知識

- [B1 第1章:半過去と物語り](../03-b1/01-telling-stories-in-the-past.md) — 複合過去と半過去の使い分け。本章の「過去の過去」はこの2枚の上にもう1層を重ねる
- 参照文法 [接続法(conjunctivul)](../grammar/subjunctive.md) — a fi pe cale să + 接続法、a începe să + 接続法の骨格
- [B2 概要](./00-b2-overview.md) — このレベルの学び方

## 本文解説

### この章の位置づけ — 時制学習の集大成

A1 から B1 まで、時制は少しずつ増えてきた。現在、複合過去、半過去、o să 未来、条件法……。B2 のこの章は、それらに**最後のピース**を加え、そのうえで**全部を1枚の地図にまとめ直す**章である。新しく本格的に運用するのは大過去(mai-mult-ca-perfectul)1つだけで、残る単純過去・未来完了は「読んで分かればよい」認識レベルにとどめる。むしろこの章の主眼は、個々の時制の暗記ではなく、**時制どうしの関係——ある出来事が別の出来事の前か後か、完結しているか継続中か——を意識して選べるようになること**、すなわちアスペクト(相、aspect)の感覚を育てることにある。

日本語話者にとっての最初の要点を言う。日本語の「〜た/〜ていた/〜てしまった」は、時間の前後関係を**あまり形で区別しない**。「駅に着いたら電車はもう出ていた」の「出ていた」は、実際には「着く」より前に完了した出来事だが、日本語では特別な形を使わない。ルーマニア語はここに専用の時制(大過去)を用意している。だから日本語話者は「前後関係を表す形がある」こと自体を新しく意識する必要がある。逆に言えば、意味そのものは日本語で完全に理解できる——ただ形が1つ増えるだけだ、と捉えれば怖くない。

### 大過去(mai-mult-ca-perfectul)— 「過去の、さらに過去」

大過去は、**過去のある基準点より前に完了していた出来事**を表す。英語の過去完了 had done にほぼ相当する。ルーマニア語の大過去は、複合過去のように助動詞を使わず、**語幹に -se- を含む語尾を直接つける単純時制**である。

第 I 群 a lucra(働く)で6人称そろえる。

| 人称 | a lucra(大過去) | IPA | 意味 |
|---|---|---|---|
| eu | lucra**sem** | /lu.kraˈsem/ | 私は(すでに)働いていた |
| tu | lucra**seși** | /lu.kraˈseʃʲ/ | |
| el, ea | lucra**se** | /luˈkra.se/ | |
| noi | lucra**serăm** | /lu.kraseˈrəm/ | |
| voi | lucra**serăți** | /lu.kraseˈrət͡sʲ/ | |
| ei, ele | lucra**seră** | /lu.kraˈse.rə/ | |

語尾は **-sem / -seși / -se / -serăm / -serăți / -seră** で、活用群を問わずこの語尾は共通である。前につく語幹は、実は**過去分詞に近い形**をもとにすると覚えやすい。過去分詞の語末 -t / -s を取り、そこに -se- 系語尾をつなぐイメージである。

| 不定詞 | 過去分詞 | 大過去(el, ea 形) | IPA | 意味 |
|---|---|---|---|---|
| a lucra | lucrat | lucrase | /luˈkra.se/ | 働いていた |
| a vedea | văzut | văzuse | /vəˈzu.se/ | 見ていた |
| a merge | mers | mersese | /merˈse.se/ | 行っていた |
| a face | făcut | făcuse | /fəˈku.se/ | してしまっていた |
| a fi | fost | fusese | /fuˈse.se/ | (すでに)〜だった |
| a spune | spus | spusese | /spuˈse.se/ | 言ってしまっていた |
| a pleca | plecat | plecase | /pleˈka.se/ | 出発してしまっていた |
| a ajunge | ajuns | ajunsese | /a.ʒunˈse.se/ | 着いていた |

とくに a fi の大過去 **fusesem / fuseseși / fusese / fuseserăm / fuseserăți / fuseseră**(私はすでに〜だった)は高頻度なので、この形だけは丸暗記しておく。

### 大過去の用法 — 「〜した時点で、それより前にすでに〜し終えていた」

大過去は単独ではあまり現れない。**複合過去や半過去で示された過去の基準点があり、それより前に完了していたこと**を大過去で示す、という2層構造で使う。

| ルーマニア語 | 自然な日本語訳 | 直訳 | 時間関係 |
|---|---|---|---|
| Când am ajuns, trenul **plecase** deja. | 私が着いたとき、電車はもう出てしまっていた | とき (助)着いた 電車-その 出てしまっていた すでに | 「着く」(複合過去)より前に「出る」(大過去)が完了 |
| Nu am putut intra, pentru că **uitasem** cheia. | 中に入れなかった、鍵を忘れてきていたからだ | 否 (助)できた 入る、なぜなら 忘れてしまっていた 鍵-その | 「入れない」より前に「忘れる」が完了 |
| Ea știa deja tot, fiindcă **citise** scrisoarea. | 彼女はもう全部知っていた、手紙を読んでいたからだ | 彼女 知っていた すでに 全部、なぜなら 読んでしまっていた 手紙-その | 「知っている」(半過去)より前に「読む」(大過去)が完了 |

日本語では「出てしまっていた」「忘れてきていた」「読んでいた」と訳せる。**基準点(複合過去/半過去)よりも一段階前に完結していた出来事——これが大過去の担当領域**である。deja(すでに)や până atunci(その時までに)といった語がよく共起する。

日本語話者への注意を1つ。日本語では前後関係を文脈に任せて「駅に着いたら電車は出ていた」で済ませられるが、ルーマニア語で `Când am ajuns, trenul a plecat.`(複合過去)と言うと、「着いた、それから電車が出た」という**継起(順番に起きた)**の意味になりかねない。「着く前にすでに出ていた」を明確にするには大過去 plecase が要る。この差は日本語には形として存在しないので、意識して作る必要がある。

### 単純過去(perfectul simplu)— 読めれば十分、作らなくてよい

単純過去は「〜した」を1語で表す総合的な過去時制である。だが**標準的な話し言葉ではほぼ使われない**。生きて使われているのは①文学作品の語りの地の文、②オルテニア(Oltenia)地方の口語、の2つに限られる。全国的な会話では、複合過去 am mers が単純過去 mersei を完全に置き換えている。

したがってこの章での目標は**「見て、これは過去の出来事だと認識できる」ことだけ**である。自分で作る練習はしない。参考までに a merge(行く)の単純過去を示す。

| 人称 | a merge(単純過去) | 意味 |
|---|---|---|
| eu | mersei | 私は行った |
| tu | merseși | |
| el, ea | **merse** | (彼・彼女は行った) |
| noi | merserăm | |
| voi | merserăți | |
| ei, ele | merseră | |

小説を読むと、地の文に3人称単純過去が頻出する。とりわけ「**〜と言った**」の zise(< a zice)は決まり文句のように現れる。

| ルーマニア語 | 意味 | もとの動詞 |
|---|---|---|
| — Bine, **zise** el. | 「よし」と彼は言った | a zice(言う) |
| Ea **se ridică** și **plecă**. | 彼女は立ち上がり、去った | a se ridica / a pleca |
| El **făcu** un pas înainte. | 彼は一歩前に出た | a face(する) |
| **Merse** până la fereastră. | 彼は窓辺まで歩いた | a merge(行く) |

**認識の手がかり**:見慣れた動詞なのに語末が **-se / -u / -ă** で終わり、しかも複合過去の助動詞(a/au)がない3人称形に出会ったら、それは単純過去だと考えてよい。意味は複合過去(a zis, a plecat)と同じ「〜した」で取ればよい。会話でこれを産出する必要はない、と割り切ることが B2 の正しい態度である。

### 未来完了(viitorul anterior)— これも認識レベルで十分

未来完了は「未来のある時点までに完了しているであろうこと」を表す。英語の will have done に相当する。形は **voi fi + 過去分詞**(a fi の未来 + 過去分詞)である。

| 人称 | a termina(未来完了) | 意味 |
|---|---|---|
| eu | voi fi terminat | 私は(その時までに)終えているだろう |
| tu | vei fi terminat | |
| el, ea | va fi terminat | |
| noi | vom fi terminat | |
| voi | veți fi terminat | |
| ei, ele | vor fi terminat | |

| ルーマニア語 | 自然な日本語訳 | 直訳 |
|---|---|---|
| Până mâine, **voi fi terminat** raportul. | 明日までには、報告書を終えているだろう | まで 明日、(未来)である 終えた 報告書-その |
| Când vei ajunge, eu **voi fi plecat** deja. | 君が着く頃には、私はもう出発しているだろう | とき (未来)着く、私 (未来)である 出発した すでに |

会話ではこの形の代わりに、単純な未来 o să termin(終えるだろう)や現在形+副詞で済ませることが多く、未来完了は書き言葉・改まった場面に寄る。ここでも目標は**読んで意味が取れること**である。「voi/vei/va fi + 過去分詞 = 未来までに完了しているだろう」と分かれば十分だ。

### アスペクト(相)を表す表現 — 時制の外側で「進み具合」を示す

時制(いつか)とは別に、出来事の**進み具合・完結度**を示すのがアスペクト(相)である。ルーマニア語は時制の種類が限られる分、アスペクトを副詞や動詞連結で細かく表す。ここが B2 で表現の幅が一気に広がるところである。

**① tocmai — ちょうど〜したところ / まさに〜している**

tocmai は動詞の直前に置き、「たった今」というごく近い過去、あるいは「まさに今」という進行を示す。

- `Tocmai am terminat.`(ちょうど終えたところだ)
- `Tocmai plecam când ai sunat.`(ちょうど出かけるところに君が電話してきた)

**② încă / deja / de mult — まだ・もう・とっくに**

| 語 | 意味 | 例 | 訳 |
|---|---|---|---|
| încă | まだ(継続) | Încă lucrez. | まだ働いている |
| încă nu | まだ〜ない | Încă nu am terminat. | まだ終えていない |
| deja | もう・すでに | Am terminat deja. | もう終えた |
| de mult | とっくに・ずっと前に | Știu de mult. | とっくに知っている |

**③ de + 期間 — 「〜前から(今も)続いている」継続**

日本語話者が最も間違えやすいのがここである。「3年前からここに住んでいる(今も住んでいる)」を、ルーマニア語は**現在形 + de + 期間**で表す。過去形ではない。

| ルーマニア語 | 自然な日本語訳 | 直訳 | 注意 |
|---|---|---|---|
| **Locuiesc** aici **de** trei ani. | 3年前からここに住んでいる | 住んでいる ここ 〜前から 3 年 | 現在形。英語の have lived にあたるが形は現在 |
| **Învăț** română **de** șase luni. | 半年前からルーマニア語を学んでいる | 学ぶ ルーマニア語 〜前から 6 か月 | 過去形にしない |
| **Nu l-am mai văzut de** mult. | 彼にはずっと会っていない | 否 彼を もう 見た 〜から 長く | 継続する否定は複合過去 + de mult |

「〜してどのくらいになるか」を尋ねるときも同じ枠組みを使う:`De cât timp locuiești aici?`(どのくらい前からここに住んでいる?)。日本語の「〜から」に引きずられて過去形にしないよう、**「今も続く継続は de + 期間 + 現在形」**と型で覚える。

**④ a fi pe cale să / a fi pe punctul de a — 〜しかけている、まさに〜しようとしている**

差し迫った未然の動作を表す。B1 で学んだ `era să...`(危うく〜するところだった)の仲間で、こちらは中立〜やや改まった言い方である。

| 表現 | 続く形 | 例 | 訳 |
|---|---|---|---|
| a fi pe cale să | + 接続法 | Sunt pe cale să plec. | まさに出かけようとしている |
| a fi pe punctul de a | + 不定詞 | Era pe punctul de a renunța. | 彼は諦めかけていた |

pe cale să のほうが口語寄りで頻度が高く、pe punctul de a は書き言葉寄りである。

**⑤ a începe să / a continua să / a termina de — 開始・継続・完了の相**

動作の「入り口・途中・出口」を、助動詞的な動詞 + 従属動詞で表す。ここが日本語の「〜し始める/〜し続ける/〜し終える」に対応する。**接続する形が動詞ごとに違う**点に注意する。

| 相 | 動詞 | 接続 | 例 | 訳 |
|---|---|---|---|---|
| 開始 | a începe să | + 接続法 | A început să plouă. | 雨が降り始めた |
| 継続 | a continua să | + 接続法 | Continuă să lucreze. | 彼は働き続けている |
| 完了 | a termina de | + **不定詞(短縮不定詞)** | Am terminat de mâncat. | 食べ終えた |

**a termina de + 動詞**だけは接続法ではなく、de のあとに**短縮不定詞(= 過去分詞と同形の形)**が来る点が要注意である:`am terminat de citit`(読み終えた)、`am terminat de scris`(書き終えた)。a începe / a continua は să + 接続法(A început să scrie / Continuă să scrie)。この非対称は丸暗記する。

## 重要ポイント

- 大過去 = 語幹 + **-sem/-seși/-se/-serăm/-serăți/-seră**。過去のある基準点(複合過去/半過去)より**前に完了**していた出来事を表す
- a fi の大過去 **fusesem…** は高頻度なので丸暗記
- 単純過去(mersei / zise / făcu)は**読めれば十分**。文学の地の文とオルテニア口語に限られ、産出は不要
- 未来完了 **voi fi + 過去分詞** も認識レベルでよい(未来までに完了しているだろう)
- 継続の **de + 期間は現在形**:Locuiesc aici de trei ani(過去形にしない)
- **a fi pe cale să + 接続法** = 〜しかけている
- 開始 a începe **să** / 継続 a continua **să** / 完了 a termina **de + 短縮不定詞** — 接続する形の非対称に注意

## 文法解説

### 大過去を「作る」手順 — 過去分詞から逆算する

大過去を自分で作れるようにするコツは、**過去分詞を経由する**ことである。

1. 不定詞から過去分詞を出す:a scrie → scris、a face → făcut、a fi → fost
2. 過去分詞の語末子音の状況を見る:
   - -t で終わる分詞(lucrat, făcut, plecat)→ -t を取り、-se- 語尾を付ける:lucra**sem**、făcu**sem**、pleca**sem**
   - -s で終わる分詞(scris, mers, spus)→ -se を重ねる形になる:scri**sesem**、mer**sesem**、spu**sesem**
3. a fi は例外として fuse- を語幹に:fuse**sem**

初学のうちは el, ea 形(-se で終わる)だけ作れれば実用上は足りる。物語や説明で大過去が要るのはたいてい3人称だからである:plecase, uitase, terminase, citise。

### 「継起」と「先行」を混同しない

大過去の本質は「基準点より**前**」である。同じ「〜して、〜した」でも、順番に起きたのか(継起)、片方が先に完了していたのか(先行)で時制が変わる。

- 継起(順番):`M-am spălat și am plecat.`(顔を洗って、それから出かけた)→ どちらも複合過去
- 先行(前に完了):`Când am plecat, mă spălasem deja.`(出かけたとき、もう顔は洗い終えていた)→ 出かける=複合過去、洗い終えた=大過去

日本語ではどちらも「洗って出かけた」「洗い終えていた」で自然に区別できるので、意味の理解は問題ない。ルーマニア語では**先行を明示したいときだけ大過去に切り替える**、と覚える。

## 全体マップ:A1〜B2 で学んだ全時制の一覧

ここまでで、ルーマニア語の主要な時制がすべて出そろった。1枚の表にまとめる。これが本書の時制学習の集大成である。

| 時制 | ルーマニア語 | 形の核 | 例(a lucra) | 意味 | 主な使用域 | 学習レベル |
|---|---|---|---|---|---|---|
| 現在 | prezent | 語幹 + 現在語尾 | lucrez | する/している | 会話の主力 | A1 |
| 複合過去 | perfectul compus | am/ai/a… + 過去分詞 | am lucrat | した(完結) | 会話の過去の主力 | A2 |
| 半過去 | imperfectul | 語幹 + -am/-eam… | lucram | していた/だった(継続・習慣) | 描写・背景 | B1 |
| 大過去 | mai-mult-ca-perfectul | 語幹 + -sem… | lucrasem | (すでに)してしまっていた | 過去の過去 | **B2** |
| 単純過去 | perfectul simplu | 語幹 + -ai/-u… | lucrai | した(完結) | 文学・オルテニア口語 | B2(認識) |
| 未来(口語) | viitor popular | o să + 接続法 | o să lucrez | するだろう/するつもり | 会話の未来の主力 | A2 |

| 条件法現在 | condițional prezent | aș/ai/ar… + 不定詞 | aș lucra | するだろうに/したい(丁寧) | 仮定・丁寧 | B1 |
| 条件法過去 | condițional perfect | aș fi + 過去分詞 | aș fi lucrat | していただろうに | 反実仮想 | B1〜B2 |

### 使い分けチャート — 「どの時制を選ぶか」を自問で決める

過去のことを言いたいとき、次の順で自問すると時制が決まる。

1. **その出来事は、別の過去の出来事より前に完了していたか?**
   - はい → **大過去**(plecase)
   - いいえ → 2へ
2. **完結した1回の出来事か、それとも継続・習慣・背景か?**
   - 完結した1回 → **複合過去**(am plecat)
   - 継続・習慣・背景・状態 → **半過去**(plecam / eram)
3. (小説の地の文を書く/読む特別な場合のみ)語りの文語 → 単純過去(plecă)

未来のことなら:

1. **会話・日常か?** → **o să + 接続法**(o să plec)
2. **書き言葉・改まった場か?** → **voi + 不定詞**(voi pleca)
3. **未来のある時点までの完了を言うか?** → **未来完了**(voi fi plecat)

仮定・丁寧なら:

1. **今・これからの仮定や丁寧な願望** → **条件法現在**(aș pleca / aș dori)
2. **過去の反実仮想(あのときこうしていたら)** → **条件法過去**(aș fi plecat)

このチャートを頭に入れておけば、複雑な複文でも「今はどの層の時間を話しているか」を見失わずにすむ。

## よくある誤用

日本語話者が高度な時制・アスペクトで特にやりがちな誤りを、誤→正 で示す。

| ❌ 誤り | ✅ 正しく | 説明 |
|---|---|---|
| 継続を過去形で:「3年前から住んでいる」に *Am locuit aici de trei ani* | Locuiesc aici de trei ani | 今も続く継続は **現在形 + de + 期間**。過去形にすると「(もう住んでいない)3年間住んだ」の意味になる |
| 先行を複合過去で:「着いたら電車は出ていた」に *Când am ajuns, trenul a plecat* | …trenul **plecase** | 基準点より前の完了は大過去。複合過去だと「着いて、それから出た」の継起に読まれる |
| 大過去語尾を複合過去と混同:*am lucrasem* | lucrasem | 大過去は助動詞を使わない単純時制。am を付けない |
| 完了に să を使う:*am terminat să mânca* | am terminat **de mâncat** | a termina は **de + 短縮不定詞**。să + 接続法ではない |
| 開始に de を使う:*a început de plouă* | a început **să** plouă | a începe は **să + 接続法**。a termina(de)と非対称 |
| 単純過去を会話で産出:友人に *Mersei la magazin* | Am mers la magazin | 会話は複合過去。単純過去は文学・地方限定 |
| a fi の大過去を規則化:*fisesem / fusem* | **fusesem** | a fi の大過去は fuse- を語幹に fusesem |

## ネイティブらしい言い換え

| 教科書的・直訳的 | ネイティブが実際によく言う形 | 注 |
|---|---|---|
| Am făcut asta înainte de aceea.(その前にそれをした) | **Făcusem** deja asta.(もうそれをやってしまっていた) | 「前に」を語彙で足すより、大過去1語で先行を示すほうが自然 |
| Chiar acum am terminat.(まさに今終えた) | **Tocmai** am terminat. | 「たった今」は tocmai が定番。chiar acum より口語的 |
| Sunt aproape să plec.(出発に近い) | Sunt **pe cale să** plec. | 「〜しかけ」は pe cale să が固定表現。aproape să はやや不自然 |
| De trei ani stau aici și încă stau.(3年いて、まだいる) | Stau aici **de** trei ani. | 継続は de + 期間の1語で言い切る。冗長に説明しない |

## フォーマル/カジュアルの違い

同じ「もうその映画は始まってしまっていた」を、場面で言い分ける。

- **カジュアル(友人との会話)**:`Când am ajuns, filmul începuse deja.`(着いたらもう始まってた)——大過去は会話でも普通に使う。
- **書き言葉・文学(小説の地の文)**:`Când intră în sală, filmul începuse de mult.`(彼が入場したとき、映画はとっくに始まっていた)——ここでは基準点が単純過去 intră になり、大過去 începuse と組む。この「単純過去(基準点)+ 大過去(先行)」の組は文語の語りに特徴的である。

会話では基準点は複合過去(am ajuns)、文語では単純過去(intră)。しかし**先行を示す大過去(începuse)はどちらでも同じ形**で使える点を押さえておく。未来完了(voi fi terminat)は逆に、会話ではほぼ使われず、報告書や改まった予定の言明に寄る。

## 練習問題

### 問題1:大過去を作れ(el, ea 形)

1. a pleca 2. a scrie 3. a face 4. a fi 5. a citi 6. a ajunge

### 問題2:大過去か複合過去か。文脈に合う形を選び、理由を一言で述べよ

1. Când am intrat, filmul (a început / începuse) deja.(入ったとき、映画はもう始まっていた)
2. Ieri (am mâncat / mâncasem) la un restaurant nou.(昨日、新しいレストランで食べた)
3. Nu am putut plăti, pentru că (am uitat / uitasem) portofelul acasă.(払えなかった、財布を家に忘れてきていたから)

### 問題3:継続の de 構文にせよ(現在形を使う)

1. 私は5年前からここで働いている。(a lucra / aici / cinci ani)
2. 彼らは2か月前からルーマニア語を学んでいる。(a învăța / română / două luni)
3. どのくらい前からブカレストに住んでいますか?(De cât timp… / a locui / la București)

### 問題4:アスペクト表現を選べ

1. (Tocmai / Deja) am terminat, acum un minut.(ちょうど今、1分前に終えた)
2. Am terminat (să citesc / de citit) cartea.(本を読み終えた)
3. A început (de plouă / să plouă).(雨が降り始めた)
4. Sunt (pe cale să / pe punctul) plec.(まさに出発しようとしている)

### 問題5:次の文の下線部の時制を答え、意味を訳せ(認識練習)

「Bătrânul **se ridică** încet, **privi** pe fereastră și **zise**: — E timpul.」

### 問題6:日本語をルーマニア語にせよ

1. 私が電話したとき、彼はもう寝てしまっていた。(a suna / a se culca)
2. 3年前からこの町に住んでいる。(a locui / orașul acesta / trei ani)
3. まさに出かけようとしていたところに、雨が降り始めた。(pe cale să / a începe să plouă)

## 解答

### 問題1
1. **plecase** 2. **scrisese**(分詞 scris → scrisese) 3. **făcuse** 4. **fusese**(a fi の大過去) 5. **citise** 6. **ajunsese**(分詞 ajuns → ajunsese)

### 問題2
1. **începuse** — 「入った」(複合過去)より前に始まっていた=先行 → 大過去。 2. **am mâncat** — ieri で示される完結した1回の出来事 → 複合過去。 3. **uitasem** — 「払えない」より前に忘れる行為が完了していた=先行 → 大過去。

### 問題3
1. **Lucrez aici de cinci ani.**(継続は現在形 + de + 期間) 2. **Învață română de două luni.**(ei 形 învață) 3. **De cât timp locuiești la București?**(疑問も現在形の枠組み)

### 問題4
1. **Tocmai**(「たった今」は tocmai。1分前=近接) 2. **de citit**(a termina は de + 短縮不定詞) 3. **să plouă**(a începe は să + 接続法) 4. **pe cale să**(pe cale să + 接続法。pe punctul なら de a plec ではなく de a pleca が必要)

### 問題5
- se ridică(**単純過去**、a se ridica)= 立ち上がった / privi(**単純過去**、a privi)= 眺めた / zise(**単純過去**、a zice)= 言った。
- 訳:「老人はゆっくりと立ち上がり、窓の外を眺め、そして言った——『時間だ』」。いずれも文学の地の文の単純過去で、会話なら s-a ridicat / a privit / a zis(複合過去)にあたる。認識できれば十分。

### 問題6
1. **Când am sunat, el se culcase deja.**(電話=複合過去、その前に寝る完了=大過去 se culcase) 2. **Locuiesc în orașul acesta de trei ani.**(継続は現在形 + de + 期間) 3. **Eram pe cale să plec când a început să plouă.**(pe cale să + 接続法、開始は a începe să + 接続法。過去の状況なので eram)

## モデルテキスト:Întârzierea(遅刻)

大過去が自然に機能する回想文である(約150語)。声に出して読み、**基準点(複合過去/半過去)より前に完了していた出来事がどこで大過去になっているか**を確認しよう。

> **Întârzierea**
>
> În dimineața aceea m-am trezit târziu. Ceasul nu sunase, fiindcă uitasem să-l pun seara. Când am ieșit din casă, autobuzul plecase deja și strada era pustie. Ploua mărunt. Îmi pregătisem prezentarea toată noaptea, iar acum eram pe cale să pierd întâlnirea cea mai importantă a anului. Am luat un taxi, dar traficul era imposibil. Când am ajuns în sfârșit la birou, colegii mei începuseră fără mine. Ședința era pe la jumătate. M-am așezat discret. Șeful tocmai spusese ceva despre proiectul meu — dar nu apucasem să aud ce. După ședință, mi-a explicat că totul mersese bine. Îmi făcusem griji degeaba.

**日本語訳**:その朝、私は寝坊した。目覚まし時計は鳴らなかった、前の晩にセットするのを忘れていたからだ。家を出たとき、バスはもう行ってしまっていて、通りは人けがなかった。小雨が降っていた。私は一晩かけてプレゼンの準備をしておいたのに、今まさに、その年で一番大事な会議を逃そうとしていた。タクシーに乗ったが、渋滞はどうしようもなかった。ようやくオフィスに着いたとき、同僚たちは私抜きで始めてしまっていた。会議は半ばだった。私はそっと座った。上司はちょうど私のプロジェクトについて何か言ったところだった——だが、それが何なのかを聞き取ることはできなかった。会議のあと、上司は、すべてうまくいったと説明してくれた。私は無駄に心配していたのだった。

**時制の色分け解説**:

| 部分 | 時制 | なぜこの時制か |
|---|---|---|
| m-am trezit / am ieșit / am luat / am ajuns / m-am așezat | 複合過去 | 話を前に進める、完結した一連の出来事(基準点) |
| era pustie / Ploua / traficul era / Ședința era | 半過去 | 背景・状況の描写(継続した状態) |
| **nu sunase / uitasem / plecase / Îmi pregătisem** | 大過去 | いずれも基準点より**前に完了**していた出来事(セット忘れ・バス発車・準備) |
| **începuseră / spusese / nu apucasem / mersese / Îmi făcusem griji** | 大過去 | オフィス到着・上司の発言・心配、いずれも各基準点より前に完了 |
| eram pe cale să pierd | アスペクト(pe cale să) | 「まさに逃そうとしていた」差し迫った未然 |
| tocmai spusese | tocmai + 大過去 | 「ちょうど言ったところだった」——近接の相を大過去に重ねる |

このテキストの背骨は、**複合過去/半過去で「今の場面」を進めながら、その一歩手前に完了していた出来事を次々に大過去で差し込む**構造である。とりわけ `Când am ieșit din casă, autobuzul plecase deja.`(家を出たとき、バスはもう行ってしまっていた)が、この章の核心「基準点(複合過去 am ieșit)+ 先行(大過去 plecase)」の典型である。`tocmai spusese`(ちょうど言ったところだった)のように、アスペクト副詞 tocmai を大過去に重ねると「近い過去の、さらに一段前」という細かな時間差まで表せることも味わってほしい。

## 次章への橋渡し

時制の全体像がそろい、「いつ・どの前後関係で・どんな進み具合か」を形で示せるようになった。ここまでは主に「出来事をどう時間軸に並べるか」の話だった。次の [第2章:議論と論証](./02-argumentation-and-discussion.md) からは、その出来事や事実について**自分がどう考え、どう主張し、どう相手の意見に反論するか**という、B2 のもう一つの大きな柱に進む。時制で組み立てた「事実の記述」の上に、接続表現(totuși, pe de altă parte など)で「論理の構造」を乗せていく——それが議論の技術である。
