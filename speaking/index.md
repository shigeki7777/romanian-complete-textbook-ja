# 会話・発話

このセクションは、発音の基礎訓練から実務レベルの交渉・プレゼンテーションまで、アウトプットとしての話す力を段階的に養うための参照集である。本編(`01-a1/` 〜 `06-c2/`)の各章には会話パートが組み込まれているが、それは主に新出文法・語彙を使った短い対話例の提示にとどまる。実際に「話せる」ようになるには、型を覚えるだけでなく、繰り返し声に出す練習と、場面ごとの実践的な会話運用の訓練が必要である。このセクションはその反復練習と場面別トレーニングを担う。

構成は難度と場面の複雑さに応じて段階的に並んでいる。`pronunciation-drills.md` は全レベルの土台となる発音訓練であり、[grammar/phonetics-and-orthography.md](../grammar/index.md) の音韻知識を実際の発話筋肉の運用に変換する。`self-introduction.md`・`daily-conversation.md`・`telephone.md`・`small-talk.md` は A1〜B1 で必要になる基礎的な対人場面を扱い、`discussion.md`・`debate.md`・`presentations.md`・`negotiation.md` は B2 以降、抽象的な話題について自分の意見を構築し他者と渡り合う力を扱う。最後の `native-like-fluency.md` は、文法的に正しいだけでなく、間の取り方・言い淀みの処理・自然なつなぎ言葉の使用など、ネイティブらしい話し方の総仕上げに位置づける。

会話の指導では「教科書的な表現」と「実際にネイティブが使う表現」の差が特に大きく出る技能であるため、各ファイルは STYLE_GUIDE の「ネイティブらしい言い換え」節を重点的に活用し、フォーマル/カジュアルの使い分けを常に明示する方針を取る。ビジネス場面(`negotiation.md` など)は [business/](../business/index.md) の専門編と、医療場面での会話は [medical/](../medical/index.md) と内容が重なる部分があるため、該当箇所は相互参照する計画である。

## 学習の目安

- **A1 の学習者**: `pronunciation-drills.md` で発話の基礎を作り、`self-introduction.md` から実際に声に出す練習を始める。
- **A2〜B1 の学習者**: `daily-conversation.md`・`telephone.md`・`small-talk.md` で場面のバリエーションを広げる。
- **B2〜C1 の学習者**: `discussion.md`・`debate.md`・`presentations.md`・`negotiation.md` に進み、意見構築・議論・実務コミュニケーションの力をつける。
- **C1〜C2 の学習者**: `native-like-fluency.md` を通じて、これまでの型の運用を自然な話し方へと昇華させる。[listening/advanced-listening-c1-c2.md](../listening/index.md) と併用すると効果的である。
