# イディオム

このセクションは、辞書的な逐語訳では意味が通らない慣用句・ことわざ・決まり文句を集めた参照集である。本編(`01-a1/` 〜 `06-c2/`)や [vocabulary/](../vocabulary/index.md) が個々の単語・基本表現を積み上げるのに対し、このセクションは単語の組み合わせによって初めて生まれる意味、つまり「知らなければ絶対に推測できない」表現を扱う。イディオムは B1 以降、学習者が「文法的には正しいのにネイティブっぽくない」と感じ始める段階で特に効果を発揮する領域であり、[speaking/](../speaking/index.md) の `native-like-fluency.md` や [writing/](../writing/index.md) の `native-like-style.md` と組み合わせて使うことで、教科書的な表現から自然な表現への橋渡しになる。

各ファイルはテーマ別に分かれており、日常会話で使う軽いイディオム(`daily-idioms.md`)から、会議・メールで使うビジネス慣用句(`business-idioms.md`)、感情の機微を表す表現(`emotional-expressions.md`)、ことわざ(`proverbs.md`)、そして最も習得が難しい皮肉・ユーモア(`irony-and-humor.md`)まで難度が段階的に上がる構成になっている。ことわざは歴史的・文化的背景を伴うことが多いため、該当する場合は [culture/](../culture/index.md) や [history/](../history/index.md) への参照を各章内に含める計画である。

イディオムの学習で最も注意すべき点は、直訳が学習者に強い誤解を与えることである。そのため各ファイルでは STYLE_GUIDE の例文フォーマットに従い、自然な日本語訳と直訳を必ず併記し、直訳から連想される誤った解釈がある場合は「よくある誤用」節で明示する方針を取る。また、すべてのイディオムにレジスター(丁寧度・口語度)を明記し、フォーマルな場では使えない表現を不用意に多用しないよう案内する。

## 学習の目安

- **B1 の学習者**: `daily-idioms.md` から着手し、日常会話で頻出する定型表現を先に身につける。ビジネス場面で学ぶ必要がある場合は `business-idioms.md` を並行する。
- **B2 の学習者**: `emotional-expressions.md`・`proverbs.md`・`colloquial-expressions.md` に進み、感情表現の幅とネイティブの口語感覚を広げる。
- **C1〜C2 の学習者**: `irony-and-humor.md` に取り組み、文脈から含意を読み取る力を養う。この段階では単独学習よりも [reading/](../reading/index.md) の `native-level-texts-c2.md` や [listening/](../listening/index.md) の上級教材と併用することを推奨する。
