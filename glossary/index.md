# 用語集(日羅対訳)

## このセクションの目的と使い方

このセクションは、本書全体で使用される専門用語を分野別に日本語⇔ルーマニア語で対訳し、辞書として引けるかたちで提供することを目的とする。レベル別本編や専門編(`medical/` `business/` `culture/`)を読み進める中で、特定の用語の意味や訳語をピンポイントで確認したい場面は多い。本セクションはそうした場面での「引く」使い方に特化しており、章として通読するものではなく、学習の合間に随時参照する参照系のセクションとして位置づけられる。

構成は本編の技能・専門編と対応させている。`grammar-terms-ja-ro.md` は、レベル別本編や `grammar/` 参照文法で使われる文法用語(接続法・条件法・格・アスペクトなど)を日本語名とルーマニア語名(または言語学用語としての英語名)の対で整理し、初出時にどのレベルで導入されるかも併記する。`medical-glossary-ja-ro.md` は `medical/` セクションと連動し、同セクションの方針に従って**誤訳リスクの高い用語には ⚠ を付す**。`business-glossary-ja-ro.md` は `business/` セクションと連動し、フォーマル/カジュアルで訳語が変わる場合はその違いも示す。`cultural-terms-ja-ro.md` は `culture/` と `history/` の両セクションで登場する、日本語に一対一で対応しない文化的・歴史的な概念語(訳出そのものが難しい語)を扱い、単なる訳語ではなく説明を伴う形で収録する。

用語集の性質上、個々の語義そのものには [SOURCE_POLICY.md](../SOURCE_POLICY.md) の出典要件は課さないが、規範的な語義については DEX(ルーマニア語詳解辞典)系の辞書を基準とする方針を用語集全体の前提とする。専門用語(医療・法律・歴史関連)で出典が必要な記述を伴う場合は、該当する本編セクション側の出典表記に従う。

## 学習の目安

通読を前提とせず、本編・専門編を読み進める中で不明な用語に出会った際に随時参照する。特に `medical-glossary-ja-ro.md` の ⚠ マーク付き用語は、実際に使う前に必ず該当項目の解説を確認する習慣を付けることを推奨する。
