# 礼儀作法 — 挨拶・訪問・贈答・食事【読み物 / 主対象 A2〜B1】

## この章で分かること

この章を読み終えると、次のことができるようになる。

- 初対面・親しい間柄それぞれの挨拶を、失礼のない形で組み立てられる。
- 家庭に招かれたときの基本的なふるまい(手土産・靴・食卓の呼吸)を、断定を避けつつ理解できる。
- 贈り物の慣習(花は奇数本、など「一般に」知られる作法)を知り、地雷を踏みにくくなる。
- 乾杯の `Noroc!` をはじめ、食事まわりの定型表現を使える。
- 宗教行事の場での基本的な身のこなしを、最低限わきまえる。

この章は日常生活に直結するため、本編 A2〜B1 の段階から並行して読むとよい。ビジネス場面の礼法は [ビジネスエチケット](../business/business-etiquette.md) に集約されているため、本章は**私的・日常の場面**を中心に扱い、重複を避けて相互参照する。

> **ステレオタイプ化への注意**: 礼儀作法は地域・家庭・世代で大きく異なる。本章の記述は「一般に広く見られる傾向」であり、すべての家庭・すべての人に当てはまる規則ではない。特に「靴を脱ぐか」「どこまで格式ばるか」は**家庭による**。迷ったら、その場でホストのやり方に合わせ、必要なら一言たずねるのが最も丁寧である。

## 1. 挨拶 — 距離感を言葉と所作で測る

挨拶は、相手との距離感を一瞬で表す所作である。ルーマニア語では、**丁寧形(dumneavoastră)とくだけた形(tu)の使い分け**が、挨拶のトーンを決める最大の要素になる(この二人称の使い分けは本編文法の重要テーマでもある)。

- **初対面・年長・目上・公的な場**: 丁寧に。`Bună ziua`(こんにちは)+ 二人称は `dumneavoastră`。握手が一般的。
- **親しい間柄・同年代・くだけた場**: `Salut` / `Bună`(やあ)+ 二人称は `tu`。親しい相手同士では**頬へのキス(左右)**の挨拶も一般的だが、これは間柄・地域・場面による。初対面や目上には握手が無難である。

日本語話者への注意: 日本の「とりあえずお辞儀」は通じない。ルーマニアでは**目を見て、握手する**のが基本的な礼儀である。視線をそらしすぎると、よそよそしい・自信がないと受け取られることがある。頬キスは、相手から差し出されたら応じる、くらいの受け身で十分で、こちらから初対面の相手に仕掛ける必要はない。

時間帯で挨拶語が変わる点も押さえたい。`Bună dimineața`(おはよう / 午前)、`Bună ziua`(こんにちは / 日中)、`Bună seara`(こんばんは / 夕方以降)。別れ際は `La revedere`(さようなら・丁寧)、`Pa` / `Pa-pa`(じゃあね・くだけた)。

## 2. 訪問 — 招かれたときの基本

家庭に招かれるのは、相手が心を開いている証である。もてなしを重んじる文化([価値観 §3](./values.md))を踏まえ、いくつかの基本を押さえておこう。いずれも**傾向**であり、家庭差が大きいことを前提に読んでほしい。

- **手土産(手ぶらで行かない)**: 一般に、招かれたら手ぶらは避け、花・ワインや酒・菓子・子どもへの小さな土産などを持参するのが好まれる傾向がある。高価すぎるものより、心のこもった適度なものがよい。
- **靴を脱ぐ習慣**: **家庭による**。玄関で靴を脱ぐ家も、そのまま上がる家もある。断定できないので、**入り口で脱ぐそぶりを見せ、ホストの反応や玄関のスリッパの有無で判断する**のが安全。「`Mă descalț?`(靴、脱ぎましょうか?)」と一言たずねれば間違いない。日本の「必ず脱ぐ」を前提にしないこと。
- **時間**: 私的な招待では、きっかり定刻より**5〜15分ほど遅れて行く**のがちょうどよい、とされることが多い(ホストの準備の余裕への配慮)。ただしビジネスの約束は時間厳守が基本(→ [ビジネスエチケット](../business/business-etiquette.md))。私的とビジネスで感覚が違う点に注意。

## 3. 贈り物 — 「一般に」知られる作法

贈り物には、広く知られた慣習がいくつかある。ここでは断定を避け「一般に」として述べるが、**知らずに破ると相手を戸惑わせうる**ものなので、覚えておく価値がある。

- **花は奇数本**: 一般に、贈る花の本数は**奇数**にする。**偶数本は葬儀・弔いの場に結びつけられる**慣習が広く知られているためである(出典: Cultural Atlas〔SBS〕, 参照 2026-07)。1本、3本、5本、7本…と数える。この「偶数=弔い」は東欧に広く見られる感覚で、日本の「4を避ける」に似た**数のタブー**として理解するとよい。ただし、大きな花束(概ね十数本以上)では本数を数えない、という現代的な緩和も語られる。迷ったら奇数にしておけば無難である。
- **菊(crizanteme)への注意**: 一般に、菊は**墓参・追悼の花**として強く結びつく傾向がある。祝いの場やふつうの手土産には避けるのが安全。
- **受け取りの作法**: 贈り物を渡すと、相手が一度遠慮する・その場で開けないこともある。これは失礼ではなく、受け取り方の一つのスタイルである。

## 4. 食事のマナーと乾杯

食卓は、もてなし文化がもっとも色濃く出る場である。基本の呼吸を押さえよう。

- **食べ始めの合図**: ホストが `Poftă bună!`(召し上がれ / どうぞ)と言ってから食べ始めるのが一般的。自分が言われたら、食べ始めてよい合図である。
- **乾杯 — Noroc!**: グラスを合わせるときの定番は **`Noroc!`**(乾杯 / 直訳は「幸運を」)。よりあらたまると `Sănătate!`(健康を)や `În sănătatea ta / dumneavoastră!`(君の / あなたの健康に)。目を見て合わせるのが礼儀とされる。
- **勧められたら少し受ける**: [価値観 §3](./values.md) で述べたとおり、断りすぎるとホストをがっかりさせうる。満腹でも `Doar puțin, mulțumesc`(ほんの少しだけ、ありがとう)と少量受けると角が立たない。
- **食後の礼**: `Mulțumesc pentru masă!`(食事をありがとう / ごちそうさま)。もてなしへの感謝を言葉にすると喜ばれる。

なお `Noroc` は多義で、乾杯のほか、**くだけた挨拶(やあ)**や、**誰かがくしゃみをしたときの「お大事に」**にも使われる。文脈で意味が変わる便利語である。

## 5. 宗教行事の場での基本

洗礼・結婚・葬儀・イースターなど、宗教行事に招かれる場面もある(→ 詳細は [宗教](./religion.md))。信仰の有無にかかわらず、**その場の作法を尊重する**のが礼儀である。基本だけ挙げる。

- **教会・修道院を訪れるとき**: 一般に、露出の少ない服装が望ましいとされ、女性はスカーフで頭を覆う場面もある(場所による)。入り口の張り紙やほかの参列者に合わせる。
- **正教会の礼拝は立って行うことが多い**: 長い儀式では座席が限られることがある。周囲に合わせて静かに立つ・十字を切る所作は真似なくてよいが、私語や写真撮影は控える。
- **葬儀・追悼**: 暗い色の服。前述のとおり、この場面でこそ偶数本の花・菊が用いられる。祝いの作法と取り違えないよう、場面で切り替える。

これらは信仰への同調を求めるものではなく、**その場を大切にする人々への敬意**の問題である。分からなければ、周囲のふるまいに静かに合わせるのが最も安全である。

## 6. 雑談の話題 — 選ぶ・避ける

もてなしや訪問の場では、雑談(conversație / small talk)が関係を温める。傾向として、安全で喜ばれやすい話題と、慎重を要する話題がある(いずれも一般論で、相手・場面による)。

- **選びやすい話題**: 食べ物・料理、旅行や出身地、家族の近況(踏み込みすぎない範囲で)、スポーツ(とくにサッカー)、相手の国・文化への関心。ルーマニアの料理や自然をほめると、もてなし文化と相まって喜ばれやすい。
- **慎重を要する話題**: 宗教の当否([宗教 §4](./religion.md))、政治的立場、共産主義時代の評価([歴史の概観 §9](../history/overview.md))、少数民族やロマ(Roma)に関する話、個人の収入。これらは相手の内面や歴史の傷に触れうるので、相手が話し出すまでこちらからは避けるのが無難である。詳しくは → [タブー](./taboo-topics.md) で扱う。

日本語話者への助言: 日本で「天気の話」が無難なように、ルーマニアでは**食・旅・相手の文化への関心**が安全な入り口になる。相手をほめる・関心を示す姿勢は、どの文化でも歓迎される。

## 7. ネイティブらしい言い換え

教科書的な丁寧表現も間違いではないが、日常ではより短くこなれた形が使われる。

- 「ありがとう」: 硬い `Vă mulțumesc foarte mult.`(たいへんありがとうございます)→ 親しい間柄では `Mersi!`(仏 merci 由来、くだけた「ありがとう」)。目上には `Mulțumesc` を保つ。
- 「どうぞ(勧める)」: `Vă rog să serviți.`(お召し上がりください)→ 家庭ではもっと軽く `Hai, servește!` / `Ia și tu!`(ほら、食べて!)。
- 「おじゃまします / 失礼します」に相当する固定句は薄く、入室時は `Bună ziua` +(招かれていれば)`Mulțumesc de invitație`(お招きありがとう)で自然。

## コラム: 日本の礼法との違い

日本語話者が特につまずきやすい差を、三点に絞って整理する。

- **お辞儀 → 握手・視線**: 日本の「お辞儀で済ませる」は通じにくい。ルーマニアでは**目を見て握手する**のが基本。視線をそらしすぎると、よそよそしい・自信がないと受け取られうる。
- **必ず脱ぐ → 家庭による**: 日本の「玄関で必ず靴を脱ぐ」は前提にできない。脱ぐ家も上がる家もあるので、`Mă descalț?` で確認するか、ホストに合わせる。
- **遠慮の着地点**: もてなしを断る遠慮の呼吸は日本と似ているが、**断りすぎると相手をがっかりさせる**点で着地が違う。日本より一歩早く「少し受ける」ほうが、関係が温まりやすい。

共通するのは、**迷ったらその場のやり方に合わせ、必要なら一言たずねる**という姿勢である。これはどの文化でも最も安全で、かつ丁寧に映る。

## 重要ポイント

- 挨拶は `dumneavoastră`(丁寧)/ `tu`(くだけた)の選択がトーンを決める。初対面は握手 + 丁寧形。
- 靴を脱ぐかは**家庭による**。決めつけず `Mă descalț?` と一言たずねる。
- 花は**奇数本**。偶数・菊は弔いを連想させる(数のタブー)。
- 乾杯は `Noroc!`、食べ始めは `Poftă bună!`。勧められたら少し受ける。
- 宗教行事は同調でなく敬意。分からなければ周囲に合わせる。

## 言語学習への接続

- **二人称の使い分け**: `tu` / `dumneavoastră` の選択は、本編でも礼儀でも中心テーマ。挨拶の練習を通じて体で覚えると、文法学習と実践がつながる。
- **反射動詞**: `a se descălța`(靴を脱ぐ)は反射代名詞の良い練習素材。`a se descurca`([価値観](./values.md))と合わせて反射動詞に慣れる。
- **定型表現(formule)**: `Poftă bună` / `Noroc` / `Mulțumesc pentru masă` は、文法を超えた「場の定型」。丸ごと覚えることで、初級でも自然な会話の入り口に立てる。

## よくある誤用

- **誤**: 花を4本・6本など偶数で贈る。→ 弔いを連想させる。**正**: 奇数本(3・5・7…)にする。
- **誤**: どの家でも当然のように靴を脱ぐ / 逆にどの家でも脱がない。**正**: `Mă descalț?` と確認するか、玄関の様子とホストに合わせる(家庭による)。
- **誤**: 私的な招待にビジネス感覚できっかり定刻に着く。**正**: 私的な招待は5〜15分ほど遅れるくらいが配慮になる場合が多い(ビジネスは時間厳守)。
- **誤**: もてなしを固辞し続ける。**正**: `Doar puțin, mulțumesc` と少し受ける。

## 練習問題(内容確認)

**問題1**: 家庭に招かれ、花を持っていく。本数について何に気をつけるか、理由とともに答えよ。

**問題2**: 玄関で靴を脱ぐべきか分からないとき、どう対処するのがよいか。ルーマニア語の一言も添えて答えよ。

**問題3**: 乾杯のときの定番の掛け声と、目上に対してよりあらたまった言い方を一つずつ挙げよ。

**問題4**: 私的な招待とビジネスの約束とで、時間に対する感覚はどう違うと本章は述べているか。

**問題5**: 初対面の相手との雑談で、選びやすい話題と慎重を要する話題を、それぞれ一つずつ挙げよ。

## 解答

**解答1**: **奇数本**にする。**偶数本は葬儀・弔いに結びつく**慣習が一般に知られているため。菊も追悼の花とされるので祝いには避ける。

**解答2**: 決めつけず、**入り口で脱ぐそぶりを見せてホストの反応や玄関のスリッパで判断する**か、`Mă descalț?`(靴、脱ぎましょうか?)と一言たずねる。習慣は家庭による。

**解答3**: 定番は `Noroc!`(乾杯)。あらたまると `Sănătate!`(健康を)や `În sănătatea dumneavoastră!`(あなたの健康に)。目を見て合わせるのが礼儀。

**解答4**: **私的な招待では時間に柔らかく**、5〜15 分ほど遅れて着くのがちょうどよいとされることが多い。一方、**ビジネス・公的な約束は時間厳守が基本**。私的とビジネスで感覚が切り替わる。

**解答5**: 選びやすい話題の例=食べ物・料理、旅行・出身地、相手の文化への関心、スポーツ(サッカー)など。慎重を要する話題の例=宗教の当否、政治的立場、共産主義時代の評価、少数民族やロマに関する話、個人の収入など。
