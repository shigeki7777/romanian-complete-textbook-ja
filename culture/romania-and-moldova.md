# ルーマニアとモルドバ共和国【レベル非依存 — 読み物】

## この章で分かること

この章を読み終えると、次のことができるようになる。

- ルーマニアとモルドバ共和国が「同一の標準語を共有する二つの国」であることを、事実に即して説明できる。
- モルドバ共和国が独立した歴史的経緯(ベッサラビア → ソ連 → 1991年独立)を、大づかみに語れる。
- 両国の社会言語的な差異を、「傾向」として断定を避けつつ理解できる。
- 学習者として「どの標準語を学べばよいか」という実務的な問いに、自分なりに答えられる。
- モルドバのアイデンティティや文化の多様性を、単純化せずにとらえる姿勢を持てる。

この章は、特定の CEFR レベルに縛られない日本語の読み物である。ただし、モルドバのアイデンティティは政治的に敏感なテーマを含むため、本編で言えば C1〜C2 相当の、含意を読み取る力がついてから読むと理解が深まる。

> **敏感なテーマについて**: 「モルドバ語かルーマニア語か」「モルドバはルーマニアと統一すべきか」といった問いは、モルドバ国内でも意見が鋭く分かれる政治的テーマである。本章は特定の立場に与せず、事実と、複数の見方があること自体を記述する。現地でこれらの話題に触れるときは、相手の立場を決めつけず、慎重に耳を傾ける姿勢を持ってほしい。

## 1. 同一の標準語を共有する二つの国

まず、学習者にとって最も実用的な事実から述べる。**ルーマニアとモルドバ共和国は、同一のルーマニア語を国家語とする二つの国である。**語彙・文法の観点で、両国で話される言語は言語学的に同一と扱われる(出典: Wikipedia "Moldovan language", 参照 2026-07)。

歴史的には、モルドバ共和国の言語は長らく「モルドバ語(limba moldovenească)」と呼ばれてきた。しかし [ルーマニア語の歴史 §10](../history/language-history.md) で詳述したとおり、**2023年、モルドバ議会は憲法を含む法令上の言語名を「モルドバ語」から「ルーマニア語」に統一した**(2023年3月16日に法律第52号を可決、3月22日に大統領が署名・公布。出典: Library of Congress / ConstitutionNet, 参照 2026-07)。したがって、**言語名は2023年以降、両国とも法的に「ルーマニア語」である。**

本教材が STYLE_GUIDE §9 に従って両国の言語を一律に「ルーマニア語」と呼ぶのは、この事実に基づく。学習者にとっての含意は明快である — **ルーマニアで通じるルーマニア語は、モルドバでも通じる。**別々の言語を二つ学ぶ必要はない。

## 2. 歴史的経緯 — ベッサラビア、ソ連、そして1991年の独立

なぜ、同じ言語を話す地域が「モルドバ共和国」という別の国になっているのか。これを理解するには、**ベッサラビア(Basarabia)**という地域の歴史をたどる必要がある。ベッサラビアは、おおむね現在のモルドバ共和国にあたる地域で、歴史的には中世モルダヴィア公国(→[歴史の概観 §3](../history/overview.md))の東部にあたる。

流れは次のとおりである(出典: Encyclopædia Britannica "Bessarabia" / "Moldova" / Wikipedia "Bessarabia", 参照 2026-07)。

- **1812年**: ブカレスト条約により、オスマン帝国の宗主権下にあったモルダヴィアの東部(ベッサラビア)が、ロシア帝国に割譲された。以後、この地はロシア帝国の一部となる。
- **1918年**: ロシア革命による帝国の崩壊のなか、ベッサラビアの地方議会(Sfatul Țării)が、ルーマニアとの統一を決議した(1918年の大統一の一部 → [歴史の概観 §7](../history/overview.md))。
- **1940年**: 独ソ不可侵条約(モロトフ・リッベントロップ協定)を背景に、ソ連が戦争の威嚇のもとでルーマニアにベッサラビアの割譲を迫り、これを併合した。ベッサラビアの中核部は**モルダヴィア・ソビエト社会主義共和国(モルダヴィア SSR)**として編成され、北部・南部はウクライナ SSR に移管された。
- **1991年8月27日**: ソ連の解体にともない、**モルダヴィア SSR は「モルドバ共和国(Republica Moldova)」として独立を宣言した**。国境はモルダヴィア SSR のものをそのまま引き継いだ。

つまり、モルドバ共和国という国は、**19〜20世紀の帝国とソ連による領土の移動の結果**として、ルーマニア本体とは別の政治的まとまりになった。言語・文化の連続性と、政治的な国境の不一致 — これがモルドバをめぐる複雑さの根源である。

## 3. 社会言語的な差異 — 「傾向」として理解する

ルーマニアとモルドバの言語は標準語として同一だが、実際の使われ方には差異の「傾向」がある。ここでは**断定を避け、あくまで傾向として**述べる。個人差・世代差・地域差が大きく、「モルドバ人はこう話す」という単純化は誤解を招くからである。

- **ロシア語借用を含む口語層(モルドバの口語)**: モルドバ共和国は、ソ連時代を通じてロシア語が広く使われた歴史を持ち、今もロシア語・ウクライナ語・ガガウズ語などを話す少数派が相当数いる多民族国家である(出典: Wikipedia "Moldovan language", 参照 2026-07)。この社会的背景から、モルドバの**口語(くだけた話し言葉)には、ルーマニア本国では使われにくいロシア語由来の語や言い回しが交じる傾向**がある。ただし、これは口語レベルの現象であり、標準語・書き言葉のレベルでは両国は共通である。
- **発音の地域差**: モルドバで聞かれる発音の特徴は、しばしば**モルダヴィア方言(graiul moldovenesc)との連続**としてとらえられる。つまりモルドバ特有の「別言語の発音」というより、ルーマニア本国のモルダヴィア地方(=カルパティア東側)から続く方言的な連続体の一部である、という見方が言語学的には自然である。
- **正書法・標準語は共通**: 書き言葉の正書法、および教育・報道で用いられる標準語は、両国で共通である(ラテン文字、DOOM3 系の規範)。したがって、標準語を学べば読み書きは両国で問題なく通用する。

繰り返すが、これらは**傾向**である。都市部の教育を受けた話者と地方の話者、若い世代と年配の世代では、言葉の使い方が大きく異なりうる。「モルドバ語=ロシア語混じり」といった図式的な理解は避けてほしい。

## 4. 学習者への実務指針

以上を踏まえた、学習者にとっての実務的な結論はシンプルである。

- **標準ルーマニア語を学べば、両国で通じる。**本教材が教える標準語は、ルーマニアでもモルドバ共和国でも通用する。どちらの国を主な目的地にするかで、学ぶべき言語を変える必要はない。
- **モルドバでは、口語でロシア語混じりの表現に出会う可能性がある。**とくにくだけた会話や、年配・ロシア語話者の多い環境では、標準語では習わない語や言い回しに遭遇するかもしれない。そのときは「これはモルドバの口語的な言い方かもしれない」と受け止め、必要なら聞き返せばよい。標準語で話せば、こちらの意図は問題なく伝わる。
- **キリル文字を学ぶ必要は、原則ない。**両国とも標準はラテン文字である。キリル文字表記に出会うのは、後述の沿ドニエストル地域など、限られた場面に限られる。

## 5. 文化・アイデンティティの多様性

言語が同一であることと、アイデンティティが単一であることは、別の問題である。モルドバ共和国の人々のアイデンティティは一様ではなく、これは政治的にも社会的にも敏感なテーマである。

大づかみに言えば、モルドバの人々のなかには、自らを**「ルーマニア人」と考える立場**と、**「モルドバ人」という独自の帰属を大切にする立場**の双方が存在する。前者はルーマニアとの言語・文化・歴史の連続性を重視し、後者はモルドバ共和国としての独立した歩みや多民族性を重視する傾向がある。加えて、ロシア語話者やガガウズ人・ウクライナ人などの少数派を含む多民族国家であることも、アイデンティティを一枚岩にしない要因である(出典: Wikipedia "Moldovan language" / Encyclopædia Britannica "Moldova", 参照 2026-07)。

2023年の言語名の法的統一(§1)も、こうした背景のなかで、通常の憲法改正手続きを踏まなかったとして批判を受け、反政府的な政治対立を伴った(出典: ConstitutionNet, 参照 2026-07)。**言語をどう呼ぶかは、モルドバでは政治的立場と結びつく問題**であり、単なる言語学の話にとどまらない。

学習者への助言はこうである — 現地でこの種の話題に触れるときは、「モルドバ語かルーマニア語か」「ルーマニアと統一すべきか」といった問いに、外部の者として断定的な意見を押し付けないこと。まず相手の考えを聞き、多様な立場があることを尊重する姿勢が、良い関係を築くうえで欠かせない。

## 6. 沿ドニエストルとキリル文字

最後に、モルドバの言語状況で特筆すべき一点に触れておく。モルドバ共和国から**分離を主張する沿ドニエストル地域(Transnistria / Transnistria)**では、今なお言語が「モルドバ語」と呼ばれ、**キリル文字**で表記されている。沿ドニエストルでは、このキリル文字表記が公用語の一つとして扱われる(出典: Wikipedia "Moldovan Cyrillic alphabet", 参照 2026-07)。

これは、[ルーマニア語の歴史 §5](../history/language-history.md) で述べたキリル文字時代の名残が、政治的な事情から特定の地域に限って現存している例である。モルドバ本体がラテン文字・「ルーマニア語」へと進んだのに対し、沿ドニエストルはソ連期のキリル文字・「モルドバ語」の枠組みを保っている。この違い自体が、この地域の複雑な政治的位置を映している。

学習者として、沿ドニエストルを旅する予定でもない限り、キリル文字を学ぶ必要は実務上ない。ただ、「モルドバのなかにも、なおキリル文字でルーマニア語(モルドバ語)を書く地域がある」という事実は、言語と政治の絡み合いを理解する good example として知っておく価値がある。

## 練習問題

**問題1**: ルーマニアとモルドバ共和国の標準語の関係を、一文で正確に説明せよ。

**問題2**: モルドバ共和国が独立国になるまでの流れを、「ベッサラビア」「ソ連」「1991年」の三語を使って要約せよ。

**問題3**: モルドバの言語について、標準語・書き言葉のレベルと、口語のレベルとで、ルーマニア本国との差異はどう異なるか。「傾向」という語を使って説明せよ。

**問題4**: 学習者が「ルーマニアとモルドバ、どちらでも通じる言葉を学びたい」と考えるとき、何を学べばよいか。

**問題5**: モルドバのなかで、今もキリル文字で言語を表記している地域はどこか。

## 解答

**解答1**: 両国は**同一のルーマニア語を国家語とする二つの国**である(語彙・文法上、言語学的に同一)。2023年以降、法的な言語名も両国とも「ルーマニア語」で共通である。

**解答2**: ベッサラビア(現モルドバ共和国にほぼ相当)は1812年にロシア帝国へ、1918年に一度ルーマニアと統一、1940年に**ソ連**に併合されてモルダヴィア SSR となり、**1991年**にソ連解体にともないモルドバ共和国として独立した。言語・文化の連続性と政治的国境の不一致が、モルドバをめぐる複雑さの根源である。

**解答3**: **標準語・書き言葉のレベルでは、両国は共通**(ラテン文字・DOOM3 系の規範)。差異が現れるのは主に**口語のレベル**で、モルドバの口語にはロシア語由来の語や言い回しが交じる**傾向**がある。ただしこれは傾向であり、個人差・世代差・地域差が大きいため、図式的に一般化してはならない。

**解答4**: **標準ルーマニア語**を学べばよい。本教材が教える標準語は、ルーマニアでもモルドバ共和国でも通用する。国ごとに別の言語を学ぶ必要はない。

**解答5**: モルドバから分離を主張する**沿ドニエストル地域(Transnistria)**。ここでは今も言語が「モルドバ語」と呼ばれ、キリル文字で表記される。
